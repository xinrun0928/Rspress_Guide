# RabbitMQ

> [!TIP]
> RabbitMQ 是采用 Erlang 语言开发的消息队列，以灵活性著称。它的核心优势在于灵活的路由机制，通过 Exchange 和 Binding 实现各种复杂的消息分发模式。本模块深入讲解 RabbitMQ 的核心概念、架构、集群等。

## 为什么选择 RabbitMQ

RabbitMQ 的核心优势：

| 优势 | 说明 |
|---|---|
| **灵活路由** | 通过 Exchange 实现复杂的路由规则 |
| **多协议支持** | 支持 AMQP、MQTT、STOMP 等协议 |
| **功能丰富** | 死信队列、延迟队列、优先级队列等 |
| **管理界面** | 自带 Web 管理界面 |
| **易用性好** | 配置灵活，易于上手 |
