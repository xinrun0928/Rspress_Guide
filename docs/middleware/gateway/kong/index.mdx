# Kong：高性能 API 网关

Kong 是由 Kong Inc. 开发的开源 API 网关，基于 Nginx + OpenResty（Lua）构建，以其极高的性能和丰富的插件生态著称。

如果你追求极致性能、需要灵活的插件扩展能力、或在 Kubernetes 环境中运行， Kong 是一个值得考虑的选择。

## 核心技术栈

| 技术 | 作用 |
|---|---|
| Nginx | 高性能 HTTP 服务器和反向代理 |
| OpenResty | Nginx + LuaJIT，提供动态脚本能力 |
| Cassandra / PostgreSQL | 数据存储，存储路由、插件配置 |
| Kong Admin API | 配置管理接口 |
