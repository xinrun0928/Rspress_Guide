# Spring Cloud Gateway

Spring Cloud Gateway 是 Spring Cloud 官方推出的第二代网关框架，基于 Spring 5 WebFlux 和 Project Reactor 构建，专为云原生和响应式架构设计。

如果你已经使用 Spring Cloud 全家桶，Gateway 是最自然的选择——它与 Nacos、Feign、Sentinel 等组件无缝集成，Java 团队可以完全用熟悉的 Spring Boot 风格解决问题。

## 核心技术栈

| 技术 | 作用 |
|---|---|
| Spring 5 WebFlux | 响应式 Web 框架，基于 Netty |
| Project Reactor | 响应式编程库，提供 Mono/Flux |
| Netty | 高性能网络框架，异步非阻塞 I/O |
| Spring Cloud CircuitBreaker | 熔断抽象层 |
