# 网关核心功能

这一章讲解 API 网关的通用核心功能，涵盖路由匹配、过滤器链、负载均衡等设计原理。无论你选择 Spring Cloud Gateway、Kong 还是 Apache APISIX，这些概念都是相通的。

掌握这些核心概念，你就能快速上手任何一款网关方案——区别只是配置语法和 API 细节。

## 内容导航

- **function.mdx** —— 网关的六大核心职责：路由、负载均衡、认证授权、限流熔断、日志监控、协议转换
- **architecture.mdx** —— 网关架构设计要点：集中式 vs 嵌入式、性能考量、高可用部署
- **route-match.mdx** —— 路由匹配规则：Path 精确/前缀/正则、Header 路由、Query 参数路由、优先级
- **filter-chain.mdx** —— 过滤器链原理：Pre/Post 过滤器、执行顺序、自定义过滤器开发（原理深入篇）
