# Canal 订阅模式

Canal 支持多种订阅模式，从简单的直连模式到复杂的高可用集群模式，满足不同业务场景的需求。

### Java 客户端

```java
public class CanalClient {

    public static void main(String[] args) {
        // 创建连接
        CanalConnector connector = CanalConnectors.newSingleConnector(
            new InetSocketAddress("localhost", 11111),
            "destination",  // instance 名
            "",             // username
            ""              // password
        );

        connector.connect();
        connector.subscribe();  // 订阅

        while (true) {
            // 拉取数据
            Message message = connector.get(100);

            if (message.getId() != -1) {
                for (Entry entry : message.getEntries()) {
                    processEntry(entry);
                }
            }

            // 确认
            connector.ack(message.getId());
        }
    }

    private static void processEntry(Entry entry) {
        String schema = entry.getHeader().getSchemaName();
        String table = entry.getHeader().getTableName();
        EventType eventType = entry.getHeader().getEventType();

        RowChange rowChange = RowChange.parseFrom(entry.getStoreValue());
        for (RowData rowData : rowChange.getRowDatasList()) {
            if (eventType == EventType.INSERT) {
                System.out.println("INSERT: " + rowData.getAfterColumnsList());
            } else if (eventType == EventType.UPDATE) {
                System.out.println("UPDATE: " + rowData.getBeforeColumnsList() + " -> " + rowData.getAfterColumnsList());
            } else if (eventType == EventType.DELETE) {
                System.out.println("DELETE: " + rowData.getBeforeColumnsList());
            }
        }
    }
}
```

### 客户端 HA

```java
public class CanalHAClient {

    public static void main(String[] args) {
        // ZooKeeper 发现可用的 Canal Server
        ClusterCanalConnector connector = CanalConnectors.newClusterConnector(
            "zk-host:2181",
            "destination",
            "",
            ""
        );

        connector.connect();
        connector.subscribe();

        // 自动 failover
        while (true) {
            Message message = connector.get(100);
            processMessage(message);
            connector.ack(message.getId());
        }
    }
}
```

## 数据过滤

### 正则过滤

```yaml title="instance.properties"
# 匹配所有表
canal.instance.filter.regex = .*\\..*

# 匹配指定库的所有表
canal.instance.filter.regex = order_db\\..*

# 匹配指定表
canal.instance.filter.regex = order_db\\.orders

# 匹配多表
canal.instance.filter.regex = order_db\\.(orders,items)
```

### 表字段过滤

```java
// 过滤字段
connector.subscribe("order_db.orders/.*");
// 等价于
connector.subscribe("order_db.orders/id,status");
```

### 过滤类型

```yaml title="instance.properties"
# 过滤 DML（INSERT/UPDATE/DELETE）
canal.instance.filter.dml = true

# 过滤 DDL
canal.instance.filter.ddl = false
```

## 增量同步配置

### 全量 + 增量

```java
public class FullAndIncrSync {

    public void sync() {
        // 1. 全量同步
        List<Order> allOrders = orderRepository.findAll();
        elasticsearchService.bulkIndex(allOrders);

        // 2. 记录同步时间
        Date syncTime = new Date();

        // 3. 增量同步
        CanalConnector connector = CanalConnectors.newSingleConnector(...);
        connector.connect();

        // 从指定时间开始订阅
        Timestamp timestamp = new Timestamp(syncTime.getTime());
        connector.subscribe();
        connector.seek(timestamp);

        while (true) {
            Message message = connector.get(100);
            for (Entry entry : message.getEntries()) {
                // 处理增量数据
                processEntry(entry);
            }
            connector.ack(message.getId());
        }
    }
}
```
