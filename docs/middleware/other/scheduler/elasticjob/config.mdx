# ElasticJob 配置与 Spring Boot 集成

ElasticJob 提供了与 Spring Boot 的深度集成，让配置更加简单。

## Maven 依赖

```xml title="pom.xml"
<dependency>
    <groupId>org.apache.shardingsphere.elasticjob</groupId>
    <artifactId>elasticjob-lite-spring-boot-starter</artifactId>
    <version>3.0.4</version>
</dependency>
```

## Spring Boot 配置

### 基本配置

```yaml title="application.yml"
spring:
  elasticjob:
    # ZooKeeper 注册中心
    reg-center:
      server-lists: localhost:2181
      namespace: elasticjob
```

### 完整配置

```yaml title="application.yml"
spring:
  elasticjob:
    # 注册中心配置
    reg-center:
      server-lists: localhost:2181
      namespace: elasticjob
      base-sleep-time-milliseconds: 1000
      max-sleep-time-milliseconds: 3000
      max-retries: 3

    # 作业配置
    jobs:
      myJob:
        # 分片数
        sharding-total-count: 4
        # 分片参数
        sharding-item-parameters: 0=Beijing,1=Shanghai,2=Guangzhou,3=Shenzhen
        # cron 表达式
        cron: 0/5 * * * * ?
        # 作业参数
        job-parameter: test
        # 是否启用
        enabled: true
        # 覆盖配置
        overwrite: true
        # 监控端口
        monitor-port: 9888
        # 最大时间差（秒）
        max-time-diff-seconds: 60
        # 错过触发
        misfire: true
        # 作业分片策略
        sharding-strategy-type: AVG_ALLOCATION

      anotherJob:
        sharding-total-count: 2
        sharding-item-parameters: ""
        cron: 0/10 * * * * ?
```

## 定义作业

### SimpleJob

```java
package com.example.job;

import org.apache.shardingsphere.elasticjob.api.ShardingContext;
import org.apache.shardingsphere.elasticjob.simple.job.SimpleJob;
import org.springframework.stereotype.Component;

@Component
public class MySimpleJob implements SimpleJob {

    @Override
    public void execute(ShardingContext context) {
        int shardingItem = context.getShardingItem();
        String shardingParameter = context.getShardingParameter();

        System.out.println("执行分片: " + shardingItem + ", 参数: " + shardingParameter);

        // 业务逻辑
        process(shardingItem);
    }

    private void process(int shardingItem) {
        // 处理逻辑
    }
}
```

```java
import org.springframework.stereotype.Component;

import java.util.List;

@Component
public class MyDataflowJob implements DataflowJob<String> {

    @Override
    public List<String> fetchData(ShardingContext context) {

        return dataRepository.find(100);

    public void processData(ShardingContext context, List<String> data) {
        // 处理数据
        for (String item : data) {
            process(item);
        }
    }

    private void process(String item) {
        // 处理逻辑
    }
}
```

### Spring 配置

```yaml title="application.yml"
spring:
  elasticjob:
    jobs:
      mySimpleJob:
        class: com.example.job.MySimpleJob
        sharding-total-count: 4
        sharding-item-parameters: 0=A,1=B,2=C,3=D
        cron: 0/5 * * * * ?

      myDataflowJob:
        class: com.example.job.MyDataflowJob
        sharding-total-count: 2

        streaming-processing: true
```

## 作业监听器

### 定义监听器

```java
@Component
public class MyJobListener implements ElasticJobListener {

    @Override
    public void beforeJobExecuted(ShardingContexts shardingContexts) {
        System.out.println("作业执行前");
    }

    @Override
    public void afterJobExecuted(ShardingContexts shardingContexts) {
        System.out.println("作业执行后");
    }
}
```

### Spring 配置

```yaml title="application.yml"
spring:
  elasticjob:
    jobs:
      myJob:
        class: com.example.job.MySimpleJob
        sharding-total-count: 3
        cron: 0/5 * * * * ?
        listener:
          simple:
            - com.example.listener.MyJobListener
```

## 作业分片策略

### 内置分片策略

| 策略 | 说明 |
|---|---|
| AVG_ALLOCATION | 平均分配 |
| CONSISTENT_HASH | 一致性哈希 |
| ROTATION | 轮询分配 |
| IDLE_BYTIME | 按时间空闲分配 |

### 自定义分片策略

```java
public class MyShardingStrategy implements ShardingStrategy {

    @Override
    public Map<Instance, List<Integer>> sharding(
            List<Integer> shardingItems,
            ShardingConfiguration shardingConfiguration) {

        Map<Instance, List<Integer>> result = new HashMap<>();

        List<Instance> instances = shardingConfiguration.getInstances();
        int i = 0;

        for (int item : shardingItems) {
            Instance instance = instances.get(i % instances.size());
            result.computeIfAbsent(instance, k -> new ArrayList<>()).add(item);
            i++;
        }

        return result;
    }
}
```

```yaml
spring:
  elasticjob:
    jobs:
      myJob:
        class: com.example.job.MySimpleJob
        sharding-total-count: 4
        sharding-strategy-type: com.example.strategy.MyShardingStrategy
```

## 监控

### 运维平台

ElasticJob 提供了运维平台：

```yaml
spring:
  elasticjob:
    # 运维 API 配置
    rest:
      host: 0.0.0.0
      port: 8899
```

### 监控 API

```bash
# 查询作业状态
curl http://localhost:8899/job/detail?jobName=myJob

# 查询分片状态
curl http://localhost:8899/sharding/detail?jobName=myJob&shardingItem=0
```

| `spring.elasticjob.jobs` | 作业配置 |
