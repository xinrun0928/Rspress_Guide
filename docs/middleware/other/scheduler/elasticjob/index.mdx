# ElasticJob

ElasticJob 是面向互联网的海量任务调度框架，由当当网开源，现为 Apache ShardingSphere 子项目。

## 为什么选择 ElasticJob？

- 去中心化架构：无需调度中心，依赖 ZooKeeper 协调
- 强大的分片能力：支持任务分片、广播任务
- 多种作业类型：Simple、Dataflow、Script
- 云原生支持：与 Kubernetes 深度集成

## 模块学习路径

```
1. ElasticJob 架构：LiteJob 与 CloudJob 两种模式
2. 分片原理：Sharding + ShardingItemParameters
3. 作业类型：Simple、Dataflow、Script
4. 作业监听器：ElasticJobListener
5. 配置与 Spring Boot 集成
6. ElasticJob-Cloud 与 Mesos
7. 注册中心原理：ZooKeeper
8. ElasticJob vs XXL-Job vs Quartz
```

> [!TIP]
> ElasticJob 的核心设计思想是「去中心化调度」，通过 ZooKeeper 实现任务的协调和选举。
