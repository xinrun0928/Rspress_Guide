# ElasticJob 作业监听器

ElasticJob 提供了作业监听器机制，可以在作业执行前后添加自定义逻辑。

## 监听器类型

```
┌─────────────────────────────────────────────────────────────────────┐
│                         ElasticJob 监听器类型                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────┐  ┌─────────────────────┐             │
│   │   ElasticJobListener │  │  ExecutionListener  │             │
│   │                      │  │                      │             │
│   │  作业级别监听        │  │  分片级别监听       │             │
│   │  before/after      │  │  before/after      │             │
│   │                      │  │                      │             │
│   │  全局生效           │  │  分片维度          │             │
│   └─────────────────────┘  └─────────────────────┘             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## ElasticJobListener

### 定义监听器

```java
public class MyJobListener implements ElasticJobListener {

    @Override
    public void beforeJobExecuted(ShardingContexts shardingContexts) {
        System.out.println("作业执行前");
        System.out.println("作业名: " + shardingContexts.getJobName());
        System.out.println("分片总数: " + shardingContexts.getShardingTotalCount());
    }

    @Override
    public void afterJobExecuted(ShardingContexts shardingContexts) {
        System.out.println("作业执行后");
        System.out.println("执行状态: " + shardingContexts.isSuccess());
    }
}
```

### 注册监听器

```java
// 方式一：注解配置
@ElasticJobConf(
    name = "myJob",
    shardingTotalCount = 3,
    cron = "0/5 * * * * ?",
    listenerClasses = MyJobListener.class
)
public class MyJob implements SimpleJob {
    @Override
    public void execute(ShardingContext context) {
        // 业务逻辑
    }
}

// 方式二：Spring 配置
<elasticjob:job id="myJob"
    class="com.example.MyJob"
    sharding-total-count="3"
    cron="0/5 * * * * ?">
    <elasticjob:listener-class class="com.example.MyJobListener"/>
</elasticjob:job>
```

## ExecutionListener

### 定义监听器

```java
public class MyExecutionListener implements ExecutionListener {

    @Override
    public void beforeJobExecution(ShardingContexts shardingContexts) {
        System.out.println("作业执行前");
    }

    @Override
    public void afterJobExecution(ShardingContexts shardingContexts, Throwable cause) {
        if (cause != null) {
            System.out.println("作业执行失败: " + cause.getMessage());
        } else {
            System.out.println("作业执行成功");
        }
    }
}
```

### 注册监听器

```java
@ElasticJobConf(
    name = "myJob",
    shardingTotalCount = 3,
    cron = "0/5 * * * * ?",
    executionListenerClasses = MyExecutionListener.class
)
public class MyJob implements SimpleJob {
    @Override
    public void execute(ShardingContext context) {
        // 业务逻辑
    }
}
```

## 全局监听器

### Spring 配置全局监听器

```xml
<!-- 全局作业监听器 -->
<bean id="globalJobListener" class="com.example.GlobalJobListener"/>

<elasticjob:job-registry>
    <elasticjob:listener
        globalListenerClass="com.example.GlobalJobListener"/>
</elasticjob:job-registry>
```

### 全局监听器示例

```java
public class GlobalJobListener implements ElasticJobListener {

    @Override
    public void beforeJobExecuted(ShardingContexts shardingContexts) {
        // 发送开始告警
        alertService.send("作业开始", "作业 " + shardingContexts.getJobName() + " 开始执行");
    }

    @Override
    public void afterJobExecuted(ShardingContexts shardingContexts) {
        if (shardingContexts.isSuccess()) {
            // 记录成功日志
            logService.log("作业成功", shardingContexts);
        } else {
            // 发送失败告警
            alertService.send("作业失败", "作业 " + shardingContexts.getJobName() + " 执行失败");
        }
    }
}
```

## 监听器执行顺序

```
┌─────────────────────────────────────────────────────────────────────┐
│                         监听器执行顺序                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   1. beforeJobExecuted (ElasticJobListener)                         │
│   2. beforeJobExecution (ExecutionListener)                         │
│   3. execute (Job)                                               │
│   4. afterJobExecution (ExecutionListener)                         │
│   5. afterJobExecuted (ElasticJobListener)                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 异常处理

```java
public class MyJobListener implements ElasticJobListener {

    @Override
    public void beforeJobExecuted(ShardingContexts shardingContexts) {
        try {
            // 可能抛出异常的操作
            prepare();
        } catch (Exception e) {
            // 记录日志，不影响主流程
            Logger.getLogger(getClass()).warning("准备失败: " + e.getMessage());
        }
    }

    @Override
    public void afterJobExecuted(ShardingContexts shardingContexts) {
        // 获取执行异常
        if (shardingContexts.getJobExecutionException() != null) {
            // 处理异常
            handleError(shardingContexts.getJobExecutionException());
        }
    }
}
```

## 总结

监听器类型对比：

| 监听器 | 作用域 | 方法 |
|---|---|---|
| ElasticJobListener | 作业级别 | beforeJobExecuted, afterJobExecuted |
| ExecutionListener | 分片级别 | beforeJobExecution, afterJobExecution |
