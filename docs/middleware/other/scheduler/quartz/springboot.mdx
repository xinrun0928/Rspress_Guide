# Quartz Spring Boot 集成

Spring Boot 提供了对 Quartz 的原生支持，只需简单配置即可使用。

## 快速开始

### 添加依赖

```xml title="pom.xml"
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-quartz</artifactId>
</dependency>
```

### 基本配置

```yaml title="application.yml"
spring:
  quartz:
    # JobStore 类型
    job-store-type: memory
    # 自动启动
    auto-startup: true
    # 启动延迟
    startup-delay: 0
    # 关闭时等待任务完成
    wait-for-jobs-to-complete-on-shutdown: true
```

### 使用 @Scheduled

Spring Boot 提供了更简单的 @Scheduled 注解：

```java
@SpringBootApplication
@EnableScheduling
public class Application {

    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}

@Service
public class TaskService {

    @Scheduled(cron = "0 0 2 * * ?")
    public void dailyTask() {
        System.out.println("每天凌晨 2 点执行");
    }

    @Scheduled(fixedRate = 5000)
    public void fixedRateTask() {
        System.out.println("每 5 秒执行一次");
    }

    @Scheduled(fixedDelay = 3000)
    public void fixedDelayTask() {
        System.out.println("上次执行完成后 3 秒再执行");
    }
}
```

## Quartz Job 方式

### 创建 Job

```java
public class MyJob implements Job {

    @Override
    public void execute(JobExecutionContext context) throws JobExecutionException {
        System.out.println("Quartz Job 执行!");
    }
}
```

### 配置 JobDetail

```java
@Configuration
public class QuartzConfig {

    @Bean
    public JobDetail myJobDetail() {
        return JobBuilder.newJob(MyJob.class)
            .withIdentity("myJob", "group1")
            .storeDurably()
            .build();
    }

    @Bean
    public Trigger myTrigger() {
        return TriggerBuilder.newTrigger()
            .forJob(myJobDetail())
            .withIdentity("myTrigger", "group1")
            .withSchedule(CronScheduleBuilder.cronSchedule("0 0 2 * * ?"))
            .build();
    }
}
```

## Spring Bean 方式

### 使用 @Component

```java
@Component
public class SpringJob {

    public void execute() {
        System.out.println("Spring Bean Job 执行!");
    }
}
```

### 配置

```java
@Configuration
public class QuartzConfig {

    @Autowired
    private SpringJob springJob;

    @Bean
    public JobDetail springJobDetail() {
        return JobBuilder.newJob(SpringJob.class)
            .withIdentity("springJob", "group1")
            .storeDurably()
            .build();
    }

    @Bean
    public MethodInvokingJobDetailFactoryBean methodInvokingJobDetail() {
        MethodInvokingJobDetailFactoryBean factory = new MethodInvokingJobDetailFactoryBean();
        factory.setTargetObject(springJob);
        factory.setTargetMethod("execute");
        factory.setConcurrent(false);  // 禁止并发
        return factory;
    }
}
```

## JobFactory

### 自定义 JobFactory

使用自定义 JobFactory 可以让 Spring 管理 Job 的依赖注入：

```java
public class AutowiringSpringBeanJobFactory extends SpringBeanJobFactory
        implements ApplicationContextAware {

    private ApplicationContext applicationContext;

    @Override
    public void setApplicationContext(ApplicationContext context) {
        this.applicationContext = context;
    }

    @Override
    protected Object createJobInstance(TriggerFiredBundle bundle) throws Exception {
        Object job = super.createJobInstance(bundle);
        applicationContext.getAutowireCapableBeanFactory().autowireBean(job);
        return job;
    }
}
```

```java
@Configuration
public class QuartzConfig {

    @Bean
    public AutowiringSpringBeanJobFactory jobFactory() {
        return new AutowiringSpringBeanJobFactory();
    }

    @Bean
    public SchedulerFactoryBean schedulerFactoryBean(JobFactory jobFactory) {
        SchedulerFactoryBean factory = new SchedulerFactoryBean();
        factory.setJobFactory(jobFactory);
        return factory;
    }
}
```

## JDBC JobStore 配置

```yaml title="application.yml"
spring:
  quartz:
    job-store-type: jdbc
    jdbc:
      initialize-schema: always
```

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/quartz
    username: root
    password: password
  quartz:
    job-store-type: jdbc
    properties:
      org:
        quartz:
          jobStore:
            class: org.quartz.impl.jdbcjobstore.JobStoreTX
            driverDelegateClass: org.quartz.impl.jdbcjobstore.StdJDBCDelegate
            tablePrefix: QRTZ_
            isClustered: true
            clusterCheckinInterval: 10000
          dataSource:
            quartzDS:
              driver: com.mysql.cj.jdbc.Driver
              URL: jdbc:mysql://localhost:3306/quartz
              user: root
              password: password
              maxConnections: 10
```

## 总结

Spring Boot 集成核心配置：

| 配置项 | 说明 |
|---|---|
| `spring.quartz.job-store-type` | 存储类型：memory/jdbc |
| `spring.quartz.auto-startup` | 自动启动 |
| `spring.quartz.jdbc.initialize-schema` | 初始化 Schema |
