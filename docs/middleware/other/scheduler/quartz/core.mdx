# Quartz 核心概念

理解 Quartz 的核心概念，是掌握 Quartz 的基础。

## 四大核心概念

```
┌─────────────────────────────────────────────────────────────────────┐
│                         Quartz 核心概念                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────┐                                                  │
│   │  Scheduler  │ → 调度器，核心控制器                             │
│   └─────────────┘                                                  │
│         │                                                           │
│         ├──→ ┌─────────────┐                                     │
│         │    │   Trigger   │ → 触发器，定义执行时间                 │
│         │    └─────────────┘                                     │
│         │         │                                               │
│         │         └──→ ┌─────────────┐                           │
│         │              │   Job       │ → 任务，真正执行的工作         │
│         │              └─────────────┘                           │
│         │                   │                                       │
│         │                   └──→ ┌─────────────┐                   │
│         │                        │  JobDetail  │ → 任务详情，包含配置 │
│         │                        └─────────────┘                   │
│         │                                                         │
│         └──→ ┌─────────────┐                                     │
│              │  JobStore  │ → 任务存储，支持内存/数据库            │
│              └─────────────┘                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Scheduler（调度器）

Scheduler 是 Quartz 的核心控制器，负责管理所有的 Trigger 和 Job。

### 获取调度器

```java
// 方式一：从默认配置获取
Scheduler scheduler = StdSchedulerFactory.getDefaultScheduler();

// 方式二：从自定义配置获取
Properties props = new Properties();
props.put("org.quartz.scheduler.instanceName", "MyScheduler");
props.put("org.quartz.threadPool.class", "org.quartz.simpl.SimpleThreadPool");
props.put("org.quartz.threadPool.threadCount", "10");
StdSchedulerFactory factory = new StdSchedulerFactory(props);
Scheduler scheduler = factory.getScheduler();
```

### 生命周期

```java
// 启动调度器
scheduler.start();

// 暂停调度器
scheduler.standby();

// 关闭调度器（参数表示是否等待正在执行的任务完成）
scheduler.shutdown(false);  // 不等待
scheduler.shutdown(true);   // 等待执行完成
```

### 基本操作

```java
// 添加任务
scheduler.scheduleJob(jobDetail, trigger);

// 删除任务
scheduler.deleteJob(jobKey);

// 暂停任务
scheduler.pauseJob(jobKey);

// 恢复任务
scheduler.resumeJob(jobKey);

// 触发任务（立即执行一次）
scheduler.triggerJob(jobKey);

// 获取任务信息
List<JobExecutionContext> runningJobs = scheduler.getCurrentlyExecutingJobs();
```

## Trigger（触发器）

Trigger 定义「什么时候执行」任务。

### Trigger 类型

| 类型 | 说明 | 使用场景 |
|---|---|---|
| SimpleTrigger | 固定时间间隔 | 「每天凌晨执行一次」 |
| CronTrigger | Cron 表达式 | 「每周一 9 点执行」 |
| DateIntervalTrigger | 日期间隔 | 「每隔 5 天执行」 |
| NthIncludedDayTrigger | 第 N 天 | 「每月第 15 天执行」 |

### 创建 Trigger

```java
// 创建 Trigger
Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")  // 名称和组
    .startNow()                            // 立即启动
    .withSchedule(SimpleScheduleBuilder.simpleSchedule()
        .withIntervalInHours(1)            // 每小时执行
        .repeatForever())                  // 永久重复
    .build();

// 也可以指定开始和结束时间
Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")
    .startAt(startTime)                    // 开始时间
    .endAt(endTime)                        // 结束时间
    .withSchedule(CronScheduleBuilder.cronSchedule("0 0 2 * * ?"))
    .build();
```

## Job（任务）

Job 定义「执行什么」。

### Job 接口

```java
public interface Job {
    void execute(JobExecutionContext context) throws JobExecutionException;
}
```

### 实现 Job

```java
public class HelloJob implements Job {

    @Override
    public void execute(JobExecutionContext context) throws JobExecutionException {
        // 业务逻辑
        System.out.println("Hello Quartz!");

        // 获取 Trigger 信息
        Trigger trigger = context.getTrigger();
        System.out.println("Trigger: " + trigger.getKey());

        // 获取 JobDetail 信息
        JobDetail jobDetail = context.getJobDetail();
        System.out.println("Job: " + jobDetail.getKey());

        // 获取参数
        JobDataMap dataMap = jobDetail.getJobDataMap();
        String message = dataMap.getString("message");
        System.out.println("Message: " + message);
    }
}
```

## JobDetail（任务详情）

JobDetail 包含 Job 的配置信息。

### 创建 JobDetail

```java
JobDetail jobDetail = JobBuilder.newJob(HelloJob.class)
    .withIdentity("helloJob", "group1")    // 名称和组
    .withDescription("这是一个示例任务")     // 描述
    .usingJobData("message", "Hello!")    // 传入参数
    .storeDurably()                        // 即使没有 Trigger 也保存
    .build();
```

### JobDataMap

```java
// 创建时传入数据
JobDetail jobDetail = JobBuilder.newJob(HelloJob.class)
    .usingJobData("count", 0)
    .usingJobData("userId", 123)
    .build();

// 在 Job 中获取
public class HelloJob implements Job {
    @Override
    public void execute(JobExecutionContext context) throws JobExecutionException {
        JobDataMap dataMap = context.getJobDetail().getJobDataMap();

        int count = dataMap.getInt("count");
        long userId = dataMap.getLong("userId");
    }
}

// 更新数据（下次执行时生效）
context.getJobDetail().getJobDataMap().put("count", count + 1);
```

## 完整示例

```java
public class QuartzExample {

    public static void main(String[] args) throws SchedulerException {
        // 1. 创建调度器
        Scheduler scheduler = StdSchedulerFactory.getDefaultScheduler();

        // 2. 创建任务
        JobDetail jobDetail = JobBuilder.newJob(HelloJob.class)
            .withIdentity("helloJob", "group1")
            .usingJobData("message", "Hello Quartz!")
            .build();

        // 3. 创建触发器（每 3 秒执行一次）
        Trigger trigger = TriggerBuilder.newTrigger()
            .withIdentity("myTrigger", "group1")
            .startNow()
            .withSchedule(SimpleScheduleBuilder.simpleSchedule()
                .withIntervalInSeconds(3)
                .repeatForever())
            .build();

        // 4. 注册并启动
        scheduler.scheduleJob(jobDetail, trigger);
        scheduler.start();

        // 5. 运行一段时间后关闭
        Thread.sleep(10000);
        scheduler.shutdown(true);
    }
}
```

## 总结

四大核心概念：

| 概念 | 说明 |
|---|---|
| Scheduler | 调度器，管理 Trigger 和 Job |
| Trigger | 触发器，定义执行时间 |
| Job | 任务，定义执行逻辑 |
| JobDetail | 任务详情，包含配置和数据 |
