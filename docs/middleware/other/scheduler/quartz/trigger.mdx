# Quartz Trigger 类型

Trigger 定义「什么时候执行」任务。Quartz 支持多种 Trigger 类型，最常用的是 SimpleTrigger 和 CronTrigger。

## SimpleTrigger

SimpleTrigger 适用于「每隔固定时间执行」的场景。

### 基本用法

```java
// 每 10 秒执行一次，共执行 5 次
Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")
    .startNow()  // 立即开始
    .withSchedule(SimpleScheduleBuilder.simpleSchedule()
        .withIntervalInSeconds(10)
        .withRepeatCount(4))  // 重复 4 次 = 总共 5 次
    .build();
```

### 完整参数

```java
Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")
    .startAt(startTime)  // 开始时间
    .endAt(endTime)       // 结束时间
    .withSchedule(SimpleScheduleBuilder.simpleSchedule()
        .withIntervalInMilliseconds(1000)  // 间隔
        .withRepeatCount(10)               // 重复次数
        .repeatForever())                  // 或永久重复
    .build();
```

### 时间设置

```java
// 5 分钟后开始，每 30 分钟执行一次
Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")
    .startAt(DateBuilder.futureDate(5, DateBuilder.IntervalUnit.MINUTE))
    .withSchedule(SimpleScheduleBuilder.simpleSchedule()
        .withIntervalInMinutes(30)
        .repeatForever())
    .build();

// 指定具体时间开始
Calendar calendar = Calendar.getInstance();
calendar.set(2024, Calendar.JANUARY, 1, 2, 0, 0);  // 2024-01-01 02:00:00
Date startTime = calendar.getTime();

Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")
    .startAt(startTime)
    .withSchedule(SimpleScheduleBuilder.simpleSchedule()
        .withIntervalInHours(24)
        .repeatForever())
    .build();
```

## CronTrigger

CronTrigger 适用于「复杂时间规则」的场景。

### Cron 表达式

```
┌─────────────────────────────────────────────────────────────────────┐
│                         Cron 表达式格式                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   秒 分 时 日 月 周 [年]                                             │
│                                                                     │
│   ┌──────────┬──────────┬──────────┬──────────┬──────────┬────────┐│
│   │   0      │   0      │   2      │   *      │   *      │   ?    ││
│   │  秒       │  分       │  时       │  日       │  月       │  周    ││
│   └──────────┴──────────┴──────────┴──────────┴──────────┴────────┘│
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 常用表达式

| 表达式 | 说明 |
|---|---|
| `0 0 * * * ?` | 每小时整点 |
| `0 0 2 * * ?` | 每天凌晨 2 点 |
| `0 30 9 ? * MON-FRI` | 工作日 9:30 |
| `0 0/5 * * * ?` | 每 5 分钟 |
| `0 0 0 1 * ?` | 每月 1 号 |
| `0 0 0 ? * 1` | 每周一 |
| `0 30 10,14,16 * * ?` | 10:30、14:30、16:30 |

### 基本用法

```java
Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")
    .withSchedule(CronScheduleBuilder.cronSchedule("0 0 2 * * ?"))
    .build();
```

### CronScheduleBuilder

```java
// 每天凌晨 2 点执行
CronScheduleBuilder cronSchedule = CronScheduleBuilder
    .cronSchedule("0 0 2 * * ?")
    .withMisfireHandlingInstructionFireAndProceed();

// 时区设置
CronScheduleBuilder cronSchedule = CronScheduleBuilder
    .cronSchedule("0 0 2 * * ?")
    .inTimeZone(TimeZone.getTimeZone("Asia/Shanghai"));
```

## Trigger 优先级

当多个 Trigger 同时触发时，优先级决定执行顺序：

```java
Trigger trigger1 = TriggerBuilder.newTrigger()
    .withIdentity("trigger1", "group1")
    .withPriority(10)
    .startNow()
    .build();

Trigger trigger2 = TriggerBuilder.newTrigger()
    .withIdentity("trigger2", "group1")
    .withPriority(5)
    .startNow()
    .build();
```

## Trigger 操作

```java
// 暂停 Trigger
scheduler.pauseTrigger(triggerKey);

// 恢复 Trigger
scheduler.resumeTrigger(triggerKey);

// 获取 Trigger 状态
TriggerState state = scheduler.getTriggerState(triggerKey);

// 重置 Trigger
scheduler.unscheduleJob(triggerKey);
```

## 总结

| Trigger 类型 | 适用场景 |
|---|---|
| SimpleTrigger | 固定间隔、次数明确的执行 |
| CronTrigger | 复杂时间规则（周、月等） |
