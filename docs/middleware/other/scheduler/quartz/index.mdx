# Quartz

Quartz 是 Java 领域最成熟的定时任务框架，功能完善、文档丰富，是理解任务调度原理的最佳选择。

## Quartz 是什么？

Quartz 是一个完全由 Java 编写的开源任务调度框架，支持：

- 丰富的调度方式（Cron、时间间隔、指定日期）
- 持久化任务到数据库
- 集群部署支持
- 监听器机制

## 模块学习路径

```
1. Quartz 核心概念：Scheduler、Trigger、Job、JobDetail
2. Trigger 类型：SimpleTrigger 与 CronTrigger
3. 任务状态与持久化
4. Spring Boot 集成
5. 集群原理：JDBC JobStore 与分布式锁
6. Misfire 策略
```
