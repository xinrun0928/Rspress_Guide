# XXL-Job

XXL-Job 是国内最流行的轻量级分布式任务调度平台，由大众点评工程师许雪里创建。

## 为什么选择 XXL-Job？

- 管理界面完善：Web UI 支持任务管理、配置
- 部署简单：调度中心 + 执行器的两层架构
- 功能丰富：任务分片、失败重试、任务依赖
- 文档完善：中文文档，社区活跃

## 模块学习路径

```
1. XXL-Job 架构：调度中心与执行器
2. 任务类型：Bean 模式、GLUE 模式、脚本
3. 路由策略：多种负载均衡策略
4. 重试机制：失败重试、阻塞处理
5. 任务编排：子任务、任务链
6. GLUE 代码开发模式
7. 高可用部署
```

> [!TIP]
> XXL-Job 的核心设计思想是「调度中心负责管理，执行器负责执行」，调度中心不执行具体任务。
