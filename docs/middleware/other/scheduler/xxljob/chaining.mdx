# XXL-Job 任务编排

XXL-Job 支持任务编排，可以通过子任务和任务链实现任务的依赖和串联。

## 子任务

子任务是简单的依赖关系：任务 A 执行完成后，自动触发任务 B。

### 配置子任务

在调度中心配置：

```
任务 A 配置：
├── 任务名称：任务 A
├── Cron：0 0 2 * * ?
└── 子任务：任务 B

任务 B 配置：
├── 任务名称：任务 B
└── 无子任务
```

### 执行效果

```
任务 A 执行：
├── 任务 A 开始
├── 任务 A 完成
│     ↓
├── 触发任务 B
├── 任务 B 开始
└── 任务 B 完成
```

### 代码实现

```java
@Component
public class SubJobHandler {

    @XxlJob("parentJobHandler")
    public ReturnT<String> execute(String param) {
        // 父任务执行
        System.out.println("任务 A 执行");

        // 框架会自动触发子任务
        return ReturnT.SUCCESS;
    }

    @XxlJob("childJobHandler")
    public ReturnT<String> childJob(String param) {
        // 子任务执行
        System.out.println("任务 B 执行");
        return ReturnT.SUCCESS;
    }
}
```

## 任务链

任务链支持更复杂的编排，包括条件执行和结果传递。

### 配置任务链

```yaml
# 任务 A 配置
任务名称: A
Cron: 0 0 2 * * ?
下游任务链:
  - 任务 B (成功时)
  - 任务 C (失败时)
```

### 代码示例

```java
@Component
public class ChainJobHandler {

    @XxlJob("jobA")
    public ReturnT<String> jobA(String param) {
        System.out.println("任务 A 执行");

        // 执行任务 B
        boolean success = XxlJobHelper.trigger("jobB");

        // 根据任务 B 结果决定下一步
        if (success) {
            XxlJobHelper.trigger("jobC");
        } else {
            XxlJobHelper.trigger("jobD");
        }

        return ReturnT.SUCCESS;
    }

    @XxlJob("jobB")
    public ReturnT<String> jobB(String param) {
        System.out.println("任务 B 执行");
        return ReturnT.SUCCESS;
    }
}
```

## 动态任务编排

### 动态触发子任务

```java
@XxlJob("dynamicChainJob")
public ReturnT<String> execute(String param) {
    // 解析参数
    List<Long> subTaskIds = parseSubTasks(param);

    // 逐个触发
    for (Long taskId : subTaskIds) {
        XxlJobHelper.trigger(taskId);
    }

    return ReturnT.SUCCESS;
}
```

### 条件触发

```java
@XxlJob("conditionalJobHandler")
public ReturnT<String> execute(String param) {
    // 业务逻辑
    boolean condition = doSomething();

    // 根据条件触发不同任务
    if (condition) {
        XxlJobHelper.trigger("successJob");
    } else {
        XxlJobHelper.trigger("failJob");
    }

    return ReturnT.SUCCESS;
}
```

## 任务执行顺序

### 顺序执行

```
触发任务 A → 任务 A 完成 → 触发任务 B → 任务 B 完成 → ...
```

### 并行执行

```java
@XxlJob("parallelJobHandler")
public ReturnT<String> execute(String param) {
    // 同时触发多个任务
    CompletableFuture.allOf(
        CompletableFuture.runAsync(() -> XxlJobHelper.trigger("jobA")),
        CompletableFuture.runAsync(() -> XxlJobHelper.trigger("jobB")),
        CompletableFuture.runAsync(() -> XxlJobHelper.trigger("jobC"))
    ).join();

    return ReturnT.SUCCESS;
}
```

### 等待多个任务完成

```java
@XxlJob("waitAllJobHandler")
public ReturnT<String> execute(String param) {
    // 触发多个任务
    String jobARet = XxlJobHelper.trigger("jobA");
    String jobBRet = XxlJobHelper.trigger("jobB");
    String jobCRet = XxlJobHelper.trigger("jobC");

    // 等待所有任务完成（通过日志或其他方式）
    waitForComplete(jobARet, jobBRet, jobCRet);

    return ReturnT.SUCCESS;
}
```

## 总结

任务编排方式：

| 方式 | 说明 |
|---|---|
| 子任务 | 简单的单向依赖 |
| 任务链 | 支持条件判断 |
| 动态触发 | 根据参数动态决定 |
