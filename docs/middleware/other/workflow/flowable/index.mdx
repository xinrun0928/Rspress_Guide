# Flowable

Flowable 是 Java 领域最活跃的开源工作流引擎，从 Activiti 6.x 分支出来，经过多年发展，已经成为企业级工作流选型的热门选择。

## 为什么选择 Flowable？

Flowable 的优势在于：

- **六大引擎**：BPMN、CMMN、DMN、Form、Content、App，覆盖多种流程场景
- **Spring Boot 原生支持**：开箱即用，零配置
- **活跃的社区**：文档完善，社区活跃
- **可扩展架构**：支持自定义引擎和插件

## 模块学习路径

```
1. Flowable 架构：六大引擎一览
2. 核心 API：部署、任务、网关
3. 高级特性：监听器、异步执行、子流程
4. Spring Boot 集成：自动配置
5. 数据库设计：核心表结构
```

让我们开始学习。
