# Flowable Spring Boot 集成

Flowable 对 Spring Boot 提供了开箱即用的支持，只需添加依赖、简单配置，就能快速集成。

## 快速开始

### 添加依赖

```xml title="pom.xml"
<dependency>
    <groupId>org.flowable</groupId>
    <artifactId>flowable-spring-boot-starter</artifactId>
    <version>7.0.0</version>
</dependency>

<!-- 完整依赖包含所有引擎 -->
<!-- Process Engine, CMMN Engine, DMN Engine, Form Engine, Content Engine, App Engine -->
```

### Spring Boot 自动配置

Flowable Spring Boot Starter 会自动配置：

```java
// 自动配置的组件
ProcessEngine               // 流程引擎
RuntimeService              // 运行时服务
TaskService                 // 任务服务
RepositoryService           // 仓储服务
HistoryService              // 历史服务
IdentityService             // 身份服务
ManagementService           // 管理服务
FormService                 // 表单服务
ContentService              // 内容服务
DynamicBpmnService          // 动态 BPMN 服务
```

### 应用入口

```java
@SpringBootApplication
public class FlowableApplication {

    public static void main(String[] args) {
        SpringApplication.run(FlowableApplication.class, args);
    }
}
```

## 配置文件

### 基本配置

```yaml title="application.yml"
spring:
  datasource:
    url: jdbc:h2:mem:flowable
    driver-class-name: org.h2.Driver
    username: sa
    password:

  flowable:
    # 自动部署 BPMN 文件
    process-definition-location-prefix: classpath:/processes/
    .check-process-definitions: true
    # 数据库 Schema 自动更新
    database-schema-update: true
    # 异步执行
    async-executor-enabled: true
    async-executor-activate: true
```

### 高级配置

```yaml title="application.yml"
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/flowable?useSSL=false
    username: root
    password: root

  flowable:
    # 数据库配置
    database-schema-update: true
    database-type: mysql

    # 异步执行配置
    async:
      executor-enabled: true
      executor-activate: true
      core-pool-size: 8
      max-pool-size: 16
      queue-size: 100

    # 历史配置
    history-level: full
    history-enabled: true

    # 自动部署配置
    process-definition-location-base-path: processes/
    process-definition-locations:
      - classpath:/processes/*.bpmn20.xml
      - classpath:/processes/*.bpmn
    .check-process-definitions: true

    # DMN 自动部署
    dmn-definition-locations:
      - classpath:/dmn/*.dmn

    # 表单自动部署
    form-definition-locations:
      - classpath:/forms/*.form
```

### 历史级别

| 级别 | 说明 | 数据量 |
|---|---|---|
| none | 不记录历史 | 最小 |
| activity | 只记录流程实例和活动实例 | 小 |
| audit | 记录活动实例和变量（可审计） | 中 |
| full | 完整历史记录 | 大 |

## 自动部署

### 目录结构

```
src/main/resources/
└── processes/
    ├── leave-process.bpmn20.xml
    ├── order-process.bpmn20.xml
    └── sub-processes/
        └── sub-approval.bpmn20.xml
```

### 部署监听器

```java
@Component
public class FlowableDeploymentListener
    implements FlowableProcessAutoDeploymentDeploymentListenerHandler {

    @Override
    public String getPathMatcher() {
        // 自定义路径匹配规则
        return "**/*.bpmn";
    }

    @Override
    public Set<String> getDeploymentNameSuffixes() {
        return Collections.singleton("bpmn");
    }
}
```

### 自定义部署

```java
@Configuration
public class FlowableConfig {

    @Bean
    public SpringProcessEngineConfiguration processEngineConfiguration(
            DataSource dataSource,
            PlatformTransactionManager transactionManager) {

        SpringProcessEngineConfiguration config =
            new SpringProcessEngineConfiguration();

        config.setDataSource(dataSource);
        config.setTransactionManager(transactionManager);
        config.setDatabaseSchemaUpdate("true");
        config.setAsyncExecutorEnabled(true);
        config.setAsyncExecutorActivate(true);
        config.setHistoryLevel("full");

        // 自定义 BPMN 部署
        config.setDeploymentMode("single-resource");

        return config;
    }
}
```

### 集成 @Async

```java
@Service
public class WorkflowService {

    @Autowired
    private RuntimeService runtimeService;

    @Autowired
    private TaskService taskService;

    // 事务内执行
    @Transactional
    public void startProcess() {
        runtimeService.startProcessInstanceByKey("leave-process");
    }

    // 异步执行（Spring @Async）
    @Async
    public void asyncNotify(String taskId) {
        taskService.complete(taskId);
    }
```

## Flowable REST API

### 启用 REST API

```xml
<dependency>
    <groupId>org.flowable</groupId>
    <artifactId>flowable-rest</artifactId>
</dependency>
```

### REST 控制器

Flowable 提供完整的 REST API：

```bash
# 流程定义
GET  /repository/process-definitions
GET  /repository/process-definitions/{processDefinitionId}
POST /repository/deployments

# 流程实例
GET  /runtime/process-instances
POST /runtime/process-instances
GET  /runtime/process-instances/{processInstanceId}
DELETE /runtime/process-instances/{processInstanceId}

# 任务
GET  /runtime/tasks
POST /runtime/tasks/{taskId}/complete
POST /runtime/tasks/{taskId}/claim
POST /runtime/tasks/{taskId}/delegate

# 历史
GET  /history/historic/process-instances
GET  /history/historic/task-instances
```

### 自定义 REST 控制器

```java
@RestController
@RequestMapping("/api/workflow")
public class WorkflowController {

    @Autowired
    private RuntimeService runtimeService;

    @Autowired
    private TaskService taskService;

    @PostMapping("/start")
    public ResponseEntity<String> startProcess(
            @RequestParam String processKey,
            @RequestBody Map<String, Object> variables) {

        ProcessInstance instance = runtimeService.startProcessInstanceByKey(
            processKey, variables
        );

        return ResponseEntity.ok(instance.getId());
    }

    @PostMapping("/complete/{taskId}")
    public ResponseEntity<Void> completeTask(
            @PathVariable String taskId,
            @RequestBody Map<String, Object> variables) {

        taskService.complete(taskId, variables);
        return ResponseEntity.ok().build();
    }
}
```

### 流程测试

```java
@SpringBootTest
class LeaveProcessTest {

    @Autowired
    private ProcessEngine processEngine;

    private RuntimeService runtimeService;
    private TaskService taskService;

    @BeforeEach
    void setUp() {
        runtimeService = processEngine.getRuntimeService();
        taskService = processEngine.getTaskService();
    }

    @Test
    void testLeaveProcess() {
        // 启动流程
        Map<String, Object> variables = new HashMap<>();
        variables.put("employee", "张三");
        variables.put("days", 5);

        ProcessInstance instance = runtimeService
            .startProcessInstanceByKey("leave-process", variables);

        // 完成任务
        Task task = taskService.createTaskQuery()
            .processInstanceId(instance.getId())
            .singleResult();

        assertNotNull(task);
        assertEquals("提交请假", task.getName());

        taskService.complete(task.getId());

        // 验证流程继续
        Task nextTask = taskService.createTaskQuery()
            .processInstanceId(instance.getId())
            .singleResult();

        assertEquals("主管审批", nextTask.getName());
    }
}
```
