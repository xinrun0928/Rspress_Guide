# Camunda

Camunda 是另一个流行的开源工作流引擎，与 Flowable 相比，它更专注于企业级场景，特别是在 BPMN + DMN 组合方面有独特优势。

## Camunda 的特点

Camunda 的核心优势在于：

- **DMN 决策表**：内置强大的决策表支持，适合复杂业务规则
- **RESTful 友好**：内置完整的 REST API，易于集成
- **企业级稳定**：商业支持稳定，适合对稳定性要求高的企业
- **多语言支持**：除 Java 外，还支持 Go、JavaScript 等语言客户端
