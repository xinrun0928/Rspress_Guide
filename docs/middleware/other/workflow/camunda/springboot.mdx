# Camunda Spring Boot 集成

Camunda 对 Spring Boot 提供了良好的支持，通过 `camunda-bpm-spring-boot-starter` 可以快速集成。

## 快速开始

### 添加依赖

```xml title="pom.xml"
<dependency>
    <groupId>org.camunda.bpm</groupId>
    <artifactId>camunda-bpm-spring-boot-starter-rest</artifactId>
    <version>7.20.0</version>
</dependency>
<dependency>
    <groupId>org.camunda.bpm</groupId>
    <artifactId>camunda-bpm-spring-boot-starter-webapp</artifactId>
    <version>7.20.0</version>
</dependency>
```

### 应用入口

```java
@SpringBootApplication
@EnableProcessApplication
public class CamundaApplication {

    public static void main(String[] args) {
        SpringApplication.run(CamundaApplication.class, args);
    }
}
```

## 配置文件

### 基本配置

```yaml title="application.yml"
spring:
  datasource:
    url: jdbc:h2:mem:camunda;DB_CLOSE_DELAY=-1
    driver-class-name: org.h2.Driver

camunda:
  # 自动部署
  auto-deployment-enabled: true
  .deployment-resource-pattern: classpath:/processes/**/*.bpmn
  # 数据库
  database-schema-update: true
  # REST API
  api:
    enabled: true
  # Web 应用
  webapp:
    enabled: true
  # 异步执行器
  job-executor:
    enabled: true
    activate: true
    deployment-aware: true
```

### 完整配置

```yaml title="application.yml"
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/camunda
    username: root
    password: root

camunda:
  # 数据库配置
  database:
    type: mysql
    schema-update: true

  # 自动部署
  auto-deployment:
    enabled: true
    resources: classpath:/processes/**/*.bpmn, classpath:/dmn/**/*.dmn

  # REST API
  api:
    enabled: true
    context-path: /engine-rest

  # Web 应用
  webapp:
    enabled: true
    index-redirect-enabled: true

  # 作业执行器
  job-executor:
    enabled: true
    activate: true
    core-pool-size: 8
    max-pool-size: 16
    queue-size: 100
    lock-time-in-millis: 60000
    max-jobs-per-acquisition: 1

  # 历史
  history:
    level: full

  # BPMN 引擎配置
  bpmn:
    fail-on-undefined-properties: true
    strict-mode: true
```

    ├── order-process.bpmn

### 自定义部署配置

```yaml title="application.yml"
camunda:
  auto-deployment:
    enabled: true
    resources: >
      classpath:/processes/production/*.bpmn,
      classpath:/processes/testing/*.bpmn,
      classpath:/dmn/*.dmn
```

### 禁用自动部署

```yaml
camunda:
  auto-deployment:
    enabled: false
```

## 核心服务注入

```java
@Service
public class WorkflowService {

    @Autowired
    private RuntimeService runtimeService;

    @Autowired
    private TaskService taskService;

    @Autowired
    private RepositoryService repositoryService;

    @Autowired
    private HistoryService historyService;

    @Autowired
    private ProcessEngine processEngine;

    public String startProcess(String processKey, Map<String, Object> variables) {
        ProcessInstance instance = runtimeService
            .startProcessInstanceByKey(processKey, variables);
        return instance.getId();
    }

    public List<Task> getTasksByAssignee(String assignee) {
        return taskService.createTaskQuery()
            .taskAssignee(assignee)
            .list();
    }

    public void completeTask(String taskId, Map<String, Object> variables) {
        taskService.complete(taskId, variables);
    }
}
```

## 表单服务

### 动态表单

```java
@Controller
@RequestMapping("/task")
public class TaskController {

    @Autowired
    private FormService formService;

    @GetMapping("/{taskId}/form")
    public Map<String, Object> getTaskForm(@PathVariable String taskId) {
        TaskFormData formData = formService.getTaskFormData(taskId);

        List<FormField> fields = formData.getFormFields();
        Map<String, Object> result = new HashMap<>();

        for (FormField field : fields) {
            result.put(field.getId(), Map.of(
                "label", field.getLabel(),
                "type", field.getType(),
                "value", field.getValue()
            ));
        }

        return result;
    }

    @PostMapping("/{taskId}/submit")
    public void submitTaskForm(
            @PathVariable String taskId,
            @RequestBody Map<String, Object> formData) {

        formService.submitTaskFormData(taskId, formData);
    }
}
```

## 事务管理

### @Transactional 注解

```java
@Service
public class OrderService {

    @Autowired
    private RuntimeService runtimeService;

    @Transactional
    public void createOrder(Order order) {
        // 保存订单
        orderRepository.save(order);

        // 启动工作流
        runtimeService.startProcessInstanceByKey(
            "order-process",
            Variables.createVariables()
                .putValue("orderId", order.getId())
        );
    }
}
```

### 事务边界

Camunda 使用 Spring 的事务管理：

```java
@Service
public class WorkflowService {

    @Autowired
    private RuntimeService runtimeService;

    @Transactional(propagation = Propagation.REQUIRES_NEW)
    public void startProcess() {
        // 在新事务中启动流程
        runtimeService.startProcessInstanceByKey("my-process");
    }
}
```

## REST API

### 启用 REST API

Camunda 提供完整的 REST API：

```bash
# 流程定义
GET    http://localhost:8080/engine-rest/process-definition
GET    http://localhost:8080/engine-rest/process-definition/key/{key}
POST   http://localhost:8080/engine-rest/process-definition/key/{key}/start

# 流程实例
GET    http://localhost:8080/engine-rest/process-instance
POST   http://localhost:8080/engine-rest/process-instance/{id}/terminate

# 任务
GET    http://localhost:8080/engine-rest/task
POST   http://localhost:8080/engine-rest/task/{id}/complete
POST   http://localhost:8080/engine-rest/task/{id}/claim

# 历史
GET    http://localhost:8080/engine-rest/history/process-instance
GET    http://localhost:8080/engine-rest/history/variable-instance
```

### Spring Security 保护

```java
@Configuration
@EnableWebSecurity
public class SecurityConfig extends WebSecurityConfigurerAdapter {

    @Override
    protected void configure(HttpSecurity http) throws Exception {
        http
            .csrf().disable()
            .authorizeRequests()
            .antMatchers("/engine-rest/**").hasRole("CAMUNDA_USER")
            .antMatchers("/camunda/**").hasRole("CAMUNDA_ADMIN")
            .anyRequest().authenticated()
            .and()
            .httpBasic();
    }
}
```

### 流程测试

```java
@SpringBootTest
@AutoConfigureTestDatabase
class LeaveProcessTest {

    @Autowired
    private ProcessEngine processEngine;

    @Autowired
    private RuntimeService runtimeService;

    @Autowired
    private TaskService taskService;

    @Test
    void testLeaveProcess() {
        // 启动流程
        Map<String, Object> variables = new HashMap<>();
        variables.put("employee", "张三");
        variables.put("days", 5);

        ProcessInstance instance = runtimeService
            .startProcessInstanceByKey("leave-process", variables);

        // 完成任务
        Task task = taskService.createTaskQuery()
            .processInstanceId(instance.getId())
            .taskDefinitionKey("submitTask")
            .singleResult();

        assertNotNull(task);

        taskService.complete(task.getId());

        // 验证下一步
        Task nextTask = taskService.createTaskQuery()
            .processInstanceId(instance.getId())
            .taskDefinitionKey("managerApproval")
            .singleResult();

        assertNotNull(nextTask);
    }
}
```

Spring Boot 集成核心配置：

| `camunda.api.enabled` | REST API 开关 |
| `camunda.webapp.enabled` | Web 应用开关 |
| `camunda.job-executor.enabled` | 异步执行器开关 |
