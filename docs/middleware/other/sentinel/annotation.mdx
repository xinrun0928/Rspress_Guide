# Sentinel 注解支持与 Dubbo 集成

Sentinel 提供了 `@SentinelResource` 注解，可以轻松地为方法添加流量控制。同时，Sentinel 与 Dubbo 深度集成，无需额外配置即可实现服务级别的保护。

## @SentinelResource 注解

### 基本用法

```java
@SentinelResource(value = "createOrder",
    blockHandler = "createOrderBlockHandler",
    fallback = "createOrderFallback")
public Result createOrder(Order order) {
    return orderService.create(order);
}

// 限流/熔断处理
public Result createOrderBlockHandler(Order order, BlockException e) {
    return Result.fail("系统繁忙，请稍后重试");
}

// 业务异常降级
public Result createOrderFallback(Order order, Throwable t) {
    log.error("创建订单失败", t);
    return Result.fail("服务暂时不可用");
}
```

### 注解属性

| 属性 | 说明 | 示例 |
|---|---|---|
| value | 资源名 | `"createOrder"` |
| blockHandler | 限流/熔断处理方法 | `"blockHandlerMethod"` |
| fallback | 业务异常降级方法 | `"fallbackMethod"` |
| exceptionsToIgnore | 忽略的异常 | `{BusinessException.class}` |
| defaultFallback | 默认降级方法 | `"defaultFallback"` |

### 完整示例

```java
@Service
public class OrderServiceImpl implements OrderService {

    @Override
    @SentinelResource(value = "createOrder",
        blockHandler = "createOrderBlockHandler",
        fallback = "createOrderFallback",
        exceptionsToIgnore = {ValidationException.class})
    public Result<Order> createOrder(Order order) {
        // 业务逻辑
        return orderRepository.save(order);
    }

    // 限流/熔断处理
    public Result<Order> createOrderBlockHandler(
            Order order,
            BlockException e,
            BlockHandler handler) {

        if (e instanceof FlowException) {
            log.warn("订单创建被限流: orderId={}", order.getId());
            return Result.fail("请求过于频繁，请稍后重试");
        }
        if (e instanceof DegradeException) {
            log.warn("订单创建触发熔断: orderId={}", order.getId());
            return Result.fail("服务降级中，请稍后重试");
        }
        return Result.fail("系统繁忙");
    }

    // 降级处理
    public Result<Order> createOrderFallback(
            Order order,
            Throwable t) {

        log.error("订单创建失败: orderId={}", order.getId(), t);

        if (t instanceof BusinessException) {
            return Result.fail(t.getMessage());
        }
        return Result.fail("服务暂时不可用");
    }
}
```

## Dubbo 集成

### 自动集成

Sentinel 与 Dubbo 自动集成，只需引入依赖：

```xml title="pom.xml"
<dependency>
    <groupId>com.alibaba.csp</groupId>
    <artifactId>sentinel-apache-dubbo-adapter</artifactId>
    <version>1.8.8</version>
</dependency>
```

### 配置

```yaml title="application.yml"
dubbo:
  provider:
    filter: sentinel-dubbo-provider

  consumer:
    filter: sentinel-dubbo-consumer
```

### Dubbo 限流

Dubbo 集成后，会自动为每个 Dubbo 接口生成资源：

```
资源名格式：
provider:com.example.OrderService:createOrder
consumer:com.example.OrderService:createOrder
```

### 自定义 Dubbo 过滤器

```java
@Activate(group = {Constants.PROVIDER})
public class CustomSentinelFilter implements Filter {

    @Override
    public Result invoke(Invoker<?> invoker, Invocation invocation)
            throws RpcException {

        String resourceName = invoker.getInterface().getName()
            + ":" + invocation.getMethodName();

        Entry entry = null;
        try {
            entry = SphU.entry(resourceName);
            // 通过
            return invoker.invoke(invocation);
        } catch (BlockException e) {
            // 被限流
            return AsyncRpcResult.newDefaultAsyncResult(
                new RpcException("服务被限流"), invocation);
        } catch (Exception e) {
            Tracer.traceEntry(e, entry);
            throw e;
        } finally {
            if (entry != null) {
                entry.exit();
            }
        }
    }
}
```

## Feign 集成

### 引入依赖

```xml title="pom.xml"
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>
<dependency>
    <groupId>com.alibaba.csp</groupId>
    <artifactId>sentinel-spring-cloud-gateway-adapter</artifactId>
    <version>1.8.8</version>
</dependency>
```

### 配置

```yaml title="application.yml"
feign:
  sentinel:
    enabled: true
```

### 使用

```java
@FeignClient(name = "order-service", fallback = OrderFeignFallback.class)
public interface OrderFeignClient {

    @RequestMapping("/api/orders/{id}")
    Order getOrder(@PathVariable Long id);
}

@Component
public class OrderFeignFallback implements OrderFeignClient {

    @Override
    public Order getOrder(Long id) {
        return Order.builder()
            .id(id)
            .status("降级状态")
            .build();
    }
}
```

## Spring Cloud Gateway 集成

### 引入依赖

```xml title="pom.xml"
<dependency>
    <groupId>com.alibaba.csp</groupId>
    <artifactId>sentinel-spring-cloud-gateway-adapter</artifactId>
    <version>1.8.8</version>
</dependency>
```

### 配置

```java
@Configuration
public class SentinelGatewayConfig {

    @Bean
    @Order(-1)
    public SentinelGatewayFilter sentinelGatewayFilter() {
        return new SentinelGatewayFilter();
    }

    @PostConstruct
    public void init() {
        // 配置限流规则
        Set<GatewayFlowRule> rules = new HashSet<>();
        rules.add(new GatewayFlowRule("order-api")
            .setCount(100)
            .setIntervalSec(1));

        GatewayRuleManager.loadRules(rules);
    }
}
```

## Web 拦截

### 注解方式

```java
@RestController
@RequestMapping("/api/orders")
public class OrderController {

    @GetMapping("/{id}")
    @SentinelResource(value = "getOrder",
        blockHandler = "getOrderBlockHandler")
    public Result<Order> getOrder(@PathVariable Long id) {
        return orderService.getOrder(id);
    }

    public Result<Order> getOrderBlockHandler(Long id, BlockException e) {
        return Result.fail("请求过于频繁");
    }
}
```

### 全局拦截

```java
@Configuration
public class SentinelWebConfig implements WebFluxConfigurer {

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(new SentinelWebInterceptor())
            .addPathPatterns("/api/**");
    }
}
```

## 配置规则

### 注解中指定

```java
@SentinelResource(value = "createOrder",
    blockHandler = "createOrderBlockHandler",
    blockHandlerClass = SentinelBlockHandler.class)
public Result createOrder(Order order) {
    return orderService.create(order);
}
```

```java
public class SentinelBlockHandler {

    public static Result createOrderBlockHandler(
            Order order,
            BlockException e,
            BlockHandler handler) {
        return Result.fail("系统繁忙");
    }
}
```

## 总结

Sentinel 注解核心属性：

| 属性 | 说明 |
|---|---|
| `value` | 资源名 |
| `blockHandler` | 限流/熔断处理方法 |
| `fallback` | 业务异常降级方法 |
| `exceptionsToIgnore` | 忽略的异常类型 |
