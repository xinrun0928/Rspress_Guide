# Kibana：数据可视化与分析

数据已经存入 Elasticsearch 了。

但产品经理问你：「给我看一下这周的用户增长趋势，按操作系统分类。」运营问：「昨晚的订单峰值是多少？主要集中在哪个时间段？」

你总不能让他们直接看 ES 返回的 JSON 吧？

Kibana 就是来解决这个问题的——它是 ES 的可视化界面，让数据从冰冷的数字变成直观的图表。

## Kibana 是什么？

Kibana 是 Elastic Stack 中的 **可视化组件**，为 Elasticsearch 提供数据搜索、分析和可视化功能。它让你不需要写代码，也能对海量数据进行探索和分析。

## 核心功能

### Discover：探索数据

即时搜索和分析存储在 ES 中的数据：

- 编写 KQL 查询语句
- 筛选时间范围
- 查看文档详情
- 保存搜索以便复用

### Visualize：图表绘制

创建各类可视化图表：

| 图表类型 | 适用场景 | 示例 |
|---------|---------|------|
| 折线图 | 趋势分析 | 日活趋势、销量走势 |
| 柱状图 | 对比分析 | 各品类销售额对比 |
| 饼图 | 占比分析 | 用户设备分布 |
| 热力图 | 密度分析 | 访问热点区域 |
| 仪表盘 | 状态展示 | 当前在线人数 |

### Dashboard：可视化大盘

将多个可视化组合在一起，形成综合监控/分析页面：

- 添加过滤器（全局生效）
- 点击穿透（关联筛选）
- 自动刷新（实时监控）
- 分享嵌入（嵌入其他系统）

## 适用场景

| 场景 | 说明 | 典型看板 |
|-----|------|---------|
| 业务分析 | 销售、用户、订单分析 | 运营 Dashboard |
| 系统监控 | 性能、错误率、可用性 | 运维监控大屏 |
| 日志分析 | 错误日志排查 | 日志搜索视图 |
| 安全审计 | 访问日志、异常检测 | 安全态势感知 |
