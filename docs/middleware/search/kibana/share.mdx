# 仪表盘分享与嵌入

辛辛苦苦做好的 Dashboard，当然要让更多人看到。Kibana 提供了多种分享方式，从简单的链接分享到嵌入式可视化，满足各种场景需求。

## 分享方式总览

| 方式 | 适用场景 | 特点 |
|---|---|---|
| 分享链接 | 内部用户访问 | 需要 Kibana 账号 |
| 公开链接 | 外部用户/无账号 | 可设置有效期和密码 |
| 截图报告 | 邮件/文档 | 静态图片，定期生成 |
| 嵌入 iframe | 网站/大屏展示 | 实时更新 |

## 分享链接

### 基本分享

点击 Dashboard 右上角的「Share」按钮，选择「Permalinks」：

```
分享方式: Permalinks
链接选项:
  - 包含当前状态: 是（时间范围、过滤器）
  - 缩短 URL: 否
```

### 分享配置

```yaml
分享选项:
  固定时间: 否（用户可调整）
  固定过滤器: 是（过滤器不可更改）
  嵌入选项:
    - 显示标题: 是
    - 显示工具栏: 是
    - 面板最大化: 否
```

## 公开链接

如果想让没有 Kibana 账号的人也能访问，需要启用公开链接：

```yaml
# kibana.yml
server.publicBaseUrl: "https://kibana.example.com"
xpack.reporting.publicBaseUrl: "https://kibana.example.com"
```

### 创建公开链接

```
Share → Public URL → Enable public link

设置:
  过期时间: 24 小时
  访问密码: 可选
  允许下载: PDF/CSV
```

:::warning 安全考虑
公开链接相当于「知道链接就能看」。确保链接不会泄露给未授权人员，或者设置访问密码。
:::

### 公开链接限制

公开链接有以下限制：

- 无法使用 Discover
- 无法创建告警
- 无法编辑 Dashboard
- 报告生成需要额外配置

## 嵌入可视化

### iframe 嵌入

最常见的嵌入方式是使用 iframe：

```html
<iframe
  src="https://kibana.example.com/app/dashboards/abc123?embed=true&_g=(filters:!(),refreshInterval:(pause:!t,value:0),time:(from:now-24h,to:now))"
  height="600"
  width="100%"
></iframe>
```

### URL 参数说明

| 参数 | 说明 | 示例 |
|---|---|---|
| embed=true | 嵌入模式，隐藏导航栏 | embed |
| _g | 全局状态（时间、过滤器） | (time:(from:now-24h,to:now)) |
| _a | 应用状态（Dashboard 配置） | (filters:!()) |
| hideToolbar | 隐藏工具栏 | true/false |
| showTopNav | 显示顶部导航 | true/false |

### 固定时间范围

```html
<!-- 固定为最近 7 天，不可调整 -->
?embed=true&_g=(time:(from:now-7d,to:now,mode:absolute))
```

### 固定过滤器

```html
<!-- 预置过滤器 -->
?embed=true&_g=(filters:!((meta:(alias:!n,disabled:!f,index:'logs-*',negate:!f,params:(query:order-service)),query:(match_phrase:(service:order-service)))))
```

## 大屏展示模式

### 配置 Dashboard

针对大屏（电视/投影）优化：

```
Dashboard → Edit → Settings

大屏配置:
  - 背景色: 深色
  - 文字大小: 大
  - 刷新频率: 30 秒
  - 启用全屏: 是
```

### 隐藏 UI 元素

```html
<!-- 隐藏顶部导航和侧边栏 -->
?embed=true&hideSidebar=true&hideTopNav=true
```

### 自动轮播

配合 JavaScript 实现 Dashboard 轮播：

```javascript
const dashboards = [
  'dash-id-1',
  'dash-id-2', 
  'dash-id-3'
];

let currentIndex = 0;

function nextDashboard() {
  currentIndex = (currentIndex + 1) % dashboards.length;
  iframe.src = `${baseUrl}/app/dashboards/${dashboards[currentIndex]}?embed=true`;
}

// 每 30 秒切换
setInterval(nextDashboard, 30000);
```

## PDF 报告

### 手动生成

```
Dashboard → Share → PDF Reports

选项:
  - 页面布局: Portrait/Landscape
  - 标题: 自动/自定义
  - 页脚: 包含生成时间
```

### 自动化报告

通过 API 定时生成报告：

```bash
curl -X POST "https://kibana.example.com/api/reporting/generate/pdf" \
  -H 'kbn-xsrf: true' \
  -H 'Authorization: Basic xxx' \
  -d '{
    "dashboardId": "dashboard-uuid",
    "title": "Weekly Report",
    "timeFrom": "now-7d",
    "timeTo": "now",
    "layout": {
      "dimensions": {
        "width": 1920,
        "height": 1080
      }
    }
  }'
```

### 报告交付

支持多种报告交付方式：

```yaml
报告交付:
  - 邮件: 按计划发送 PDF 到指定邮箱
  - S3: 上传到 S3 存储
  - Webhook: 调用外部接口
```

## CSV 报告

### 导出数据

```
Dashboard → Share → CSV Reports

选项:
  - 最大行数: 10000
  - 分隔符: 逗号
  - 编码: UTF-8
```

### API 导出

```bash
curl -X POST "https://kibana.example.com/api/reporting/generate/csv" \
  -H 'kbn-xsrf: true' \
  -d '{
    "savedSearchId": "search-uuid",
    "columns": ["timestamp", "status", "service", "message"],
    "fetchFromAllNodes": true
  }'
```

## 常见问题

### Q: 分享的链接可以撤销吗？

可以。进入分享配置，点击「Revoke」按钮。已分发的链接将立即失效。

### Q: 嵌入 iframe 时如何处理登录？

推荐使用 Kibana 空间的无密码认证或 API Key。或者使用 Reporting API 生成静态报告替代实时嵌入。

### Q: 如何追踪谁在查看分享的 Dashboard？

启用审计日志，记录所有访问：

```yaml
审计日志:
  events:
    - dashboard_viewed
```

分享是一种能力，让数据产生价值。但分享也意味着责任——确保数据安全、不泄露敏感信息，是每个分享者的义务。
