# 谈薪与收尾

技术面一路过关斩将，HR 却打来电话：「我们综合评估后，暂时不考虑了。」

你懵了——技术明明没问题，怎么就挂在最后一步？

这不是个别现象。我见过太多候选人在技术上无可挑剔，却在谈薪环节功亏一篑。他们输在哪里？不是能力，是**博弈意识**。他们把谈薪当成了「等通知」，而实际上，这是整场面试中**唯一能主动出击争取利益的环节**。错过了这个窗口期，你可能一辈子少拿几十万。

## 谈薪的本质：信息战

很多人以为谈薪是「讨价还价」，其实不是。

谈薪是一场**信息战**。谁掌握更多信息、谁更不害怕失去这个 offer，谁就更有谈判优势。

几个关键原则：

- **永远不要主动报数字**，等对方先开口。如果 HR 坚持，给区间不给点值。
- **报总包，不报 base**。总包 = 基本工资 + 年终奖 + 股票 + 补贴 + 福利。
- **谈总包，不谈月薪**。月薪的谈判空间有限，总包才是真正的战场。
- **手上有其他 offer 是最好的谈判筹码**。不是威胁，是市场定价的参考。
- **不要只盯着钱**。签字费、股票兑现节奏、试用期工资、起始级别，都是可以谈的。

## 谈薪前必须知道的事

### 1. 先做调研

谈判的基础是**信息对称**。在开口之前，你需要知道：

- 这个岗位在行业里的薪资范围是多少？
- 这家公司大概是什么薪资水平（大厂 / 中厂 / 创业公司）？
- 你目前的薪资构成是什么（基本工资 / 奖金 / 股票 / 补贴）？

获取信息的渠道：脉脉、牛客、OfferShow 等平台上的薪资爆料，猎头的市场报告，朋友在同类公司工作的参考。

**目标**：拿到一个「有底气」的区间，而不是凭感觉报价。

### 2. 锚定策略

心理学上有个概念叫「锚定效应」——第一个数字会锚定后面的谈判空间。所以，**你的第一个报价，决定了谈判的起点**。

报一个比你实际期望**略高**的数字，留出对方的压价空间。

## 谈薪资的心理学技巧

### 技巧一：制造稀缺性

> 「我很希望能加入贵司，但同时也在和其他公司接触。我需要在 XX 号之前做决定。」

这样能给 HR 一个时间压力，加速他们的决策。

### 技巧二：展示诚意

> 「我对这个机会很感兴趣，薪资上我可以有一定弹性，但希望我们能找到一个双方都满意的方案。」

诚意的表达能让 HR 愿意帮你争取。

### 技巧三：不要暴露你的底线

永远不要先说出你的底线数字。先探对方的空间，再决定自己的让步空间。

### 技巧四：沉默是金

当你报价后，HR 说「这个数字有点高」，不要急着解释或让步。**沉默**，让对方先开口。

## 面试结束后的跟进节奏

拿到 offer 之后还没有结束。以下几个时间节点需要注意：

- **面试结束后 1-2 天**：发送感谢邮件，简要回顾亮点，加深印象
- **等待超过 1 周没消息**：主动跟进，询问面试结果
- **拿到 offer 后**：认真评估，必要时进入谈薪环节
- **背调前**：提前告知证明人，做好准备工作

```
谈薪成功 = 市场调研 + 锚定策略 + 让对方先开口 + 保持谈判空间

```

拿到 offer 不是终点，谈出最优条件才是。
