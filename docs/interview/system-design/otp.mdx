# 一次性密码（OTP）系统设计

你有没有想过：

- 登录时收到的 6 位数字验证码是怎么生成的？
- 为什么验证码有时效性？
- 为什么有时收不到验证码？

这些问题的背后，都是**一次性密码（OTP）系统**在默默工作。

## 一、OTP 的基本概念

### 1.1 什么是 OTP？

:::tip 一次性密码（OTP）
**定义**

一种只能使用一次的密码，每个密码只能用于一次认证

**特点**
1. 一次性：使用后立即失效
2. 时效性：只在限定时间内有效
3. 随机性：每次生成不同的密码
4. 动态性：根据时间或计数器动态生成
:::

### 1.2 OTP 类型

:::tip OTP 类型对比
**1. HOTP（HMAC-based OTP）**
- 基于计数器
- 每次认证后计数器 +1
- 适合：硬件令牌

**2. TOTP（Time-based OTP）**
- 基于时间
- 每 30 秒生成一个新密码
- 适合：手机 App、Google Authenticator

**3. SMS OTP（短信验证码）**
- 服务端生成
- 通过短信发送
- 适合：用户注册、登录
:::

## 二、TOTP 算法详解

### 2.1 算法原理

:::tip TOTP 算法原理
**算法公式**

`TOTP = HOTP(Secret, floor(当前时间戳 / 30秒))`

**计算步骤**

1. 将时间戳除以 30，取整
2. 将结果转为字节数组
3. 使用 HMAC-SHA1 签名（密钥为 Secret）
4. 动态截取（Dynamic Truncation）获取 6 位数字

**Secret**
- 用户注册时生成
- Base32 编码
- 长度：16-32 字符
:::

### 2.2 实现

```java
/**
 * TOTP 算法实现
 */
public class TOTPAlgorithm {

    /**
     * 生成 TOTP
     */
    public String generateTOTP(String secret, long timestamp, int digits, int period) {
        // 1. 计算时间步
        long timeStep = timestamp / period;

        // 2. 时间步转为字节数组（大端序）
        byte[] timeBytes = new byte[8];
        for (int i = 7; i >= 0; i--) {
            timeBytes[i] = (byte) (timeStep & 0xff);
            timeStep >>= 8;
        }

        // 3. Base32 解码 Secret
        byte[] secretBytes = Base32.decode(secret);

        // 4. HMAC-SHA1 计算
        byte[] hash = hmacSha1(secretBytes, timeBytes);

        // 5. 动态截断
        int offset = hash[hash.length - 1] & 0x0f;
        int binary = ((hash[offset] & 0x7f) << 24)
                   | ((hash[offset + 1] & 0xff) << 16)
                   | ((hash[offset + 2] & 0xff) << 8)
                   | (hash[offset + 3] & 0xff);

        // 6. 取指定位数
        int totp = binary % (int) Math.pow(10, digits);

        // 7. 左补零
        return String.format("%0" + digits + "d", totp);
    }

    /**
     * 验证 TOTP
     */
    public boolean verifyTOTP(String secret, String token,
                             int tolerance, int digits, int period) {
        long currentTime = System.currentTimeMillis() / 1000;

        // 允许前后 tolerance 个时间步
        for (int i = -tolerance; i <= tolerance; i++) {
            long timestamp = currentTime + (i * period);
            String expected = generateTOTP(secret, timestamp * 1000, digits, period);

            if (expected.equals(token)) {
                return true;
            }
        }

        return false;
    }

    /**
     * HMAC-SHA1
     */
    private byte[] hmacSha1(byte[] key, byte[] data) {
        try {
            Mac mac = Mac.getInstance("HmacSHA1");
            SecretKeySpec keySpec = new SecretKeySpec(key, "HmacSHA1");
            mac.init(keySpec);
            return mac.doFinal(data);
        } catch (NoSuchAlgorithmException | InvalidKeyException e) {
            throw new RuntimeException(e);
        }
    }
}
```

```java
/**
 * Secret 生成与二维码
 */
public class TOTPSecretGenerator {

    /**
     * 生成随机 Secret
     */
    public String generateSecret() {
        // 1. 生成 20 字节随机数
        SecureRandom random = new SecureRandom();
        byte[] bytes = new byte[20];
        random.nextBytes(bytes);

        // 2. Base32 编码
        return Base32.encode(bytes);
    }

    /**
     * 生成 otpauth:// URI
     *
     * 格式：otpauth://totp/{issuer}:{account}?secret={secret}&issuer={issuer}&algorithm={algo}&digits={digits}&period={period}
     */
    public String generateOTPAuthURI(String secret, String account, String issuer) {
        try {
            String accountEncoded = URLEncoder.encode(account, "UTF-8");
            String issuerEncoded = URLEncoder.encode(issuer, "UTF-8");

            return String.format(
                "otpauth://totp/%s:%s?secret=%s&issuer=%s&algorithm=SHA1&digits=6&period=30",
                issuerEncoded, accountEncoded, secret, issuerEncoded
            );
        } catch (UnsupportedEncodingException e) {
            throw new RuntimeException(e);
        }
    }

    /**
     * 生成二维码
     */
    public byte[] generateQRCode(String otpAuthURI, int width, int height) {
        QRCodeWriter qrCodeWriter = new QRCodeWriter();

        try {
            BitMatrix bitMatrix = qrCodeWriter.encode(otpAuthURI,
                BarcodeFormat.QR_CODE, width, height);

            // 转为 PNG
            ByteArrayOutputStream output = new ByteArrayOutputStream();
            MatrixToImageWriter.writeToStream(bitMatrix, "PNG", output);
            return output.toByteArray();

        } catch (WriterException | IOException e) {
            throw new RuntimeException(e);
        }
    }
}
```

## 三、短信验证码系统

### 3.1 架构设计

:::tip 短信验证码系统架构
**用户端**

**API 网关**

**验证码服务**

**数据层**

**最终到达用户手机**

### 3.2 核心实现

```java
/**
 * 短信验证码服务
 */
public class SMSOTPService {

    private RedisTemplate<String, String> redis;
    private SMSGateway smsGateway;
    private SMSLogRepository logRepository;
    private RateLimitService rateLimitService;

    /**
     * 发送验证码
     */
    public Result sendCode(String phone) {
        // 1. 限流检查
        if (!rateLimitService.allowSend(phone)) {
            return Result.fail("发送太频繁，请稍后再试");
        }

        // 2. 生成验证码
        String code = generateCode(6);

        // 3. 存储到 Redis（有效期 5 分钟）
        String key = "sms:code:" + phone;
        redis.opsForValue().set(key, code, Duration.ofMinutes(5));

        // 4. 记录发送日志
        logSend(phone, code);

        // 5. 发送短信
        try {
            smsGateway.send(phone, "您的验证码是 " + code + "，5 分钟内有效");
            return Result.success();
        } catch (Exception e) {
            return Result.fail("发送失败，请稍后重试");
        }
    }

    /**
     * 验证验证码
     */
    public Result verifyCode(String phone, String code) {
        String key = "sms:code:" + phone;
        String storedCode = redis.opsForValue().get(key);

        if (storedCode == null) {
            return Result.fail("验证码已过期");
        }

        if (!storedCode.equals(code)) {
            // 记录错误尝试
            recordFailedAttempt(phone);
            return Result.fail("验证码错误");
        }

        // 验证成功，删除验证码
        redis.delete(key);

        // 解锁用户（如有锁定）
        unlockUser(phone);

        return Result.success();
    }

    /**
     * 生成随机验证码
     */
    private String generateCode(int length) {
        Random random = new Random();
        StringBuilder code = new StringBuilder();

        for (int i = 0; i < length; i++) {
            code.append(random.nextInt(10));
        }

        return code.toString();
    }
}
```

### 3.3 安全措施

```java
/**
 * 安全防护
 */
public class SecurityProtection {

    private RedisTemplate<String, String> redis;

    /**
     * 限流检查
     */
    public boolean allowSend(String phone) {
        String key = "sms:limit:send:" + phone;

        // 每小时最多发送 5 条
        Long count = redis.opsForValue().increment(key);
        if (count == 1) {
            redis.expire(key, Duration.ofHours(1));
        }

        return count <= 5;
    }

    /**
     * 每日发送限制
     */
    public boolean allowDailySend(String phone) {
        String key = "sms:limit:daily:" + phone;

        // 每天最多发送 10 条
        Long count = redis.opsForValue().increment(key);
        if (count == 1) {
            redis.expire(key, Duration.ofDays(1));
        }

        return count <= 10;
    }

    /**
     * 错误次数限制
     */
    public void recordFailedAttempt(String phone) {
        String key = "sms:verify:fail:" + phone;

        Long count = redis.opsForValue().increment(key);
        redis.expire(key, Duration.ofMinutes(30));

        // 5 次错误后锁定
        if (count >= 5) {
            lockPhone(phone, Duration.ofHours(1));
        }
    }

    /**
     * 锁定手机号
     */
    public void lockPhone(String phone, Duration duration) {
        String key = "sms:lock:" + phone;
        redis.opsForValue().set(key, "locked", duration);
    }

    /**
     * 检查是否锁定
     */
    public boolean isLocked(String phone) {
        String key = "sms:lock:" + phone;
        return Boolean.TRUE.equals(redis.hasKey(key));
    }

    /**
     * 解锁手机号
     */
    public void unlockPhone(String phone) {
        String key = "sms:lock:" + phone;
        redis.delete(key);
    }
}
```

```java
/**
 * 短信发送日志
 */
public class SMSLogService {

    private SMSLogRepository logRepository;

    /**
     * 记录发送日志
     */
    public void logSend(String phone, String code) {
        // 注意：不要记录明文验证码！
        SMSLog log = new SMSLog();
        log.setPhone(maskPhone(phone)); // 手机号脱敏
        log.setCodeHash(hashCode(code)); // 只存哈希
        log.setType(SMSType.VERIFICATION_CODE);
        log.setStatus(SMSStatus.SENT);
        log.setSentAt(LocalDateTime.now());

        logRepository.save(log);
    }

    /**
     * 查询发送记录
     */
    public List<SMSLog> querySendHistory(String phone, LocalDateTime start,
                                         LocalDateTime end) {
        return logRepository.findByPhoneAndTimeRange(
            maskPhone(phone), start, end);
    }

    /**
     * 手机号脱敏
     */
    private String maskPhone(String phone) {
        // 138****5678
        if (phone.length() >= 11) {
            return phone.substring(0, 3) + "****" + phone.substring(7);
        }
        return phone;
    }

    /**
     * 验证码哈希
     */
    private String hashCode(String code) {
        // SHA-256 哈希
        return DigestUtils.sha256Hex(code);
    }
}
```

## 四、多设备支持

### 4.1 TOTP 多设备同步

```java
/**
 * 多设备支持
 */
public class MultiDeviceSupport {

    /**
     * 设备管理
     */
    public class DeviceManager {

        /**
         * 绑定设备
         */
        public void bindDevice(long userId, DeviceInfo device) {
            // 1. 生成设备密钥
            String deviceSecret = generateDeviceSecret();

            // 2. 存储设备信息
            DeviceBind bind = new DeviceBind();
            bind.setUserId(userId);
            bind.setDeviceId(device.getDeviceId());
            bind.setDeviceSecret(encryptSecret(deviceSecret));
            bind.setBoundAt(LocalDateTime.now());

            deviceRepository.save(bind);
        }

        /**
         * 获取用户的所有设备
         */
        public List<DeviceInfo> getUserDevices(long userId) {
            List<DeviceBind> binds = deviceRepository.findByUserId(userId);

            return binds.stream()
                .map(bind -> new DeviceInfo(
                    bind.getDeviceId(),
                    decryptSecret(bind.getDeviceSecret()),
                    bind.getBoundAt()))
                .collect(Collectors.toList());
        }

        /**
         * 解绑设备
         */
        public void unbindDevice(long userId, String deviceId) {
            deviceRepository.deleteByUserIdAndDeviceId(userId, deviceId);
        }
    }

    /**
     * 生成设备密钥
     */
    private String generateDeviceSecret() {
        byte[] bytes = new byte[20];
        new SecureRandom().nextBytes(bytes);
        return Base32.encode(bytes);
    }
}
```

### 4.2 恢复码

```java
/**
 * 恢复码
 */
public class RecoveryCodeService {

    private static final int RECOVERY_CODE_COUNT = 10;

    /**
     * 生成恢复码
     */
    public List<String> generateRecoveryCodes(long userId) {
        List<String> codes = new ArrayList<>();

        for (int i = 0; i < RECOVERY_CODE_COUNT; i++) {
            // 生成 8 位恢复码
            String code = generateCode(8);
            codes.add(code);

            // 存储哈希
            RecoveryCode record = new RecoveryCode();
            record.setUserId(userId);
            record.setCodeHash(hashCode(code));
            record.setUsed(false);
            record.setCreatedAt(LocalDateTime.now());

            recoveryCodeRepository.save(record);
        }

        // 返回明文（只显示一次）
        return codes;
    }

    /**
     * 使用恢复码
     */
    public boolean useRecoveryCode(long userId, String code) {
        RecoveryCode record = recoveryCodeRepository
            .findByUserIdAndCodeHashAndUsed(userId, hashCode(code), false)
            .orElseThrow(() -> new InvalidRecoveryCodeException());

        // 标记已使用
        record.setUsed(true);
        record.setUsedAt(LocalDateTime.now());
        recoveryCodeRepository.save(record);

        return true;
    }

    /**
     * 重新生成恢复码（使用后）
     */
    public void regenerateRecoveryCodes(long userId) {
        // 删除旧恢复码
        recoveryCodeRepository.deleteByUserId(userId);

        // 生成新的
        return generateRecoveryCodes(userId);
    }
}
```

## 五、面试追问方向

### 问题一：「TOTP 和 HOTP 哪个更安全？」

**TOTP（时间型）**

- 优势：不需要计数器同步
- 劣势：依赖时间准确性

**HOTP（计数型）**

- 优势：不依赖时间
- 劣势：需要同步计数器

**安全性对比**

- TOTP 通常更安全，因为有时间窗口限制
- HOTP 如果计数器不同步可能导致多个有效密码

### 问题二：「如何防止验证码被薅羊毛？」

防护措施：
1. 限流：每小时/每天发送次数限制
2. 图形验证码：发送前先验证人机
3. IP 限制：同一 IP 限制发送数量
4. 行为分析：异常行为检测
5. 打码平台识别：检测模拟器、代理等

### 问题三：「为什么有些验证码收不到？」

**原因分析**

**手机问题**
- 信号不好
- 短信收件箱满
- 拦截软件

**运营商问题**
- 发送频率限制
- 垃圾短信过滤

**平台问题**
- 通道质量
- 发送时间

**解决方案**
1. 多个短信通道备份
2. 增加语音验证
3. 增加邮箱验证

## 六、总结

:::tip OTP 系统设计要点
**OTP 类型**
- HOTP：基于计数器
- TOTP：基于时间
- SMS OTP：短信验证码

**TOTP 算法**
- HMAC-SHA1 签名
- 动态截断
- 30 秒时间窗口

**安全措施**
- 限流控制
- 错误次数限制
- 锁定机制
- 日志审计

**多设备支持**
- 设备绑定
- 恢复码
- 设备管理
:::

OTP 系统的设计，本质上是在「用户体验」与「安全性」之间找平衡。太复杂的安全措施会流失用户，太简单的验证又容易被攻击。**你有没有想过，如果你是攻击者，你会用什么方式绕过 OTP 系统？** 理解攻击者的思路，才能设计出更安全的系统。
