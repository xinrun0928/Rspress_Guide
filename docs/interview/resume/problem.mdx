# 简历常见问题与修改建议

:::note
简历写完之后，最好找 3 个人帮你 review：你身边的同学、比你资深的人、有 HR 或猎头背景的人。
:::

## 十大高频问题

### 问题一：简历太长或太短

**症状：** 应届生写三页，或者工作三年只写半页。

**影响：** 没人有耐心看完，或者让人觉得没什么可写的。

**建议：**

- 应届生 / 0-3 年：1 页
- 3-5 年：1.5-2 页
- 5 年以上：2 页，核心放在最近两段经历上

### 问题二：技术栈堆砌，没有重点

**症状：** 技术栈写了几十项，看不出主次。

**影响：** 面试官无法快速判断你的核心竞争力。

**建议：** 技术栈分类 + 按掌握程度分层，保留 10-15 项核心技能，删除不熟悉的。

### 问题三：项目描述太笼统

**症状：** 「负责系统开发」「参与需求评审」这类描述太空。

**影响：** 面试官没有提问的切入点。

**建议：** 用 STAR 法则改写，加上具体数据和技术细节。

### 问题四：项目经历没有亮点

**症状：** 写的都是 CRUD，没有技术含量。

**影响：** 面试官会认为你的能力止步于 CRUD。

**建议：** 深挖每个项目的技术难点，可以从以下角度：

- 性能优化：接口 RT 降低了多少
- 高并发：QPS 支撑多少
- 数据量：存储了多少数据
- 架构设计：用了什么设计模式/架构模式
- 问题解决：遇到过什么线上问题，怎么排查和解决的

### 问题五：数据夸大或造假

**症状：** 项目 QPS 虚高（「日均十亿级请求」），项目规模明显不匹配。

**影响：** 面试官追问立刻露馅，影响诚信。

**建议：** 数据可以保守估计，但要真实。能讲清楚来龙去脉的数据才是好数据。

### 问题六：个人信息遗漏关键项

**症状：** 没写手机号、没写邮箱、文件名是「简历.pdf」。

**影响：** HR 联系不到你。

**建议：** 检查清单：

- [ ] 手机号（最好加分隔符如 138-xxxx-xxxx）
- [ ] 邮箱（用专业邮箱，别用 QQ 邮箱昵称）
- [ ] 简历文件名：`姓名-岗位-工作年限.pdf`

### 问题七：自我评价太虚

**症状：** 「热爱技术」「学习能力强」「善于沟通」—— 全是套话。

**影响：** 等于没写，反而降低印象分。

**建议：** 用事实说话，或者直接删掉自我评价。

### 问题八：简历格式混乱

**症状：** 字号不统一、日期不一致、缩进乱、标点乱。

**影响：** 暴露态度问题。

**建议：**

- 统一使用一种字体（推荐等线或 Arial）
- 统一日期格式（如 2023.06 - 2024.06）
- 用 Markdown 写，导出 PDF 格式最稳定
- 字号 10.5pt - 12pt，行距 1.2-1.5

### 问题九：项目经历全是「我」

**症状：** 每个项目都写「我设计」「我实现」「我优化」。

**影响：** 如果是团队项目，显得不真实，而且面试官会追问。

**建议：** 明确区分个人贡献和团队协作。可以这样写：

> 主导订单模块设计与实现（独立），日均处理 30 万订单...
> 参与用户中心模块开发（2 人协作），负责缓存层设计与实现...

### 问题十：简历不匹配岗位 JD

**症状：** 用同一份简历投所有公司，不看 JD 要求。

**影响：** 通过率极低。

**建议：** 每投一家公司，根据 JD 调整简历的侧重点：

- JD 强调高并发 → 突出高并发项目
- JD 强调中间件 → 突出中间件使用经验
- JD 强调团队管理 → 突出带团队经验

## 修改清单

完成初稿后，对照以下清单逐项检查：

```markdown title="resume-checklist.md"
格式检查
□ 文件名为「姓名-岗位-年限.pdf」
□ 手机号和邮箱格式正确
□ 日期格式统一
□ 字号、字距一致
□ 没有错别字和乱码

内容检查
□ 技术栈数量控制在 10-15 项
□ 每个项目有量化数据
□ 每个项目有 1-2 个技术亮点
□ STAR 法则描述项目经历
□ 没有使用「精通」除非真的精通

针对性检查
□ 根据目标公司 JD 调整了侧重点
□ 删除了与岗位无关的内容
□ GitHub / 博客链接可正常访问（如有）
□ 没有敏感信息（身份证、详细住址等）
```

## 总结

简历修改是一个不断迭代的过程：

1. **写初稿**：先按自己的想法写出来
2. **找反馈**：让 2-3 个人帮你 review
3. **改问题**：根据反馈修改
4. **再打磨**：针对不同公司再做微调

记住：**简历不是写你做过什么，而是写你做成了什么**。
