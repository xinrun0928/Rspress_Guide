# HR 面与谈薪

技术面都过了，HR 却告诉你：「我们综合评估后，暂时不考虑了。」

技术过硬却在 HR 环节被筛掉，是一件很亏的事。

很多人觉得 HR 面就是「聊聊天」，其实不然。HR 面的本质是**匹配度验证**和**风险排查**——公司要确认你是合适的人，不是来「骑驴找马」的，不是入职后很快会离职的，不是有隐性问题的。

更重要的是，谈薪是整个面试流程中**你唯一能主动出击的环节**。前面都是被动回答，薪资谈判是你为数不多可以主动争取的时刻。

## HR 面的三个核心问题

**问题一：为什么离职？（必问）**

这是 HR 最在意的问题之一。答案透露的是你的稳定性和价值观。

❌ 雷区说法：
> 「前公司领导太垃圾，加班太多，钱还少。」

✅ 安全说法：
> 「希望寻找一个更大的平台，在技术深度和业务广度上都有更多发展空间。和前领导、同事都保持良好关系。」

---

**问题二：职业规划是什么？（必问）**

HR 想知道的是：你有没有想清楚自己要什么？你会不会把这份工作当跳板？

✅ 推荐说法：
> 「未来 3 年，我希望能在一个技术方向上深耕（比如高并发/架构/某个业务领域），成为一个领域的专家。具体路径是通过解决实际业务问题积累经验。」

---

**问题三：你还有什么问题想问我的？（必问）**

这是一个反向筛选公司和你争取信息的机会。

✅ 值得问的问题：
- 「这个岗位的团队规模和协作模式是怎样的？」
- 「试用期转正的标准是什么？」
- 「公司未来半年的技术方向或项目重点是什么？」
- 「团队的成长机制是什么样的？」


## 谈薪策略

**谈薪资不是讨价还价，是一场信息战。**

谁先开口、谁掌握更多信息、谁更不害怕失去这个 offer，谁就更有谈判优势。

几个关键原则：

```
1. 永远不要主动报数字，等对方先开口
2. 报 range，不报固定值，给自己留余地
3. 谈总包，不谈 base，total compensation 更重要
4. 不要只盯着钱，签字费、股票兑现节奏、晋升空间都是谈判筹码
5. 手上有其他 offer 是最好的谈判武器
```


## 本模块核心公式

```
HR 面 = 真诚表达 + 规避雷区 + 信息收集 + 主动谈判
```

拿到 offer 不是终点，谈出最优条件才是。
