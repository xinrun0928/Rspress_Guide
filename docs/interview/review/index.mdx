# 面试复盘

面试一次，少一次。但每一次失败，都应该是下一次成功的台阶。

我见过太多人面试挂了，回去沮丧两天，然后继续投下一家、继续挂。这不是「努力」，这是「重复错误」。**真正的进步，来自每一次面试后的复盘。**

## 复盘为什么重要

面试失败的常见原因：

- 算法题刷了但没有按类型总结，面试遇到变形题就懵
- 项目经验很丰富但讲不清楚，面试官听不出你的价值
- 技术原理懂但说不透，追问就卡壳
- HR 面完全没准备，薪资谈判直接报底价

这些问题，都不是靠「临场发挥」能解决的。**你需要在面试后把每个失败的环节都复盘清楚，找到真正的问题所在。**

---

## 复盘的正确姿势

### 1. 趁热打铁

面试结束后 24 小时内复盘，越早越好。记忆会衰减，细节会模糊。

### 2. 客观评价

不要因为「整体感觉还行」就忽略细节问题，也不要因为「挂了」就否定自己。

### 3. 重点突破

不需要每道题都深度复盘，把精力放在「答不好的题」和「高频考点」上。

### 4. 持续迭代

复盘本身也要迭代。发现自己复盘时总是漏掉某个维度，就加上。

## 面试成功 = 硬实力 + 软实力 + 好心态 + 好运气

```
面试成功 = 硬实力（会做）+ 软实力（讲清）+ 心态稳（不慌）+ 好运气（题目对胃口）
```

这四个因素缺一不可：

- **硬实力**：算法能写对，原理能讲清，项目能说明白——这是基础
- **软实力**：能把复杂问题讲清楚，遇到追问能从容应对——这是区分普通候选人和优秀候选人的分水岭
- **心态稳**：紧张是正常的，但不能让紧张影响发挥——心态稳的人，即便遇到不会的题也能展示思考过程
- **好运气**：题目正好是你擅长的方向、面试官状态好、业务正好缺人——这部分不能控制，但可以通过增加面试次数来提高概率

**失败了，难过一下，然后继续。** 你走的每一步，都不是白走的。
