# 面试着装与礼仪注意事项

> 第一印象在面试中至关重要。研究表明，面试官在前 7 秒内就会形成对你的判断。

着装和礼仪看似是「表面功夫」，但它传递的信号是：你是否重视这个机会、你是否职业化、你是否能融入团队。这些软信号，有时候比硬实力更容易影响结果。

## 一、线下面试着装

### 技术岗：Smart Casual 是安全牌

很多人以为面试要穿正装，但对于技术岗来说，穿全套西装打领带反而显得格格不入。不过「随意」不等于「随便」。

**推荐：Smart Casual（精致休闲）**

| 穿着 | 推荐 | 避免 |
|---|---|---|
| 上装 | 纯色衬衫（白 / 浅蓝 / 浅灰）、休闲西装外套 | 花哨格子衬衫、大 Logo T 恤、背心 |
| 下装 | 深色休闲裤（黑 / 深蓝 / 卡其）、修身牛仔裤 | 短裤、运动裤、破洞牛仔裤 |
| 鞋子 | 皮鞋、干净的休闲鞋 | 拖鞋、凉鞋、运动鞋（除非公司文化允许） |
| 配饰 | 简约手表、小背包 | 夸张首饰、电子烟 |

**基本原则**：穿得比平时上班好一点，但不要比面试官正式。

### 不同公司的风格判断

- **大厂（阿里、腾讯、字节等）**：通常偏休闲，穿 Smart Casual 即可
- **金融 / 国企 / 外企**：可以适当正式一些，Business Casual 更安全
- **创业公司**：通常非常随意，甚至穿 T 恤短裤都可以——但保险起见还是稍微正式一点
- **日韩企业**：通常偏正式，建议 Business Casual

如果你不确定，**按比公司平均水平略正式一档来穿**，永远不会错。

## 二、线上面试礼仪

线上面试的「礼仪」更多体现在技术细节和细节行为上。

### 技术准备

- [ ] 提前测试网络，确保流畅
- [ ] 准备好稳定的设备和清晰的麦克风
- [ ] 背景干净整洁（书架或纯色墙面最佳）
- [ ] 提前测试摄像头角度（视线要平视，不要仰拍或俯拍）
- [ ] 备好面试软件的备用方案（Zoom / 腾讯会议 / 飞书等都备着）

:::warning 技术事故比答错题更致命
面试中网络断了、软件崩溃了，这种事故会让整个面试节奏乱掉。提前测试、备好方案，是最基本的职业化体现。
:::

### 线上面试的行为细节

| 细节 | 正确做法 | 错误做法 |
|---|---|---|
| 眼神 | 看摄像头（而不是屏幕） | 盯着屏幕或低头看简历 |
| 坐姿 | 坐直，略前倾 | 瘫在椅子上 |
| 面部 | 适度微笑，表情自然 | 表情僵硬或过于夸张 |
| 手势 | 可以适当用手势辅助表达 | 手不知道往哪放 |
| 环境 | 安静、独立、无干扰 | 有宠物叫声、门铃声、人声嘈杂 |
| 设备 | 提前关掉不必要的通知 | 突然弹出微信消息 |

### 常见问题

**Q：线上面试要不要穿正装下装？**

A：技术岗通常不需要穿全套正装，但建议穿整洁的上装。如果下身穿睡裤，万一站起来拿东西就尴尬了。

**Q：候选人那边看不到我的桌面吧？**

A：共享屏幕前检查一遍，不要留有暴露隐私的文件。不需要清空桌面，但不要有太尴尬的内容。

## 三、面试过程中的礼仪

### 进入与离开

- 提前 5-10 分钟到达（不要过早也不要迟到）
- 进入前敲门或等对方说「请进」
- 如果是线下面试，进门后和面试官握手（如果有）
- 结束时表示感谢，给出明确的结束信号（「感谢您的时间，期待您的消息」）

### 候场期间

- 关掉手机或调成静音
- 不要东张西望或表现得焦躁不安
- 可以翻看一下准备的笔记，但不要大声朗读

### 面谈期间

| 细节 | 正确做法 | 错误做法 |
|---|---|---|
| 倾听 | 认真听完面试官的问题再回答 | 对方还没说完就抢答 |
| 思考 | 可以短暂停顿组织语言 | 沉默太久不知道怎么回答 |
| 打断 | 不要打断面试官 | 插话、抢话 |
| 反馈 | 适度点头、眼神回应 | 没有任何反馈，面试官不知道你在不在听 |
| 手机 | 面试全程手机静音并收起来 | 手机突然响了 |

### 回答问题的礼仪

- 不知道就说不知道，不要硬撑
- 不会的时候，展示思考过程比硬编答案更好
- 如果面试官纠正你的错误，大方承认并感谢
- 遇到追问不要慌，这是好事——说明他在认真考察你

## 四、细节中的加分项

### 一份简历打印件

线下面试时，准备一份简历打印件（纸质版）递给面试官。虽然面试官通常已有电子版，但这份「小动作」能体现你的细心和专业度。

### 一支笔 + 本子

面试过程中可以用笔在本子上记录关键信息，展示你在认真倾听。记录不是做样子，而是真的在记——面试尾声可能会有「你有什么想问我的」环节，你记下的内容可以帮助你问出好问题。

### 准备一杯水

线下面试时手边放一杯水。如果回答问题说到一半口渴了，喝口水不会扣分，反而能给自己一个喘息和组织语言的时间。

### 准时但不过早

提前 5-10 分钟到是最理想的。过早到会显得你太焦虑，过晚到是灾难。

## 五、常见礼仪失误盘点

| 失误 | 影响 | 应对 |
|---|---|---|
| 迟到 | 直接扣分，严重时取消面试 | 提前踩点、规划路线，线上面试提前测试设备 |
| 手机响了 | 分心 + 不尊重 | 静音 + 收起来 |
| 眼神飘忽 | 显得不自信或心不在焉 | 眼神聚焦在面试官面部（线下面试）或摄像头（线上面试） |
| 过度紧张导致语速过快 | 面试官听不清你的回答 | 有意识地放慢语速，深呼吸 |
| 攻击前公司 / 前领导 | 暴露负能量 | 只说正向的离职原因 |
| 没有反向提问 | 显得不上心 | 提前准备 2-3 个有深度的问题 |

---

## 最后

礼仪不是表演，而是**职业化的自然体现**。真正有素养的候选人，不需要刻意表演——他们的行为举止本身就是对这次机会的尊重。

把礼仪当成一种习惯，而不是面试前的临时准备。这样不管面对什么场合，你都能从容应对。
