# 面试前准备

面试不是临时抱佛脚能应付的，它的准备周期通常需要 4-8 周。

很多人在面试前一周才开始着急——刷两道算法、背几个八股文，然后就上场了。结果可想而知：算法题卡壳、项目讲不清楚、HR 问题答得漏洞百出。

**面试是一场全链路的考察，硬实力、软实力、临场心态缺一不可。** 你需要在每个环节都做好准备，才能在竞争中脱颖而出。

## 为什么要提前准备

面试失败的常见原因：

- 算法题刷了但没有按类型总结，面试遇到变形题就懵
- 项目经验很丰富但讲不清楚，面试官听不出你的价值
- 技术原理懂但说不透，追问就卡壳
- HR 面完全没准备，薪资谈判直接报底价

这些问题，都不是靠「临场发挥」能解决的。**你需要在面试前把每个环节都准备好，形成肌肉记忆。**

## 准备周期的推荐节奏

如果你有 **4 周**以上的时间，建议分阶段准备：

| 周数 | 重点 | 时间占比 |
|---|---|---|
| 第 1 周 | 诊断 + 算法基础补齐 | 算法 70%，项目 20%，基础 10% |
| 第 2 周 | 专项突破 + 项目梳理 | 算法 50%，项目 40%，基础 10% |
| 第 3 周 | 综合练习 + 系统设计（高级岗） | 算法 40%，项目 30%，系统设计 20% |
| 第 4 周 | 模拟面试 + 查漏补缺 | 模拟 50%，复盘 30%，冲刺 20% |

如果时间更短（1-2 周），优先保算法和项目，HR 面和礼仪放到最后一天过一遍即可。

## 面试前一周的 Checklist

- [ ] 按类型过一遍算法高频题（不用全刷，重点是已掌握的类型不生疏）
- [ ] 每个项目能用 STAR 法则讲清楚
- [ ] 准备 2-3 个「技术亮点」深入聊
- [ ] 模拟面试一次（和朋友互面或用平台）
- [ ] 了解公司业务和岗位要求（不是海投）
- [ ] 准备好反向提问（2-3 个有深度的问题）
- [ ] HR 面的常见问题话术过一遍
- [ ] 调整作息，保证面试时精神好

准备好了这些，面试就是你的主场。
