# 公共资源

本目录为全站静态资源存储目录，用于存放各类公共资源文件。

## 内容范围

| 目录 | 内容 |
|------|------|
| `images/` | Logo、全站通用图片等 |
| `icon.png` | 站点图标（favicon 用途） |

## 使用说明

站内引用静态资源时，请使用以下路径格式：

```markdown
![描述文字](/images/logo.png)
![图标](/public/icon.png)
```

资源文件统一存放在本目录下，各模块文章按需引用。请确保引用路径正确，避免图片加载失败。
