## 二、支付流程

```mermaid
sequenceDiagram
    participant U as 用户
    participant APP as App
    participant GW as 支付网关
    participant ORDER as 订单服务
    participant PAY as 支付服务
    participant CH as 渠道
    participant CB as 回调

    U->>APP: 选择支付
    APP->>GW: 发起支付
    GW->>ORDER: 查询订单
    ORDER-->>GW: 订单信息
    GW->>PAY: 创建支付单
    PAY->>CH: 调起支付
    CH-->>APP: 返回支付凭据
    APP->>CH: 完成支付
    CH->>CB: 异步回调
    CB->>PAY: 处理回调
    PAY->>ORDER: 更新订单状态
```
