## 一、分布式 ID 要求

| 要求 | 说明 |
|------|------|
| 全局唯一 | 不同机器、不同时间不重复 |
| 趋势递增 | 有利于索引和范围查询 |
| 高可用 | 服务挂了也能生成 |
| 高性能 | 延迟低、吞吐高 |
| 可解释性 | 最好包含时间信息 |

## 三、雪花算法详解

### 算法原理

```
64 位 = 1 位符号 + 41 位时间戳 + 10 位机器 ID + 12 位序列号

| 1 bit | 41 bits | 10 bits | 12 bits |
|-------|---------|---------|---------|
| 0     | 时间戳   | 机器ID   | 序列号   |
```

### Java 实现

```java
public class Snowflake {
    private static final long EPOCH = 1609459200000L; // 2021-01-01
    private static final long WORKER_ID_BITS = 10L;
    private static final long SEQUENCE_BITS = 12L;

    private static final long MAX_WORKER_ID = ~(-1L << WORKER_ID_BITS);
    private static final long MAX_SEQUENCE = ~(-1L << SEQUENCE_BITS);

    private final long workerId;
    private long sequence = 0L;
    private long lastTimestamp = -1L;

    public Snowflake(long workerId) {
        if (workerId < 0 || workerId > MAX_WORKER_ID) {
            throw new IllegalArgumentException("workerId 超出范围");
        }
        this.workerId = workerId;
    }

    public synchronized long nextId() {
        long timestamp = System.currentTimeMillis();

        if (timestamp < lastTimestamp) {
            timestamp = lastTimestamp;
        }

        if (timestamp == lastTimestamp) {
            sequence = (sequence + 1) & MAX_SEQUENCE;
            if (sequence == 0) {
                timestamp = tilNextMillis();
            }
        } else {
            sequence = 0L;
        }

        lastTimestamp = timestamp;

        return ((timestamp - EPOCH) << (WORKER_ID_BITS + SEQUENCE_BITS))
            | (workerId << SEQUENCE_BITS)
            | sequence;
    }

    private long tilNextMillis() {
        long timestamp = System.currentTimeMillis();
        while (timestamp <= lastTimestamp) {
            timestamp = System.currentTimeMillis();
        }
        return timestamp;
    }
}
```

## 四、TinyID 方案

### 原理

```java
@Service
public class TinyIdService {
    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    private static final String ID_KEY = "tinyid:";

    public List<Long> getNextId(int count) {
        String key = ID_KEY + "segment";
        Long max = redisTemplate.opsForValue().increment(key, count);
        if (max == null) {
            return Collections.emptyList();
        }

        List<Long> ids = new ArrayList<>(count);
        long start = max - count + 1;
        for (long i = start; i <= max; i++) {
            ids.add(i);
        }
        return ids;
    }
}
```

## 五、时钟回拨处理

@Service
public class SnowflakeWithProtection {
    private volatile long lastTimestamp = -1L;
    private volatile long sequence = 0L;

    public synchronized long nextId() {
        long timestamp = System.currentTimeMillis();

        // 时钟回拨
        if (timestamp < lastTimestamp) {
            // 方案1：等待
            timestamp = lastTimestamp;

            // 方案2：使用备用机器ID
            // return generateWithBackupWorker(timestamp);
        }

        if (timestamp == lastTimestamp) {
            sequence = (sequence + 1) & MAX_SEQUENCE;
            if (sequence == 0) {
                timestamp = tilNextMillis(lastTimestamp);
            }
        } else {
            sequence = 0L;
        }

        lastTimestamp = timestamp;
        return generate(timestamp, workerId, sequence);
    }
}
