### 2. 热点 Key 处理

```java
@Service
public class HotKeyService {
    private static final String HOT_KEY_PREFIX = "hot:key:";

    // 热点 Key 本地缓存
    private LoadingCache<String, String> localCache = Caffeine.newBuilder()
        .maximumSize(10000)
        .expireAfterWrite(Duration.ofSeconds(5))
        .build();

    public String getProductInfo(Long productId) {
        String hotKey = HOT_KEY_PREFIX + productId;
        String cacheKey = "product:info:" + productId;

        // 1. 先查本地缓存
        String localValue = localCache.getIfPresent(cacheKey);
        if (localValue != null) {
            return localValue;
        }

        // 2. 查 Redis
        String value = redisTemplate.opsForValue().get(cacheKey);
        if (value != null) {
            // 写入本地缓存
            localCache.put(cacheKey, value);
        }

        return value;
    }
}
```

## 三、缓存优化

### 1. 多级缓存

```java
@Configuration
public class MultiLevelCacheConfig {
    // L1: 本地缓存（Caffeine）
    @Bean
    public Cache<Long, Product> localCache() {
        return Caffeine.newBuilder()
            .maximumSize(10000)
            .expireAfterWrite(Duration.ofSeconds(5))
            .build();
    }

    // L2: Redis 分布式缓存
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    // L3: MySQL
    @Autowired
    private ProductMapper productMapper;
}

@Service
public class ProductService {
    @Autowired
    private Cache<Long, Product> localCache;
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    @Autowired
    private ProductMapper productMapper;

    public Product getProduct(Long productId) {
        // L1: 本地缓存
        Product product = localCache.getIfPresent(productId);
        if (product != null) {
            return product;
        }

        // L2: Redis
        String key = "product:" + productId;
        product = (Product) redisTemplate.opsForValue().get(key);
        if (product != null) {
            localCache.put(productId, product);
            return product;
        }

        // L3: 数据库
        product = productMapper.selectById(productId);
        if (product != null) {
            redisTemplate.opsForValue().set(key, product);
            localCache.put(productId, product);
        }

        return product;
    }
}
```

### 2. 缓存预热

```java
@Component
public class CacheWarmer {
    @Autowired
    private ProductMapper productMapper;
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    @PostConstruct
    public void warmUp() {
        // 秒杀活动前 10 分钟预热
        List<Product> hotProducts = productMapper.findHotProducts();

        for (Product product : hotProducts) {
            String key = "product:info:" + product.getId();
            redisTemplate.opsForValue().set(key, product);

            String stockKey = "seckill:stock:" + product.getId();
            redisTemplate.opsForValue().set(stockKey, product.getStock());
        }
    }
}
```

## 五、风控系统

### 1. 用户画像

```java
@Service
public class RiskControlService {
    @Autowired
    private UserProfileMapper userProfileMapper;

    public RiskLevel evaluateUser(Long userId) {
        UserProfile profile = userProfileMapper.selectByUserId(userId);

        int score = 0;

        // 新用户扣分
        if (profile.getAccountAgeDays() < 30) score -= 20;

        // 频繁购买扣分
        if (profile.getDailyOrderCount() > 5) score -= 30;

        // 高信用加分
        if (profile.getCreditScore() > 800) score += 20;

        // 黑名单直接拒绝
        if (profile.isBlackList()) return RiskLevel.REJECT;

        if (score >= 20) return RiskLevel.ALLOW;
        if (score >= 0) return RiskLevel.REVIEW;
        return RiskLevel.REJECT;
    }
}
```

### 2. 实时风控

```java
@Service
public class RealTimeRiskControl {
    @Autowired
    private RiskControlService riskControlService;
    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    public RiskCheckResult check(SeckillRequest request) {
        Long userId = request.getUserId();
        Long productId = request.getProductId();

        // 1. 风控评估
        RiskLevel level = riskControlService.evaluateUser(userId);
        if (level == RiskLevel.REJECT) {
            return RiskCheckResult.reject("账户异常");
        }

        // 2. 设备指纹检查
        if (!validateDevice(request.getDeviceFingerprint())) {
            return RiskCheckResult.reject("设备异常");
        }

        // 3. 行为分析
        if (analyzeBehavior(request)) {
            return RiskCheckResult.review("需要人工审核");
        }

        return RiskCheckResult.allow();
    }

    private boolean validateDevice(String fingerprint) {
        // 检查设备是否在黑名单
        String key = "device:blacklist:" + fingerprint;
        return !Boolean.TRUE.equals(redisTemplate.hasKey(key));
    }

    private boolean analyzeBehavior(SeckillRequest request) {
        // 分析用户行为：点击频率、下单间隔等
        String key = "behavior:" + request.getUserId();
        Long count = redisTemplate.opsForValue().increment(key);

        if (count != null && count == 1) {
            redisTemplate.expire(key, Duration.ofMinutes(1));
        }

        // 1 分钟内点击超过 100 次，标记为可疑
        return count != null && count > 100;
    }
}
```

## 六、总结对比

| 问题 | 解决方案 | 效果 |
|------|----------|------|
| 热点 Key | 多级缓存 + 本地缓存 | 降低 Redis 压力 |
| 缓存击穿 | 缓存预热 + 互斥锁 | 保证可用性 |
| 热点隔离 | 独立集群 + 独立数据库 | 故障隔离 |
| 薅羊毛 | 风控系统 + 设备指纹 | 识别恶意用户 |
