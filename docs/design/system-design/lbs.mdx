## 二、GeoHash 算法

### 算法原理

将经纬度编码为字符串，相邻字符串表示相邻区域。

```
地球 → 32进制编码 → 8位字符串

精度对照：
- 1位：范围约 5000km
- 4位：范围约 156km
- 6位：范围约 39km
- 8位：范围约 4.9km
```

## 三、Redis Geo 实现

```java
@Service
public class GeoService {
    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    private static final String GEO_KEY = "geo:poi:";

    // 添加 POI
    public void addPoi(Long poiId, double longitude, double latitude) {
        String key = GEO_KEY + "daily"; // 按天分桶
        redisTemplate.opsForGeo().add(key, new Point(longitude, latitude), poiId.toString());
    }

    // 附近查询
    public List<PoiResult> nearbySearch(double longitude, double latitude, double radius, int limit) {
        String key = GEO_KEY + "daily";
        GeoResults results = redisTemplate.opsForGeo()
            .radius(key, new Circle(new Point(longitude, latitude), new Distance(radius, Metrics.KILOMETERS)),
                GeoRadiusCommandArgs.newGeoRadiusArgs()
                    .includeDistance()
                    .includeCoordinates()
                    .sortAscending()
                    .limit(limit));

        return convertToPoiResults(results);
    }
}
```
