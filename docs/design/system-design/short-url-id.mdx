## 三、雪花算法（Snowflake）

### 算法原理

```
64 bit = 1(符号位) + 41(时间戳) + 10(机器ID) + 12(序列号)

| 1 bit | 41 bits | 10 bits | 12 bits |
| ---- | ------- | ------- | ------- |
| 0    | 时间戳   | 机器ID   | 序列号   |
```

### Java 实现

```java
public class SnowflakeIdGenerator {
    // 开始时间戳（2020-01-01）
    private static final long EPOCH = 1577836800000L;

    // 机器 ID 占用的位数
    private static final long WORKER_ID_BITS = 10L;
    // 最大机器 ID
    private static final long MAX_WORKER_ID = ~(-1L << WORKER_ID_BITS);

    // 序列号占用的位数
    private static final long SEQUENCE_BITS = 12L;
    // 序列号最大值
    private static final long MAX_SEQUENCE = ~(-1L << SEQUENCE_BITS);

    // 机器 ID 左移位数
    private static final long WORKER_ID_SHIFT = SEQUENCE_BITS;
    // 时间戳左移位数
    private static final long TIMESTAMP_SHIFT = SEQUENCE_BITS + WORKER_ID_BITS;

    private final long workerId;
    private long sequence = 0L;
    private long lastTimestamp = -1L;

    public SnowflakeIdGenerator(long workerId) {
        if (workerId < 0 || workerId > MAX_WORKER_ID) {
            throw new IllegalArgumentException("workerId 超出范围");
        }
        this.workerId = workerId;
    }

    public synchronized long nextId() {
        long timestamp = timeGen();

        // 时钟回拨检查
        if (timestamp < lastTimestamp) {
            timestamp = lastTimestamp;
        }

        // 同一毫秒内，序列号递增
        if (timestamp == lastTimestamp) {
            sequence = (sequence + 1) & MAX_SEQUENCE;
            // 序列号溢出，等待下一毫秒
            if (sequence == 0) {
                timestamp = tilNextMillis(lastTimestamp);
            }
        } else {
            sequence = 0L;
        }

        lastTimestamp = timestamp;

        return ((timestamp - EPOCH) << TIMESTAMP_SHIFT)
            | (workerId << WORKER_ID_SHIFT)
            | sequence;
    }

    private long tilNextMillis(long lastTimestamp) {
        long timestamp = timeGen();
        while (timestamp <= lastTimestamp) {
            timestamp = timeGen();
        }
        return timestamp;
    }

    private long timeGen() {
        return System.currentTimeMillis();
    }
}
```

### 解析雪花 ID

```java
public class SnowflakeIdParser {
    private static final long EPOCH = 1577836800000L;
    private static final long WORKER_ID_BITS = 10L;
    private static final long SEQUENCE_BITS = 12L;

    public static void parse(long id) {
        long timestamp = (id >> (SEQUENCE_BITS + WORKER_ID_BITS)) + EPOCH;
        long workerId = (id >> SEQUENCE_BITS) & ((1 << WORKER_ID_BITS) - 1);
        long sequence = id & ((1 << SEQUENCE_BITS) - 1);

        System.out.println("时间戳: " + new Date(timestamp));
        System.out.println("机器ID: " + workerId);
        System.out.println("序列号: " + sequence);
    }
}
```

## 五、时钟回拨处理

### 问题

```
问题：机器时间回拨，导致生成重复 ID
场景：NTP 服务同步时间、虚拟机暂停恢复
```

### 解决方案

### 方案一：等待回拨时间过去

```java
private long tilNextMillis(long lastTimestamp) {
    long timestamp = timeGen();
    // 等待直到时间大于上次时间
    while (timestamp <= lastTimestamp) {
        timestamp = timeGen();
    }
    return timestamp;
}
```

### 方案二：使用历史最大时间

```java
private static final long EPOCH = 1577836800000L;
private volatile long lastTimestamp = -1L;

public synchronized long nextId() {
    long timestamp = timeGen();

    // 如果时间回拨，使用历史最大时间
    if (timestamp < lastTimestamp) {
        timestamp = lastTimestamp;
    }

    if (timestamp == lastTimestamp) {
        sequence = (sequence + 1) & MAX_SEQUENCE;
        if (sequence == 0) {
            timestamp = tilNextMillis(lastTimestamp);
        }
    } else {
        sequence = 0L;
    }

    lastTimestamp = timestamp;
    return generate(timestamp, workerId, sequence);
}
```

### 方案三：扩展时间戳位数

```java
// 如果发现时间回拨，使用备用机器 ID
if (timestamp < lastTimestamp) {
    // 使用备用机器 ID
    long backupWorkerId = workerId + MAX_WORKER_ID / 2;
    return generate(timestamp, backupWorkerId, sequence);
}
```

## 六、实战应用

### 短链发号器实现

```java
@Service
public class ShortUrlIdGenerator {
    private SnowflakeIdGenerator snowflake;

    @PostConstruct
    public void init() {
        // 从配置或注册中心获取 workerId
        long workerId = getWorkerIdFromConfig();
        snowflake = new SnowflakeIdGenerator(workerId);
    }

    public String generateShortCode() {
        long id = snowflake.nextId();
        // 转换为 Base62
        return Base62.encode(id);
    }

    private long getWorkerIdFromConfig() {
        // 从配置中心或注册中心获取
        return 1L;
    }
}
```
