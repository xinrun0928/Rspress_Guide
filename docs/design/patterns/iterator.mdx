## 二、迭代器模式实现

```java
// 迭代器接口
public interface Iterator<T> {
    boolean hasNext();
    T next();
}

// 集合接口
public interface Collection<T> {
    Iterator<T> createIterator();
}

// 具体实现
public class UserCollection implements Collection<User> {
    private User[] users = new User[100];
    private int size = 0;

    @Override
    public Iterator<User> createIterator() {
        return new UserIterator(this);
    }

    // 内部类实现迭代器
    private class UserIterator implements Iterator<User> {
        private int index = 0;

        @Override
        public boolean hasNext() {
            return index < size;
        }

        @Override
        public User next() {
            return users[index++];
        }
    }
}
```

## 三、Java 内置迭代器

```java
// List 的迭代器
List<String> list = new ArrayList<>();
Iterator<String> it = list.iterator();

while (it.hasNext()) {
    String item = it.next();
}

// forEach 遍历
list.forEach(System.out::println);

// JDK 8+ forEachRemaining
it.forEachRemaining(System.out::println);
```

## 四、Fail-Fast vs Fail-Safe

| 对比 | Fail-Fast | Fail-Safe |
|------|------------|-----------|
| 实现 | modCount 检查 | 复制集合 |
| 异常 | ConcurrentModificationException | 不抛异常 |
| 内存 | 无额外开销 | 需要复制 |
| 示例 | ArrayList | CopyOnWriteArrayList |
