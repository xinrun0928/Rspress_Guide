# 简历投递后的跟进策略

「简历投了，没消息，是不是就没戏了？」这是很多求职者的困惑。

其实，简历投递只是求职的开始。**跟进是让机会「起死回生」的关键**。但跟进不是骚扰，而是专业度的体现。

## 面试官最关心的 3 个问题

> **问题 1**：简历投递后多久没消息需要跟进？
> **问题 2**：如何得体地跟进，不让 HR 觉得烦？
> **问题 3**：跟进后仍然没有回复，还有必要继续等吗？

## 跟进的时间节点

### 标准跟进时间线

```
Day 0：简历投递
    ↓
Day 3-5：第一次跟进（通过招聘平台消息或邮件）
    ↓
Day 7-10：第二次跟进（如果第一次没回复）
    ↓
Day 14：第三次跟进（可选）
    ↓
Day 30：放弃等待，简历进入冷冻期
```

### 不同渠道的跟进策略

| 渠道 | 跟进方式 | 跟进频率 |
|------|----------|----------|
| **官网投递** | 官方渠道/邮件 | 每周一次 |
| **内推** | 联系内推人 | 3-5 天一次 |
| **猎头** | 通过猎头询问 | 配合猎头节奏 |
| **招聘网站** | 平台消息 | 3-5 天一次 |

## 跟进话术模板

### 模板 1：初次跟进（Day 3-5）

**邮件主题**：简历跟进 - [姓名] - [岗位名称]

```
尊敬的 HR，您好！

我是 [姓名]，于 [日期] 通过 [渠道] 投递了贵司 [岗位名称] 的简历。

非常期待能有机会与您进一步沟通，了解岗位详情。

如果您方便的话，希望能得到一些反馈，无论是否合适都感谢您的时间和考虑。

祝好！
[姓名]
[联系方式]
```

### 模板 2：内推跟进（通过内推人）

```
[内推人姓名]，您好！

感谢您帮我内推 [公司] 的 [岗位]。

想跟您了解下进度，目前简历是否已经被 HR 看到？有没有后续安排？

如果不方便打扰，请忽略这条消息，非常感谢您的帮助！
```

### 模板 3：猎头跟进

```
[猎头顾问]，您好！

想了解下 [公司] [岗位] 的进展，目前是什么状态？

如果需要补充材料或调整简历，请随时告知，我会第一时间处理。
```

### 模板 4：面试后跟进

```
尊敬的面试官，您好！

感谢您今天抽出宝贵时间面试我。

面试过程中，您对 [团队/业务方向] 的介绍让我印象深刻，也让我更加确信这正是我希望加入的平台。

如果后续有任何问题需要沟通，欢迎随时联系我。

期待能有机会成为团队的一员！
祝工作顺利！
[姓名]
```

## ⚠️ 跟进中的常见错误

### 错误 1：跟进太频繁

**表现**：每天问一次，HR 已经拉黑。

**后果**：被 HR 拉入黑名单，永远失去这家公司的机会。

**正确做法**：每次跟进间隔至少 3 天，总跟进次数不超过 3 次。

### 错误 2：跟进态度不专业

**错误话术**：

```
❌ 「我投了简历怎么没人理？」
❌ 「什么时候给我面试机会？」
❌ 「你们是不是不想招人了？」
```

**正确话术**：

```
✅ 「您好，想了解下简历的筛选进度，如有需要补充的材料请告知。」
✅ 「非常期待能有机会沟通，请问方便告知后续安排吗？」
```

### 错误 3：重复投递同一岗位

**表现**：一周没回复，再投一次。

**后果**：被系统标记为「重复投递」，两份简历都被作废。

**正确做法**：只投递一次，通过跟进获取信息。

### 错误 4：Hr 不回就放弃了

**表现**：Hr 没回复，就再也不管了。

**后果**：可能只是 Hr 太忙没看到，错过面试机会。

**正确做法**：至少跟进 2-3 次，保持礼貌但不放弃。

## 不同场景的跟进策略

### 场景 1：简历显示「已投递」但无后续

**策略**：等待 3-5 天后，通过招聘平台消息跟进。

**话术**：模板 1 或模板 4。

### 场景 2：Hr 显示「已读」但无回复

**策略**：可以跟进，但语气要温和，表示理解 Hr 很忙。

**话术**：

```
您好！理解您工作繁忙，如有需要补充的材料请随时告知。
如果您认为我的背景与岗位不匹配，也感谢您的考虑，祝工作顺利！
```

### 场景 3：Hr 明确说「不合适」

**策略**：不要纠缠，礼貌感谢，询问不匹配的原因（可选）。

**话术**：

```
感谢您的回复！方便的话，能告知不匹配的原因吗？方便我后续改进。
祝您招聘顺利！
```

### 场景 4：Hr 答应面试但一直约时间

**策略**：主动提供几个时间段，表现出积极性和配合度。

**话术**：

```
您好！非常期待面试，请问您这周方便的时间是什么时候？
我可以配合您的日程，包括晚间或周末都可以。
```

### 场景 5：Hr 面试后说「等通知」

**策略**：面试后 3-5 天跟进，询问结果。

**话术**：

```
您好！想了解下 [日期] 的面试结果，方便告知吗？
如有任何问题需要补充材料，请随时告知。
```

## 💡 加分技巧：让 Hr 愿意回复

### 技巧 1：提供增量信息

不要只说「想了解进度」，而是提供一些新信息：

```
✅ 「我刚完成了一个 XX 项目，取得 XX 成果，想同步给您。」
✅ 「最近面试了 XX 公司，对 XX 有了更深入的理解，希望能与贵司深入交流。」
```

### 技巧 2：展示对公司的兴趣

```
✅ 「我了解到贵司最近在 XX 方向有新的突破，非常感兴趣。」
✅ 「关注贵司已经有一段时间了，希望能有机会深入交流。」
```

### 技巧 3：提供价值

```
✅ 「我这边有一些 [相关技术/业务] 的经验，可能对团队有帮助。」
✅ 「如果您方便，我可以做一个简短的技术分享，展示我的能力。」
```

## 跟进状态解读

| HR 回复 | 含义 | 后续行动 |
|---------|------|----------|
| 「简历已收到，在筛选中」 | 正常流程，继续等待 | 1 周后再次跟进 |
| 「已推荐给业务部门」 | 进入下一阶段 | 等业务部门安排面试 |
| 「不合适」 | 这次岗位不匹配 | 问原因，改进简历 |
| 「暂无合适岗位」 | 短期无机会 | 保持联系，等待机会 |
| 「已读不回」 | 可能真的不合适 | 停止跟进，换其他机会 |

## 跟进检查清单

- [ ] 投递后记录投递时间、渠道、岗位
- [ ] 投递后 3-5 天进行第一次跟进
- [ ] 跟进时语气专业、礼貌
- [ ] 提供增量信息，展示诚意
- [ ] 每条消息间隔至少 3 天
- [ ] 总跟进次数不超过 3 次
- [ ] 被拒绝后礼貌感谢，问清原因
- [ ] 记录跟进结果，更新状态

## 常见问题

**Q：Hr 一直不回，是不是就没希望了？**

A：不一定。很多公司 Hr 人手不够，无法及时处理所有简历。跟进 2-3 次后如果仍无回复，可以暂时放弃，简历进入公司人才库，未来有新岗位可能还会被捞起来。

**Q：可以同时跟进多个岗位吗？**

A：可以，但每个岗位要分别跟进，不要让 Hr 觉得你「广撒网」。

**Q：内推的简历需要跟进吗？**

A：需要。内推只是让简历更容易被看到，但不代表会自动推进。保持和内推人的沟通，了解进度。

**Q：跟进次数太多会被拉黑吗？**

A：如果每次跟进都间隔合理（>= 3 天）、语气专业，一般不会被拉黑。但如果频率太高（如每天一次）、语气不友好（如质问、催促），则有可能。
