# Offer 综合评估维度

拿到多个 Offer 该怎么选？只看薪资？那可能是最不理智的选择。

## 面试官最关心的 3 个问题

> **问题 1**：评估 Offer 应该考虑哪些因素？
> **问题 2**：薪资重要还是平台重要？
> **问题 3**：如何做出最优选择？

## Offer 评估七维度

### 维度 1：薪资待遇（25%）

**评估内容**：

| 项目 | 说明 |
|------|------|
| Base | 年薪或月薪 |
| Bonus | 年终奖 |
| Sign-on | 入职奖金 |
| Stock/RSU | 股票期权 |
| Benefits | 社保公积金、商业保险等 |

**评估方法**：

```
第一步：计算 Total Package
第二步：评估涨幅（与当前薪资对比）
第三步：评估长期增长潜力
```

### 维度 2：职业发展（25%）

**评估内容**：

| 项目 | 说明 |
|------|------|
| 技术成长 | 能学到什么 |
| 级别晋升 | 能否晋升 |
| 项目机会 | 有没有挑战性项目 |
| 背书价值 | 对未来跳槽有帮助吗 |

**评估方法**：

```
- 这家公司/团队的技术栈是什么？
- 能接触到大流量、大规模系统吗？
- 进去是什么级别？上升空间如何？
```

### 维度 3：公司平台（20%）

**评估内容**：

| 项目 | 说明 |
|------|------|
| 公司规模 | 大公司/中型/创业公司 |
| 行业地位 | 行业排名、口碑 |
| 技术实力 | 技术是否领先 |
| 资金状况 | 是否盈利、融资情况 |

**评估方法**：

```
- 公司是否稳定？会裁员吗？
- 技术在行业内是什么水平？
- 有没有技术护城河？
```

### 维度 4：团队氛围（15%）

**评估内容**：

| 项目 | 说明 |
|------|------|
| 直属领导 | 领导风格、能力 |
| 团队规模 | 人多人少 |
| 协作方式 | 团队配合如何 |
| 技术分享 | 有没有技术交流 |

**评估方法**：

```
- 可以通过面试观察
- 可以问面试官或 HR
- 可以通过脉脉、牛客了解
```

### 维度 5：业务前景（10%）

**评估内容**：

| 项目 | 说明 |
|------|------|
| 业务增长 | 业务是否快速增长 |
| 商业模式 | 能否盈利 |
| 竞争格局 | 竞争对手是谁 |
| 政策风险 | 政策对业务的影响 |

### 维度 6：工作地点（3%）

**评估内容**：

| 项目 | 说明 |
|------|------|
| 城市 | 一线/二线城市 |
| 通勤 | 通勤时间和方式 |
| 住房 | 房租、房价 |
| 生活成本 | 生活成本差异 |

### 维度 7：其他因素（2%）

| 项目 | 说明 |
|------|------|
| 通勤灵活性 | 能否远程 |
| 休假制度 | 年假多少 |
| 培训机会 | 有没有培训预算 |
| 股权激励 | 长期激励如何 |

## Offer 对比工具

### 对比矩阵

| 维度 | 权重 | Offer A | Offer B | Offer C |
|------|------|---------|---------|---------|
| 薪资待遇 | 25% | 90 | 85 | 80 |
| 职业发展 | 25% | 85 | 95 | 75 |
| 公司平台 | 20% | 90 | 85 | 70 |
| 团队氛围 | 15% | 80 | 90 | 85 |
| 业务前景 | 10% | 85 | 80 | 75 |
| 工作地点 | 3% | 90 | 85 | 95 |
| 其他因素 | 2% | 80 | 85 | 80 |
| **加权总分** | 100% | **87** | **88** | **77** |

### 计算公式

```
加权总分 = Σ(分数 × 权重)

Offer A = 90×25% + 85×25% + 90×20% + 80×15% + 85×10% + 90×3% + 80×2% = 87
```

## ⚠️ Offer 选择的常见误区

### 误区 1：只看薪资

**表现**：「A 公司薪资最高，我选 A」

**问题**：忽略了职业发展、平台价值等长期因素

### 误区 2：薪资至上

**表现**：「我要最大化薪资」

**问题**：可能损失平台价值、成长空间

### 误区 3：忽视领导因素

**表现**：「公司好就行，领导无所谓」

**问题**：直属领导是最影响日常工作的人

### 误区 4：只看公司名气

**表现**：「大公司就行，团队/业务无所谓」

**问题**：同一大公司，不同团队差距可能很大

### 误区 5：忽视稳定性

**表现**：「薪资高就行，不怕裁员」

**问题**：如果公司裁员，高薪资也白搭

## 💡 特殊情况的处理

### 情况 1：薪资差距大

```
如果 A 公司薪资比 B 公司高 50%：
- 评估这个差距是否值得放弃其他因素
- 考虑 A 公司的高薪资能持续多久
```

### 情况 2：平台差距大

```
如果 A 公司平台比 B 公司高很多：
- 平台溢价可能值得牺牲一些薪资
- 考虑未来的成长和背书价值
```

### 情况 3：职业早期 vs 晚期

```
职业早期：优先考虑平台和发展
职业晚期：可以更看重薪资和稳定性
```

## 常见问题

**Q：薪资差多少值得换平台？**

A：通常 20-30% 的涨幅值得考虑。但如果平台差距太大，可以接受更小的涨幅。

**Q：offer deadline 很紧怎么办？**

A：可以礼貌地请求延期，说明你有其他 offer 在看。大多数公司愿意等 1-2 周。

**Q：同时有多个 offer 应该怎么选？**

A：用七维框架全面评估，不要只看薪资。
