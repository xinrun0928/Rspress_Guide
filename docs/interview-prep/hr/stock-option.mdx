# 股票/期权的计算方式

「给你价值 100 万的股票期权」——这句话值多少钱？

股票期权是薪资谈判中最复杂的部分，也是最容易「被坑」的地方。很多人拿到期权 offer 后算了半天算不明白，最后发现实际价值远低于预期。

## 面试官最关心的 3 个问题

> **问题 1**：股票期权的价值如何计算？
> **问题 2**：归属（Vesting）是什么意思？如何计算每年能拿到多少？
> **问题 3**：期权 vs RSU 有什么区别？哪个更好？

## 基本概念

### 常见术语

| 术语 | 全称 | 说明 |
|------|------|------|
| **Stock** | 股票 | 公司直接给股票 |
| **RSU** | Restricted Stock Unit | 限制性股票单元，满足条件后发放股票 |
| **Option** | 期权 | 可以在特定价格购买股票的权利 |
| **Strike Price** | 行权价 | 期权可以购买股票的价格 |
| **Grant** | 期权授予 | 一次性给多少期权 |
| **Vesting** | 归属 | 期权逐步到手的过程 |
| **Cliff** | 悬崖期 | 归属前需要等待的时间 |
| **Exercise** | 行权 | 使用期权购买股票 |

## RSU vs 期权

### RSU（限制性股票单元）

**定义**：公司直接给你股票，但需要满足一定条件才能到手。

**特点**：

- 你拿到的是「真实的股票」
- 归属后就成为你的财产
- 股票价值 = 当前股价 × 归属股数
- 如果公司倒闭，股票可能变成废纸

**计算示例**：

```
RSU 授予：10 万股
归属计划：4 年归属，每年 25%
当前股价：$10

第一年末：你获得 2.5 万股，价值 = 2.5 × $10 = $25 万
第二年末：你获得 2.5 万股，价值 = 2.5 × $12 = $30 万
第三年末：你获得 2.5 万股，价值 = 2.5 × $15 = $37.5 万
第四年末：你获得 2.5 万股，价值 = 2.5 × $20 = $50 万

4 年总计：价值约 $142.5 万
```

### 期权（Stock Option）

**定义**：公司给你一种「购买股票的权利」，但需要支付行权价。

**特点**：

- 你拿到的是「权利」，不是股票
- 如果股价高于行权价，你才能赚钱
- 如果股价低于行权价，期权就是废纸
- 行权时需要缴税

**计算示例**：

```
期权授予：10 万股
行权价：$5
归属计划：4 年归属，每年 25%
当前股价：$10

第一年末：获得 2.5 万股行权权利
- 如果行权：成本 = 2.5 万 × $5 = $12.5 万
- 股票价值 = 2.5 万 × $10 = $25 万
- 收益 = $25 万 - $12.5 万 = $12.5 万（未扣税）
```

### RSU vs 期权对比

| 维度 | RSU | 期权 |
|------|-----|------|
| **风险** | 低风险（拿到的是股票） | 高风险（可能是废纸） |
| **收益上限** | 无上限 | 无上限 |
| **收益下限** | 可能为零（公司倒闭） | 一定为零（行权价 > 股价） |
| **税务** | 归属时按工资收入缴税 | 行权时按资本利得缴税 |
| **适用公司** | 上市公司为主 | 创业公司/上市公司都有 |

## 归属机制（Vesting）

### 常见归属计划

| 归属计划 | 说明 | 示例 |
|----------|------|------|
| **4 年归属，1 年悬崖** | 满 1 年后开始归属 | 常见于美国公司 |
| **4 年均衡归属** | 每年归属 25% | 常见于美国公司 |
| **3 年归属** | 每年归属 33.3% | 常见于某些公司 |
| **分月归属** | 每月归属一部分 | 某些公司 |

### 归属计算示例

**4 年归属，1 年悬崖**：

```
授予：10 万股
第一年末：0 股（cliff）
第二年末：2.5 万股
第三年末：2.5 万股
第四年末：2.5 万股
第五年末：2.5 万股

总计：10 万股
```

**4 年均衡归属**：

```
授予：10 万股
第一年末：2.5 万股
第二年末：2.5 万股
第三年末：2.5 万股
第四年末：2.5 万股

总计：10 万股
```

### Vesting Schedule 计算器

```
假设：
- 授予总量：10 万股
- Vesting 周期：4 年
- 1 年悬崖
- 每年归属：25%

Year 1：0 股（还在 cliff 期）
Year 2：10 万 × 25% = 2.5 万股
Year 3：10 万 × 25% = 2.5 万股
Year 4：10 万 × 25% = 2.5 万股
Year 5：10 万 × 25% = 2.5 万股
```

## 如何计算期权价值

### 步骤 1：确定授予数量

```
期权价值 = 授予数量 × (当前股价 - 行权价) × 归属比例
```

### 步骤 2：考虑归属时间

```
年度期权价值 = 授予总量 × 年度归属比例 × (当前股价 - 行权价)
```

### 步骤 3：考虑股票增长

```
假设年增长率 10%：
Year 1：10 万 × 25% × ($10 × 1.1 - $5) = ...
Year 2：10 万 × 25% × ($10 × 1.1² - $5) = ...
```

## ⚠️ 期权陷阱

### 陷阱 1：只看授予总量

**表现**：「给你 100 万股期权」听起来很多。

**问题**：行权价、归属条件、股价都可能让期权变成废纸。

**正确做法**：计算实际价值，而不是授予总量。

### 陷阱 2：忽视行权价

**表现**：「1 块钱的行权价」看起来很便宜。

**问题**：如果公司发展不好，股价可能低于行权价。

**正确做法**：计算「内在价值」= (当前股价 - 行权价) × 数量。

### 陷阱 3：忽视归属期

**表现**：「4 年后全部归属」。

**问题**：4 年内离职，期权可能全部或部分作废。

**正确做法**：了解归属计划和留任要求。

### 陷阱 4：不了解行权成本

**表现**：期权行权要花钱。

**问题**：行权时需要按行权价购买股票，这是一笔不小的开支。

**正确做法**：计算行权成本和税后收益。

### 陷阱 5：忽视稀释

**表现**：我有 10 万股期权。

**问题**：如果公司融资稀释，你的 10 万股可能变成 5 万股。

**正确做法**：了解稀释机制，计算稀释后的实际比例。

## 💡 期权谈判技巧

### 技巧 1：谈 RSU 而不是期权

```
✅ "我希望 RSU 而不是期权，因为 RSU 的风险更小。"
❌ "给我期权就行。"
```

### 技巧 2：谈更高的授予量

```
✅ "如果 base 无法提高，我希望能增加 RSU 的授予量。"
```

### 技巧 3：谈更短的归属期

```
✅ "我希望 3 年归属而不是 4 年。"
```

### 技巧 4：谈更低的行权价（期权）

```
✅ "行权价能否按上一轮融资的价格计算，而不是更高的价格？"
```

## 常见问题

**Q：创业公司的期权值不值得拿？**

A：看公司阶段、团队、估值、发展潜力。风险高，潜在收益也高。如果公司上市或被收购，期权可能价值很高；如果倒闭，就变成废纸。

**Q：离开公司后期权怎么处理？**

A：通常有 30-90 天的行权窗口期。必须在窗口期内决定是否行权，否则期权作废。

**Q：期权要交多少税？**

A：比较复杂。行权时按工资收入缴个税，后续卖出按资本利得缴税。具体请咨询专业税务顾问。
