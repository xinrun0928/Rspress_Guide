# 背调注意事项

「背调过了，offer 就稳了。」——这句话只对了一半。

背调（背景调查）是 offer 发放前的最后一道关卡。很多人技术面试、HR 面试都通过了，却因为背调出了问题而被取消 offer。

## 面试官最关心的 3 个问题

> **问题 1**：背调主要查什么？
> **问题 2**：哪些问题会导致背调失败？
> **问题 3**：如何提前准备背调？

## 背调的内容

### 常规背调项目

| 项目 | 说明 | 风险等级 |
|------|------|----------|
| **身份信息** | 身份证、学历、工作经历 | 🔴 |
| **学历验证** | 毕业院校、学位、毕业时间 | 🔴 |
| **工作经历** | 工作时间、职位、离职原因 | 🔴 |
| **薪资核实** | 上一份工作的薪资 | 🟡 |
| **离职原因** | 是否正常离职 | 🟡 |
| **上级评价** | 前领导对候选人的评价 | 🟡 |
| **竞业协议** | 是否有竞业限制 | 🔴 |
| **犯罪记录** | 是否有犯罪记录 | 🟡 |
| **征信记录** | 是否有严重逾期 | 🟢 |

### 不同公司的背调深度

| 公司类型 | 背调深度 | 说明 |
|----------|----------|------|
| 大厂（阿里、腾讯等） | 深 | 查得比较仔细 |
| 外企 | 深 | 背调公司比较专业 |
| 中小公司 | 浅 | 可能只做基础验证 |
| 创业公司 | 浅或无 | 可能跳过背调 |

## ⚠️ 背调风险点

### 风险 1：学历造假

**表现**：

- 谎报毕业院校
- 谎报学历（专科说本科）
- 谎报学位

**后果**：

- 直接取消 offer
- 可能被拉入公司黑名单

**真实案例**：

```
候选人：某候选人声称是 985 大学计算机硕士
实际：普通本科，通过网络教育获得学历
结果：背调发现，取消 offer
```

### 风险 2：工作经历造假

**表现**：

- 延长工作时间
- 虚构公司
- 谎报职位
- 夸大薪资

**后果**：

- 直接取消 offer
- 如果已入职，可能被开除

**真实案例**：

```
候选人：声称在某公司担任技术负责人
实际：只是普通开发，没有带团队
结果：背调发现，取消 offer
```

### 风险 3：隐瞒竞业协议

**表现**：

- 隐瞒与上一家公司签订的竞业限制协议
- 声称已经解除竞业

**后果**：

- 可能面临法律诉讼
- 新公司可能取消 offer

**真实案例**：

```
候选人：从某大厂跳槽，隐瞒了竞业协议
实际：签有竞业协议，离职后不能加入竞争对手
结果：新公司收到原公司律师函，取消 offer
```

### 风险 4：薪资虚报

**表现**：

- 虚报上一份工作的薪资
- 虚报年终奖、股票等

**后果**：

- 如果基于薪资谈 offer，可能被追讨
- 可能影响信任

### 风险 5：隐瞒离职原因

**表现**：

- 把被辞退说成主动离职
- 隐瞒严重违纪记录

**后果**：

- 如果背景调查发现诚信问题
- 可能取消 offer

## 如何提前准备背调

### 准备清单

- [ ] 整理好所有工作经历的起止时间
- [ ] 确认每家公司的职位名称
- [ ] 确认离职原因
- [ ] 确认学历、学位信息
- [ ] 确认是否有竞业协议
- [ ] 准备好薪资证明（offer、工资条等）

### 背调前自检

```
学历检查：
- [ ] 毕业院校正确
- [ ] 毕业时间正确
- [ ] 学位信息正确

工作经历检查：
- [ ] 每家公司的起止时间正确
- [ ] 职位名称正确
- [ ] 离职原因正确

其他检查：
- [ ] 是否有竞业协议
- [ ] 是否有需要交接的东西
- [ ] 薪资信息是否准备好
```

## 背调常见问题

### Q：背调会查多久？

A：一般 1-2 周，有些公司可能更长。

### Q：背调会联系谁？

A：通常是 hr、前领导、同事。高级别岗位可能联系更多。

### Q：背调前需要通知前公司吗？

A：不需要，背调机构会自己联系。

### Q：可以提供背调的问题清单吗？

A：可以提前向 hr 了解背调的流程和内容。

### Q：可以解释背调中的问题吗？

A：可以。如果有特殊情况（如公司重组导致时间不一致），可以提前向 hr 说明。

## ⚠️ 背调中的陷阱

### 陷阱 1：不知道有竞业协议

**预防**：

- 离职前仔细阅读劳动合同
- 询问 HR 或法务是否有竞业限制

### 陷阱 2：时间记不清

**预防**：

- 提前整理好所有工作经历的时间线
- 如果时间有出入，提前向 hr 说明

### 陷阱 3：联系人写错

**预防**：

- 选择与你有良好工作关系的人作为背调联系人
- 提前和背调联系人打招呼

### 陷阱 4：薪资证明造假

**预防**：

- 不要虚报薪资
- 准备好真实的薪资证明

## 💡 背调加分技巧

### 技巧 1：主动说明特殊情况

```
如果工作时间有出入：
✅ "我在 XX 公司的工作时间是 2019.03-2022.06，但由于公司内部调整，
   社保缴纳记录可能显示不同的公司名称，这是同一集团下的子公司。"

✅ "我的职位是高级工程师，但公司内部职级可能叫 XX，实际职责是 XX。"
```

### 技巧 2：选择合适的背调联系人

```
选择：
- ✅ 与你有良好关系的前领导
- ✅ 了解你工作表现的同时
- ✅ 可以客观评价你的人

不选：
- ❌ 与你有矛盾的人
- ❌ 职位太低不了解你的人
- ❌ 可能给负面评价的人
```

### 技巧 3：提前和联系人沟通

```
✅ "公司可能会做背调，如果有人联系你，麻烦帮我客观评价一下。"
```

## 常见问题

**Q：背调失败后还有机会吗？**

A：大多数情况下，背调失败意味着 offer 取消。但如果是小问题（如时间记错），可以尝试解释。

**Q：可以拒绝背调吗？**

A：可以，但大多数公司的 offer 都会附带背调通过的条件。如果拒绝背调，可能无法入职。

**Q：背调机构会透露结果吗？**

A：一般不会透露具体内容，只会告知是否通过。
