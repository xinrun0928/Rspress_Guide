# 薪资构成解析

「年薪 50 万」——这四个字背后，可能藏着完全不同的含义。

有人 base 50 万，有人 base 35 万 + 年终 15 万，有人 base 30 万 + stock 20 万。看起来都是 50 万，实际上千差万别。

了解薪资构成，是谈薪资的第一步。

## 面试官最关心的 3 个问题

> **问题 1**：薪资由哪些部分构成？每个部分有什么特点？
> **问题 2**：如何计算 total package？
> **问题 3**：不同公司的薪资结构有什么差异？

## 薪资构成详解

### 1. Base Salary（基本工资）

**定义**：固定的月薪或年薪。

**特点**：

| 维度 | 说明 |
|------|------|
| 稳定性 | 100% 确定的收入 |
| 涨幅 | 每年调薪，一般 5-15% |
| 税前/税后 | 通常是税前 |
| 五险一金 | 按 base 基数缴纳 |

**计算方式**：

```
月薪 = 年薪 / 12（或 / 16/18/20，视公司而异）
```

### 2. Bonus（年终奖/绩效奖）

**定义**：根据公司业绩和个人绩效发放的奖金。

**特点**：

| 维度 | 说明 |
|------|------|
| 稳定性 | 不确定，取决于公司业绩和个人绩效 |
| 发放时间 | 通常年底发放 |
| 范围 | 0-6 个月不等 |
| 缴税 | 和 base 一起算税 |

**常见公司**：

| 公司 | 年薪构成 | 年终奖范围 |
|------|----------|------------|
| 阿里 | 16 薪 | 0-6 个月 |
| 腾讯 | 12-18 薪 | 视部门 |
| 字节 | 15 薪 | 通常 0-1 个月 |
| 美团 | 15.5 薪 | 通常 0-3 个月 |

### 3. Sign-on Bonus（入职奖金）

**定义**：一次性发放的入职奖励。

**特点**：

| 维度 | 说明 |
|------|------|
| 发放方式 | 一次性 or 分 2 年发放 |
| 税率 | 一次性发放税率较高 |
| 留任条款 | 可能要求服务满 N 年 |

**注意事项**：

- 如果第二年离职，可能需要退还部分或全部 sign-on
- 分两年发放的可以规避部分风险

### 4. Stock/RSU（股票/期权）

**定义**：公司给的长期激励。

**常见类型**：

| 类型 | 说明 | 风险 |
|------|------|------|
| **RSU** | 限制性股票，达到条件后归属 | 股票波动 |
| **期权** | 可以在特定价格购买股票 | 可能变成废纸 |
| **ESPP** | 员工购股计划 | 有限 |

**归属机制（Vesting）**：

```
典型归属计划：4 年归属，1 年悬崖

Year 1: 0%（第1年末开始归属）
Year 2: 25%
Year 3: 25%
Year 4: 25%
Year 5: 25%

或者：
每年归属 25%，4 年全部归属
```

### 5. Benefits（福利）

**常见福利**：

| 福利 | 说明 |
|------|------|
| 五险一金 | 社保公积金，通常 base 的 10-12% |
| 商业保险 | 医疗险、意外险等 |
| 餐补/交通补 | 每月几百到几千不等 |
| 房补/租房补贴 | 部分公司有 |
| 股票/期权 | 外企常见 |
| 年假 | 带薪年假 |
| 其他 | 健身、体检等 |

## Total Package 计算

### 计算公式

```
Total Package = (Base × 月数) + Bonus + Sign-on + (Stock Value / Vesting Years) + Benefits 价值
```

### 示例计算

**Offer A**：

```
Base: 40 万 / 16 薪 = 月薪 2.5 万
Bonus: 2 个月 = 8 万
Sign-on: 10 万（一次性）
RSU: 20 万 / 4 年 = 5 万/年
Benefits: 约 5 万/年（社保公积金 + 商业险）

Total Package = 40 + 8 + 10 + 5 + 5 = 68 万/年
```

**Offer B**：

```
Base: 50 万 / 12 薪 = 月薪 4.17 万
Bonus: 0
Sign-on: 0
RSU: 0
Benefits: 约 6 万/年

Total Package = 50 + 0 + 0 + 0 + 6 = 56 万/年
```

**对比**：Offer A 的第一年 package 更高（68 vs 56），但 Offer B 的 base 更高。

## ⚠️ 薪资谈判中的陷阱

### 陷阱 1：只谈 Base

**表现**：只看 base 多少，不关注其他部分。

**后果**：可能被 base 高但 total package 低的公司「坑」了。

**正确做法**：了解完整的 package，计算 total compensation。

### 陷阱 2：忽视税后收入

**表现**：只算税前薪资。

**后果**：实际到手可能差很多。

**正确做法**：了解税后收入。五险一金、个税会扣除约 20-30%。

### 陷阱 3：不了解归属机制

**表现**：只看 stock 总值，不看归属计划。

**后果**：实际每年到手可能很少。

**正确做法**：了解 vesting schedule 和公司的长期发展。

### 陷阱 4：sign-on 陷阱

**表现**：被 sign-on 的数字吸引。

**后果**：如果要求绑满 N 年，提前离职要退还。

**正确做法**：了解 sign-on 的发放条件和留任要求。

## 如何比较不同 Offer

### 比较维度

| 维度 | 说明 |
|------|------|
| **Total Package** | 第一年实际到手的总价值 |
| **Base/Base 增长** | base 的增长空间 |
| **股票/期权价值** | 公司发展和市值增长潜力 |
| **平台价值** | 公司/团队/业务的发展潜力 |
| **成长空间** | 级别晋升和薪资增长空间 |
| **Work-Life Balance** | 工作强度和加班情况 |
| **福利价值** | 商业险、年假等 |

### 比较工具

```
第一年 Package = Base × 月数 + Bonus + Sign-on + Stock 第一年归属 + Benefits

长期价值 = 每年 Package + 股票增值潜力 + 级别增长空间
```

## 常见问题

**Q：如何了解公司具体的薪资结构？**

A：可以通过脉脉、牛客、Glassdoor 等平台查询，也可以问猎头或内部员工。

**Q：五险一金按 base 交和按实际工资交有什么区别？**

A：按 base 交，公司成本更低，但你到手的社保公积金也少。面试时可以问清楚缴纳基数。

**Q：股票和期权的区别是什么？**

A：RSU 是直接给你股票，达到条件后归属；期权是你可以用特定价格购买股票，如果股价高于行权价你就赚钱。
