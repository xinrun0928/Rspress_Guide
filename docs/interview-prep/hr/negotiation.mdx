# 谈判话术

「薪资谈判是最后一次加薪机会。」——这句话一点都不夸张。

很多人技术面、HR 面都表现不错，却在谈判环节「怂」了：不敢开口、不敢要价、不敢还价。结果本该多拿 10-20% 的薪资，因为没谈而白白损失。

## 面试官最关心的 3 个问题

> **问题 1**：什么时候是谈薪资的最佳时机？
> **问题 2**：有哪些谈判话术可以使用？
> **问题 3**：如果 HR 不同意，应该怎么办？

## 谈判的基本原则

### 原则 1：永远不要先开口

> **让 HR 先给数字**。

HR 问你期望，说明他们对你有兴趣，想留住你。如果你先开口，就失去了主动权。

```
❌ 错误：
"我的期望是 60 万。"

✅ 正确：
"我想先了解一下这个岗位的薪资结构和预算范围，然后我可以根据情况给出一个合理的期望。"
```

### 原则 2：永远不要接受第一个 offer

> **第一个数字往往不是最好的数字**。

HR 给的第一个数字通常是「试探价」，后面还有谈判空间。

```
❌ 错误：
"好的，我接受。"

✅ 正确：
"谢谢您的 offer。这个数字我可以接受，但我想看看是否还有一些调整空间。"
```

### 原则 3：永远带着筹码

> **谈判需要筹码**。

常见筹码：

| 筹码 | 说明 |
|------|------|
| 其他 offer | 你有其他公司的 offer |
| 市场行情 | 你了解市场薪资水平 |
| 独特价值 | 你有独特的技能或经验 |
| 时机 | 你现在有工作，不急于跳槽 |

### 原则 4：永远保持双赢

> **谈判不是零和游戏**。

最好的谈判是双方都觉得赢。不要把 HR 当成敌人，而是当成合作伙伴。

```
❌ 错误：
"不给我 80 万我就不去。"

✅ 正确：
"我理解预算有限。如果我们换一种方式，比如在 sign-on bonus 或股票方面做一些调整，是否可行？"
```

## 谈判话术模板

### 场景 1：HR 问期望薪资

**话术**：

```
HR：你的期望薪资是多少？

✅ 标准话术：
"我通过市场调研了解到，P6 级别（XX 技术栈）的薪资范围大约在 40-60 万之间。
根据我的经验和市场行情，我的期望 total package 大约在 XX 万左右。
当然，我也想先了解一下这个岗位的薪资结构和预算范围，我们可以在这个基础上进一步沟通。"

✅ 保守话术：
"我想先了解一下贵司这个岗位的薪资结构和预算范围，然后我可以根据情况给出一个合理的期望。"
```

### 场景 2：HR 给出的数字低于期望

**话术**：

```
HR：我们能给到 45 万。

✅ 回应话术：
"谢谢您的 offer。45 万比我期望的低一些，我希望能到 55 万。
我知道可能有预算限制，如果我们换一种方式，比如：
- 在 sign-on bonus 方面做一些补偿？
- 或者在 base 和 bonus 的比例上做调整？
- 或者在股票期权方面增加一些？

我非常希望能够加入贵司，也希望我们能找到一个双方都满意的方案。"
```

### 场景 3：HR 说无法再调整

**话术**：

```
HR：非常抱歉，这个岗位的预算已经到了上限，我们确实给不到更高的数字。

✅ 回应话术：
"我理解预算限制。那请问：
- 未来是否有调薪的机会？一般调薪周期和幅度是怎样的？
- 除了薪资，package 里还有其他可以协商的部分吗？比如入职时间、sign-on bonus、RSU 等？
- 如果我入职后表现优秀，是否有晋升和调薪的机会？"

✅ 放弃话术（如果差距太大）：
"我理解。感谢您和团队的时间，这个数字对我目前来说可能不太合适。
如果以后有机会，希望还能和贵司合作。"
```

### 场景 4：用其他 offer 谈判

**话术**：

```
HR：我们能给到 50 万。

✅ 谈判话术：
"谢谢您的 offer。我目前也在看其他公司的机会，其中有一家给了 XX 万的 package。
我对贵司的平台更感兴趣，希望能够在 package 上有一定的竞争力。
如果贵司能调整到 XX 万，我愿意接受这个 offer 并尽快入职。"
```

### 场景 5：要求更多时间考虑

**话术**：

```
HR：请问我什么时候可以得到您的答复？

✅ 话术：
"我需要和家人商量一下，预计需要 X 天时间。
如果 deadline 比较紧，能否宽限到 XX 号？我会尽快给您答复。"

✅ 加分话术：
"我非常希望能加入贵司，但这个决定对我来说很重要。
如果方便的话，我想和您约一个时间，详细了解一下 package 的细节，然后再做决定。"
```

## ⚠️ 谈判中的常见错误

### 错误 1：急于接受

```
❌ 表现：
"好的，我接受。"

✅ 正确做法：
"感谢您的 offer。这个数字我可以接受，但我希望看看是否还有一些调整空间。"
```

### 错误 2：威胁对方

```
❌ 表现：
"不给到 XX 万，我就不去。"
"我要把你们的 offer 发到网上。"

✅ 正确做法：
"我理解可能有些困难，但我相信我们能找到双方都满意的方案。"
```

### 错误 3：透露底牌

```
❌ 表现：
"我最低能接受 40 万。"
"我现在薪资只有 30 万。"

✅ 正确做法：
"我的期望是 XX 万，这是基于市场行情和我自身价值的合理期望。"
```

### 错误 4：情绪化

```
❌ 表现：
"你们怎么给这么低！"
"我这么优秀，你们就给这点钱？"

✅ 正确做法：
保持专业和礼貌，用事实和数据说话。
```

### 错误 5：只盯着薪资

```
❌ 表现：
"薪资不涨 30% 我不考虑。"

✅ 正确做法：
"如果 base 无法调整，我们能否在其他方面协商？比如 sign-on、股票、假期等。"
```

## 💡 加分技巧

### 技巧 1：多轮谈判

```
第一轮：了解初始数字
第二轮：提出期望和理由
第三轮：协商细节（base/bonus/stock/sign-on）
```

### 技巧 2：谈判多个维度

```
如果 base 谈不动：
- ✅ 可以谈 sign-on bonus
- ✅ 可以谈 RSU
- ✅ 可以谈更多假期
- ✅ 可以谈入职时间（晚入职几天）
- ❌ 不要只盯着 base
```

### 技巧 3：展示诚意

```
✅ "我非常希望加入贵司，如果 package 能调整到 XX 万，我愿意接受并尽快入职。"
❌ "你们自己看着办吧。"
```

### 技巧 4：设置 deadline 压力

```
✅ "我目前还有其他 offer 在等，但贵司是我最想来的。如果能在 XX 号之前确认，我可以推掉其他 offer。"
```

## 常见问题

**Q：谈判会不会影响 offer？**

A：一般不会。HR 的职责之一就是谈判，他们理解候选人想要更好的 package。但要注意方式，不要过于强硬或威胁。

**Q：谈多少次合适？**

A：2-3 轮比较合适。如果谈了 3 轮还没有结果，可以考虑是否值得继续谈。

**Q：可以同时和多家公司谈吗？**

A：可以，而且应该这样做。把 offer 之间的竞争关系作为谈判筹码。

**Q：如果最后还是达不到期望怎么办？**

A：如果差距不大（比如 5-10%），可以考虑接受，因为其他因素可能弥补薪资差距。如果差距太大，可以礼貌拒绝，保持后续联系。
