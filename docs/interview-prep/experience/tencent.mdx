# 腾讯 面经汇总

腾讯是中国互联网的巨头之一，其技术面试注重基础和实战能力。本章汇总了腾讯的面经，帮你了解真实面试情况。

## 面试官最关心的 3 个问题

> **问题 1**：腾讯各事业群的面试风格有什么差异？
> **问题 2**：腾讯的面试考察重点是什么？
> **问题 3**：如何准备腾讯的技术面试？

## 面试流程总览

### 常规流程（3-5 轮）

```
简历筛选
    ↓
在线笔试（部分岗位）
    ↓
一面（技术初面，45-60分钟）
    ↓
二面（技术复面，45-60分钟）
    ↓
三面（技术终面，45-60分钟）
    ↓
HR面（30分钟）
    ↓
Offer
```

### 各事业群流程差异

| 事业群 | 流程 | 说明 |
|--------|------|------|
| **WXG（微信）** | 3-4 轮 | 流程短，节奏快 |
| **CSIG（云与智慧产业）** | 4-5 轮 | 流程较长 |
| **IEG（互动娱乐）** | 4 轮 | 偏业务 |
| **PCG（平台与内容）** | 4 轮 | 偏内容 |
| **TEG（技术工程）** | 4 轮 | 技术导向 |

## 各事业群面试风格

### WXG（微信事业群）

| 特点 | 说明 |
|------|------|
| 面试难度 | 高，竞争激烈 |
| 考察重点 | 基础扎实、系统设计、性能优化 |
| 技术栈 | C++、Go 为主，部分 Java |
| 加班强度 | 中 |

### CSIG（云与智慧产业）

| 特点 | 说明 |
|------|------|
| 面试难度 | 中高 |
| 考察重点 | 分布式系统、云原生、解决方案 |
| 技术栈 | Java、Go、云技术 |
| 加班强度 | 中高 |

### IEG（互动娱乐）

| 特点 | 说明 |
|------|------|
| 面试难度 | 中 |
| 考察重点 | 游戏后台、实时系统、稳定性 |
| 技术栈 | C++、Go、Unity |
| 加班强度 | 高 |

### TEG（技术工程）

| 特点 | 说明 |
|------|------|
| 面试难度 | 中高 |
| 考察重点 | 底层技术、基础设施 |
| 技术栈 | C++、Go、底层系统 |
| 加班强度 | 中 |

## 技术面高频考点

### 通用考点

| 分类 | 高频考点 |
|------|----------|
| **数据结构与算法** | 链表、树、动态规划、滑动窗口 |
| **网络** | TCP/UDP、HTTP、HTTPS、TLS |
| **操作系统** | 进程线程、内存管理、IO 模型 |
| **数据库** | MySQL 索引、事务、SQL 优化 |
| **缓存** | Redis 原理、持久化、分布式锁 |
| **消息队列** | Kafka/RocketMQ 原理 |
| **微服务** | 注册中心、熔断降级、网关 |

### Java 后端额外考点

| 分类 | 高频考点 |
|------|----------|
| **Java 基础** | HashMap、JUC、并发编程 |
| **JVM** | 内存模型、GC 算法、类加载 |
| **Spring** | IOC、AOP、事务 |

## 面经精选

### P6 面经（Java后端）

```
【一面】（60分钟）
- 自我介绍
- TCP 三次握手四次挥手
- TCP 和 UDP 的区别
- HTTP 和 HTTPS 的区别
- HTTPS 的加密过程
- 进程和线程的区别
- 线程池的参数和原理
- synchronized 和 ReentrantLock 的区别
- 算法题：合并两个有序数组

【二面】（60分钟）
- 项目介绍，深挖 XX 项目
- Redis 哨兵模式和集群模式的区别
- Redis 分布式锁的实现
- MySQL 事务隔离级别
- MVCC 原理
- 算法题：二叉树的层序遍历

【三面】（45分钟）
- 项目深挖
- 系统设计：设计一个秒杀系统
- 如何保证缓存一致性？
- 分布式事务的解决方案
- 算法题：LRU 缓存

【HR面】（30分钟）
- 为什么离职？
- 为什么选择腾讯？
- 期望薪资？
- 对微信有什么了解？
```

### P6 面经（Go后端 - WXG）

```
【一面】（60分钟）
- 自我介绍
- Go 的 GMP 模型
- Go 协程和线程的区别
- Go 的 channel 底层实现
- Go 的 GC 机制
- Redis 常用数据结构
- Redis 持久化机制
- 算法题：爬楼梯

【二面】（60分钟）
- 项目介绍
- 分布式 ID 生成方案
- 如何设计一个长连接系统？
- 如何保证消息可靠性？
- 算法题：二叉树最近公共祖先

【三面】（45分钟）
- 项目深挖
- 系统设计：设计一个即时通讯系统
- 架构设计能力评估
- 对微信的了解
```

## ⚠️ 常见坑点

### 技术面坑点

| 坑点 | 说明 | 应对 |
|------|------|------|
| 算法题 | 腾讯重视算法 | 重点刷高频算法题 |
| 基础不扎实 | 网络、操作系统基础 | 系统复习 |
| 项目深度不够 | 被追问项目细节答不上来 | 深挖项目 |

### HR面坑点

| 坑点 | 说明 | 应对 |
|------|------|------|
| 稳定性问题 | 腾讯重视稳定性 | 准备合理的离职理由 |
| 对腾讯不了解 | 显得没有诚意 | 提前了解业务 |

## 常见问题

**Q：腾讯面试难度如何？**

A：微信事业群难度最高，其他事业群难度中等偏上。

**Q：腾讯对学历有要求吗？**

A：通常本科及以上，但不是硬性要求。

**Q：腾讯面试流程大概多久？**

A：微信事业群通常 1-2 周，其他事业群通常 2-4 周。
