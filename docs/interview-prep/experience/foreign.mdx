# 外企面试特点

外企（Google、Meta、Amazon、Microsoft 等）是很多程序员的dream company，但它们的面试风格和国内大厂有显著差异。

## 面试官最关心的 3 个问题

> **问题 1**：外企面试和国内大厂有什么不同？
> **问题 2**：如何准备外企的技术面试？
> **问题 3**：英语在面试中有多重要？

## 外企面试特点

### 主要差异

| 维度 | 国内大厂 | 外企 |
|------|----------|------|
| **语言** | 中文 | 英文为主 |
| **算法难度** | 中高 | 高 |
| **系统设计** | 偏业务 | 偏架构 |
| **BQ 面试** | 较少 | 必考 |
| **薪资结构** | Base 为主 | Base + RSU + Sign-on |
| **工作文化** | 加班文化 | WLB |

### 外企类型

| 类型 | 示例 | 特点 |
|------|------|------|
| **FLAG** | Google、Meta、Apple | 难度最高 |
| **其他美国大厂** | Amazon、Microsoft | 难度中高 |
| **欧洲大厂** | SAP、ABB | 相对容易 |
| **日韩外企** | 三星、索尼 | 偏稳定 |

## 技术面准备

### 算法（最重要）

| 难度 | 推荐资源 |
|------|----------|
| 入门 | LeetCode Easy |
| 中等 | LeetCode Medium |
| 困难 | LeetCode Hard |

**高频类型**：

- 数组、字符串
- 链表、树
- 动态规划
- 回溯
- 滑动窗口
- DFS/BFS

### 系统设计

| 题目类型 | 考察点 |
|----------|--------|
| 设计短链系统 | 存储、编码、路由 |
| 设计 Twitter | Feed、搜索、通知 |
| 设计聊天系统 | 长连接、消息存储 |
| 设计限流器 | 令牌桶、滑动窗口 |

### 计算机基础

| 分类 | 高频考点 |
|------|----------|
| 操作系统 | 进程线程、内存管理、IO 模型 |
| 计算机网络 | TCP/UDP、HTTP、HTTPS |
| 数据库 | 索引、事务、SQL 优化 |
| 并发 | 多线程、锁、并发容器 |

## BQ 面试准备

### 核心框架

外企非常重视 BQ 面试，常用框架：

| 框架 | 说明 |
|------|------|
| **STAR** | Situation-Task-Action-Result |
| **SAR** | Situation-Action-Result |
| **YODY** | Yourstory, Opportunity, Drive, Yield |

### 常见 BQ 题

| 类型 | 示例 |
|------|------|
| Leadership | 主导的项目、解决冲突 |
| 团队协作 | 跨团队协作、培养新人 |
| 问题解决 | 技术挑战、线上故障 |
| 成长学习 | 学习新技能、克服困难 |

### Amazon 特殊要求

Amazon 使用 16 条 Leadership Principles，几乎每轮面试都会考察：

```
1. Customer Obsession
2. Ownership
3. Invent and Simplify
4. Are Right, A Lot
5. Learn and Be Curious
6. Hire and Develop the Best
7. Insist on Highest Standards
8. Bias for Action
9. Frugality
10. Earn Trust
11. Dive Deep
12. Have Backbone, Disagree and Commit
13. Deliver Results
14. Strive to be Earth's Best Employer
15. Success and Scale Bring Broad Responsibility
```

## 英语面试准备

### 重要性

| 场景 | 英语重要程度 |
|------|-------------|
| 简历 | 必须英文 |
| 电话面 | 高 |
| 技术面 | 中（算法题可以英文） |
| BQ 面 | 高 |
| 现场面 | 高 |

### 准备方法

| 方法 | 说明 |
|------|------|
| 刷题用英文 | 熟悉英文术语 |
| BQ 答案准备中英两份 | 先用中文写，再翻译 |
| 模拟面试 | 用英语练习回答 |
| 技术词汇 | 熟悉常见技术的英文说法 |

### 技术词汇

| 中文 | 英文 |
|------|------|
| 数组 | Array |
| 链表 | Linked List |
| 树 | Tree |
| 图 | Graph |
| 哈希表 | Hash Table |
| 动态规划 | Dynamic Programming |
| 递归 | Recursion |
| 迭代 | Iteration |

## 薪资结构

### 外企薪资构成

```
Total Package = Base + Bonus + RSU/Stock + Sign-on
```

| 部分 | 说明 |
|------|------|
| **Base** | 按月发放，税前 |
| **Bonus** | 年终奖，一般 10-20% |
| **RSU** | 限制性股票，4 年归属 |
| **Sign-on** | 入职奖金，一次性 |

### 典型 Package（美国大厂 L3/L4）

```
Base: $150,000 - $200,000
Bonus: $15,000 - $30,000/年
RSU: $100,000 - $200,000（4 年）
Sign-on: $10,000 - $50,000

Total Package: $250,000 - $400,000/年
```

## 常见问题

**Q：英语不好能进外企吗？**

A：可以，但需要达到基本沟通水平。技术岗对英语要求相对较低，但 BQ 面需要能用英语表达。

**Q：外企面试难度如何？**

A：FLAG 难度最高，其他美国大厂难度中高，欧洲大厂难度中等。

**Q：外企薪资如何？**

A：通常比国内大厂高，但要注意税后实际收入。
