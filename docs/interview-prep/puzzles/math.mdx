## 数学题类型

### 类型 1：基础运算

包括求和、数列、排列组合等。

**示例**：1 + 2 + ... + 100 = ?

```
方法 1：等差数列求和公式
S = n × (a1 + an) / 2 = 100 × (1 + 100) / 2 = 5050

方法 2：配对法
1 + 100 = 101
2 + 99 = 101
...
50 + 51 = 101
共 50 对，和 = 50 × 101 = 5050
```

### 类型 2：进制转换

**示例**：如何判断一个数是不是 2 的幂次？

```
方法 1：循环除 2
while (n % 2 == 0) n /= 2;
return n == 1;

方法 2：位运算（最优）
n > 0 && (n & (n-1)) == 0

原理：
- 如果 n 是 2 的幂次，如 8 = 1000
- n-1 = 7 = 0111
- n & (n-1) = 0
- 如果 n 不是 2 的幂次，如 6 = 0110
- n-1 = 5 = 0101
- n & (n-1) = 0100 ≠ 0
```

### 类型 3：概率与期望

见「概率题」章节。

### 类型 4：逻辑数学

涉及推理和证明。

**示例**：证明：任意 6 个人的聚会中，一定有 3 个人互相认识或 3 个人互相不认识。

**证明**：

```
这是经典的 Ramsey 定理：R(3,3) = 6

考虑其中一个人 A，他与其他 5 个人的关系只能是「认识」或「不认识」。

根据抽屉原理，5 个人中至少有 3 个人与 A 的关系相同（认识或不认识）。

假设有 B、C、D 三个人与 A 的关系是「认识」。

如果 B、C、D 中存在两个人互相认识，如 B 认识 C，则 A、B、C 三人互相认识。
如果 B、C、D 中不存在两人互相认识，则 B、C、D 三人互相不认识。

因此，无论如何都存在 3 个人互相认识或 3 个人互相不认识。
```

## 经典例题精讲

### 🔴 高频题 1：水壶问题

**题目**：有一个 7 升的水壶和一个 3 升的水壶，如何得到恰好 5 升的水？

**分析**：

```
步骤：
1. 装满 7 升壶：7L 壶 = 7，3L 壶 = 0
2. 从 7L 壶倒入 3L 壶：7L 壶 = 4，3L 壶 = 3
3. 倒掉 3L 壶：7L 壶 = 4，3L 壶 = 0
4. 从 7L 壶倒入 3L 壶：7L 壶 = 1，3L 壶 = 3
5. 倒掉 3L 壶：7L 壶 = 1，3L 壶 = 0
6. 从 7L 壶倒入 3L 壶：7L 壶 = 0，3L 壶 = 1
7. 装满 7L 壶：7L 壶 = 7，3L 壶 = 1
8. 从 7L 壶倒入 3L 壶（只能倒 2 升）：7L 壶 = 5，3L 壶 = 3

得到 5 升水！
```

**答案**：见上述步骤。

### 🔴 高频题 2：赛金条问题

**题目**：你让工人为你工作 7 天，回报是一根金条。你只能将金条切两次，请问如何给工人付费？

**分析**：

```
最少需要切 2 次。

最优切法：切成 1/7、2/7、4/7 三段。

第一天：给 1/7
第二天：给 2/7，换回 1/7
第三天：给 1/7（现在总共给了 3/7）
第四天：给 4/7，换回 1/7 和 2/7（现在总共给了 4/7）
第五天：给 1/7（总共给了 5/7）
第六天：给 2/7，换回 1/7（总共给了 6/7）
第七天：给 1/7（总共给了 7/7）
```

**答案**：切成 1/7、2/7、4/7 三段，按上述方式支付。

### 🟡 中频题 3：蚂蚁相遇问题

**题目**：一排 10 个木块排成一列，相邻木块间隔 1 米。一只蚂蚁在 5 号木块上，速度 1 米/秒，相邻的 4 号和 6 号木块上各有一只蚂蚁，速度也是 1 米/秒，方向朝向 5 号蚂蚁。如果两只蚂蚁相遇，它们会掉头。请问 5 号蚂蚁多久后掉下木块？

**分析**：

```
关键洞察：蚂蚁相遇后掉头，等价于它们「穿过」彼此。

从相对运动的角度考虑：
- 三只蚂蚁的速度都是 1 米/秒
- 5 号蚂蚁距离左端 4 米，距离右端 5 米
- 4 号蚂蚁向左走，6 号蚂蚁向右走

等效情况：
- 5 号蚂蚁相当于在向两端走的蚂蚁中随机选择方向
- 每只蚂蚁独立行动

5 号蚂蚁掉下木块的概率：
- 向左：需要走 4 米，速度 1 米/秒，需要 4 秒
- 向右：需要走 5 米，速度 1 米/秒，需要 5 秒

但两只蚂蚁相遇会掉头，这个情况比较复杂。

更简单的思考：
- 把两只蚂蚁看作「穿过」彼此
- 5 号蚂蚁最终会从某一端掉下
- 从左端掉下的时间 = 4 秒
- 从右端掉下的时间 = 5 秒
- 但掉下的方向是随机的

实际上，蚂蚁相遇掉头的问题，可以转化为「它们穿过彼此继续前进」。
所以 5 号蚂蚁从左端掉下的概率 = 4/9（左侧有 4 米距离）
从右端掉下的概率 = 5/9（右侧有 5 米距离）

期望时间 = 4 × (4/9) + 5 × (5/9) = 16/9 + 25/9 = 41/9 ≈ 4.56 秒

但更简单的方法是：5 号蚂蚁速度是 1 米/秒，它距离左端 4 米，距离右端 5 米。
等效于有一只蚂蚁从左端出发，需要走 4 米；有一只从右端出发，需要走 5 米。
5 号蚂蚁等效于这两只蚂蚁中的一只。
所以期望时间是 41/9 秒。
```

**答案**：期望时间约 4.56 秒。

### 🟢 低频题 4：圆周率估算

**题目**：如何用蒙特卡洛方法估算圆周率？

**分析**：

```
蒙特卡洛方法基于概率和几何：

1. 在边长为 2 的正方形中画一个内接圆
   - 正方形面积 = 4
   - 圆的面积 = π × r² = π × 1² = π

2. 向正方形内随机投点
   - 点落在圆内的概率 = 圆面积 / 正方形面积 = π/4

3. 投 N 个点，落在圆内的有 M 个
   - M/N ≈ π/4
   - π ≈ 4M/N
```

**答案**：π ≈ 4 × (圆内点数) / (总点数)。

## ⚠️ 常见陷阱

### 陷阱 1：硬算而不找规律

**错误**：计算 1+2+...+100 用循环硬算。

**正确**：用等差数列公式。

### 陷阱 2：忽视边界条件

**示例**：判断一个数是否为 2 的幂次时，忘记 0 和负数的情况。

**正确**：n > 0 && (n & (n-1)) == 0

### 陷阱 3：复杂化简单问题

**示例**：用水壶量水时，想得太复杂。

**正确**：一步一步操作，不要想太多。

## 解题方法论

### 方法 1：找规律

很多数学问题有隐藏的规律，找到规律就能快速解决。

```
1+2+...+n → 等差数列公式
1²+2²+...+n² → 立方公式
2ⁿ → 位运算技巧
```

### 方法 2：逆向思维

从结果倒推过程。

```
水壶问题：目标 5 升
→ 上一步是什么？需要 2 升 + 3 升，或 5 升 + 0 升
→ 继续倒推...
```

### 方法 3：等效转换

把复杂问题转化为简单问题。

```
蚂蚁相遇掉头 → 等效于蚂蚁穿过彼此
```

### 方法 4：举例子

从简单情况开始找规律。

```
n = 1, 2, 3, ... 逐步分析
```

## 常见问题

**Q：数学题需要很强的数学基础吗？**

A：大多数面试数学题只需要初高中数学水平。关键是思维方式和找规律的能力。

**Q：如何提高数学思维？**

A：多练习，积累常见题型的解法。

**Q：面试中的数学题和算法有什么关系？**

A：很多算法题本质上是数学问题，如位运算、概率、排列组合等。
