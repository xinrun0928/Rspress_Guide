# 博弈题

「两个人轮流报数，每次报 1-3 个，谁报到 100 谁赢，有什么必胜策略？」「两瓶毒酒，如何用最少的测试找出哪瓶有毒？」——博弈题考察的不是数学计算，而是**逻辑推理和策略思维**。

## 面试官最关心的 3 个问题

> **问题 1**：博弈题有哪些常见的类型和解题套路？
> **问题 2**：如何找到博弈中的「必胜策略」？
> **问题 3**：为什么「囚徒困境」是经典的博弈论案例？

## 博弈论基础

### 基本概念

| 概念 | 定义 | 示例 |
|------|------|------|
| **博弈** | 多方决策相互影响的情境 | 报数游戏、囚徒困境 |
| **策略** | 博弈中可选择的行动方案 | 先手策略、后手策略 |
| **收益** | 博弈结束后各方获得的结果 | 赢/输、得/失 |
| **均衡** | 各方都满意的最优策略组合 | Nash 均衡 |

### 博弈类型

| 类型 | 特点 | 示例 |
|------|------|------|
| **零和博弈** | 一方所得 = 另一方所失 | 报数游戏、棋类 |
| **非零和博弈** | 可以双赢或多输 | 囚徒困境、贸易战 |
| **完全信息** | 所有玩家知道全部信息 | 报数游戏 |
| **不完全信息** | 部分信息未知 | 石头剪刀布 |

## 经典例题精讲

### 🔴 高频题 1：报数必胜策略

**题目**：两个人轮流报数，每次报 1-3 个数（不能不报），报到 100 的人获胜。请问先报的人有没有必胜策略？

**分析**：

```
解题关键：找到「关键数」

报数规律：
- 报到 100 赢
- 报到 99 后，对方只能报 100（因为每次最多报 3 个）
- 所以报到 96 就能保证报到 99

依次类推，关键数序列：
100, 96, 92, 88, ..., 4

只要先手报到 4，就能依次报到 8, 12, 16, ..., 100

必胜策略：
先手报到 4，然后每次报到「4 的倍数 + 4」的位置
```

**答案**：先手必胜，策略是报到 4，然后报到 8, 12, 16, ..., 100。

**追问**：如果每次报 1-4 个数呢？

```
关键数：100, 95, 90, 85, ..., 5
先手报到 5，然后报到 10, 15, ..., 100
```

### 🔴 高频题 2：天平称重问题

**题目**：有 12 个球，其中一个是次品（重量与其他球不同，但不知道是轻还是重）。用天平称重 3 次，如何找出次品球？

**分析**：

```
第一步：将球分成三组：A(1,2,3,4)、B(5,6,7,8)、C(9,10,11,12)

称第一次：A vs B

情况 1：A = B
→ 次品在 C 组
→ 称第二次：C vs A（假设 A 是标准球）
  - 如果 C > A，次品在 C 且是重的，取 C 中的球两两称
  - 如果 C < A，次品在 C 且是轻的，取 C 中的球两两称
→ 称第三次：确定次品球

情况 2：A > B
→ 次品在 A 或 B，但不知道是轻是重
→ 称第二次：将 A(1,2,3)、B(4,5,6) 称
  - 如果 A > B，次品在 1,2,3 中且是重的，或在 4 中且是轻的
  - 称第三次：1 vs 2...
  - 如果 A < B，次品在 4,5,6 中且是轻的，或在 1 中且是重的
  - 称第三次：4 vs 5...

情况 3：A < B（分析与情况 2 类似）
```

**答案**：可以通过三次称重找出次品球。

### 🟡 中频题 3：海盗分金问题

**题目**：5 个海盗抢得 100 枚金币，他们约定按顺序提出分配方案，如果超过一半的人反对，则提出方案的人被扔进海喂鲨鱼。第一个海盗如何分配才能让自己不被扔下海？

**分析**：

```
逆向推理：

第 5 个海盗：
- 如果只剩他和第 4 个海盗，无论第 4 个怎么分，他都会反对让第 4 个喂鱼
- 所以第 4 个海盗必须让第 5 个支持自己

第 4 个海盗：
- 他知道第 5 个海盗一定会反对
- 所以他只需要给自己分 100 枚，第 5 个分 0 枚

第 3 个海盗：
- 他知道如果自己死了，第 4 个会得到 100 枚，第 5 个会得到 0 枚
- 所以他需要争取第 5 个的支持
- 分配方案：自己 99 枚，第 4 个 0 枚，第 5 个 1 枚
  → 第 5 个得到 1 枚 > 0 枚，支持第 3 个
  → 第 4 个反对，但没超过一半

第 2 个海盗：
- 如果他死了，第 3 个会得到 99 枚，第 4 个 0 枚，第 5 个 1 枚
- 他需要争取第 4 个和第 5 个的支持
- 分配方案：自己 99 枚，第 3 个 0 枚，第 4 个 1 枚，第 5 个 0 枚
  → 第 4 个得到 1 枚 > 0 枚，支持第 2 个
  → 第 5 个得到 0 枚 = 0 枚，可能反对

第 1 个海盗：
- 如果他死了，第 2 个会得到 99 枚，第 3 个 0 枚，第 4 个 1 枚，第 5 个 0 枚
- 他需要争取第 3 个、第 4 个和第 5 个中的任意两个
- 分配方案：自己 98 枚，第 2 个 0 枚，第 3 个 1 枚，第 4 个 0 枚，第 5 个 1 枚
  → 第 3 个得到 1 枚 > 0 枚
  → 第 5 个得到 1 枚 > 0 枚
```

**答案**：第一个海盗的分配方案是：自己 98 枚，第 2 个 0 枚，第 3 个 1 枚，第 4 个 0 枚，第 5 个 1 枚。

### 🟡 中频题 4：囚徒困境

**题目**：两个嫌疑犯被隔离审讯，如果两人都坦白，各判 5 年；如果一人坦白一人沉默，坦白者 2 年、沉默者 10 年；如果两人都沉默，各判 1 年。请问最终结果是什么？

**分析**：

```
收益矩阵（-表示年数，越大越好）：

              囚犯 B
              坦白    沉默
囚犯 A 坦白   -5,-5   -10,0
      沉默   0,-10   -1,-1

分析囚犯 A：
- 如果 B 坦白，A 坦白得 -5 年 > 沉默得 -10 年 → 坦白
- 如果 B 沉默，A 坦白得 -2 年 > 沉默得 -1 年 → 坦白

无论 B 怎么选，A 的最优策略都是坦白。
同样，B 的最优策略也是坦白。

所以最终结果是：两人都坦白，各判 5 年。

但如果两人都选择沉默，只需要各判 1 年。
这就是「囚徒困境」——个体最优导致集体非最优。
```

**答案**：两人都坦白，各判 5 年。这是 Nash 均衡，但不是集体最优解。

## ⚠️ 常见陷阱

### 陷阱 1：只看眼前

**错误**：报数游戏中，只考虑下一步，不考虑整体策略。

**正确**：从最终目标倒推，找到「必争之地」（关键数）。

### 陷阱 2：忽视对手的理性选择

**错误**：假设对手会做出非理性选择。

**正确**：假设所有参与者都是理性的，都会选择最优策略。

### 陷阱 3：混淆零和与非零和

**错误**：在囚徒困境中，用零和博弈的思路分析。

**正确**：囚徒困境是非零和博弈，存在「个体理性导致集体非理性」的情况。

## 解题方法论

### 方法 1：逆向推理法

从最终状态倒推，找到每个阶段的「最优选择」。

```
步骤：
1. 确定最终目标（如报到 100 赢）
2. 确定「关键位置」（如 100, 96, 92...）
3. 逆向推导每一步应该如何选择
```

### 方法 2：递归法

将复杂问题分解为子问题，通过递归求解。

```
适用场景：海盗分金、博弈树等

核心思想：
- 如果当前状态是「胜利状态」，我赢
- 如果当前状态能转到「胜利状态」，我也能赢
- 如果所有可能状态都是「失败状态」，我输
```

### 方法 3：博弈树法

画出所有可能的决策路径，分析每一步。

```
适用场景：棋类游戏、复杂决策

步骤：
1. 画出决策树
2. 评估叶子节点的收益
3. 从叶子节点向根节点回溯，选择最优分支
```

## 常见问题

**Q：博弈题需要数学基础吗？**

A：大多数博弈题只需要基本的数学运算和逻辑推理，不需要高等数学。关键是找到规律和策略。

**Q：如何练习博弈题？**

A：可以从经典的报数游戏、天平称重问题开始，理解核心思路后再扩展。

**Q：面试中遇到不会的博弈题怎么办？**

A：展示思考过程。从最简单的情况开始分析（如只有 1 个、2 个的情况），然后逐步扩展。
