# 概率题

「两个人同时抛硬币，正面朝上的概率相同吗？」「一封信投入邮箱，邮递员取信的概率是多少？」——概率题是面试中让无数候选人头疼的题型。

但概率题其实有「套路」可循。本章将系统讲解概率题的解题方法，配合经典例题，帮助你从容应对。

## 面试官最关心的 3 个问题

> **问题 1**：概率题有哪些常见的解题思路？
> **问题 2**：如何避免概率题中的「直觉陷阱」？
> **问题 3**：面试中遇到不会的概率题应该怎么办？

## 概率论基础回顾

### 核心概念

| 概念 | 定义 | 示例 |
|------|------|------|
| **概率** | 事件发生的可能性，用 0-1 之间的数表示 | 抛硬币正面朝上的概率是 0.5 |
| **独立事件** | 事件 A 的发生不影响事件 B 的概率 | 两次抛硬币相互独立 |
| **条件概率** | 在事件 B 发生的条件下，事件 A 发生的概率 | P(A\|B) |
| **期望值** | 随机变量取值的加权平均 | 掷骰子的期望值是 3.5 |

### 核心公式

| 公式 | 适用场景 |
|------|----------|
| `P(A or B) = P(A) + P(B) - P(A and B)` | 或事件的概率 |
| `P(A and B) = P(A) × P(B)` | 且事件的概率（独立事件） |
| `P(A\|B) = P(A and B) / P(B)` | 条件概率 |
| `E(X) = Σ x × P(x)` | 期望值计算 |

## 解题方法论

### 方法 1：枚举法

当样本空间较小时，枚举所有可能情况。

**适用场景**：n 较小（如硬币、骰子问题）

**示例**：抛两枚硬币，至少有一个正面的概率？

```
样本空间（4 种可能）：
- 正正
- 正反
- 反正
- 反反

至少一个正面的情况（3 种）：正正、正反、反正
概率 = 3/4
```

### 方法 2：对立事件法

求「至少有一个」的的概率时，先求「一个都没有」的概率，然后用 1 减去。

**适用场景**：「至少有一个」「至少有一次」等问题

**示例**：抛 10 枚硬币，至少有一个正面的概率？

```
「至少有一个正面」的对立事件是「全部都是反面」
P(全部反面) = (1/2)^10
P(至少有一个正面) = 1 - (1/2)^10
```

### 方法 3：条件概率法

使用条件概率公式 `P(A|B) = P(A and B) / P(B)`

**适用场景**：涉及「已知 XX 发生」的问题

**示例**：生两个孩子，已知至少有一个是男孩，求两个都是男孩的概率？

```
设 A = 两个都是男孩
设 B = 至少有一个男孩

样本空间：男男、男女、男男（第一个男第二个女/第一个女第二个男，但题目没有区分顺序）、女女
但更准确的样本空间（考虑顺序）：
- 男男
- 男女
- 男男（重复了）

重新考虑：不区分顺序
样本空间（4 种）：男男、男女、女男、女女
P(B) = 3/4（除了女女都是至少有一个男孩）
P(A and B) = P(男男) = 1/4
P(A|B) = (1/4) / (3/4) = 1/3
```

### 方法 4：期望值法

通过期望值简化问题。

**适用场景**：求平均次数、平均时间等问题

**示例**：抛硬币直到出现正面，平均需要抛几次？

```
设平均需要抛 n 次

第一种情况（概率 1/2）：第一次就抛正面，只需要 1 次
第二种情况（概率 1/2）：第一次抛反面，还需要再抛 n 次

E = 1/2 × 1 + 1/2 × (1 + n)
E = 1/2 + 1/2 + 1/2 × n
E = 1 + 1/2 × n
E - 1/2 × n = 1
1/2 × n = 1
n = 2

所以平均需要抛 2 次
```

## 经典例题精讲

### 🔴 高频题 1：三门问题

**题目**：你参加一个游戏，有三扇门，其中一扇门后面有汽车，另外两扇门后面是山羊。你选择一扇门，主持人（知道每扇门后面是什么）会打开另外一扇有山羊的门。现在给你一个选择：坚持原来的选择，还是换另一扇门？换与不换的概率分别是多少？

**分析**：

```
初始选择：选中汽车的概率 = 1/3，选中山羊的概率 = 2/3

如果坚持选择：
- 选中汽车 → 坚持后赢
- 选中山羊 → 坚持后输
- 坚持赢的概率 = 1/3

如果换选择：
- 初始选中汽车 → 换后输（2/3 的门被打开，你只能换到山羊门）
- 初始选中山羊 → 换后赢（主持人打开另一只山羊，你换到汽车）
- 换后赢的概率 = 2/3
```

**答案**：换选择的胜率是 2/3，不换的胜率是 1/3。**应该换！**

### 🔴 高频题 2：生日悖论

**题目**：一个房间需要有多少人，才能保证至少两个人的生日是同一天的概率大于 50%？

**分析**：

```
设 n 个人，至少有一对生日相同的概率 = 1 - P(所有人的生日都不同)

P(所有人的生日都不同) = 365/365 × 364/365 × ... × (365-n+1)/365

当 n = 23 时：
P(不同) = 365/365 × 364/365 × ... × 343/365 ≈ 0.492
P(相同) = 1 - 0.492 ≈ 0.508 > 50%
```

**答案**：只需要 23 个人！

**延伸追问**：

> 100 人的房间，至少一对生日相同的概率是多少？

```
P(不同) = 365/365 × 364/365 × ... × 266/365 ≈ 0.0000003
P(相同) ≈ 99.99997%
```

### 🟡 中频题 3：两个骰子问题

**题目**：同时抛两个骰子，已知第一个骰子是 4 点，两个骰子之和大于 8 的概率是多少？

**分析**：

```
已知第一个骰子是 4，第二个骰子可能是 1-6 中的任意一个

需要满足：4 + 第二个骰子 > 8
即：第二个骰子 > 4
所以第二个骰子可以是 5 或 6

满足条件的可能：2 种（4,5）和（4,6）
总可能：6 种

概率 = 2/6 = 1/3
```

### 🟡 中频题 4：随机事件期望

**题目**：你参加一个游戏，抛硬币正面朝上得 1 分，继续抛；反面朝上不得分，结束游戏。请问你期望得到多少分？

**分析**：

```
设期望值为 E

第一次抛硬币：
- 得到 1 分的概率 = 1/2
- 得到 1 分后重新开始，期望还是 E

E = 1/2 × (1 + E) + 1/2 × 0
E = 1/2 + 1/2 × E
E - 1/2 × E = 1/2
1/2 × E = 1/2
E = 1
```

**答案**：期望得到 1 分。

### 🟢 低频题 5：帽子问题

**题目**：有 n 个人参加宴会，每个人都有一顶帽子。把所有帽子收集起来，随机发放。没有人拿到自己帽子的概率是多少？

**分析**：

```
这是经典的「错排问题」

当 n = n 时，错排数 D(n) = n! × Σ(k=0 to n) (-1)^k / k!

D(n) / n! = Σ(k=0 to n) (-1)^k / k!

当 n 很大时，这个值趋近于 1/e ≈ 0.3679

所以 10 个人中，至少一个人拿到自己帽子的概率 = 1 - D(10)/10! ≈ 0.6321
```

**答案**：没人拿对帽子的概率约为 36.8%。

## ⚠️ 常见陷阱

### 陷阱 1：混淆独立事件

**错误**：「抛两次硬币都是正面的概率是 1/2 + 1/2 = 1」

**正确**：`P(两次都是正面) = P(第一次正面) × P(第二次正面) = 1/2 × 1/2 = 1/4`

### 陷阱 2：忽视条件

**错误**：「抛两枚硬币，至少有一枚正面的概率是 2/3」

**正确**：

```
样本空间：正正、正反、反正、反反
至少一枚正面：正正、正反、反正（3 种）
概率 = 3/4

或者用对立事件：
P(至少一枚正面) = 1 - P(两枚都是反面) = 1 - 1/4 = 3/4
```

### 陷阱 3：条件概率误用

**错误**：「生两个孩子，至少有一个是男孩，两个都是男孩的概率是 1/2」

**正确**：P(两个都是男孩 | 至少有一个男孩) = 1/3

### 陷阱 4：直觉偏差

**问题**：三门问题中，很多人的直觉是换与不换概率一样（1/2）。

**正确解释**：主持人打开门的行为提供了信息。换门胜率 2/3，不换胜率 1/3。

## 常见问题

**Q：面试中遇到完全没见过的概率题怎么办？**

A：先分析题目条件，列出已知信息，然后尝试用枚举法（如果可能）、对立事件法、条件概率法等方法。不要慌，展示你的思考过程比得到正确答案更重要。

**Q：概率题需要大量计算吗？**

A：大多数面试概率题不需要精确计算到小数点后很多位，通常用分数或简单小数（如 0.5、1/3）表示即可。

**Q：如何练习概率题？**

A：刷 LeetCode 的概率相关题目，理解常见题型的解法。
