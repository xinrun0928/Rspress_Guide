# Java 基础

你知道面试官为什么最爱问 `new String("abc")` 创建了几个对象吗？

这个问题看似简单，却能延伸出字符串常量池、intern() 方法、堆与常量池的关系——一道题考完 Java 基础的半壁江山。

Java 基础不是「Hello World」的堆砌，它是理解 JVM、框架、设计模式的根基。从数据类型到面向对象，从 String 源码到异常体系——这一模块，把你从「会用 Java」带到「懂 Java」。

## 学习路线建议

```
第一阶段：数据类型 → 面向对象 → String（基础三件套）
         ↓
第二阶段：基本类型与包装类 → 异常体系（细节深化）
         ↓
第三阶段：枚举与注解 → 泛型 → 序列化（新特性串联）
```

## 面试核心公式

```
String 不可变 = final + char[]/byte[] + 安全性
         ↓
new String("x") = 1 或 2 个对象（常量池是关键）
         ↓
Integer 缓存 = -128 ~ 127（超出范围创建新对象）
         ↓
自动装箱陷阱 = Integer == Integer（可能翻车）
```
