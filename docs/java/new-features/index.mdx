# Java 17+ 新特性

JDK 21 已经发布，虚拟线程（Virtual Thread）终于不再预览——Java 并发编程即将迎来革命。

但你知道吗？Java 16 引入的 Record 类型、Java 17 的 Sealed Class、Java 21 的 Pattern Matching for switch——这些新特性正在让 Java 变得越来越像一门现代化的语言。

从 Project Loom 的虚拟线程，到 Pattern Matching 的语法增强，到 Record 和 Sealed Class 的类型系统升级——Java 正在变得越来越现代、越来越简洁。

---

## 模块内容

### 版本演进总览

快速了解每个版本的核心变化。

- [JDK 9~11 新特性](/java/new-features/jdk9-11)：模块化系统、var 推断、集合工厂方法、Stream 增强、HTTP Client
- [JDK 12~16 新特性](/java/new-features/jdk12-16)：Switch 表达式、Records、Sealed Classes 预览版、instanceof 模式匹配
- [JDK 17~21 LTS 新特性](/java/new-features/jdk17-21)：Sealed Classes 正式版、Virtual Thread（Loom）、Pattern Matching for switch

每一节都是独立的深度专题。

## 新特性学习公式

```
JDK 8 → JDK 17 必学 = Lambda → Switch 表达式 → Record → Sealed Class
         ↓
JDK 21 重点 = Virtual Thread（并发革命）+ Pattern Matching（语法简化）
         ↓
新项目选型 = JDK 21 LTS（最新稳定）/ JDK 17 LTS（旧项目迁移）
         ↓
面试准备 = JDK 8（基础）+ JDK 17+（进阶亮点）
```

## 新特性学习优先级

| 优先级 | 特性 | 原因 |
|---|---|---|
| ⭐⭐⭐ 必学 | 虚拟线程 | JDK 21 LTS，革新并发编程范式 |

| ⭐ 可选 | JDK 12~16 预览特性 | 为 JDK 17+ 打基础 |
