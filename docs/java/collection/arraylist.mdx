## 一、核心问题

### 🔴 ArrayList 的底层是什么？

ArrayList 的底层是 **动态数组**，使用 `Object[] elementData` 存储元素。

```java title="ArrayList.java"
public class ArrayList<E> extends AbstractList<E>
        implements List<E>, RandomAccess, Cloneable, Serializable {
    
    // 存储元素的数组
    transient Object[] elementData;
    
    // 元素数量
    private int size;
    
    // 默认容量
    private static final int DEFAULT_CAPACITY = 10;
    
    // 空数组（共享）
    private static final Object[] EMPTY_ELEMENTDATA = {};
    
    // 默认空数组（共享）
    private static final Object[] DEFAULTCAPACITY_EMPTY_ELEMENTDATA = {};
}
```

### 数组 vs 链表

| 特性 | 数组 | 链表 |
|------|------|------|
| **存储方式** | 连续内存 | 分散内存 |
| **查询** | O(1) | O(n) |
| **插入/删除** | O(n)（需移动元素） | O(1)（只需修改指针） |
| **内存开销** | 小 | 大（需要存储指针） |
| **CPU 缓存** | 友好（连续） | 不友好（跳跃） |

## 二、扩容机制

### 🔴 ArrayList 是如何扩容的？

ArrayList 在添加元素时，如果数组已满，会触发扩容。扩容规则是**容量变为原来的 1.5 倍**（JDK 8）。

### 扩容源码

```java title="ArrayList.java"
public boolean add(E e) {
    // 确保容量足够
    ensureCapacityInternal(size + 1);
    elementData[size++] = e;
    return true;
}

private void ensureCapacityInternal(int minCapacity) {
    // 如果是默认空数组，使用默认容量 10
    if (elementData == DEFAULTCAPACITY_EMPTY_ELEMENTDATA) {
        minCapacity = Math.max(DEFAULT_CAPACITY, minCapacity);
    }
    
    ensureExplicitCapacity(minCapacity);
}

private void ensureExplicitCapacity(int minCapacity) {
    modCount++;  // 修改计数
    
    // 需要扩容
    if (minCapacity - elementData.length > 0)
        grow(minCapacity);
}

// 核心扩容方法
private void grow(int minCapacity) {
    int oldCapacity = elementData.length;
    
    // JDK 8: newCapacity = oldCapacity + oldCapacity / 2
    // 即 newCapacity = oldCapacity * 1.5
    int newCapacity = oldCapacity + (oldCapacity >> 1);
    
    // 如果 1.5 倍还不够，直接使用 minCapacity
    if (newCapacity - minCapacity < 0)
        newCapacity = minCapacity;
    
    // 防止溢出
    if (newCapacity - MAX_ARRAY_SIZE > 0)
        newCapacity = hugeCapacity(minCapacity);
    
    // Arrays.copyOf 拷贝数据
    elementData = Arrays.copyOf(elementData, newCapacity);
}
```

### 扩容时机

| 操作 | 是否触发扩容 |
|------|-------------|
| `add(E e)` | 如果 size + 1 > length |
| `add(int index, E e)` | 如果插入位置后有元素需要移动 |
| `addAll(Collection)` | 如果 totalSize > length |

### trimToSize

```java
// 收缩到实际大小，节省内存
public void trimToSize() {
    modCount++;
    if (size < elementData.length) {
        elementData = Arrays.copyOf(elementData, size);
    }
}
```

## 三、核心方法源码

### add 方法

```java
public void add(int index, E element) {
    // 边界检查
    rangeCheckForAdd(index);
    
    ensureCapacityInternal(size + 1);
    
    // 将 index 及之后的元素后移一位
    System.arraycopy(elementData, index,
                      elementData, index + 1,
                      size - index);
    
    elementData[index] = element;
    size++;
}

private void rangeCheckForAdd(int index) {
    if (index < 0 || index > size)
        throw new IndexOutOfBoundsException(outOfBoundsMsg(index));
}
```

### get 方法

```java
public E get(int index) {
    rangeCheck(index);
    return elementData(index);
}

E elementData(int index) {
    return (E) elementData[index];
}
```

### remove 方法

```java
public E remove(int index) {
    rangeCheck(index);
    modCount++;
    E oldValue = elementData(index);
    
    int numMoved = size - index - 1;
    if (numMoved > 0)
        // 将 index 之后的元素前移
        System.arraycopy(elementData, index + 1,
                         elementData, index,
                         numMoved);
    
    elementData[--size] = null;  // GC 回收
    return oldValue;
}

public boolean remove(Object o) {
    if (o == null) {
        for (int index = 0; index < size; index++)
            if (elementData[index] == null) {
                fastRemove(index);
                return true;
            }
    } else {
        for (int index = 0; index < size; index++)
            if (o.equals(elementData[index])) {
                fastRemove(index);
                return true;
            }
    }
    return false;
}
```

## 四、ArrayList vs Vector

### 对比表

| 特性 | ArrayList | Vector |
|------|-----------|--------|
| **线程安全** | 非同步 | 同步（synchronized） |
| **性能** | 较快 | 较慢（每次操作都要加锁） |
| **容量增长** | 1.5 倍 | 2 倍（可指定） |
| **枚举** | 不支持 fail-fast | 支持 fail-fast（但不同步 modCount） |
| **引入版本** | JDK 1.2 | JDK 1.0 |

### Vector 扩容源码

```java
// Vector 扩容
private void grow(int minCapacity) {
    int oldCapacity = elementData.length;
    // Vector 默认增长 2 倍
    int newCapacity = oldCapacity + 
        ((capacityIncrement > 0) ? capacityIncrement : oldCapacity);
}

// capacityIncrement 可以自定义
public Vector(int initialCapacity, int capacityIncrement) {
    super();
    if (initialCapacity < 0)
        throw new IllegalArgumentException();
    elementData = new Object[initialCapacity];
    this.capacityIncrement = capacityIncrement;
}
```

### 替代方案

```java
// 线程安全的 ArrayList
List<String> list = Collections.synchronizedList(new ArrayList<>());

// 或使用 CopyOnWriteArrayList（读多写少场景）
List<String> cowList = new CopyOnWriteArrayList<>();
```

## 五、面试题精讲

### 🔴 第一层：ArrayList 的底层是什么？

> **参考答案**：
>
> ArrayList 的底层是**动态数组**（`Object[] elementData`）。它实现了 `RandomAccess` 接口，支持随机访问，查询效率 O(1)。但插入和删除需要移动元素，时间复杂度 O(n)。

### 🟡 第二层：ArrayList 的扩容机制

> **参考答案**：
>
> ArrayList 在添加元素时，如果当前容量不足，会触发扩容：
>
> 1. 新容量 = 旧容量 + 旧容量 / 2（即 1.5 倍）
> 2. 如果 1.5 倍仍不够，直接使用所需容量
> 3. 最后通过 `Arrays.copyOf()` 拷贝数据
>
> JDK 8 的扩容机制避免了每次都翻倍造成的空间浪费。

### 🟡 第三层：ArrayList 和 LinkedList 的区别

> **参考答案**：
>
> | 维度 | ArrayList | LinkedList |
> |------|-----------|------------|
> | **底层结构** | 动态数组 | 双向链表 |
> | **随机访问** | O(1) | O(n) |
> | **头部插入** | O(n) | O(1) |
> | **尾部插入** | 均摊 O(1) | O(1) |
> | **内存开销** | 小（无指针） | 大（需存储前后指针） |
> | **CPU 缓存** | 友好 | 不友好 |
> | **适用场景** | 查询多、迭代多 | 插入删除多 |

### 💡 第四层：为什么不直接使用数组？

> **参考答案**：
>
> 数组长度固定，无法动态扩展。ArrayList 封装了数组，提供了：
> 1. **自动扩容**：无需关心容量管理
> 2. **边界检查**：防止数组越界
> 3. **封装方法**：add、remove 等操作封装好了
> 4. **fail-fast**：迭代时检测并发修改

## 六、常见错误与陷阱

### ⚠️ 陷阱 1：扩容导致内存浪费

```java
// 错误：预估容量过小
ArrayList<String> list = new ArrayList<>();
for (int i = 0; i < 1000000; i++) {
    list.add("item" + i);  // 扩容多次，效率低
}

// 正确：预估容量
ArrayList<String> list = new ArrayList<>(1000000);
for (int i = 0; i < 1000000; i++) {
    list.add("item" + i);  // 无需扩容
}
```

### ⚠️ 陷阱 2：遍历时删除元素

```java
List<String> list = new ArrayList<>(Arrays.asList("a", "b", "c"));

// 错误：fail-fast
for (String s : list) {
    if ("b".equals(s)) {
        list.remove(s);  // ConcurrentModificationException
    }
}

// 正确方式 1：使用迭代器
Iterator<String> it = list.iterator();
while (it.hasNext()) {
    if ("b".equals(it.next())) {
        it.remove();
    }
}

// 正确方式 2：倒序删除
for (int i = list.size() - 1; i >= 0; i--) {
    if ("b".equals(list.get(i))) {
        list.remove(i);
    }
}

// 正确方式 3：removeIf (JDK 9+)
list.removeIf("b"::equals);
```

### ⚠️ 陷阱 3：ArrayList 不是线程安全的

```java
// 错误：多线程不安全
List<Integer> list = new ArrayList<>();
Thread t1 = new Thread(() -> {
    for (int i = 0; i < 1000; i++) list.add(i);
});
Thread t2 = new Thread(() -> {
    for (int i = 0; i < 1000; i++) list.add(i);
});
t1.start(); t2.start();
t1.join(); t2.join();
System.out.println(list.size());  // 可能小于 2000
```

## 七、手写简化版 ArrayList

```java title="SimpleArrayList.java"
public class SimpleArrayList<E> {
    
    private static final int DEFAULT_CAPACITY = 10;
    private Object[] elementData;
    private int size;
    
    public SimpleArrayList() {
        elementData = new Object[DEFAULT_CAPACITY];
    }
    
    public SimpleArrayList(int initialCapacity) {
        if (initialCapacity < 0) {
            throw new IllegalArgumentException("Illegal Capacity: " + initialCapacity);
        }
        elementData = new Object[initialCapacity];
    }
    
    public boolean add(E e) {
        ensureCapacityInternal(size + 1);
        elementData[size++] = e;
        return true;
    }
    
    @SuppressWarnings("unchecked")
    public E get(int index) {
        rangeCheck(index);
        return (E) elementData[index];
    }
    
    @SuppressWarnings("unchecked")
    public E remove(int index) {
        rangeCheck(index);
        E oldValue = (E) elementData[index];
        int numMoved = size - index - 1;
        if (numMoved > 0) {
            System.arraycopy(elementData, index + 1,
                           elementData, index, numMoved);
        }
        elementData[--size] = null;
        return oldValue;
    }
    
    public int size() {
        return size;
    }
    
    private void ensureCapacityInternal(int minCapacity) {
        if (size == elementData.length) {
            int newCapacity = elementData.length + (elementData.length >> 1);
            elementData = Arrays.copyOf(elementData, newCapacity);
        }
    }
    
    private void rangeCheck(int index) {
        if (index >= size || index < 0)
            throw new IndexOutOfBoundsException();
    }
}
```

## 八、对比总结表

| 维度 | ArrayList | LinkedList | Vector |
|------|-----------|------------|--------|
| **底层结构** | 动态数组 | 双向链表 | 动态数组 |
| **线程安全** | 否 | 否 | 是 |
| **get 时间复杂度** | O(1) | O(n) | O(1) |
| **add 时间复杂度** | 均摊 O(1) | O(1) | 均摊 O(1) |
| **remove 时间复杂度** | O(n) | O(1) | O(n) |
| **内存开销** | 小 | 大 | 小 |
| **迭代器 fail-fast** | 是 | 是 | 是 |
