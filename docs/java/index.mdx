# Java

Java，自 1995 年诞生以来，一直是企业级开发领域的中流砥柱。"一次编写，到处运行"的理念，让 Java 成为当时最具革命性的编程语言之一。二十多年后的今天，Java 依然活力十足——从银行系统到互联网应用，从大数据处理到移动开发，Java 的身影无处不在。

Java 的知识体系庞大而深邃。从基础的语法特性，到复杂的并发编程；从 JVM 的内存管理，到类加载机制；从集合框架的设计哲学，到 I/O 系统的底层原理……每一个领域都有大量值得深入探索的内容。

本章节将带你系统学习 Java 核心技术栈：

**基础语法** 部分覆盖了 String、枚举与注解、异常处理、泛型与语法糖等核心概念，打牢 Java 编程的地基。**集合框架** 则深入剖析 List、Set、Map、Queue 四大集合家族的设计原理与使用技巧，让你面对复杂的数据处理需求时能选出最合适的容器。**并发编程** 是 Java 最具挑战性的领域之一，从线程基础到 synchronized 锁机制，从 JUC 并发工具到线程池，每一个知识点都是面试中的高频考点。**JVM 原理** 则揭开 Java 虚拟机的神秘面纱，理解字节码、运行时数据区、垃圾回收机制，是从"会用"到"精通"的关键一步。**新特性** 部分则聚焦 JDK 9 到 JDK 21 的现代化演进，跟上 Java 发展的最新趋势。

无论你是 Java 初学者，还是想系统梳理知识体系的开发者，这套内容都能帮助你构建完整的 Java 技术图谱。每一篇文章都会从"为什么"讲起，让你不仅知道怎么用，更理解背后的原理与权衡。

让我们开始吧。
