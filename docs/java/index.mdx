## 学习建议

### 学习顺序

1. **集合框架**：从 HashMap 入手，理解数据结构与算法
2. **并发编程**：理解 JMM 后学习 synchronized、线程池
3. **JVM**：内存区域 → 垃圾回收 → 类加载
4. **Java 基础**：OOP、异常、反射
5. **I/O 与网络**：BIO → NIO → Netty
6. **新特性**：Lambda、Stream

### 准备策略

| 级别 | 要求 |
|------|------|
| **P5** | 理解核心概念，能回答第一层和第二层追问 |
| **P6** | 掌握源码细节，能回答第三层追问，有加分项 |

### 面试技巧

1. **从问题出发**：不要只背概念，要理解设计背景
2. **对比优先**：同类技术要有对比表格（如 synchronized vs ReentrantLock）
3. **追问链**：每道题准备 2-3 层追问链
4. **源码加持**：关键原理能讲出 JDK 源码片段
