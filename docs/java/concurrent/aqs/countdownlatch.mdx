# AQS 实现 CountDownLatch 源码解析

`CountDownLatch`（倒计时门闩）是 AQS 共享模式的经典应用。它允许一个或多个线程等待其他线程完成一组操作。理解 CountDownLatch 的实现，能帮助我们深入理解 AQS 的共享模式。

## CountDownLatch 结构

```java
public class CountDownLatch {
    
    // 内部使用 Sync，继承 AQS
    private final Sync sync;
    
    // Sync 实现
    private static final class Sync extends AbstractQueuedSynchronizer {
        
        Sync(int count) {
            setState(count);
        }
        
        // 共享模式获取
        protected int tryAcquireShared(int ignore) {
            return getState() == 0 ? 1 : -1;
        }
        
        // 共享模式释放
        protected boolean tryReleaseShared(int decrements) {
            for (;;) {
                int c = getState();
                if (c == 0) {
                    return false;  // 已经是0了
                }
                int nextc = c - 1;
                if (compareAndSetState(c, nextc)) {
                    return nextc == 0;  // 返回是否完全倒计时
                }
            }
        }
    }
}
```

## 核心机制

### state 的含义

```java
// state 表示倒计时的初始值
// 每次调用 countDown()，state - 1
// 当 state = 0 时，所有等待的线程被唤醒

// 初始化
Sync(int count) {
    setState(count);  // 设置初始倒计时值
}

// 获取
protected int tryAcquireShared(int ignore) {
    // state = 0，返回 1（成功）
    // state != 0，返回 -1（失败）
    return getState() == 0 ? 1 : -1;
}

// 释放
protected boolean tryReleaseShared(int decrements) {
    // CAS 递减 state
    // 返回 state 是否变为 0
}
```

## await() 方法的执行流程

```java
public void await() throws InterruptedException {
    sync.acquireSharedInterruptibly(1);
}

// AQS.acquireSharedInterruptibly()
public final void acquireSharedInterruptibly(int arg) 
        throws InterruptedException {
    if (Thread.interrupted()) {
        throw new InterruptedException();
    }
    
    // 1. 尝试获取（检查 state 是否为0）
    if (tryAcquireShared(arg) < 0) {
        // 2. 获取失败，加入队列等待
        doAcquireSharedInterruptibly(arg);
    }
}

// CountDownLatch.tryAcquireShared()
protected int tryAcquireShared(int ignore) {
    return getState() == 0 ? 1 : -1;
    // state == 0，返回 1（成功，等待线程继续执行）
    // state != 0，返回 -1（失败，等待线程进入队列）
}
```

## doAcquireSharedInterruptibly：队列等待

```java
private void doAcquireSharedInterruptibly(int arg) 
        throws InterruptedException {
    
    // 1. 添加共享节点到队列
    final Node node = addWaiter(Node.SHARED);
    
    try {
        for (;;) {
            final Node p = node.predecessor();
            
            if (p == head) {
                // 2. 再次尝试获取
                int r = tryAcquireShared(arg);
                if (r >= 0) {
                    // 3. 获取成功，设置头节点
                    setHeadAndPropagate(node, r);
                    p.next = null;
                    return;
                }
            }
            
            // 4. 检查并更新等待状态
            if (shouldParkAfterFailedAcquire(p, node)) {
                // 5. 挂起线程
                if (parkAndCheckInterrupt()) {
                    throw new InterruptedException();
                }
            }
        }
    } catch (Throwable t) {
        cancelAcquire(node);
        throw t;
    }
}

// 设置头节点并传播
private void setHeadAndPropagate(Node node, int propagate) {
    Node h = head;
    setHead(node);
    
    // 传播：唤醒后继的共享节点
    if (propagate > 0 && h != null && h.waitStatus < 0) {
        unparkSuccessor(h);
    }
}
```

## countDown() 方法的执行流程

```java
public void countDown() {
    sync.releaseShared(1);
}

// AQS.releaseShared()
public final boolean releaseShared(int arg) {
    // 1. 尝试释放
    if (tryReleaseShared(arg)) {
        // 2. 释放成功，唤醒等待线程
        doReleaseShared();
        return true;
    }
    return false;
}

// CountDownLatch.tryReleaseShared()
protected boolean tryReleaseShared(int decrements) {
    for (;;) {
        int c = getState();
        if (c == 0) {
            return false;  // 已经是0了
        }
        int nextc = c - 1;
        if (compareAndSetState(c, nextc)) {
            return nextc == 0;  // 返回是否完全倒计时
        }
    }
}
```

## doReleaseShared：唤醒等待线程

```java
private void doReleaseShared() {
    for (;;) {
        Node h = head;
        
        if (h != null && h != tail) {
            int ws = h.waitStatus;
            
            if (ws == Node.SIGNAL) {
                // 唤醒后继节点
                if (compareAndSetWaitStatus(h, SIGNAL, 0)) {
                    unparkSuccessor(h);
                }
            } else if (ws == 0 &&
                       !compareAndSetWaitStatus(h, 0, Node.PROPAGATE)) {
                // 传播状态
            }
        }
        
        if (h == head) {
            break;
        }
    }
}
```

## await(timeout) 带超时等待

```java
public boolean await(long timeout, TimeUnit unit) 
        throws InterruptedException {
    
    return sync.tryAcquireSharedNanos(1, unit.toNanos(timeout));
}

// AQS.tryAcquireSharedNanos()
public final boolean tryAcquireSharedNanos(int arg, long nanosTimeout) 
        throws InterruptedException {
    
    if (Thread.interrupted()) {
        throw new InterruptedException();
    }
    
    if (tryAcquireShared(arg) >= 0) {
        return true;
    }
    
    // 计算截止时间
    long deadline = System.nanoTime() + nanosTimeout;
    
    for (;;) {
        if (tryAcquireShared(arg) >= 0) {
            return true;
        }
        
        long nanosTimeout = deadline - System.nanoTime();
        if (nanosTimeout <= 0) {
            return false;  // 超时
        }
        
        if (shouldParkAfterFailedAcquire(...) && 
            nanosTimeout > spinForTimeoutThreshold) {
            LockSupport.parkNanos(this, nanosTimeout);
        }
        
        if (Thread.interrupted()) {
            throw new InterruptedException();
        }
    }
}
```

## 完整示例

```java
public class CountDownLatchDemo {
    
    public void example() throws InterruptedException {
        CountDownLatch latch = new CountDownLatch(3);
        
        // 启动3个工作线程
        for (int i = 0; i < 3; i++) {
            final int id = i;
            new Thread(() -> {
                try {
                    System.out.println("工作线程 " + id + " 完成任务");
                    latch.countDown();
                } catch (Exception e) {
                    e.printStackTrace();
                }
            }).start();
        }
        
        // 等待所有工作线程完成
        System.out.println("等待工作线程完成...");
        latch.await();
        System.out.println("所有工作线程完成，继续执行");
    }
    
    public void withTimeout() throws InterruptedException {
        CountDownLatch latch = new CountDownLatch(3);
        
        // 带超时等待
        boolean completed = latch.await(5, TimeUnit.SECONDS);
        
        if (completed) {
            System.out.println("所有任务完成");
        } else {
            System.out.println("等待超时");
        }
    }
}
```

## 总结

CountDownLatch 的 AQS 实现：

1. **state**：表示倒计时初始值
2. **tryAcquireShared**：检查 state 是否为0
3. **tryReleaseShared**：CAS 递减 state
4. **共享模式**：所有等待线程同时被唤醒

关键特点：
- **一次性**：state 归零后不能重置
- **共享模式**：唤醒所有等待线程
- **支持超时**：tryAcquireSharedNanos

理解 CountDownLatch 的实现，就理解了 AQS 共享模式的核心机制。
