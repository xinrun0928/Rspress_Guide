# AQS 实现 ReentrantLock 源码解析

`ReentrantLock` 是 AQS 最典型的应用，它完美展示了如何通过继承 AQS 实现一个功能完整的独占锁。本文将从源码角度深入解析 ReentrantLock 如何利用 AQS 实现可重入、公平/非公平策略。

## ReentrantLock 结构

```java
public class ReentrantLock implements Lock, java.io.Serializable {
    
    private final Sync sync;
    
    // Sync 是抽象基类
    abstract static class Sync extends AbstractQueuedSynchronizer {
        abstract void lock();
        
        // 非公平获取
        final boolean nonfairTryAcquire(int acquires) { ... }
        
        // 释放
        protected final boolean tryRelease(int releases) { ... }
    }
    
    // 非公平锁
    static final class NonfairSync extends Sync {
        protected final boolean tryAcquire(int acquires) {
            return nonfairTryAcquire(acquires);
        }
    }
    
    // 公平锁
    static final class FairSync extends Sync {
        protected final boolean tryAcquire(int acquires) {
            // 公平版本
        }
    }
}
```

## 公平锁 vs 非公平锁

### 非公平锁：NonfairSync

```java
static final class NonfairSync extends Sync {
    
    final void lock() {
        // 直接尝试 CAS 获取锁
        if (compareAndSetState(0, 1)) {
            setExclusiveOwnerThread(Thread.currentThread());
            return;
        }
        acquire(1);
    }
    
    protected final boolean tryAcquire(int acquires) {
        return nonfairTryAcquire(acquires);
    }
}

// Sync 的非公平获取实现
final boolean nonfairTryAcquire(int acquires) {
    Thread current = Thread.currentThread();
    int c = getState();
    
    if (c == 0) {
        // 状态为0，锁未被持有
        // 直接 CAS，不检查等待队列
        if (compareAndSetState(0, acquires)) {
            setExclusiveOwnerThread(current);
            return true;
        }
    } else if (current == getExclusiveOwnerThread()) {
        // 当前线程已持有锁，可重入
        int nextc = c + acquires;
        setState(nextc);
        return true;
    }
    return false;
}
```

### 公平锁：FairSync

```java
static final class FairSync extends Sync {
    
    final void lock() {
        acquire(1);
    }
    
    protected final boolean tryAcquire(int acquires) {
        Thread current = Thread.currentThread();
        int c = getState();
        
        if (c == 0) {
            // 状态为0，检查是否有更早等待的线程
            if (!hasQueuedPredecessors() &&
                compareAndSetState(0, acquires)) {
                setExclusiveOwnerThread(current);
                return true;
            }
        } else if (current == getExclusiveOwnerThread()) {
            // 可重入
            int nextc = c + acquires;
            setState(nextc);
            return true;
        }
        return false;
    }
}

// 关键：检查等待队列
public final boolean hasQueuedPredecessors() {
    Node t = tail;
    Node h = head;
    Node s;
    
    // 如果 head == tail，队列为空或只有一个节点
    // 如果 h.next == null，正在初始化
    // 如果 s.thread != current，有更早等待的线程
    return h != t && 
           ((s = h.next) == null || 
            s.thread != Thread.currentThread());
}
```

### 公平锁 vs 非公平锁的区别

```java
public class FairVsUnfair {
    
    // 非公平锁：允许"插队"
    // ┌────────────────────────────────────────┐
    // │ 线程B尝试获取时，即使线程A在等待            │
    // │ 线程B也可能直接 CAS 成功                  │
    // └────────────────────────────────────────┘
    
    // 公平锁：严格按照 FIFO
    // ┌────────────────────────────────────────┐
    // │ 线程B尝试获取时，先检查队列               │
    // │ 如果有更早等待的线程A，线程B必须排队      │
    // └────────────────────────────────────────┘
}
```

## 可重入的实现

```java
// state 表示重入次数
// 每次获取锁，state + 1
// 每次释放锁，state - 1
// 当 state = 0 时，锁被完全释放

protected final boolean tryRelease(int releases) {
    int c = getState() - releases;
    
    // 检查是否是持有锁的线程
    if (Thread.currentThread() != getExclusiveOwnerThread()) {
        throw new IllegalMonitorStateException();
    }
    
    boolean free = false;
    if (c == 0) {
        // 状态为0，完全释放
        free = true;
        setExclusiveOwnerThread(null);
    }
    setState(c);
    return free;
}
```

### 可重入示例

```java
public class ReentrantDemo {
    
    private final ReentrantLock lock = new ReentrantLock();
    
    public void outer() {
        lock.lock();
        try {
            System.out.println("outer 获取锁，state = " + lock.getHoldCount());
            inner();  // 重入
        } finally {
            lock.unlock();
        }
    }
    
    public void inner() {
        lock.lock();
        try {
            System.out.println("inner 获取锁，state = " + lock.getHoldCount());
        } finally {
            lock.unlock();
        }
    }
    
    public static void main(String[] args) {
        new ReentrantDemo().outer();
        // 输出：
        // outer 获取锁，state = 1
        // inner 获取锁，state = 2
    }
}
```

## lock() 方法的执行流程

### 非公平锁的 lock()

```java
// NonfairSync.lock()
public void lock() {
    // 1. 第一次 CAS 获取（不检查队列）
    if (compareAndSetState(0, 1)) {
        setExclusiveOwnerThread(Thread.currentThread());
        return;
    }
    
    // 2. 获取失败，调用 AQS 的 acquire
    acquire(1);
}

// AQS.acquire()
public final void acquire(int arg) {
    // 3. 调用 tryAcquire（子类实现）
    if (!tryAcquire(arg)) {
        // 4. 获取失败，加入等待队列
        Node node = addWaiter(Node.EXCLUSIVE);
        
        // 5. 在队列中等待
        if (acquireQueued(node, arg)) {
            selfInterrupt();
        }
    }
}

// NonfairSync.tryAcquire()
protected final boolean tryAcquire(int acquires) {
    return nonfairTryAcquire(acquires);
}

// Sync.nonfairTryAcquire()
final boolean nonfairTryAcquire(int acquires) {
    Thread current = Thread.currentThread();
    int c = getState();
    
    if (c == 0) {
        // 再次尝试 CAS（可能有其他线程释放了锁）
        if (compareAndSetState(0, acquires)) {
            setExclusiveOwnerThread(current);
            return true;
        }
    } else if (current == getExclusiveOwnerThread()) {
        // 可重入
        int nextc = c + acquires;
        setState(nextc);
        return true;
    }
    return false;
}
```

### acquireQueued：队列中的等待

```java
final boolean acquireQueued(final Node node, int arg) {
    boolean failed = true;
    try {
        boolean interrupted = false;
        for (;;) {
            // 获取前驱节点
            final Node p = node.predecessor();
            
            if (p == head && tryAcquire(arg)) {
                // 如果是队列第一个节点，尝试获取
                setHead(node);
                p.next = null;  // 帮助 GC
                failed = false;
                return interrupted;
            }
            
            // 获取失败，检查是否需要挂起
            if (shouldParkAfterFailedAcquire(p, node) &&
                parkAndCheckInterrupt()) {
                interrupted = true;
            }
        }
    } finally {
        if (failed) {
            cancelAcquire(node);
        }
    }
}

// 检查是否需要挂起
private static boolean shouldParkAfterFailedAcquire(Node pred, Node node) {
    int ws = pred.waitStatus;
    
    if (ws == Node.SIGNAL) {
        // 前驱节点会唤醒我们
        return true;
    }
    
    if (ws > 0) {
        // 前驱节点已取消，跳过
        do {
            node.prev = pred = pred.prev;
        } while (pred.waitStatus > 0);
        pred.next = node;
    } else {
        // 设置前驱节点状态为 SIGNAL
        compareAndSetWaitStatus(pred, ws, Node.SIGNAL);
    }
    return false;
}
```

## unlock() 方法的执行流程

```java
public void unlock() {
    sync.release(1);
}

// AQS.release()
public final boolean release(int arg) {
    if (tryRelease(arg)) {
        Node h = head;
        if (h != null && h.waitStatus != 0) {
            // 唤醒后继节点
            unparkSuccessor(h);
        }
        return true;
    }
    return false;
}

// Sync.tryRelease()
protected final boolean tryRelease(int releases) {
    int c = getState() - releases;
    
    if (Thread.currentThread() != getExclusiveOwnerThread()) {
        throw new IllegalMonitorStateException();
    }
    
    boolean free = false;
    if (c == 0) {
        free = true;
        setExclusiveOwnerThread(null);
    }
    setState(c);
    return free;
}

// 唤醒后继节点
private void unparkSuccessor(Node node) {
    int ws = node.waitStatus;
    if (ws < 0) {
        compareAndSetWaitStatus(node, ws, 0);
    }
    
    Node s = node.next;
    if (s == null || s.waitStatus > 0) {
        // 找到最前面的有效节点
        s = null;
        for (Node t = tail; t != null && t != node; t = t.prev) {
            if (t.waitStatus <= 0) {
                s = t;
            }
        }
    }
    
    if (s != null) {
        LockSupport.unpark(s.thread);
    }
}
```

## 总结

ReentrantLock 通过 AQS 实现：

1. **Sync**：抽象基类，实现 tryRelease
2. **NonfairSync**：非公平锁，不检查等待队列
3. **FairSync**：公平锁，使用 hasQueuedPredecessors() 检查
4. **state**：表示重入次数
5. **独占模式**：同一时刻只有一个线程持有

理解 ReentrantLock 的源码，就理解了 AQS 的实际应用。
