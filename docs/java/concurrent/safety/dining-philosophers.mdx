# 哲学家就餐问题

五个哲学家围坐一张圆桌，面前摆着意面和筷子。每个哲学家需要同时拿到左右两边的筷子才能吃面。

问题是：当五个哲学家同时饿了，都拿起了左边的筷子，等待右边的筷子——然后所有人都永远等下去。

**哲学家就餐问题** 是并发领域的经典问题，由 Dijkstra 于 1965 年提出。它描述了资源竞争和死锁的复杂性。

## 问题模型

```
           哲学家 0
          🍝
    筷子0          筷子1
哲学家 4  ⬛          ⬛ 哲学家 1
    筷子4          筷子1
          ⬜ ⬜ ⬜ ⬜ ⬜
          圆桌
    筷子3          筷子2
哲学家 3              哲学家 2
```

**规则**：
1. 哲学家要么思考，要么吃饭
2. 拿起筷子时需要同时拿到左右两根
3. 不能抢别人正在使用的筷子
4. 吃完后放下筷子，让别人用

## 最直观的解法（有问题）

### 尝试一：简单模拟

```java
public class PhilosopherProblem {
    public static void main(String[] args) {
        final Philosopher[] philosophers = new Philosopher[5];
        final Chopstick[] chopsticks = new Chopstick[5];

        for (int i = 0; i < 5; i++) {
            chopsticks[i] = new Chopstick(i);
        }

        for (int i = 0; i < 5; i++) {
            philosophers[i] = new Philosopher(
                i,
                chopsticks[i],
                chopsticks[(i + 1) % 5]
            );
            new Thread(philosophers[i], "哲学家-" + i).start();
        }
    }

    static class Philosopher implements Runnable {
        private final int id;
        private final Chopstick leftChopstick;
        private final Chopstick rightChopstick;

        Philosopher(int id, Chopstick left, Chopstick right) {
            this.id = id;
            this.leftChopstick = left;
            this.rightChopstick = right;
        }

        @Override
        public void run() {
            try {
                while (true) {
                    think();
                    pickUpChopsticks();
                    eat();
                    putDownChopsticks();
                }
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
            }
        }

        private void think() throws InterruptedException {
            System.out.println("哲学家 " + id + " 正在思考");
            Thread.sleep((long) (Math.random() * 1000));
        }

        private void pickUpChopsticks() throws InterruptedException {
            System.out.println("哲学家 " + id + " 试图拿起筷子");
            leftChopstick.pickUp();
            rightChopstick.pickUp();
        }

        private void eat() throws InterruptedException {
            System.out.println("哲学家 " + id + " 正在吃饭");
            Thread.sleep((long) (Math.random() * 1000));
        }

        private void putDownChopsticks() {
            System.out.println("哲学家 " + id + " 放下筷子");
            leftChopstick.putDown();
            rightChopstick.putDown();
        }
    }

    static class Chopstick {
        private final int id;
        private boolean taken = false;

        Chopstick(int id) {
            this.id = id;
        }

        public synchronized void pickUp() throws InterruptedException {
            while (taken) {
                wait();
            }
            taken = true;
        }

        public synchronized void putDown() {
            taken = false;
            notify();
        }
    }
}
```

**问题**：当所有哲学家同时拿起左边的筷子后，都会等待右边的筷子，形成死锁。

## 解法一：服务生解法（破坏循环等待）

引入一个**仲裁者（服务生）**，只有拿到许可才能拿筷子：

```java
public class ServerSolution {
    public static void main(String[] args) {
        final PhilosopherWithServer[] philosophers = new PhilosopherWithServer[5];
        final Chopstick[] chopsticks = new Chopstick[5];
        final Semaphore table = new Semaphore(4);  // 最多4个人同时进餐

        for (int i = 0; i < 5; i++) {
            chopsticks[i] = new Chopstick(i);
        }

        for (int i = 0; i < 5; i++) {
            philosophers[i] = new PhilosopherWithServer(
                i, chopsticks[i], chopsticks[(i + 1) % 5], table
            );
            new Thread(philosophers[i], "哲学家-" + i).start();
        }
    }

    static class PhilosopherWithServer implements Runnable {
        private final int id;
        private final Chopstick left;
        private final Chopstick right;
        private final Semaphore table;

        PhilosopherWithServer(int id, Chopstick left, Chopstick right, Semaphore table) {
            this.id = id;
            this.left = left;
            this.right = right;
            this.table = table;
        }

        @Override
        public void run() {
            try {
                while (true) {
                    think();
                    requestChopsticks();
                    eat();
                    returnChopsticks();
                }
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
            }
        }

        private void think() throws InterruptedException {
            Thread.sleep((long) (Math.random() * 1000));
        }

        private void requestChopsticks() throws InterruptedException {
            table.acquire();  // 请求许可
            left.pickUp();
            right.pickUp();
        }

        private void eat() throws InterruptedException {
            Thread.sleep((long) (Math.random() * 1000));
        }

        private void returnChopsticks() {
            left.putDown();
            right.putDown();
            table.release();  // 归还许可
        }
    }

    static class Chopstick {
        private final int id;
        private boolean taken = false;

        Chopstick(int id) {
            this.id = id;
        }

        public synchronized void pickUp() throws InterruptedException {
            while (taken) {
                wait();
            }
            taken = true;
        }

        public synchronized void putDown() {
            taken = false;
            notify();
        }
    }
}
```

**核心思想**：限制同时就餐的哲学家数量为 4 个（n - 1 个），破坏循环等待条件。

## 解法二：筷子排序解法（破坏持有并等待）

给筷子编号，让哲学家**先拿编号小的筷子**：

```java
public class OrderingSolution {
    public static void main(String[] args) {
        final PhilosopherOrdered[] philosophers = new PhilosopherOrdered[5];
        final Chopstick[] chopsticks = new Chopstick[5];

        for (int i = 0; i < 5; i++) {
            chopsticks[i] = new Chopstick(i);
        }

        for (int i = 0; i < 5; i++) {
            // 传递筷子顺序，按筷子 ID 排序
            Chopstick first = chopsticks[Math.min(i, (i + 1) % 5)];
            Chopstick second = chopsticks[Math.max(i, (i + 1) % 5)];
            philosophers[i] = new PhilosopherOrdered(i, first, second);
            new Thread(philosophers[i], "哲学家-" + i).start();
        }
    }

    static class PhilosopherOrdered implements Runnable {
        private final int id;
        private final Chopstick first;
        private final Chopstick second;

        PhilosopherOrdered(int id, Chopstick first, Chopstick second) {
            this.id = id;
            this.first = first;
            this.second = second;
        }

        @Override
        public void run() {
            try {
                while (true) {
                    think();
                    // 总是先拿 first，再拿 second
                    first.pickUp();
                    second.pickUp();
                    eat();
                    second.putDown();
                    first.putDown();
                }
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
            }
        }

        private void think() throws InterruptedException {
            Thread.sleep((long) (Math.random() * 1000));
        }

        private void eat() throws InterruptedException {
            Thread.sleep((long) (Math.random() * 1000));
        }
    }
}
```

**核心思想**：统一筷子获取顺序，避免循环等待。

## 解法三：非对称解法（破坏循环等待）

奇数号哲学家先拿左边，偶数号哲学家先拿右边：

```java
public class AsymmetricSolution {
    public static void main(String[] args) {
        final PhilosopherAsymmetric[] philosophers = new PhilosopherAsymmetric[5];
        final Chopstick[] chopsticks = new Chopstick[5];

        for (int i = 0; i < 5; i++) {
            chopsticks[i] = new Chopstick(i);
        }

        for (int i = 0; i < 5; i++) {
            philosophers[i] = new PhilosopherAsymmetric(i, chopsticks[i], chopsticks[(i + 1) % 5]);
            new Thread(philosophers[i], "哲学家-" + i).start();
        }
    }

    static class PhilosopherAsymmetric implements Runnable {
        private final int id;
        private final Chopstick left;
        private final Chopstick right;

        PhilosopherAsymmetric(int id, Chopstick left, Chopstick right) {
            this.id = id;
            this.left = left;
            this.right = right;
        }

        @Override
        public void run() {
            try {
                while (true) {
                    think();

                    // 奇数哲学家先左后右，偶数先右后左
                    if (id % 2 == 0) {
                        right.pickUp();
                        left.pickUp();
                    } else {
                        left.pickUp();
                        right.pickUp();
                    }

                    eat();

                    left.putDown();
                    right.putDown();
                }
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
            }
        }

        private void think() throws InterruptedException {
            Thread.sleep((long) (Math.random() * 1000));
        }

        private void eat() throws InterruptedException {
            Thread.sleep((long) (Math.random() * 1000));
        }
    }
}
```

**核心思想**：打破对称性，总有一个哲学家能先拿到两根筷子。

## 解法四：ReentrantLock + tryLock（破坏不可抢占）

使用 `tryLock()` 尝试获取锁，失败就重试：

```java
import java.util.concurrent.locks.ReentrantLock;

public class TryLockSolution {
    public static void main(String[] args) {
        final PhilosopherTryLock[] philosophers = new PhilosopherTryLock[5];
        final ChopstickWithLock[] chopsticks = new ChopstickWithLock[5];

        for (int i = 0; i < 5; i++) {
            chopsticks[i] = new ChopstickWithLock(i);
        }

        for (int i = 0; i < 5; i++) {
            philosophers[i] = new PhilosopherTryLock(
                i, chopsticks[i], chopsticks[(i + 1) % 5]
            );
            new Thread(philosophers[i], "哲学家-" + i).start();
        }
    }

    static class PhilosopherTryLock implements Runnable {
        private final int id;
        private final ChopstickWithLock left;
        private final ChopstickWithLock right;

        PhilosopherTryLock(int id, ChopstickWithLock left, ChopstickWithLock right) {
            this.id = id;
            this.left = left;
            this.right = right;
        }

        @Override
        public void run() {
            try {
                while (true) {
                    think();
                    tryToEat();
                    putDownChopsticks();
                }
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
            }
        }

        private void think() throws InterruptedException {
            Thread.sleep((long) (Math.random() * 1000));
        }

        private void tryToEat() throws InterruptedException {
            while (true) {
                // 尝试获取左筷子
                if (left.tryPickUp()) {
                    System.out.println("哲学家 " + id + " 拿起左筷子 " + left.id);

                    // 尝试获取右筷子
                    if (right.tryPickUp()) {
                        System.out.println("哲学家 " + id + " 拿起右筷子 " + right.id + "，开始吃饭");
                        return;  // 成功获取两根筷子
                    } else {
                        // 获取右筷子失败，释放左筷子
                        left.putDown();
                        System.out.println("哲学家 " + id + " 获取右筷子失败，放弃");
                    }
                }

                // 等待一小段时间后重试
                Thread.sleep(10);
            }
        }

        private void putDownChopsticks() throws InterruptedException {
            System.out.println("哲学家 " + id + " 放下筷子");
            left.putDown();
            right.putDown();
            Thread.sleep((long) (Math.random() * 1000));
        }
    }

    static class ChopstickWithLock {
        final int id;
        private final ReentrantLock lock = new ReentrantLock();

        ChopstickWithLock(int id) {
            this.id = id;
        }

        public boolean tryPickUp() {
            return lock.tryLock();
        }

        public void putDown() {
            lock.unlock();
        }
    }
}
```

**核心思想**：使用 tryLock 避免永久等待，失败后重试，破坏不可抢占条件。

## 解法五：使用 Semaphore（资源信号量）

```java
import java.util.concurrent.Semaphore;

public class SemaphoreSolution {
    public static void main(String[] args) {
        final Semaphore[] chopsticks = new Semaphore[5];
        for (int i = 0; i < 5; i++) {
            chopsticks[i] = new Semaphore(1);  // 每根筷子只能被一个哲学家持有
        }

        for (int i = 0; i < 5; i++) {
            final int id = i;
            new Thread(() -> {
                try {
                    while (true) {
                        think(id);
                        // 按顺序获取筷子
                        chopsticks[id].acquire();  // 左筷子
                        chopsticks[(id + 1) % 5].acquire();  // 右筷子
                        eat(id);
                        chopsticks[(id + 1) % 5].release();  // 右筷子
                        chopsticks[id].release();  // 左筷子
                    }
                } catch (InterruptedException e) {
                    Thread.currentThread().interrupt();
                }
            }, "哲学家-" + i).start();
        }
    }

    private static void think(int id) throws InterruptedException {
        Thread.sleep((long) (Math.random() * 1000));
    }

    private static void eat(int id) throws InterruptedException {
        System.out.println("哲学家 " + id + " 正在吃饭");
        Thread.sleep((long) (Math.random() * 1000));
    }
}
```

## 五种解法对比

| 解法 | 破坏的条件 | 优点 | 缺点 |
| --- | --- | --- | --- |
| 服务生解法 | 循环等待 | 简单，资源可控 | 需要额外的仲裁者 |
| 筷子排序 | 持有并等待 | 无需额外资源 | 需要修改资源标识 |
| 非对称解法 | 循环等待 | 简单直接 | 不够通用 |
| tryLock 解法 | 不可抢占 | 灵活 | 可能饥饿 |
| Semaphore | 互斥 | 并发友好 | 需要 5 个信号量 |

## 总结

哲学家就餐问题是理解死锁的绝佳案例：

- **死锁原因**：所有哲学家同时拿起左边的筷子，等待右边的筷子
- **解决思路**：破坏死锁的四个必要条件之一

实际并发编程中，**筷子排序**和**服务生解法**是最常用的策略。如果使用显式锁，**tryLock + 重试**也是一个不错的选择。
