# 并行流 ParallelStream 原理

`ParallelStream` 是 Java 8 引入的数据并行处理方式，底层基于 `ForkJoinPool`。

## 基本用法

```java
public class ParallelStreamDemo {
    
    public void basicUsage() {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
        
        // 串行流
        numbers.stream()
            .map(x -> x * 2)
            .collect(Collectors.toList());
        
        // 并行流
        numbers.parallelStream()
            .map(x -> x * 2)
            .collect(Collectors.toList());
        
        // 或者
        numbers.stream().parallel()
            .map(x -> x * 2)
            .collect(Collectors.toList());
    }
}
```

## 性能注意事项

```java
public class PerformanceTips {
    
    // 1. 小数据量不适合并行
    public void smallData() {
        // 100个元素，并行开销可能大于收益
        List<Integer> small = Arrays.asList(1, 2, 3);
        small.parallelStream().map(x -> x * 2);  // 不值得
    }
    
    // 2. 避免有状态的 lambda
    public void statefulLambda() {
        List<Integer> result = new ArrayList<>();
        // 错误：有状态lambda在并行下不安全
        // list.parallelStream().forEach(x -> result.add(x));
    }
    
    // 3. 顺序无关的操作
    public void orderIndependent() {
        // 求和适合并行
        IntStream.range(0, 1000000)
            .parallel()
            .sum();
    }
}
```

## 总结

| 场景 | 推荐 |
|------|------|
| 大数据量 | ParallelStream |
| 小数据量 | Stream |
| 顺序敏感 | Stream |
| 顺序不敏感 | ParallelStream |
