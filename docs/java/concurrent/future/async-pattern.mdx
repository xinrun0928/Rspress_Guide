# Java 异步编程模式总结

Java 提供了多种异步编程方式，本章总结各种模式的使用场景和最佳实践。

## 同步 vs 异步

```java
public class SyncVsAsync {
    
    // 同步：简单直接，阻塞等待
    public String syncProcess() {
        return queryDB();
    }
    
    // 异步：非阻塞，提高吞吐量
    public CompletableFuture<String> asyncProcess() {
        return CompletableFuture.supplyAsync(() -> queryDB());
    }
}
```

## 异步模式选择

```java
public class PatternSelection {
    
    // 1. 任务链（thenCompose）
    public void chainPattern() {
        CompletableFuture.supplyAsync(() -> step1())
            .thenCompose(r -> CompletableFuture.supplyAsync(() -> step2(r)));
    }
    
    // 2. 并行组合（allOf）
    public void parallelPattern() {
        CompletableFuture.allOf(
            CompletableFuture.supplyAsync(() -> queryDB()),
            CompletableFuture.supplyAsync(() -> queryCache())
        );
    }
    
    // 3. 快速失败（anyOf）
    public void fastFailPattern() {
        CompletableFuture.anyOf(
            CompletableFuture.supplyAsync(() -> queryPrimary()),
            CompletableFuture.supplyAsync(() -> queryBackup())
        );
    }
    
    private String step1() { return "1"; }
    private String step2(String s) { return s; }
}
```

## 最佳实践

1. **合理选择线程池**
2. **做好异常处理**
3. **避免线程阻塞**
4. **适当组合任务**
