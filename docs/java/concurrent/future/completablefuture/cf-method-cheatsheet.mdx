# CompletableFuture 常用方法速查表

本文汇总了 `CompletableFuture` 的常用方法，方便快速查阅。

## 创建方法

| 方法 | 说明 |
|------|------|
| `supplyAsync(Supplier)` | 异步执行，有返回值 |
| `runAsync(Runnable)` | 异步执行，无返回值 |
| `completedFuture(T)` | 创建已完成的Future |
| `failedFuture(ex)` | 创建已失败的Future |

## 转换方法

| 方法 | 说明 |
|------|------|
| `thenApply(fn)` | 同步转换
| `thenApplyAsync(fn)` | 异步转换
| `thenCompose(fn)` | 扁平化链式调用
| `thenCombine(other, fn)` | 组合两个Future |

## 消费方法

| 方法 | 说明 |
|------|------|
| `thenAccept(consumer)` | 消费结果
| `thenRun(action)` | 不消费结果，只执行动作 |

## 异常处理

| 方法 | 说明 |
|------|------|
| `exceptionally(fn)` | 处理异常，返回默认值
| `handle(biFn)` | 处理结果或异常
| `whenComplete(biConsumer)` | 结果或异常时执行 |

## 组合方法

| 方法 | 说明 |
|------|------|
| `allOf(futures...)` | 等待所有完成
| `anyOf(futures...)` | 任一完成

## 示例代码

```java
public class Cheatsheet {
    
    public void creation() {
        // 创建
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> "hello");
        CompletableFuture<Void> f2 = CompletableFuture.runAsync(() -> {});
        CompletableFuture<String> f3 = CompletableFuture.completedFuture("done");
    }
    
    public void transformation() {
        CompletableFuture.supplyAsync(() -> 1)
            .thenApply(x -> x * 2)           // 2
            .thenApplyAsync(x -> x + 1);    // 3
    }
    
    public void composition() {
        CompletableFuture.supplyAsync(() -> "hello")
            .thenCompose(s -> CompletableFuture.supplyAsync(() -> s + " world"));
    }
    
    public void combination() {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> "a");
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> "b");
        
        f1.thenCombine(f2, (a, b) -> a + b);  // "ab"
    }
    
    public void exceptionHandling() {
        CompletableFuture.supplyAsync(() -> {
            throw new RuntimeException();
        })
        .exceptionally(ex -> "fallback")      // 返回默认值
        .handle((result, ex) -> {
            if (ex != null) return "error";
            return result;
        });
    }
    
    public void waiting() throws Exception {
        CompletableFuture.allOf(
            CompletableFuture.supplyAsync(() -> 1),
            CompletableFuture.supplyAsync(() -> 2)
        ).join();
    }
}
```
