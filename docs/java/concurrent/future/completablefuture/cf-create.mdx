# CompletableFuture 创建方式

`CompletableFuture` 提供了多种创建方式，从简单的已完成Future到复杂的异步链。理解这些创建方式，才能灵活应对各种异步编程场景。

## 静态工厂方法

```java
public class CreationMethods {
    
    // 1. supplyAsync - 有返回值的异步任务
    public void supplyAsyncDemo() {
        CompletableFuture<String> future = CompletableFuture.supplyAsync(() -> {
            // 异步执行，返回结果
            return "Hello";
        });
    }
    
    // 2. runAsync - 无返回值的异步任务
    public void runAsyncDemo() {
        CompletableFuture<Void> future = CompletableFuture.runAsync(() -> {
            // 异步执行，无返回值
            System.out.println("执行中...");
        });
    }
    
    // 3. completedFuture - 已完成的Future
    public void completedFutureDemo() {
        CompletableFuture<String> future = CompletableFuture.completedFuture("已完成");
        // 立即返回结果
    }
    
    // 4. failedFuture - 已失败的Future
    public void failedFutureDemo() {
        CompletableFuture<String> future = CompletableFuture.failedFuture(
            new RuntimeException("错误")
        );
    }
}
```

## supplyAsync：异步供应

```java
public class SupplyAsyncDemo {
    
    public void basicUsage() throws Exception {
        CompletableFuture<String> future = CompletableFuture.supplyAsync(() -> {
            // 模拟耗时操作
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
            }
            return "result";
        });
        
        // 获取结果
        String result = future.get();
        System.out.println(result);
    }
    
    // 指定线程池
    public void withExecutor() {
        ExecutorService executor = Executors.newFixedThreadPool(4);
        
        CompletableFuture<String> future = CompletableFuture.supplyAsync(() -> {
            return "result";
        }, executor);
    }
    
    // 使用默认线程池
    public void defaultPool() {
        // 默认使用 ForkJoinPool.commonPool()
        CompletableFuture.supplyAsync(() -> {
            return process();
        });
    }
}
```

## runAsync：无返回值异步

```java
public class RunAsyncDemo {
    
    public void basicUsage() throws Exception {
        CompletableFuture<Void> future = CompletableFuture.runAsync(() -> {
            // 异步执行，不返回结果
            for (int i = 0; i < 100; i++) {
                System.out.println("执行中: " + i);
            }
        });
        
        // 等待完成
        future.get();
    }
    
    // 链式处理
    public void chainedUsage() {
        CompletableFuture.runAsync(() -> {
            System.out.println("第一步");
        })
        .thenRun(() -> System.out.println("第二步"))
        .thenRun(() -> System.out.println("第三步"));
    }
}
```

## 已完成/已失败的 Future

```java
public class CompletedDemo {
    
    // completedFuture - 立即返回结果
    public void immediateResult() {
        CompletableFuture<String> future = CompletableFuture.completedFuture("立即结果");
        
        // get() 立即返回，不会阻塞
        String result = future.get();
        System.out.println(result);
    }
    
    // failedFuture - 立即失败
    public void immediateFailure() {
        CompletableFuture<String> future = CompletableFuture.failedFuture(
            new RuntimeException("立即失败")
        );
        
        try {
            future.get();
        } catch (Exception e) {
            System.out.println("捕获异常: " + e.getCause().getMessage());
        }
    }
}
```

## 创建空 Future

```java
public class EmptyFuture {
    
    // 创建一个永不完成的Future
    public void neverComplete() {
        CompletableFuture<String> future = new CompletableFuture<>();
        // 永远不会完成，除非手动调用 complete()
    }
    
    // 手动完成
    public void manualComplete() {
        CompletableFuture<String> future = new CompletableFuture<>();
        
        // 模拟异步处理
        new Thread(() -> {
            try {
                Thread.sleep(1000);
                future.complete("完成");  // 手动标记完成
            } catch (InterruptedException e) {
                future.completeExceptionally(e);
            }
        }).start();
    }
}
```

## allOf 和 anyOf

```java
public class AllOfAnyOf {
    
    public void allOfDemo() throws Exception {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> "A");
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> "B");
        CompletableFuture<String> f3 = CompletableFuture.supplyAsync(() -> "C");
        
        // 等待所有完成
        CompletableFuture<Void> allDone = CompletableFuture.allOf(f1, f2, f3);
        allDone.join();  // 阻塞直到所有完成
        
        // 获取结果
        String result = Stream.of(f1, f2, f3)
            .map(CompletableFuture::join)
            .collect(Collectors.joining(","));
        System.out.println(result);  // A,B,C
    }
    
    public void anyOfDemo() {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(1000);
            return "A";
        });
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> "B");
        
        // 等待任一完成
        CompletableFuture<Object> anyDone = CompletableFuture.anyOf(f1, f2);
        Object result = anyDone.join();
        System.out.println("第一个完成: " + result);  // B (可能)
    }
}
```

## 总结

创建 CompletableFuture 的方式：

1. **supplyAsync()**：异步任务，有返回值
2. **runAsync()**：异步任务，无返回值
3. **completedFuture()**：已完成
4. **failedFuture()**：已失败
5. **`new CompletableFuture<>()`**：手动控制
6. **allOf()**：等待所有
7. **anyOf()**：等待任一
