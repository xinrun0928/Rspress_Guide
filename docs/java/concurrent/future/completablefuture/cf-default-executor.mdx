# CompletableFuture 默认线程池：ForkJoinPool.commonPool()

`CompletableFuture` 的异步任务默认使用 `ForkJoinPool.commonPool()` 执行。理解这个默认线程池的特性，对于性能调优至关重要。

## 默认线程池

```java
public class DefaultPoolDemo {
    
    public void defaultPool() {
        // 默认使用 ForkJoinPool.commonPool()
        // 无需显式创建线程池
        
        CompletableFuture.supplyAsync(() -> {
            // 由 commonPool 执行
            return process();
        });
        
        // commonPool 的并行度 = CPU核心数 - 1
        int parallelism = ForkJoinPool.commonPool().getParallelism();
        System.out.println("并行度: " + parallelism);
    }
}
```

## commonPool 的特性

```java
public class CommonPoolProperties {
    
    public void properties() {
        ForkJoinPool pool = ForkJoinPool.commonPool();
        
        // 并行度：CPU核心数 - 1
        int parallelism = pool.getParallelism();
        
        // 线程数上限
        int poolSize = pool.getPoolSize();
        
        // 是否使用公共线程池
        boolean isCommon = pool == ForkJoinPool.commonPool();
        
        System.out.println("并行度: " + parallelism);
        System.out.println("线程数: " + poolSize);
    }
}
```

## 使用默认线程池的注意事项

```java
public class DefaultPoolCautions {
    
    public void caution1() {
        // 注意：commonPool 是共享的
        // 不要提交长时间运行的任务，会阻塞其他任务
        
        CompletableFuture.supplyAsync(() -> {
            // 模拟长时间任务 - 不要这样做！
            Thread.sleep(10000);
            return "done";
        });
    }
    
    public void caution2() {
        // 正确的做法：使用自定义线程池
        ExecutorService executor = Executors.newFixedThreadPool(4);
        
        CompletableFuture.supplyAsync(() -> {
            return "done";
        }, executor);
    }
}
```

## 性能调优

```java
public class PerformanceTuning {
    
    public void tuning() {
        // 默认 commonPool 的并行度是 CPU核心数 - 1
        // 如果有大量短任务，可以增加并行度
        
        System.setProperty("java.util.concurrent.ForkJoinPool.common.parallelism", "8");
        
        // 或者在代码中创建专用线程池
        ExecutorService dedicatedPool = new ForkJoinPool(8);
        
        CompletableFuture.supplyAsync(() -> "task", dedicatedPool);
    }
}
```

## 总结

1. **默认线程池**：`ForkJoinPool.commonPool()`
2. **并行度**：CPU核心数 - 1
3. **注意**：共享资源，避免阻塞
4. **调优**：可用系统属性或自定义线程池
