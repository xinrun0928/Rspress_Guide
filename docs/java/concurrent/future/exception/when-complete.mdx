# whenComplete：无论成功失败都执行回调

`whenComplete` 允许在 `CompletableFuture` 完成时（无论成功还是失败）执行回调，访问结果或异常信息。

## 基本用法

```java
public class WhenCompleteDemo {
    
    public void basicUsage() {
        CompletableFuture.supplyAsync(() -> {
            if (Math.random() > 0.5) {
                throw new RuntimeException("随机错误");
            }
            return "success";
        })
        .whenComplete((result, ex) -> {
            if (ex != null) {
                System.err.println("异常: " + ex.getMessage());
            } else {
                System.out.println("结果: " + result);
            }
        })
        .join();
    }
}
```

## 实战：日志记录

```java
public class LoggingExample {
    
    public void withLogging() {
        CompletableFuture.supplyAsync(() -> {
            return processRequest();
        })
        .whenComplete((result, ex) -> {
            if (ex != null) {
                System.err.println("请求失败: " + ex.getMessage());
            } else {
                System.out.println("请求成功: " + result);
            }
        });
    }
    
    private String processRequest() {
        return "processed";
    }
}
```

## 总结

`whenComplete` 无论成功失败都会执行，适合日志记录、资源清理等场景。
