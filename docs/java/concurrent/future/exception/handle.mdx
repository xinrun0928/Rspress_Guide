# handle：携带异常信息的统一处理

`handle` 是 `CompletableFuture` 中最通用的异常处理方法，它同时接收结果和异常，允许根据情况决定返回值。

## 基本用法

```java
public class HandleDemo {
    
    public void basicUsage() {
        CompletableFuture.supplyAsync(() -> {
            if (Math.random() > 0.5) {
                throw new RuntimeException("错误");
            }
            return "success";
        })
        .handle((result, ex) -> {
            if (ex != null) {
                return "fallback: " + ex.getMessage();
            }
            return result;
        })
        .thenAccept(System.out::println)
        .join();
    }
}
```

## 与 exceptionally 对比

```java
public class Compare {
    
    public void comparison() {
        // exceptionally: 只处理异常
        CompletableFuture.supplyAsync(() -> "result")
            .exceptionally(ex -> "fallback");
        
        // handle: 同时处理结果和异常
        CompletableFuture.supplyAsync(() -> "result")
            .handle((result, ex) -> {
                if (ex != null) {
                    return "fallback";
                }
                return result.toUpperCase();
            });
    }
}
```

## 总结

`handle` 是最通用的处理方法，可以根据结果或异常返回不同的值。
