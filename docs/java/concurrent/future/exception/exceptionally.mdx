# exceptionally：异常时返回默认值

`exceptionally` 是 `CompletableFuture` 中处理异常最简单的方式，允许在异常时返回一个默认值。

## 基本用法

```java
public class ExceptionallyDemo {
    
    public void basicUsage() {
        CompletableFuture.supplyAsync(() -> {
            if (Math.random() > 0.5) {
                throw new RuntimeException("随机错误");
            }
            return "success";
        })
        .exceptionally(ex -> {
            // 处理异常，返回默认值
            System.err.println("发生错误: " + ex.getMessage());
            return "fallback";
        })
        .thenAccept(System.out::println)
        .join();
    }
}
```

## 实战：服务降级

```java
public class ServiceDegradation {
    
    public void degradedService() {
        CompletableFuture.supplyAsync(() -> {
            // 调用远程服务
            return callRemoteService();
        })
        .exceptionally(ex -> {
            // 服务调用失败，返回默认值
            System.err.println("远程服务调用失败: " + ex.getMessage());
            return getDefaultValue();
        })
        .thenAccept(value -> {
            System.out.println("结果: " + value);
        });
    }
    
    private String callRemoteService() {
        throw new RuntimeException("服务不可用");
    }
    
    private String getDefaultValue() {
        return "默认结果";
    }
}
```

## 总结

`exceptionally` 在异常时返回默认值，实现服务降级。
