# allOf / anyOf：多任务组合

`allOf` 和 `anyOf` 是 `CompletableFuture` 中用于组合多个独立任务的强大工具。

## allOf：等待所有任务完成

```java
public class AllOfDemo {
    
    public void allOfDemo() throws Exception {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(1000);
            return "A";
        });
        
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(500);
            return "B";
        });
        
        CompletableFuture<String> f3 = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(800);
            return "C";
        });
        
        // 等待所有完成
        CompletableFuture.allOf(f1, f2, f3).join();
        
        // 获取所有结果
        String a = f1.join();
        String b = f2.join();
        String c = f3.join();
        
        System.out.println(a + b + c);  // "ABC"
    }
}
```

## anyOf：任一任务完成

```java
public class AnyOfDemo {
    
    public void anyOfDemo() {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(1000);
            return "A";
        });
        
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(500);
            return "B";
        });
        
        CompletableFuture<String> f3 = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(800);
            return "C";
        });
        
        // 任一完成
        CompletableFuture.anyOf(f1, f2, f3)
            .thenAccept(result -> {
                System.out.println("第一个完成: " + result);  // B (最快)
            })
            .join();
    }
}
```

## 实战：快速失败

```java
public class FastFail {
    
    public void demo() {
        CompletableFuture<String>[] tasks = new CompletableFuture[] {
            CompletableFuture.supplyAsync(() -> queryPrimary()),
            CompletableFuture.supplyAsync(() -> querySecondary()),
            CompletableFuture.supplyAsync(() -> queryTertiary())
        };
        
        // 任一成功即可
        CompletableFuture.anyOf(tasks)
            .thenAccept(result -> {
                System.out.println("获得结果: " + result);
            })
            .exceptionally(ex -> {
                System.err.println("所有查询失败");
                return null;
            });
    }
    
    private String queryPrimary() { return "primary"; }
    private String querySecondary() { return "secondary"; }
    private String queryTertiary() { return "tertiary"; }
}
```

## 总结

| 方法 | 说明 | 返回 |
|------|------|------|
| `allOf` | 等待所有任务完成 | `CompletableFuture<Void>` |
| `anyOf` | 任一任务完成 | `CompletableFuture<Object>` |
