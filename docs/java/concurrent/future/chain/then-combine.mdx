# thenCombine / thenAcceptBoth：组合两个 Future

`thenCombine` 和 `thenAcceptBoth` 用于组合两个独立的 `CompletableFuture`，让它们并行执行后合并结果。

## thenCombine：合并结果

```java
public class ThenCombineDemo {
    
    public void thenCombineDemo() {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> "Hello");
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> "World");
        
        // 组合两个 Future，结果合并
        String result = f1.thenCombine(f2, (a, b) -> a + " " + b)
            .join();
        
        System.out.println(result);  // "Hello World"
    }
    
    public void moreComplex() {
        CompletableFuture<Integer> f1 = CompletableFuture.supplyAsync(() -> 10);
        CompletableFuture<Integer> f2 = CompletableFuture.supplyAsync(() -> 20);
        
        Integer sum = f1.thenCombine(f2, Integer::sum)
            .join();
        System.out.println("Sum: " + sum);  // 30
    }
}
```

## thenAcceptBoth：消费两个结果

```java
public class ThenAcceptBothDemo {
    
    public void thenAcceptBothDemo() {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> "Hello");
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> "World");
        
        // 消费两个结果，但不返回新值
        f1.thenAcceptBoth(f2, (a, b) -> {
            System.out.println("a = " + a);
            System.out.println("b = " + b);
        })
        .join();
    }
}
```

## thenCombine vs thenAcceptBoth

```java
public class Comparison {
    
    public void demo() {
        CompletableFuture<String> f1 = CompletableFuture.supplyAsync(() -> "a");
        CompletableFuture<String> f2 = CompletableFuture.supplyAsync(() -> "b");
        
        // thenCombine: 有返回值
        CompletableFuture<String> result1 = f1.thenCombine(f2, (a, b) -> a + b);
        
        // thenAcceptBoth: 无返回值
        CompletableFuture<Void> result2 = f1.thenAcceptBoth(f2, (a, b) -> {
            System.out.println(a + b);
        });
    }
}
```

## 实战：聚合查询

```java
public class AggregatedQuery {
    
    public void query() {
        CompletableFuture<String> userFuture = CompletableFuture.supplyAsync(() -> 
            queryUserInfo());
        
        CompletableFuture<String> orderFuture = CompletableFuture.supplyAsync(() -> 
            queryUserOrders());
        
        CompletableFuture<String> statsFuture = CompletableFuture.supplyAsync(() -> 
            queryUserStats());
        
        // 等待所有查询完成，聚合结果
        userFuture.thenCombine(orderFuture, (user, orders) -> {
            return "用户: " + user + ", 订单: " + orders;
        })
        .thenCombine(statsFuture, (result, stats) -> {
            return result + ", 统计: " + stats;
        })
        .thenAccept(System.out::println);
    }
    
    private String queryUserInfo() { return "张三"; }
    private String queryUserOrders() { return "10单"; }
    private String queryUserStats() { return "VIP"; }
}
```

## 总结

| 方法 | 参数 | 返回 | 使用场景 |
|------|------|------|---------|
| `thenCombine` | BiFunction | CompletableFuture | 需要返回新值 |
| `thenAcceptBoth` | BiConsumer | `CompletableFuture<Void>` | 只消费结果 |
