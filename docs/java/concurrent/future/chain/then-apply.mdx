# thenApply / thenApplyAsync：同步与异步转换

`thenApply` 和 `thenApplyAsync` 是 `CompletableFuture` 中最常用的转换方法，用于对前一个阶段的结果进行转换处理。理解它们的区别，对于正确使用异步编程至关重要。

## 基本用法

```java
public class ThenApplyDemo {
    
    public void basicUsage() {
        CompletableFuture.supplyAsync(() -> {
            return 1;
        })
        .thenApply(x -> {
            return x * 2;  // 2
        })
        .thenApply(x -> {
            return x + 1;  // 3
        })
        .thenAccept(System.out::println);  // 输出 3
    }
}
```

## thenApply：同步转换

```java
public class SyncTransform {
    
    public void thenApply() {
        CompletableFuture.supplyAsync(() -> {
            return 10;
        })
        .thenApply(x -> {
            // 在调用线程执行（不创建新线程）
            System.out.println("thenApply 执行: " + Thread.currentThread().getName());
            return x * 2;
        })
        .join();  // 阻塞直到完成
    }
}
```

## thenApplyAsync：异步转换

```java
public class AsyncTransform {
    
    public void thenApplyAsync() {
        CompletableFuture.supplyAsync(() -> {
            return 10;
        })
        .thenApplyAsync(x -> {
            // 在 ForkJoinPool.commonPool() 执行
            System.out.println("thenApplyAsync 执行: " + Thread.currentThread().getName());
            return x * 2;
        })
        .join();
    }
    
    // 指定线程池
    public void withExecutor() {
        ExecutorService executor = Executors.newFixedThreadPool(4);
        
        CompletableFuture.supplyAsync(() -> 10, executor)
            .thenApplyAsync(x -> x * 2, executor)  // 在指定线程池执行
            .join();
    }
}
```

## 两者区别

```java
public class Difference {
    
    public void demo() {
        // thenApply: 同步执行，使用前一个阶段的线程
        CompletableFuture.supplyAsync(() -> {
            System.out.println("supplyAsync: " + Thread.currentThread().getName());
            return 1;
        })
        .thenApply(x -> {
            System.out.println("thenApply: " + Thread.currentThread().getName());
            return x * 2;
        });
        
        // thenApplyAsync: 异步执行，使用公共线程池
        CompletableFuture.supplyAsync(() -> {
            System.out.println("supplyAsync: " + Thread.currentThread().getName());
            return 1;
        })
        .thenApplyAsync(x -> {
            System.out.println("thenApplyAsync: " + Thread.currentThread().getName());
            return x * 2;
        });
    }
}
```

## 实战：多步骤数据处理

```java
public class DataProcessing {
    
    public void process() {
        CompletableFuture.supplyAsync(() -> {
            // 第一步：从数据库查询
            return queryFromDB();
        })
        .thenApply(data -> {
            // 第二步：数据转换
            return transform(data);
        })
        .thenApply(data -> {
            // 第三步：数据验证
            return validate(data);
        })
        .thenAccept(result -> {
            // 第四步：保存结果
            saveResult(result);
        })
        .exceptionally(ex -> {
            System.err.println("处理失败: " + ex.getMessage());
            return null;
        });
    }
    
    private String queryFromDB() { return "原始数据"; }
    private String transform(String data) { return "转换后"; }
    private String validate(String data) { return "验证通过"; }
    private void saveResult(String result) { System.out.println(result); }
}
```

## 总结

| 方法 | 执行线程 | 使用场景 |
|------|---------|---------|
| `thenApply` | 前一个阶段的线程 | 轻量转换 |
| `thenApplyAsync` | 公共线程池/指定线程池 | 需要并行 |
