# thenCompose：扁平化链式调用

`thenCompose` 是 `CompletableFuture` 中用于处理**依赖关系**的关键方法。它解决了 `thenApply` 嵌套 `CompletableFuture` 导致的"Future of Future"问题。

## 问题：嵌套 Future

```java
public class NestedProblem {
    
    public void problem() {
        // thenApply 返回 CompletableFuture<T>
        // 如果 T 本身是 CompletableFuture，会变成 CompletableFuture<CompletableFuture<T>>
        
        CompletableFuture<CompletableFuture<String>> nested = 
            CompletableFuture.supplyAsync(() -> "hello")
                .thenApply(s -> CompletableFuture.supplyAsync(() -> s + " world"));
        
        // 需要双重 join
        String result = nested.join().join();
    }
}
```

## 解决：thenCompose 扁平化

```java
public class ThenComposeSolution {
    
    public void thenComposeDemo() {
        // thenCompose 扁平化链式调用
        String result = CompletableFuture.supplyAsync(() -> "hello")
            .thenCompose(s -> CompletableFuture.supplyAsync(() -> s + " world"))
            .join();
        
        System.out.println(result);  // "hello world"
    }
}
```

## thenApply vs thenCompose

```java
public class ApplyVsCompose {
    
    public void compare() {
        // thenApply: 转换函数返回 T
        CompletableFuture<Integer> f1 = CompletableFuture.supplyAsync(() -> "hello")
            .thenApply(s -> s.length());  // s -> Integer
        
        // thenCompose: 转换函数返回 CompletableFuture<T>
        CompletableFuture<Integer> f2 = CompletableFuture.supplyAsync(() -> "hello")
            .thenCompose(s -> CompletableFuture.supplyAsync(() -> s.length()));  // s -> CompletableFuture<Integer>
    }
}
```

## 实战：多服务调用

```java
public class MultiServiceCall {
    
    public void serviceChain() {
        CompletableFuture.supplyAsync(() -> {
            // 第一步：查询用户
            return queryUser();
        })
        .thenCompose(user -> {
            // 第二步：依赖用户信息，查询订单
            return CompletableFuture.supplyAsync(() -> queryOrders(user));
        })
        .thenCompose(orders -> {
            // 第三步：依赖订单，查询商品详情
            return CompletableFuture.supplyAsync(() -> queryProducts(orders));
        })
        .thenAccept(products -> {
            System.out.println("商品: " + products);
        })
        .exceptionally(ex -> {
            System.err.println("调用失败: " + ex.getMessage());
            return null;
        });
    }
    
    private String queryUser() { return "user"; }
    private String queryOrders(String user) { return "orders"; }
    private String queryProducts(String orders) { return "products"; }
}
```

## 总结

| 方法 | 函数签名 | 返回类型 |
|------|---------|---------|
| `thenApply` | `Function<T, R>` | `CompletableFuture<R>` |
| `thenCompose` | `Function<T, CompletableFuture<R>>` | `CompletableFuture<R>` |
