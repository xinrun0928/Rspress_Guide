# 多服务并行调用：thenCompose 链式编排实战

在微服务架构中，经常需要调用多个服务并将结果组合。`thenCompose` 是实现这种链式调用的利器。

## 场景：用户中心

```java
public class UserCenter {
    
    public void getUserDashboard(String userId) {
        CompletableFuture.supplyAsync(() -> {
            // 1. 查询用户基本信息
            return queryUserInfo(userId);
        })
        .thenCompose(user -> {
            // 2. 根据用户信息查询订单
            return CompletableFuture.supplyAsync(() -> 
                queryUserOrders(user.getId())
            );
        })
        .thenCompose(orders -> {
            // 3. 查询推荐商品
            return CompletableFuture.supplyAsync(() -> 
                queryRecommendedProducts(orders)
            );
        })
        .thenAccept(recommendations -> {
            // 4. 组装并展示
            System.out.println("推荐: " + recommendations);
        })
        .exceptionally(ex -> {
            System.err.println("获取仪表盘失败: " + ex.getMessage());
            return null;
        });
    }
    
    private User queryUserInfo(String userId) { return new User(userId); }
    private List<Order> queryUserOrders(String userId) { return new ArrayList<>(); }
    private List<Product> queryRecommendedProducts(List<Order> orders) { return new ArrayList<>(); }
    
    static class User { String id; User(String id) { this.id = id; } }
    static class Order {}
    static class Product {}
}
```

## 总结

`thenCompose` 适合处理依赖关系的链式调用。
