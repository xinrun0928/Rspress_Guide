# allOf 聚合多路查询结果并处理异常

`allOf` 适合并行调用多个独立服务，然后聚合结果。

## 场景：聚合搜索

```java
public class AggregatedSearch {
    
    public void search(String query) {
        CompletableFuture<List<Product>> webSearch = CompletableFuture.supplyAsync(() ->
            searchFromWeb(query)
        );
        
        CompletableFuture<List<Product>> dbSearch = CompletableFuture.supplyAsync(() ->
            searchFromDB(query)
        );
        
        CompletableFuture<List<Product>> cacheSearch = CompletableFuture.supplyAsync(() ->
            searchFromCache(query)
        );
        
        // 等待所有搜索完成
        CompletableFuture.allOf(webSearch, dbSearch, cacheSearch)
            .thenAccept(v -> {
                // 聚合结果
                List<Product> results = new ArrayList<>();
                results.addAll(webSearch.join());
                results.addAll(dbSearch.join());
                results.addAll(cacheSearch.join());
                System.out.println("找到 " + results.size() + " 个结果");
            })
            .exceptionally(ex -> {
                System.err.println("搜索失败: " + ex.getMessage());
                return null;
            });
    }
    
    private List<Product> searchFromWeb(String query) { return new ArrayList<>(); }
    private List<Product> searchFromDB(String query) { return new ArrayList<>(); }
    private List<Product> searchFromCache(String query) { return new ArrayList<>(); }
    
    static class Product {}
}
```

## 总结

`allOf` 适合并行查询后的结果聚合。
