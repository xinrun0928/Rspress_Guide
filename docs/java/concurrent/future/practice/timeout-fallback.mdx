# anyOf 实现超时降级与快速失败

`anyOf` 适合实现超时降级和快速失败策略。

## 场景：多源查询，取最快响应

```java
public class FastestResponse {
    
    public void query(String id) {
        CompletableFuture<String> primarySource = CompletableFuture.supplyAsync(() ->
            queryPrimary(id)  // 慢：1秒
        );
        
        CompletableFuture<String> backupSource = CompletableFuture.supplyAsync(() ->
            queryBackup(id)   // 快：200毫秒
        );
        
        // 任一完成即可
        CompletableFuture.anyOf(primarySource, backupSource)
            .thenAccept(result -> {
                System.out.println("最快响应: " + result);
            });
    }
    
    private String queryPrimary(String id) throws InterruptedException {
        Thread.sleep(1000);
        return "primary";
    }
    
    private String queryBackup(String id) throws InterruptedException {
        Thread.sleep(200);
        return "backup";
    }
}
```

## 场景：超时降级

```java
public class TimeoutDegrade {
    
    public void withTimeout() {
        CompletableFuture<String> slowService = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(5000);
            return "slow result";
        });
        
        CompletableFuture<String> timeout = CompletableFuture.supplyAsync(() -> {
            Thread.sleep(1000);
            return "timeout";
        });
        
        CompletableFuture.anyOf(slowService, timeout)
            .thenAccept(result -> {
                if ("timeout".equals(result)) {
                    System.out.println("服务超时，使用降级结果");
                } else {
                    System.out.println("正常结果: " + result);
                }
            });
    }
}
```

## 总结

`anyOf` 适合实现快速失败和超时降级策略。
