# Future 接口方法详解：get()、cancel()、isDone()

`Future` 接口定义了异步操作结果的契约。它的方法不多，但每个都有其特定的行为和注意事项。理解这些方法的细节，才能正确使用 Future。

## Future 接口定义

```java
public interface Future<V> {
    
    // 获取结果，会阻塞
    V get() throws InterruptedException, ExecutionException;
    
    // 带超时的获取
    V get(long timeout, TimeUnit unit) 
        throws InterruptedException, ExecutionException, TimeoutException;
    
    // 判断任务是否完成
    boolean isDone();
    
    // 判断任务是否被取消
    boolean isCancelled();
    
    // 取消任务
    boolean cancel(boolean mayInterruptIfRunning);
}
```

## get() 方法

### 基本用法

```java
public class GetMethod {
    
    public void basicGet() throws Exception {
        FutureTask<String> future = new FutureTask<>(() -> "result");
        new Thread(future).start();
        
        // get() 会阻塞，直到任务完成
        String result = future.get();
        System.out.println(result);
    }
    
    // 带超时
    public void getWithTimeout() throws Exception {
        FutureTask<String> future = new FutureTask<>(() -> {
            Thread.sleep(5000);
            return "result";
        });
        new Thread(future).start();
        
        try {
            // 最多等待 2 秒
            String result = future.get(2, TimeUnit.SECONDS);
        } catch (TimeoutException e) {
            System.out.println("获取结果超时");
        }
    }
}
```

### get() 的异常

```java
public class GetExceptions {
    
    public void exceptionHandling() {
        FutureTask<String> future = new FutureTask<>(() -> {
            throw new RuntimeException("任务失败");
        });
        new Thread(future).start();
        
        try {
            future.get();
        } catch (InterruptedException e) {
            // 线程被中断
            Thread.currentThread().interrupt();
        } catch (ExecutionException e) {
            // 任务执行中抛出异常
            Throwable cause = e.getCause();
            System.out.println("任务异常: " + cause.getMessage());
        }
    }
}
```

## cancel() 方法

### 基本用法

```java
public class CancelMethod {
    
    public void basicCancel() {
        FutureTask<String> future = new FutureTask<>(() -> {
            // 模拟长时间运行的任务
            for (int i = 0; i < Integer.MAX_VALUE; i++) {
                if (Thread.currentThread().isInterrupted()) {
                    throw new InterruptedException("任务被中断");
                }
            }
            return "完成";
        });
        
        new Thread(future).start();
        
        // 取消任务
        boolean cancelled = future.cancel(true);  // true: 尝试中断正在运行的线程
        System.out.println("取消结果: " + cancelled);
    }
    
    // cancel(true) vs cancel(false)
    public void cancelVsInterrupt() {
        // cancel(true):
        //   - 如果任务未开始，会阻止其开始
        //   - 如果任务正在运行，会尝试中断执行线程
        
        // cancel(false):
        //   - 如果任务未开始，会阻止其开始
        //   - 如果任务正在运行，不做任何事
    }
}
```

### cancel() 的行为

```java
public class CancelBehavior {
    
    public void cancelBehavior() throws Exception {
        // 场景1：任务未开始
        FutureTask<String> notStarted = new FutureTask<>(() -> {
            Thread.sleep(10000);
            return "done";
        });
        
        notStarted.cancel(false);  // 阻止任务开始
        // isDone() = true
        // isCancelled() = true
        // get() 抛出 CancellationException
        
        // 场景2：任务已完成
        FutureTask<String> completed = new FutureTask<>(() -> "done");
        completed.run();
        
        completed.cancel(false);  // 无效，任务已完成
        // isCancelled() = false
        
        // 场景3：任务正在运行
        FutureTask<String> running = new FutureTask<>(() -> {
            for (int i = 0; i < Integer.MAX_VALUE; i++) {
                if (Thread.currentThread().isInterrupted()) {
                    return "interrupted";
                }
            }
            return "done";
        });
        
        new Thread(running).start();
        running.cancel(true);  // 尝试中断
    }
}
```

## isDone() 方法

```java
public class IsDoneMethod {
    
    public void isDoneDemo() throws Exception {
        FutureTask<String> future = new FutureTask<>(() -> {
            Thread.sleep(2000);
            return "result";
        });
        
        System.out.println("初始状态: " + future.isDone());  // false
        
        new Thread(future).start();
        
        while (!future.isDone()) {
            System.out.println("任务执行中...");
            Thread.sleep(500);
        }
        
        System.out.println("完成: " + future.get());
    }
    
    // isDone() 返回 true 的情况：
    // 1. 任务正常完成
    // 2. 任务抛出异常
    // 3. 任务被取消
}
```

## isCancelled() 方法

```java
public class IsCancelledMethod {
    
    public void isCancelledDemo() throws Exception {
        FutureTask<String> future = new FutureTask<>(() -> {
            Thread.sleep(5000);
            return "done";
        });
        
        new Thread(future).start();
        Thread.sleep(100);
        
        future.cancel(true);
        
        // 状态检查
        System.out.println("isDone: " + future.isDone());      // true
        System.out.println("isCancelled: " + future.isCancelled());  // true
        
        // 取消后调用 get() 会抛出 CancellationException
        try {
            future.get();
        } catch (CancellationException e) {
            System.out.println("任务被取消");
        }
    }
}
```

## 综合示例

```java
public class FutureComprehensiveDemo {
    
    private final ExecutorService executor = Executors.newFixedThreadPool(4);
    
    public void comprehensiveDemo() {
        Callable<String> longTask = () -> {
            for (int i = 0; i < 10; i++) {
                if (Thread.currentThread().isInterrupted()) {
                    return "被中断";
                }
                Thread.sleep(500);
            }
            return "完成";
        };
        
        Future<String> future = executor.submit(longTask);
        
        // 模拟主线程处理
        for (int i = 0; i < 5; i++) {
            if (future.isDone()) {
                try {
                    String result = future.get();
                    System.out.println("任务完成: " + result);
                    break;
                } catch (Exception e) {
                    e.printStackTrace();
                }
            } else {
                System.out.println("任务执行中...");
                Thread.sleep(200);
            }
        }
        
        // 超时检查
        try {
            String result = future.get(1, TimeUnit.SECONDS);
        } catch (TimeoutException e) {
            System.out.println("超时，取消任务");
            future.cancel(true);
        } catch (Exception e) {
            e.printStackTrace();
        }
        
        // 最终状态
        System.out.println("isDone: " + future.isDone());
        System.out.println("isCancelled: " + future.isCancelled());
    }
}
```

## 总结

Future 方法详解：

1. **get()**：阻塞获取结果
2. **get(timeout, unit)**：带超时的获取
3. **isDone()**：检查是否完成
4. **isCancelled()**：检查是否被取消
5. **cancel()**：取消任务

状态关系：
- isDone() = true 的情况：正常完成、异常完成、取消
- isCancelled() = true：只针对取消
