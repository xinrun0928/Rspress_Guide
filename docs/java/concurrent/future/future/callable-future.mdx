# Callable、Future、FutureTask 三者关系

在 Java 并发编程中，`Callable`、`Future` 和 `FutureTask` 是异步编程的基础组件。它们之间的关系常常让初学者困惑。`Callable` 是任务的抽象，`Future` 是异步结果的契约，`FutureTask` 则是连接两者的桥梁。理解它们的关系，是深入学习并发编程的第一步。

## 三者的定位

```java
// Callable：定义任务的接口，类似 Runnable，但可以有返回值
@FunctionalInterface
public interface Callable<V> {
    V call() throws Exception;
}

// Future：异步结果的接口，定义了获取结果的方法
public interface Future<V> {
    V get() throws InterruptedException, ExecutionException;
    V get(long timeout, TimeUnit unit) throws ...;
    boolean isDone();
    boolean isCancelled();
    boolean cancel(boolean mayInterruptIfRunning);
}

// FutureTask：Callable 和 Future 的实现类
public class FutureTask<V> implements RunnableFuture<V> {
    // RunnableFuture 继承自 Runnable 和 Future
}

// RunnableFuture
public interface RunnableFuture<V> extends Runnable, Future<V> {
    void run();
}
```

## Callable vs Runnable

```java
public class CallableVsRunnable {
    
    // Runnable：不能返回结果，不能抛出受检异常
    public void runnableExample() {
        Runnable task = () -> {
            System.out.println("执行任务");
        };
        new Thread(task).start();
    }
    
    // Callable：可以返回结果，可以抛出受检异常
    public void callableExample() throws Exception {
        Callable<String> task = () -> {
            return "任务结果";
        };
        
        String result = task.call();  // 直接调用
        System.out.println(result);
    }
}
```

## FutureTask：桥梁

```java
public class FutureTaskDemo {
    
    // FutureTask 是 Callable 和 Future 的桥梁
    // 它包装 Callable，使其可以被 Thread 或 Executor 执行
    // 同时实现了 Future，可以获取异步结果
    
    public void basicUsage() throws Exception {
        // 1. 创建 Callable
        Callable<Integer> callable = () -> {
            Thread.sleep(1000);
            return 42;
        };
        
        // 2. 创建 FutureTask
        FutureTask<Integer> futureTask = new FutureTask<>(callable);
        
        // 3. 用 Thread 执行
        new Thread(futureTask).start();
        
        // 4. 获取结果（会阻塞）
        Integer result = futureTask.get();
        System.out.println(result);  // 42
    }
}
```

## Future 接口详解

```java
public class FutureMethods {
    
    public void futureMethodDemo() throws Exception {
        FutureTask<String> future = new FutureTask<>(() -> {
            Thread.sleep(1000);
            return "result";
        });
        
        new Thread(future).start();
        
        // get()：获取结果，一直阻塞直到完成
        String result = future.get();  // 阻塞
        
        // get(timeout)：带超时获取
        String result2 = future.get(5, TimeUnit.SECONDS);
        
        // isDone()：检查是否完成
        while (!future.isDone()) {
            System.out.println("任务还在执行...");
            Thread.sleep(100);
        }
        
        // isCancelled()：检查是否被取消
        boolean cancelled = future.isCancelled();
        
        // cancel()：取消任务
        boolean cancelled2 = future.cancel(true);  // true 表示中断执行
    }
}
```

## 使用场景

### 场景1：计算结果

```java
public class ComputeResult {
    
    public void compute() throws Exception {
        Callable<Long> task = () -> {
            long sum = 0;
            for (long i = 0; i < 1_000_000_000; i++) {
                sum += i;
            }
            return sum;
        };
        
        FutureTask<Long> future = new FutureTask<>(task);
        new Thread(future).start();
        
        // 做其他事情
        doOtherWork();
        
        // 获取计算结果
        Long result = future.get();
        System.out.println("计算结果: " + result);
    }
    
    private void doOtherWork() {
        System.out.println("执行其他工作...");
    }
}
```

### 场景2：带超时的操作

```java
public class TimeoutOperation {
    
    public void withTimeout() throws Exception {
        Callable<String> task = () -> {
            // 模拟耗时操作
            Thread.sleep(3000);
            return "完成";
        };
        
        FutureTask<String> future = new FutureTask<>(task);
        ExecutorService executor = Executors.newSingleThreadExecutor();
        executor.submit(future);
        
        try {
            // 最多等待 2 秒
            String result = future.get(2, TimeUnit.SECONDS);
            System.out.println("结果: " + result);
        } catch (TimeoutException e) {
            System.out.println("操作超时");
            future.cancel(true);
        } finally {
            executor.shutdown();
        }
    }
}
```

### 场景3：取消任务

```java
public class CancelTask {
    
    public void cancelDemo() throws Exception {
        Callable<String> longTask = () -> {
            for (int i = 0; i < Integer.MAX_VALUE; i++) {
                if (Thread.currentThread().isInterrupted()) {
                    throw new InterruptedException("任务被取消");
                }
                // 执行计算
            }
            return "完成";
        };
        
        FutureTask<String> future = new FutureTask<>(longTask);
        Thread thread = new Thread(future);
        thread.start();
        
        Thread.sleep(100);  // 模拟等待
        
        // 取消任务
        boolean cancelled = future.cancel(true);
        System.out.println("任务被取消: " + cancelled);
        System.out.println("是否完成: " + future.isDone());
        System.out.println("是否取消: " + future.isCancelled());
    }
}
```

## 与 ExecutorService 配合

```java
public class Executor配合 {
    
    private final ExecutorService executor = Executors.newFixedThreadPool(4);
    
    // 提交 Callable，返回 Future
    public void submitCallable() throws Exception {
        Future<String> future = executor.submit(() -> {
            Thread.sleep(1000);
            return "Hello";
        });
        
        String result = future.get();  // 阻塞获取结果
        System.out.println(result);
    }
    
    // 提交 Runnable，返回 Future
    public void submitRunnable() throws Exception {
        Future<?> future = executor.submit(() -> {
            System.out.println("执行任务");
        }, null);
        
        future.get();  // 等待完成
    }
    
    // 提交 FutureTask
    public void submitFutureTask() throws Exception {
        Callable<Integer> callable = () -> 42;
        FutureTask<Integer> task = new FutureTask<>(callable);
        
        executor.submit(task);  // 可以提交 FutureTask
        Integer result = task.get();
        System.out.println(result);
    }
}
```

## 总结

三者的关系：

1. **Callable**：定义有返回值、可抛异常的任务
2. **Future**：定义异步结果的行为
3. **FutureTask**：实现 RunnableFuture，连接 Callable 和 Future

使用流程：
1. 创建 Callable
2. 包装为 FutureTask
3. 提交给 Executor 或 Thread 执行
4. 调用 Future.get() 获取结果
