# Spring Boot Banner：启动画面也可以很个性

每次启动 Spring Boot 应用，你都会看到一个 ASCII 艺术风格的 Banner。你有没有想过，这个 Banner 是怎么来的？怎么自定义它？

这一节，我们来聊聊 Spring Boot Banner 的定制。

## 一、Banner 概述

### 默认 Banner

Spring Boot 的默认 Banner 是一个大号的 Spring Logo：

```
  .   ____          _            __ _ _
 /\\ / ___'_ __ _ _(_)_ __  __ _ \ \ \ \
( ( )\___ | '_ | '_| | '_ \/ _` | \ \ \ \
 \\/  ___)| |_)| | | | | || (_| |  ) ) ) )
  '  |____| .__|_| |_|_| |_\__, | / / / /
 =========|_|==============|___/=/_/_/_/
 :: Spring Boot ::                (v3.2.0)
```

### Banner 的作用

1. **品牌标识**：展示 Spring Boot 的品牌
2. **版本信息**：清晰显示当前版本
3. **视觉分隔**：在日志中起到视觉分隔作用

## 二、自定义 Banner 文件

### 文件位置

Spring Boot 按以下顺序查找 Banner 文件：

1. `classpath:banner.txt`
2. `classpath:banner.gif`
3. `classpath:banner.jpg`
4. `classpath:banner.png`
5. `file:./banner.txt`（项目根目录）

### 创建文本 Banner

```text title="src/main/resources/banner.txt"
 ____             _        _     ____            _
|  _ \ ___   ___ | | _____| |_  / ___|  ___ _ __(_) ___
| |_) / _ \ / _ \| |/ / _ \ __| \___ \ / _ \ '__| |/ _ \
|  __/ (_) | (_) |   <  __/ |_   ___) |  __/ |  | | (_) |
|_|   \___/ \___/|_|\_\___|\__| |____/ \___|_|  |_|\___/

:: ${application.title} :: (v${application.version})
:: Spring Boot :: (v${spring-boot.version})
```

### 使用图片 Banner

支持 GIF、JPG、PNG 格式的图片。Spring Boot 会自动将图片转换为 ASCII 艺术：

```bash
# 将图片放在 resources 目录下
src/main/resources/banner.png
```

> [!TIP]
> 图片 Banner 会自动调整大小以适应控制台宽度。如果图片过大，ASCII 转换可能不太美观，建议使用文本 Banner。

## 三、Banner 变量

Spring Boot 提供了多个内置变量：

### 可用变量

| 变量 | 说明 | 示例输出 |
|---|---|---|
| `${application.version}` | 应用版本 | 1.0.0 |
| `${application.title}` | 应用标题 | My Application |
| `${spring-boot.version}` | Spring Boot 版本 | 3.2.0 |
| `${spring-boot.formatted-version}` | 带前缀的版本 | (v3.2.0) |
| `${MANIFEST.MF.xxx}` | Manifest 属性 | |
| `${Ansi.NAME}` | ANSI 颜色代码 | |
| `${AnsiColor.NAME}` | ANSI 颜色代码（旧） | |

### Manifest 属性

如果 `MANIFEST.MF` 包含以下属性，可以直接引用：

```properties
Implementation-Title: My Application
Implementation-Version: 1.0.0
Implementation-Author: Spring Team
```

引用方式：

```text
Title:      ${MANIFEST.MF.Implement-Title}
Version:    ${MANIFEST.MF.Implementation-Version}
Author:     ${MANIFEST.MF.Implementation-Author}
```

### ANSI 颜色

支持 ANSI 颜色代码，让 Banner 更生动：

```text
// ${AnsiColor.BRIGHT_GREEN} 绿色
// ${AnsiColor.BRIGHT_BLUE} 蓝色
// ${AnsiColor.BRIGHT_YELLOW} 黄色
// ${AnsiColor.BRIGHT_RED} 红色

  .   ____          _            __ _ _
 /\\ / ___'_ __ _ _(_)_ __  __ _ \ \ \ \
( ( )\___| '_ | '_| | '_ \/ _` | \ \ \ \
 \\/  ___)| |_)| | | | | || (_| |  ) ) ) )
  '  |____| .__|_| |_|_| |_\__, | / / / /
 =========|_|==============|___/=/_/_/_/

:: ${AnsiColor.GREEN} Spring Boot :: ${AnsiColor.DEFAULT}
         ${application.version}
```

### ANSI 样式

```text
${AnsiStyle.NORMAL}     普通
${AnsiStyle.BOLD}       粗体
${AnsiStyle.ITALIC}     斜体
${AnsiStyle.BOLD_ITALIC} 粗体+斜体
```

## 四、动态 Banner

### 实现 Banner 接口

可以通过实现 `Banner` 接口创建动态 Banner：

```java title="src/main/java/com/example/banner/CustomBanner.java"
package com.example.banner;

import org.springframework.boot.Banner;
import org.springframework.boot.SpringApplication;
import org.springframework.core.env.Environment;
import java.io.PrintStream;

public class CustomBanner implements Banner {

    @Override
    public void printBanner(Environment environment, Class<?> sourceClass,
            PrintStream out) {
        out.println("========================================");
        out.println("           My Custom Application         ");
        out.println("========================================");
        out.println();
        out.println("  Version  : " + environment.getProperty("spring.application.version", "1.0.0"));
        out.println("  Author   : " + environment.getProperty("spring.application.author", "Unknown"));
        out.println("  Profile  : " + environment.getActiveProfiles().length > 0
                ? environment.getActiveProfiles()[0] : "default");
        out.println();
        out.println("========================================");
    }
}
```

### 注册动态 Banner

**方式一：通过 main 方法设置**

```java title="src/main/java/com/example/Application.java"
package com.example;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;
import com.example.banner.CustomBanner;

@SpringBootApplication
public class Application {

    public static void main(String[] args) {
        SpringApplication app = new SpringApplication(Application.class);
        app.setBanner(new CustomBanner());
        app.run(args);
    }
}
```

**方式二：通过 SpringApplicationBuilder 设置**

```java
new SpringApplicationBuilder(Application.class)
    .banner(new CustomBanner())
    .run(args);
```

## 五、配置文件配置

### 控制 Banner 显示

```yaml title="application.yml"
spring:
  main:
    banner-mode: console  # console / log / off
```

| 值 | 说明 |
|---|
| `console` | 输出到控制台（默认） |
| `log` | 输出到日志 |
| `off` | 禁用 Banner |

### 代码配置

```java
SpringApplication app = new SpringApplication(Application.class);
app.setBannerMode(Banner.Mode.LOG);
app.run(args);
```

## 八、常见问题

### 问题 1：Banner 不显示

**原因**：`spring.main.banner-mode` 设置为 `off`。

**解决方案**：

```yaml
spring:
  main:
    banner-mode: console
```

### 问题 2：中文乱码

**原因**：控制台编码不支持中文。

**解决方案**：

```bash
java -Dfile.encoding=UTF-8 -jar app.jar
```

### 问题 3：版本变量为空

**原因**：`MANIFEST.MF` 中没有定义相应属性。

**解决方案**：确保 `pom.xml` 中配置了 Maven JAR 插件：

```xml title="pom.xml"
<build>
    <plugins>
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
        </plugin>
    </plugins>
</build>
```

## 九、总结

Banner 定制的方式：

| 方式 | 配置位置 | 优先级 |
|---|---|---|
| 文本文件 | `classpath:banner.txt` | 1 |
| 图片文件 | `classpath:banner.gif/jpg/png` | 2 |
| 代码实现 | `Banner` 接口 + `setBanner()` | 3 |
| 配置文件 | `spring.main.banner-mode` | 控制显示/隐藏 |

Banner 虽然不是核心功能，但它能为你的应用增添品牌识别度。一个精心设计的 Banner，不仅美观，还能传达版本、作者等重要信息。
