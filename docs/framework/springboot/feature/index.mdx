# Spring Boot 常用功能

这一章介绍 Spring Boot 开发中最常用的功能，包括日志配置、异常处理、静态资源、CORS 跨域、拦截器与过滤器、WebSocket 和 GraphQL 等。这些功能在日常开发中几乎每天都会用到。

## 延伸思考

Spring Boot 的「约定优于配置」理念在常用功能中体现得淋漓尽致。你不需要手动配置日志框架，不需要配置异常处理器，不需要配置静态资源路径——Spring Boot 都为你准备好了。

但理解这些功能的底层原理，才能在特殊场景下正确地定制和扩展。
