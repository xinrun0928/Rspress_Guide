# 属性注入：@Value 与 @PropertySource

Spring 提供了多种属性注入方式：`@Value` 注解、`@PropertySource`、`Environment` 接口等。

这一节，我们来学习 Spring 的属性注入机制。

## 一、@Value 注解

### 1.1 基本用法

```java
@Component
public class UserService {
    @Value("${app.name}")
    private String appName;

    @Value("${app.version:1.0}")
    private String version;  // 带默认值
}
```

### 1.2 SpEL 表达式

```java
@Component
public class UserService {
    // 简单表达式
    @Value("#{systemProperties['user.region']}")
    private String region;

    // 计算表达式
    @Value("#{10 * 2}")
    private int multiplier;
}
```

### 1.3 类型转换

`@Value` 支持自动类型转换：

```java
@Value("${app.port}")
private int port;  // String -> int

@Value("${app.enabled}")
private boolean enabled;  // String -> boolean

@Value("${app.rate}")
private double rate;  // String -> double
```

## 二、@PropertySource

### 2.1 基本用法

```java
@Configuration
@PropertySource("classpath:app.properties")
@PropertySource("file:./config.properties")  // 支持多个
public class AppConfig {
    // 使用 @Value 或 Environment 获取属性
}
```

### 2.2 指定编码

```java
@Configuration
@PropertySource(value = "classpath:app.properties", encoding = "UTF-8")
public class AppConfig { }
```

### 2.3 ignoreResourceNotFound

```java
@Configuration
@PropertySource(value = "classpath:app.properties", ignoreResourceNotFound = true)
public class AppConfig { }
```

---

## 三、Environment 接口

### 3.1 获取属性

```java
@Service
public class UserService implements EnvironmentAware {
    private Environment environment;

    @Override
    public void setEnvironment(Environment environment) {
        this.environment = environment;
    }

    public void doSomething() {
        String appName = environment.getProperty("app.name");
        int port = environment.getProperty("app.port", Integer.class);
    }
}
```

### 3.2 Profiles 支持

```java
@Bean
@Profile("dev")
public DataSource devDataSource() { }

@Bean
@Profile("prod")
public DataSource prodDataSource() { }
```

---

## 四、属性占位符

```properties
# application.properties
app.name=MyApp
app.description=${app.name} is running
```

Spring 支持在属性值中使用 `${}` 占位符。

## 五、面试追问方向

**「@Value 和 @ConfigurationProperties 的区别？」**

@Value 用于注入单个属性；@ConfigurationProperties 用于批量注入一组相关属性，支持宽松绑定和校验。

**「Spring 如何解析 @Value 属性？」**

通过 `PropertySourcesPlaceholderConfigurer` 或 `PropertySource` 机制，从 Environment 中查找属性值。
