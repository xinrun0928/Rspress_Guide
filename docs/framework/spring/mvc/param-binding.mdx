# 参数绑定

前端传来的参数是如何绑定到 Controller 方法参数的？`@RequestParam`、`@RequestBody`、`@PathVariable` 都是怎么工作的？

这一节，我们来全面学习 Spring MVC 的参数绑定机制。

## 一、基础参数绑定

### 1.1 基本类型

```java
// URL: /user?id=1&name=张三
@GetMapping("/user")
public User getUser(Long id, String name) {
    return new User(id, name);
}
```

### 1.2 实体对象

```java
// URL: /user?id=1&name=张三
// 或 POST form data
@PostMapping("/user")
public User createUser(User user) {
    return userService.save(user);
}
```

---

## 二、常用注解

### 2.1 @RequestParam

```java
// 获取请求参数
@GetMapping("/user")
public User getUser(@RequestParam("id") Long userId) {
    return userService.findById(userId);
}

// required 和 defaultValue
@GetMapping("/search")
public List<User> search(
    @RequestParam(required = false, defaultValue = "10") int pageSize) {
    return userService.search(pageSize);
}
```

### 2.2 @PathVariable

```java
// 获取 URL 路径变量
@GetMapping("/user/{id}")
public User getUser(@PathVariable Long id) {
    return userService.findById(id);
}

// 多个路径变量
@GetMapping("/user/{userId}/order/{orderId}")
public Order getOrder(@PathVariable Long userId, @PathVariable Long orderId) {
    return orderService.findById(userId, orderId);
}
```

### 2.3 @RequestBody

```java
// 获取请求体（JSON）
@PostMapping("/user")
public User createUser(@RequestBody User user) {
    return userService.save(user);
}
```

### 2.4 @RequestHeader

```java
// 获取请求头
@GetMapping("/user")
public User getUser(@RequestHeader("Authorization") String token) {
    return userService.findByToken(token);
}
```

### 2.5 @CookieValue

```java
// 获取 Cookie
@GetMapping("/user")
public User getUser(@CookieValue("session_id") String sessionId) {
    return userService.findBySession(sessionId);
}
```

## 三、参数绑定流程

```java
// RequestMappingHandlerAdapter.invokeHandlerMethod()
protected ModelAndView invokeHandlerMethod(...) {
    // 1. 创建参数解析器列表
    HandlerMethodArgumentResolver[] resolvers = getDefaultArgumentResolvers();

    // 2. 解析每个参数
    for (MethodParameter parameter : handlerMethod.getMethodParameters()) {
        HandlerMethodArgumentResolver resolver = getResolver(parameter);
        resolver.resolveArgument(parameter, ...);
    }
}
```

---

## 四、自定义参数解析器

```java
@Component
public class CurrentUserArgumentResolver implements HandlerMethodArgumentResolver {
    @Override
    public boolean supportsParameter(MethodParameter parameter) {
        return parameter.hasParameterAnnotation(CurrentUser.class);
    }

    @Override
    public Object resolveArgument(...) {
        return SecurityContextHolder.getContext().getAuthentication().getPrincipal();
    }
}

// 使用
@GetMapping("/user")
public User getUser(@CurrentUser User currentUser) {
    return currentUser;
}
```

## 五、面试追问方向

**「@RequestParam 和 @RequestBody 的区别？」**

`@RequestParam` 绑定 query string 或 form data；`@RequestBody` 绑定请求体（JSON/XML），需要配合消息转换器使用。

**「参数绑定的流程是什么？」**

获取参数解析器列表 → 根据参数类型/注解选择解析器 → 解析器解析参数。
