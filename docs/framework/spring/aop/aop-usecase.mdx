## 二、事务管理

### 2.1 基本用法

```java
@Service
public class UserService {
    @Transactional(rollbackFor = Exception.class)
    public void createUser(User user) {
        userDao.save(user);
        sendEmail(user);
    }
}
```

### 2.2 自定义事务切面

```java
@Aspect
@Component
@Slf4j
public class TransactionAspect {

    @Around("@annotation(Transactional)")
    public Object around(ProceedingJoinPoint pjp, Transactional tx) throws Throwable {
        Connection conn = null;
        try {
            conn = dataSource.getConnection();
            conn.setAutoCommit(false);

            // 将连接绑定到当前线程
            TransactionSynchronizationManager
                .bindResource(dataSource, new DataSourceTransactionObject(conn));

            Object result = pjp.proceed();

            conn.commit();
            return result;
        } catch (Exception e) {
            if (conn != null) conn.rollback();
            throw e;
        } finally {
            TransactionSynchronizationManager
                .unbindResource(dataSource);
            if (conn != null) conn.setAutoCommit(true);
        }
    }
}
```

## 三、日志记录

### 3.1 操作日志

```java
@Aspect
@Component
public class OperationLogAspect {

    @Autowired
    private OperationLogService logService;

    @Around("@annotation(OperationLog)")
    public Object around(ProceedingJoinPoint pjp, OperationLog logAnnotation)
            throws Throwable {
        // 记录开始
        OperationLogEntity log = new OperationLogEntity();
        log.setOperation(logAnnotation.value());
        log.setMethod(pjp.getSignature().toShortString());
        log.setStartTime(LocalDateTime.now());

        Object result = null;
        try {
            result = pjp.proceed();
            log.setSuccess(true);
            return result;
        } catch (Exception e) {
            log.setSuccess(false);
            log.setErrorMessage(e.getMessage());
            throw e;
        } finally {
            log.setEndTime(LocalDateTime.now());
            log.setDuration(
                Duration.between(log.getStartTime(), log.getEndTime()).toMillis()
            );
            // 异步记录日志
            CompletableFuture.runAsync(() -> logService.save(log));
        }
    }
}

// 使用
@OperationLog("创建用户")
public User createUser(User user) {
    return userDao.save(user);
}
```

## 四、安全控制

### 4.1 权限校验

```java
@Aspect
@Component
@RequiredArgsConstructor
public class SecurityAspect {

    private final PermissionService permissionService;

    @Before("@annotation(RequiresPermission)")
    public void checkPermission(JoinPoint jp, RequiresPermission perm) {
        String requiredPermission = perm.value();

        if (!permissionService.hasPermission(requiredPermission)) {
            throw new AccessDeniedException(
                "权限不足: 需要 " + requiredPermission);
        }
    }

    @Before("@annotation(RequiresRoles)")
    public void checkRoles(JoinPoint jp, RequiresRoles roles) {
        for (String role : roles.value()) {
            if (roles.requireAll() && !permissionService.hasRole(role)) {
                throw new AccessDeniedException("需要所有角色: " + Arrays.toString(roles.value()));
            }
            if (!roles.requireAll() && permissionService.hasRole(role)) {
                return; // 有一个角色即可
            }
        }
    }
}

// 使用
@RequiresPermission("user:create")
public User createUser(User user) {
    return userDao.save(user);
}

@RequiresRoles(value = {"admin", "manager"}, requireAll = true)
public void deleteUser(Long id) {
    userDao.deleteById(id);
}
```

## 五、缓存加速

### 5.1 自定义缓存切面

```java
@Aspect
@Component
@RequiredArgsConstructor
public class CacheAspect {

    private final CacheManager cacheManager;

    @Around("@annotation(Cacheable)")
    public Object aroundCacheable(ProceedingJoinPoint pjp, Cacheable cacheable)
            throws Throwable {
        String cacheName = cacheable.value();
        String key = buildKey(cacheable.key(), pjp.getArgs());

        Cache cache = cacheManager.getCache(cacheName);
        if (cache != null) {
            ValueWrapper wrapper = cache.get(key);
            if (wrapper != null) {
                return wrapper.get();
            }
        }

        Object result = pjp.proceed();

        if (cache != null && result != null) {
            cache.put(key, result);
        }

        return result;
    }

    @AfterReturning(pointcut = "@annotation(CacheEvict)", returning = "result")
    public void evictCache(JoinPoint jp, CacheEvict evict, Object result) {
        if (evict.afterInvocation() && result != null) {
            // 清除缓存
        } else if (!evict.afterInvocation()) {
            // 清除缓存
        }
    }
}

// 注解定义
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface Cacheable {
    String value();
    String key() default "";
    long ttl() default 3600;
}
```

### 6.2 异步日志

```java
@Service
public class AsyncLogService {

    private final BlockingQueue<LogEntry> queue = new LinkedBlockingQueue<>();

    @PostConstruct
    public void init() {
        Executors.newSingleThreadExecutor().submit(() -> {
            while (true) {
                try {
                    LogEntry entry = queue.take();
                    doLog(entry);
                } catch (InterruptedException e) {
                    break;
                }
            }
        });
    }

    public void logAsync(LogEntry entry) {
        queue.offer(entry);
    }
}

@Aspect
@Component
public class AsyncLogAspect {

    @Autowired
    private AsyncLogService asyncLogService;

    @AfterReturning("@annotation(OperationLog)")
    public void afterOperation(JoinPoint jp) {
        asyncLogService.logAsync(buildLogEntry(jp));
    }
}
```

## 七、性能监控

### 7.1 方法耗时监控

```java
@Aspect
@Component
@Slf4j
public class PerformanceMonitorAspect {

    @Around("@annotation(Monitored)")
    public Object around(ProceedingJoinPoint pjp, Monitored monitored)
            throws Throwable {
        String method = pjp.getSignature().toShortString();
        long start = System.currentTimeMillis();

        try {
            return pjp.proceed();
        } finally {
            long duration = System.currentTimeMillis() - start;
            long threshold = monitored.threshold();

            if (duration > threshold) {
                log.warn("方法执行慢: {} 耗时 {}ms（阈值 {}ms）",
                    method, duration, threshold);
            }

            // 记录指标
            Metrics.gauge("method.duration", duration, Tags.of(
                "method", method,
                "slow", duration > threshold ? "true" : "false"
            ));
        }
    }
}

// 使用
@Monitored(threshold = 1000) // 超过 1 秒记录警告
public List<User> findUsers() {
    return userDao.findAll();
}
```

### 7.2 链路追踪

```java
@Aspect
@Component
public class TracingAspect {

    private final Tracer tracer;

    @Around("@annotation(Traced)")
    public Object around(ProceedingJoinPoint pjp, Traced traced) throws Throwable {
        Span span = tracer.startSpan(pjp.getSignature().getName());
        Scope scope = tracer.activate(span);

        try {
            // 注入 traceId 到日志
            MDC.put("traceId", span.context().traceId());

            // 添加标签
            span.tag("class", pjp.getSignature().getDeclaringTypeName());
            span.tag("method", pjp.getSignature().getName());

            Object result = pjp.proceed();

            span.setTag("success", true);
            return result;
        } catch (Exception e) {
            span.setTag("error", true);
            span.setTag("error.message", e.getMessage());
            throw e;
        } finally {
            scope.close();
            span.finish();
            MDC.remove("traceId");
        }
    }
}
```

## 八、参数处理

### 8.1 自动脱敏

```java
@Aspect
@Component
public class DataMaskingAspect {

    @AfterReturning(pointcut = "@annotation(SensitiveData)", returning = "result")
    public Object maskSensitiveData(JoinPoint jp, SensitiveData annotation, Object result) {
        if (result instanceof User) {
            User user = (User) result;
            if (annotation.phone()) {
                user.setPhone(maskPhone(user.getPhone()));
            }
            if (annotation.idCard()) {
                user.setIdCard(maskIdCard(user.getIdCard()));
            }
        }
        return result;
    }

    private String maskPhone(String phone) {
        if (phone == null || phone.length() < 7) return phone;
        return phone.substring(0, 3) + "****" + phone.substring(7);
    }
}

// 使用
@SensitiveData(phone = true, idCard = true)
public User getUser(Long id) {
    return userDao.findById(id);
}
```

### 8.2 参数转换

```java
@Aspect
@Component
public class ParameterConversionAspect {

    @Around("@annotation(ConvertParam)")
    public Object around(ProceedingJoinPoint pjp, ConvertParam annotation)
            throws Throwable {
        Object[] args = pjp.getArgs();
        Class<?>[] paramTypes = ((MethodSignature) pjp.getSignature())
            .getParameterTypes();

        // 遍历所有参数
        for (int i = 0; i < args.length; i++) {
            if (args[i] instanceof String) {
                Class<?> targetType = annotation.targets()[i];
                args[i] = convert(args[i], targetType);
            }
        }

        return pjp.proceed(args);
    }
}
```

## 九、总结

AOP 的实际应用场景非常广泛：

| 场景 | 典型注解 | 作用 |
|---|---|---|
| 事务管理 | `@Transactional` | 自动开启/提交/回滚事务 |
| 日志记录 | `@OperationLog` | 记录操作日志 |
| 安全控制 | `@RequiresPermission` | 权限校验 |
| 缓存加速 | `@Cacheable` | 自动缓存结果 |
| 异步执行 | `@Async` | 方法异步执行 |
| 性能监控 | `@Monitored` | 方法耗时统计 |
| 参数处理 | `@SensitiveData` | 参数/返回值处理 |

掌握这些场景的 AOP 实现，能够让你的代码更加简洁、解耦。
