# Spring дәӢеҠЎз®ЎзҗҶпјҡ@Transactional жҳҜеҰӮдҪ•дҝқиҜҒдёҖиҮҙжҖ§зҡ„

`@Transactional` еҶҷеңЁж–№жі•дёҠпјҢSpring жҖҺд№Ҳе°ұзҹҘйҒ“иҝҷжҳҜдёӘдәӢеҠЎпјҹ

еҫҲеӨҡдәәеҶҷд»Јз Ғж—¶пјҢ`@Transactional` жҳҜж Үй…Қпјҡ

```java
@Transactional
public void transfer(String from, String to, double amount) {
    accountDao.deduct(from, amount);
    accountDao.credit(to, amount);
}
```

иҝҷзңӢиө·жқҘеҫҲз®ҖеҚ•пјҢдҪҶиғҢеҗҺзҡ„еҺҹзҗҶжҳҜд»Җд№ҲпјҹSpring жҳҜеҰӮдҪ•зҹҘйҒ“иҰҒеңЁж–№жі•ејҖе§Ӣж—¶ејҖеҗҜдәӢеҠЎпјҢеңЁж–№жі•з»“жқҹж—¶жҸҗдәӨдәӢеҠЎпјҢеңЁејӮеёёж—¶еӣһж»ҡдәӢеҠЎзҡ„пјҹ

д»ҠеӨ©пјҢжҲ‘д»¬д»Һжәҗз ҒеұӮйқўпјҢжҠҠ Spring дәӢеҠЎз®ЎзҗҶзҡ„е®Ңж•ҙй“ҫи·ҜжӢҶи§ЈдёҖйҒҚгҖӮ

## TransactionInterceptorпјҡдәӢеҠЎжӢҰжҲӘеҷЁ

`TransactionInterceptor` жҳҜ Spring дәӢеҠЎзҡ„ж ёеҝғжӢҰжҲӘеҷЁгҖӮе®ғе®һзҺ°дәҶ `MethodInterceptor` жҺҘеҸЈпјҢеңЁж–№жі•и°ғз”ЁеүҚеҗҺж·»еҠ дәӢеҠЎйҖ»иҫ‘пјҡ

```java title="TransactionInterceptor.invoke() жәҗз Ғ"
public class TransactionInterceptor extends TransactionAspectSupport
        implements MethodInterceptor, Serializable {

    @Override
    public Object invoke(MethodInvocation invocation) throws Throwable {
        // иҺ·еҸ–зӣ®ж Үзұ»
        Class<?> targetClass = (invocation.getThis() != null ?
            AopUtils.getTargetClass(invocation.getThis()) : null);

        // ж ёеҝғж–№жі•пјҡжү§иЎҢеёҰдәӢеҠЎзҡ„ж–№жі•
        return invokeWithinTransaction(invocation.getMethod(), targetClass, invocation::proceed);
    }
}
```

`invokeWithinTransaction` жҳҜдәӢеҠЎеӨ„зҗҶзҡ„ж ёеҝғйҖ»иҫ‘пјҡ

```java title="TransactionAspectSupport.invokeWithinTransaction() жәҗз Ғ"
protected Object invokeWithinTransaction(Method method, @Nullable Class<?> targetClass,
        final TransactionAspectSupport.InvocationCallback invocation) throws Throwable {

    // 1. иҺ·еҸ–дәӢеҠЎеұһжҖ§
    TransactionAttributeSource tas = getTransactionAttributeSource();
    TransactionAttribute txAttr = (tas != null ?
        tas.getTransactionAttribute(method, targetClass) : null);

    // 2. иҺ·еҸ–дәӢеҠЎз®ЎзҗҶеҷЁ
    PlatformTransactionManager tm = determineTransactionManager(txAttr);

    // 3. жһ„е»әдәӢеҠЎдҝЎжҒҜ
    String joinpointIdentification = methodIdentification(method, targetClass, txAttr);

    if (txAttr == null || !(tm instanceof CallbackPreferringPlatformTransactionManager)) {
        // ж ҮеҮҶдәӢеҠЎеӨ„зҗҶ
        TransactionInfo txInfo = createTransactionIfNecessary(tm, txAttr, joinpointIdentification);

        Object retVal;
        try {
            // 4. жү§иЎҢзӣ®ж Үж–№жі•
            retVal = invocation.proceed();
        } catch (Throwable ex) {
            // 5. ејӮеёёеӨ„зҗҶпјҡеӣһж»ҡ
            completeTransactionAfterThrowing(txInfo, ex);
            throw ex;
        } finally {
            // 6. жё…зҗҶдәӢеҠЎдҝЎжҒҜ
            cleanupTransactionInfo(txInfo);
        }

        // 7. жӯЈеёёз»“жқҹпјҡжҸҗдәӨдәӢеҠЎ
        commitTransactionAfterReturning(txInfo);

        return retVal;
    } else {
        // CallbackPreferringTransactionManager зҡ„еӨ„зҗҶ...
    }
}
```

## PlatformTransactionManagerпјҡдәӢеҠЎз®ЎзҗҶеҷЁжҺҘеҸЈ

`PlatformTransactionManager` жҳҜдәӢеҠЎз®ЎзҗҶзҡ„ж ёеҝғжҺҘеҸЈпјҡ

```java title="PlatformTransactionManager жҺҘеҸЈ"
public interface PlatformTransactionManager {

    // иҺ·еҸ–дәӢеҠЎзҠ¶жҖҒ
    TransactionStatus getTransaction(@Nullable TransactionDefinition definition) throws TransactionException;

    // жҸҗдәӨдәӢеҠЎ
    void commit(TransactionStatus status) throws TransactionException;

    // еӣһж»ҡдәӢеҠЎ
    void rollback(TransactionStatus status) throws TransactionException;
}
```

Spring жҸҗдҫӣдәҶеӨҡз§Қе®һзҺ°пјҡ

| е®һзҺ°зұ» | йҖӮз”ЁеңәжҷҜ |
|---|---|
| `DataSourceTransactionManager` | JDBC/MyBatis еҚ•ж•°жҚ®жәҗ |
| `JpaTransactionManager` | JPA/Hibernate |
| `JtaTransactionManager` | еҲҶеёғејҸдәӢеҠЎпјҲXAпјү |
| `HibernateTransactionManager` | HibernateпјҲе·Іеәҹејғпјү |

protected Object doGetTransaction() {
    DataSourceTransactionObject txObject = new DataSourceTransactionObject();

### doBegin()пјҡејҖе§ӢдәӢеҠЎ

```java title="DataSourceTransactionManager.doBegin() жәҗз Ғ"
@Override
protected void doBegin(Object transaction, TransactionDefinition definition) {
    DataSourceTransactionObject txObject = (DataSourceTransactionObject) transaction;
    Connection con = null;

    try {
        if (txObject.getConnectionHolder() == null ||
                txObject.getConnectionHolder().isSynchronizedWithTransaction()) {
            // иҺ·еҸ–ж–°зҡ„ж•°жҚ®еә“иҝһжҺҘ
            Connection newCon = obtainDataSource().getConnection();
            txObject.setConnectionHolder(new ConnectionHolder(newCon), true);
        }

        con = txObject.getConnectionHolder().getConnection();

        // и®ҫзҪ®йҡ”зҰ»зә§еҲ«
        Integer previousIsolationLevel = DataSourceUtils.prepareConnectionForTransaction(
            con, definition);
        txObject.setPreviousIsolationLevel(previousIsolationLevel);

        // е…ій—ӯиҮӘеҠЁжҸҗдәӨ
        if (con.getAutoCommit()) {
            txObject.setMustRestoreAutoCommit(true);
            con.setAutoCommit(false);
        }

        // еҲӨж–ӯжҳҜеҗҰжҳҜеҸӘиҜ»дәӢеҠЎ
        if (definition.isReadOnly()) {
            // и®ҫзҪ®еҸӘиҜ»
            prepareReadOnly(con);
        }

        // е°ҶиҝһжҺҘж Үи®°дёәдәӢеҠЎдёӯ
        txObject.getConnectionHolder().setSynchronizedWithTransaction(true);
        con.createSavepoint();

    } catch (Throwable ex) {
        // ејӮеёёеӨ„зҗҶ
        closeConnectionAfterError(con, txObject);
        throw new CannotCreateTransactionException(...);
    }
}
```

### doSuspend() / doResume()пјҡжҢӮиө·дёҺжҒўеӨҚдәӢеҠЎ

еҪ“еөҢеҘ—дәӢеҠЎдҪҝз”Ё `REQUIRES_NEW` дј ж’ӯиЎҢдёәж—¶пјҢйңҖиҰҒжҢӮиө·еҪ“еүҚдәӢеҠЎпјҡ

```java title="DataSourceTransactionManager.doSuspend() жәҗз Ғ"
@Override
protected Object doSuspend(Object transaction) {
    DataSourceTransactionObject txObject = (DataSourceTransactionObject) transaction;

    // и§ЈйҷӨ ThreadLocal з»‘е®ҡ
    txObject.setConnectionHolder(null);

    // жҢӮиө·еҪ“еүҚиҝһжҺҘ
    Connection suspendedConnection = doSuspend(txObject);

    return suspendedConnection;
}

@Override
protected void doResume(@Nullable Object transaction, Object suspendedResources) {
    // жҒўеӨҚжҢӮиө·зҡ„иҝһжҺҘ
    Connection suspendedConnection = (Connection) suspendedResources;
    // йҮҚж–°з»‘е®ҡеҲ° ThreadLocal
    TransactionSynchronizationManager.bindResource(
        obtainDataSource(),
        new ConnectionHolder(suspendedConnection)
    );
}
```

## TransactionSynchronizationManagerпјҡдәӢеҠЎеҗҢжӯҘз®ЎзҗҶеҷЁ

`TransactionSynchronizationManager` жҳҜ Spring дәӢеҠЎз®ЎзҗҶзҡ„ж ёеҝғе·Ҙе…·зұ»пјҢе®ғдҪҝз”Ё `ThreadLocal` жқҘз®ЎзҗҶдәӢеҠЎдёҠдёӢж–Үпјҡ

```java title="TransactionSynchronizationManager ж ёеҝғж–№жі•"
public abstract class TransactionSynchronizationManager {

    // иө„жәҗпјҲDataSource -> ConnectionHolderпјү
    private static final ThreadLocal<Map<Object, Object>> resources =
        new NamedThreadLocal<>("Transactional resources");

    // дәӢеҠЎеҗҢжӯҘеҷЁ
    private static final ThreadLocal<Set<TransactionSynchronization>> synchronizations =
        new NamedThreadLocal<>("Transaction synchronizations");

    // еҪ“еүҚдәӢеҠЎеҗҚз§°
    private static final ThreadLocal<String> currentTransactionName =
        new NamedThreadLocal<>("Current transaction name");

    // еҪ“еүҚдәӢеҠЎжҳҜеҗҰеҸӘиҜ»
    private static final ThreadLocal<Boolean> currentTransactionReadOnly =
        new NamedThreadLocal<>("Current transaction read-only status");

    // еҪ“еүҚдәӢеҠЎдј ж’ӯиЎҢдёә
    private static final ThreadLocal<Integer> currentTransactionIsolationLevel =
        new NamedThreadLocal<>("Current transaction isolation level");

    // з»‘е®ҡиө„жәҗеҲ°еҪ“еүҚзәҝзЁӢ
    public static void bindResource(Object key, Object value) {
        Map<Object, Object> map = resources.get();
        if (map == null) {
            map = new HashMap<>();
            resources.set(map);
        }
        map.put(key, value);
    }

    // д»ҺеҪ“еүҚзәҝзЁӢиҺ·еҸ–иө„жәҗ
    @Nullable
    public static Object unbindResource(Object key) {
        Map<Object, Object> map = resources.get();
        if (map == null) {
            return null;
        }
        Object value = map.remove(key);
        if (map.isEmpty()) {
            resources.remove();
        }
        return value;
    }
}
```

## @Transactional еұһжҖ§иҜҰи§Ј

```java
@Transactional(
    propagation = Propagation.REQUIRED,    // дј ж’ӯиЎҢдёә
    isolation = Isolation.DEFAULT,         // йҡ”зҰ»зә§еҲ«
    timeout = -1,                         // и¶…ж—¶ж—¶й—ҙ
    readOnly = false,                      // жҳҜеҗҰеҸӘиҜ»
    rollbackFor = Exception.class,         // еӣһж»ҡзҡ„ејӮеёёзұ»еһӢ
    noRollbackFor = RuntimeException.class // дёҚеӣһж»ҡзҡ„ејӮеёёзұ»еһӢ
)
```

### rollbackFor е’Ң noRollbackFor

```java
// й»ҳи®ӨеҸӘеӣһж»ҡ RuntimeException е’Ң Error
// жүҖд»ҘдёӢйқўиҝҷдёӘдёҚдјҡеӣһж»ҡ
@Transactional
public void saveUser(User user) {
    throw new Exception("дҝқеӯҳеӨұиҙҘ");  // дёҚдјҡеӣһж»ҡпјҒ
}

// еә”иҜҘиҝҷж ·й…ҚзҪ®
@Transactional(rollbackFor = Exception.class)
public void saveUser(User user) throws Exception {
    throw new Exception("дҝқеӯҳеӨұиҙҘ");  // дјҡеӣһж»ҡ
}
```

## йқўиҜ•иҝҪй—®ж–№еҗ‘

- `TransactionInterceptor` зҡ„ `invoke()` ж–№жі•жҳҜеҰӮдҪ•е·ҘдҪңзҡ„пјҹ
- `PlatformTransactionManager` жңүе“Әдәӣе®һзҺ°пјҹеҗ„иҮӘйҖӮеҗҲд»Җд№ҲеңәжҷҜпјҹ
- `TransactionSynchronizationManager` зҡ„ `ThreadLocal` жҳҜеҰӮдҪ•з®ЎзҗҶдәӢеҠЎдёҠдёӢж–Үзҡ„пјҹ
- `DataSourceTransactionManager.doBegin()` дёӯеҒҡдәҶе“Әдәӣж“ҚдҪңпјҹ
- д»Җд№ҲжҳҜдәӢеҠЎзҡ„дј ж’ӯиЎҢдёәпјҹ`REQUIRED` е’Ң `REQUIRES_NEW` жңүд»Җд№ҲеҢәеҲ«пјҹ
- `@Transactional` зҡ„ `rollbackFor` еұһжҖ§й»ҳи®ӨеҸӘеӣһж»ҡе“ӘдәӣејӮеёёпјҹ

---

Spring дәӢеҠЎз®ЎзҗҶзҡ„еҺҹзҗҶе°ұеҲҶжһҗеҲ°иҝҷйҮҢгҖӮдҪҶиҝҳжңүдёҖдёӘйҮҚиҰҒзҡ„й—®йўҳпјҡдәӢеҠЎзҡ„дј ж’ӯиЎҢдёәгҖӮ`REQUIRED`гҖҒ`REQUIRES_NEW`гҖҒ`NESTED`вҖҰвҖҰиҝҷдәӣдј ж’ӯиЎҢдёәеҲ°еә•жңүд»Җд№ҲеҢәеҲ«пјҹ
