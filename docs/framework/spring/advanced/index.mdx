# Spring 高级特性

除了核心的 IoC、AOP、MVC 之外，Spring 还提供了许多强大的高级特性：事件机制、资源加载、国际化、类型转换、验证框架等。

这一章，我们来学习这些「不那么常见但同样重要」的高级特性。

## 为什么学习高级特性？

这些高级特性可能在日常开发中不常用，但它们是构建大型框架和库的基础：

- **事件机制**：解耦业务逻辑
- **类型转换**：处理复杂数据类型
- **验证框架**：数据校验

理解这些特性，能帮助你更好地使用 Spring，并有能力构建自己的扩展。
