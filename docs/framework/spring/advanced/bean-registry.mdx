# BeanDefinitionRegistryPostProcessor：动态注册 Bean

Spring 启动后，还能注册新的 Bean 吗？

正常情况下，Bean 是在容器启动时创建的。但有时候，你需要在启动过程中动态地添加 Bean——比如根据配置文件动态创建数据源，或者动态注册 MyBatis 的 Mapper 接口。

Spring 提供了 `BeanDefinitionRegistryPostProcessor` 来实现这一点。

这一节，我们深入解析动态注册 Bean 的机制。

---

## BeanFactoryPostProcessor vs BeanDefinitionRegistryPostProcessor

### 两者区别

| 接口 | 执行时机 | 作用 |
|---|---|---|
| `BeanFactoryPostProcessor` | BeanDefinition 加载后、Bean 创建前 | 修改 BeanDefinition |
| `BeanDefinitionRegistryPostProcessor` | 更早，所有 BeanDefinition 加载前 | 动态注册 BeanDefinition |

### 执行顺序

```
容器启动
    │
    ▼
BeanDefinitionRegistryPostProcessor（动态注册）
    │
    ▼
BeanFactoryPostProcessor（修改定义）
    │
    ▼
Bean 实例化
    │
    ▼
BeanPostProcessor（实例后处理）
```

## BeanDefinitionRegistryPostProcessor

### 接口定义

```java
public interface BeanDefinitionRegistryPostProcessor extends BeanFactoryPostProcessor {
    // 在所有 BeanDefinition 加载后、Bean 实例化前调用
    void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry)
        throws BeansException;
}
```

### 核心方法

```java
public interface BeanDefinitionRegistry {
    // 注册新的 BeanDefinition
    void registerBeanDefinition(String beanName, BeanDefinition beanDefinition)
        throws BeanDefinitionStoreException;

    // 移除 BeanDefinition
    void removeBeanDefinition(String beanName) throws NoSuchBeanDefinitionException;

    // 获取 BeanDefinition
    BeanDefinition getBeanDefinition(String beanName)
        throws NoSuchBeanDefinitionException;

    // 判断是否存在
    boolean containsBeanDefinition(String beanName);

    // 获取所有 Bean 名称
    String[] getBeanDefinitionNames();

    // 获取 BeanDefinition 数量
    int getBeanDefinitionCount();

    // 判断名称是否被使用
    boolean isBeanNameInUse(String beanName);
}
```

## BeanDefinition

### 类型

| 类型 | 说明 |
|---|---|
| `RootBeanDefinition` | 根定义，可合并子定义 |
| `GenericBeanDefinition` | 通用定义，推荐使用 |
| `ChildBeanDefinition` | 子定义，继承父定义 |
| `ConfigurationClassBeanDefinition` | 配置类中的 @Bean |
| `AnnotatedBeanDefinition` | 带注解的 Bean 定义 |

### 基本属性

```java
public interface BeanDefinition extends AttributeAccessor {
    // Bean 类名
    String getBeanClassName();

    // 作用域
    String getScope();

    // 是否懒加载
    boolean isLazyInit();

    // 依赖注入
    String[] getDependsOn();

    // 构造方法参数
    ConstructorArgumentValues getConstructorArgumentValues();

    // 属性值
    MutablePropertyValues getPropertyValues();

    // 初始化方法
    String getInitMethodName();

    // 销毁方法
    String getDestroyMethodName();
}
```

## 使用示例

### 动态注册简单 Bean

```java
@Component
public class CustomBeanDefinitionRegistryPostProcessor
        implements BeanDefinitionRegistryPostProcessor {

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry)
            throws BeansException {
        // 注册一个 Bean
        BeanDefinition beanDefinition = new GenericBeanDefinition();
        beanDefinition.setBeanClassName("com.example.MyService");
        beanDefinition.setScope(BeanDefinition.SCOPE_SINGLETON);

        registry.registerBeanDefinition("myService", beanDefinition);
    }

    @Override
    public void postProcessBeanFactory(ConfigurableListableBeanFactory beanFactory)
            throws BeansException {
        // 可以在这里对 BeanFactory 做进一步处理
    }
}
```

### 使用 BeanDefinitionBuilder

```java
@Component
public class DynamicDataSourceRegistrar
        implements BeanDefinitionRegistryPostProcessor {

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry)
            throws BeansException {
        // 使用 Builder 创建
        BeanDefinitionBuilder builder = BeanDefinitionBuilder
            .genericBeanDefinition(DataSource.class);

        builder.addPropertyValue("url", "jdbc:mysql://localhost:3306/test");
        builder.addPropertyValue("username", "root");
        builder.addPropertyValue("password", "password");
        builder.setScope(BeanDefinition.SCOPE_SINGLETON);
        builder.setLazyInit(false);

        BeanDefinition beanDefinition = builder.getBeanDefinition();

        registry.registerBeanDefinition("dataSource", beanDefinition);
    }

    @Override
    public void postProcessBeanFactory(ConfigurableListableBeanFactory beanFactory)
            throws BeansException {
        // 进一步处理
    }
}
```

## 动态注册 Mapper 接口（MyBatis 原理）

### MyBatis 的扫描注册

MyBatis 的 `@MapperScan` 就是通过 `BeanDefinitionRegistryPostProcessor` 实现的：

```java
public class MapperScannerConfigurer
        implements BeanDefinitionRegistryPostProcessor, EnvironmentAware {

    private String basePackage;
    private String sqlSessionFactoryBeanName;

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry) {
        // 创建扫描器
        ClassPathMapperScanner scanner = new ClassPathMapperScanner(registry);

        // 设置过滤器
        scanner.addIncludeFilter(new InterfaceTypeFilter());

        // 扫描并注册
        Set<BeanDefinitionHolder> beanDefinitions = scanner
            .findCandidateComponents(basePackage);

        for (BeanDefinitionHolder holder : beanDefinitions) {
            // 注册为 MapperFactoryBean
            AbstractBeanDefinition beanDefinition =
                BeanDefinitionBuilder
                    .genericBeanDefinition(MapperFactoryBean.class)
                    .addPropertyValue("mapperInterface", holder.getBeanDefinition().getBeanClassName())
                    .getBeanDefinition();

            registry.registerBeanDefinition(holder.getBeanName(), beanDefinition);
        }
    }
}
```

### 手写简化版

```java
@Component
public class MapperScanProcessor
        implements BeanDefinitionRegistryPostProcessor {

    private static final String MAPPER_PACKAGE = "com.example.mapper";

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry)
            throws BeansException {
        // 扫描 Mapper 接口
        Set<Class<?>> mapperClasses = scanMappers(MAPPER_PACKAGE);

        for (Class<?> mapperClass : mapperClasses) {
            // 注册 MapperFactoryBean
            BeanDefinitionBuilder builder = BeanDefinitionBuilder
                .genericBeanDefinition(MapperFactoryBean.class);

            builder.addPropertyValue("mapperInterface", mapperClass);
            builder.setAutowireMode(AbstractBeanDefinition.AUTOWIRE_BY_TYPE);

            AbstractBeanDefinition beanDefinition = builder.getBeanDefinition();

            // 注册
            String beanName = mapperClass.getSimpleName();
            beanName = Character.toLowerCase(beanName.charAt(0)) + beanName.substring(1);
            registry.registerBeanDefinition(beanName, beanDefinition);
        }
    }

    private Set<Class<?>> scanMappers(String basePackage) {
        // 扫描逻辑
        // ...
        return new HashSet<>();
    }
}
```

## ImportBeanDefinitionRegistrar

### 接口定义

```java
public interface ImportBeanDefinitionRegistrar {
    // 手动注册 BeanDefinition
    void registerBeanDefinitions(AnnotationMetadata importingClassMetadata,
                                BeanDefinitionRegistry registry);
}
```

### 使用 @Import

```java
// 定义 Registrar
public class MyImportBeanDefinitionRegistrar implements ImportBeanDefinitionRegistrar {

    @Override
    public void registerBeanDefinitions(AnnotationMetadata importingClassMetadata,
                                      BeanDefinitionRegistry registry) {
        // 获取导入类的注解信息
        Map<String, Object> attrs = importingClassMetadata.getAnnotationAttributes(
            EnableMyFeature.class.getName());

        String config = (String) attrs.get("config");

        // 注册 Bean
        BeanDefinitionBuilder builder = BeanDefinitionBuilder
            .genericBeanDefinition(MyService.class);
        builder.addPropertyValue("config", config);

        registry.registerBeanDefinition("myService", builder.getBeanDefinition());
    }
}

// 定义注解
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.TYPE)
@Import(MyImportBeanDefinitionRegistrar.class)
public @interface EnableMyFeature {
    String config() default "";
}

// 使用
@EnableMyFeature(config = "custom")
@Configuration
public class AppConfig { }
```

## @MapperScan 原理

### MyBatis 自动配置

```java
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.TYPE)
@Import(MapperScannerRegistrar.class)
@Repeatable(MapperScans.class)
public @interface MapperScan {
    String[] value() default {};
    String[] basePackages() default {};
    Class<?>[] basePackageClasses() default {};
}
```

### MapperScannerRegistrar

```java
public class MapperScannerRegistrar implements ImportBeanDefinitionRegistrar {

    @Override
    public void registerBeanDefinitions(AnnotationMetadata metadata,
                                      BeanDefinitionRegistry registry) {
        // 获取 @MapperScan 注解属性
        Map<String, Object> attrs = metadata.getAnnotationAttributes(
            MapperScan.class.getName());

        String[] basePackages = (String[]) attrs.get("basePackages");

        // 创建并注册 MapperScannerConfigurer
        BeanDefinitionBuilder builder = BeanDefinitionBuilder
            .genericBeanDefinition(MapperScannerConfigurer.class);

        builder.addPropertyValue("basePackage", StringUtils.join(basePackages, ","));

        registry.registerBeanDefinition(
            MapperScannerConfigurer.class.getName(),
            builder.getBeanDefinition()
        );
    }
}
```

## 动态注册 FactoryBean

### FactoryBean 接口

```java
public interface FactoryBean<T> {
    // 返回工厂创建的实例
    T getObject() throws Exception;

    // 返回实例类型
    Class<?> getObjectType();

    // 是否单例
    default boolean isSingleton() {
        return true;
    }
}
```

### 注册 FactoryBean

```java
@Component
public class DynamicServiceRegistrar
        implements BeanDefinitionRegistryPostProcessor {

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry)
            throws BeansException {
        // 注册 FactoryBean
        BeanDefinitionBuilder builder = BeanDefinitionBuilder
            .genericBeanDefinition(DynamicServiceFactoryBean.class);

        builder.addPropertyValue("config", loadConfig());

        AbstractBeanDefinition beanDefinition = builder.getBeanDefinition();
        beanDefinition.setAttribute("factoryBeanObjectType", DynamicService.class);

        registry.registerBeanDefinition("dynamicService", beanDefinition);
    }

    private Map<String, Object> loadConfig() {
        // 动态加载配置
        return new HashMap<>();
    }
}
```

## 常见使用场景

### 1. 动态数据源

```java
@Component
public class DynamicDataSourceRegistrar
        implements BeanDefinitionRegistryPostProcessor {

    @Value("${datasource.type:hikari}")
    private String dataSourceType;

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry)
            throws BeansException {
        if ("hikari".equals(dataSourceType)) {
            registerHikariDataSource(registry);
        } else if ("druid".equals(dataSourceType)) {
            registerDruidDataSource(registry);
        }
    }

    private void registerHikariDataSource(BeanDefinitionRegistry registry) {
        BeanDefinitionBuilder builder = BeanDefinitionBuilder
            .genericBeanDefinition(HikariDataSource.class);

        builder.addPropertyValue("jdbcUrl", "${spring.datasource.url}");
        builder.addPropertyValue("username", "${spring.datasource.username}");
        builder.addPropertyValue("password", "${spring.datasource.password}");

        registry.registerBeanDefinition("dataSource", builder.getBeanDefinition());
    }

    private void registerDruidDataSource(BeanDefinitionRegistry registry) {
        // 注册 DruidDataSource
    }
}
```

### 2. 动态 Bean 注册

```java
@Component
public class ExternalBeanRegistrar
        implements BeanDefinitionRegistryPostProcessor {

    @Value("${external.beans:}")
    private String externalBeans;

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry)
            throws BeansException {
        if (StringUtils.isBlank(externalBeans)) {
            return;
        }

        String[] beans = externalBeans.split(",");
        for (String bean : beans) {
            String[] parts = bean.split(":");
            String beanName = parts[0];
            String className = parts[1];

            BeanDefinitionBuilder builder = BeanDefinitionBuilder
                .genericBeanDefinition(className);

            registry.registerBeanDefinition(beanName, builder.getBeanDefinition());
        }
    }
}
```

## 面试追问方向

**「BeanFactoryPostProcessor 和 BeanDefinitionRegistryPostProcessor 的区别？」**

- `BeanDefinitionRegistryPostProcessor`：更早执行，用于动态注册 BeanDefinition
- `BeanFactoryPostProcessor`：在 BeanDefinition 加载后、Bean 创建前执行，用于修改 BeanDefinition

**「如何动态注册 Bean？」**

通过 `BeanDefinitionRegistry.registerBeanDefinition()` 注册 `BeanDefinition`。

**「MyBatis 的 @MapperScan 是怎么实现的？」**

通过 `@Import(MapperScannerRegistrar.class)` + `ImportBeanDefinitionRegistrar` 动态注册 `MapperFactoryBean`。

---

## 总结

动态注册 Bean 的方式：

| 方式 | 说明 |
|---|---|
| `BeanDefinitionRegistryPostProcessor` | 在容器启动早期动态注册 |
| `ImportBeanDefinitionRegistrar` | 通过 `@Import` 导入时注册 |
| `BeanDefinitionRegistry.registerBeanDefinition()` | 注册 BeanDefinition |

使用场景：
- MyBatis Mapper 扫描
- 动态数据源
- 外部配置 Bean
