# Spring 格式化：Formatter 与 FormattingConversionService

当数据从表单提交到 Controller，或者从 Controller 返回到视图时，格式怎么转换？

日期怎么从 "2024-01-15" 变成 `LocalDate`？数字怎么从 `BigDecimal` 变成 "¥1,234.56"？

这就是 Spring 格式化系统的职责。

这一节，我们深入解析 Spring 的格式化机制。

## Formatter vs Converter

### Converter：类型转换

Converter 负责**类型之间的转换**，没有格式化的概念：

```java
// String -> Date（转换）
public class StringToDateConverter implements Converter<String, Date> {
    @Override
    public Date convert(String source) {
        return new SimpleDateFormat("yyyy-MM-dd").parse(source);
    }
}
```

### Formatter：解析与打印

Formatter 负责**字符串和对象之间的解析与打印**：

```java
// String <-> Date（解析和打印）
public class DateFormatter implements Formatter<Date> {
    @Override
    public Date parse(String text, Locale locale) {
        // String -> Date
        return new SimpleDateFormat("yyyy-MM-dd", locale).parse(text);
    }

    @Override
    public String print(Date object, Locale locale) {
        // Date -> String
        return new SimpleDateFormat("yyyy-MM-dd", locale).print(object, locale);
    }
}
```

### 对比

| 特性 | Converter | Formatter |
|---|---|---|
| 方向 | S → T | String ↔ T |
| 用途 | 通用类型转换 | 解析/打印 |
| Locale 支持 | 不支持 | 支持 |
| 使用场景 | 业务逻辑 | UI 展示 |

## Formatter 接口

### 接口定义

```java
public interface Formatter<T> extends Printer<T>, Parser<T> {
    // 继承自 Printer 和 Parser
}

public interface Printer<T> {
    String print(T object, Locale locale);
}

public interface Parser<T> {
    T parse(String text, Locale locale) throws ParseException;
}
```

### 基本实现

```java
public class DateFormatter implements Formatter<Date> {
    private final String pattern;

    public DateFormatter(String pattern) {
        this.pattern = pattern;
    }

    @Override
    public Date parse(String text, Locale locale) throws ParseException {
        SimpleDateFormat dateFormat = new SimpleDateFormat(pattern, locale);
        dateFormat.setLenient(false);
        return dateFormat.parse(text);
    }

    @Override
    public String print(Date object, Locale locale) {
        SimpleDateFormat dateFormat = new SimpleDateFormat(pattern, locale);
        return dateFormat.format(object);
    }
}
```

## 注解驱动的格式化

### @NumberFormat

数字格式化注解：

```java
public class Order {
    // 默认格式
    @NumberFormat
    private Double price;

    // 货币格式
    @NumberFormat(style = Style.CURRENT)
    private Double currency;

    // 百分比格式
    @NumberFormat(style = Style.PERCENT)
    private Double discount;

    // 自定义模式
    @NumberFormat(pattern = "#,###.##")
    private Double amount;
}
```

### @DateTimeFormat

日期时间格式化注解：

```java
public class User {
    // ISO 日期格式
    @DateTimeFormat(iso = ISO.DATE)
    private LocalDate birthDate;

    // ISO 时间格式
    @DateTimeFormat(iso = ISO.TIME)
    private LocalTime loginTime;

    // ISO 日期时间格式
    @DateTimeFormat(iso = ISO.DATE_TIME)
    private LocalDateTime createTime;

    // 自定义格式
    @DateTimeFormat(pattern = "yyyy/MM/dd")
    private Date registerDate;
}
```

### ISO 枚举值

| 值 | 说明 | 示例 |
|---|---|---|
| `ISO.DATE` | ISO 日期 | 2024-01-15 |
| `ISO.TIME` | ISO 时间 | 14:30:00 |
| `ISO.DATE_TIME` | ISO 日期时间 | 2024-01-15T14:30:00 |

## FormattingConversionService

### FormattingConversionServiceFactoryBean

```java
@Configuration
public class FormattingConfig {

    @Bean
    public FormattingConversionService formattingConversionService() {
        FormattingConversionServiceFactoryBean factory =
            new FormattingConversionServiceFactoryBean();

        // 设置是否注册默认格式化器
        factory.setRegisterDefaultFormatters(true);

        // 添加额外的格式化注解
        factory.setFormatterRegistrars(Set.of(
            new Jsr310DateTimeFormatAnnotationFormatterFactory(),
            new NumberFormatAnnotationFormatterFactory()
        ));

        return factory.createConversionService();
    }
}
```

### DefaultFormattingConversionService

简化配置：

```java
@Bean
public FormattingConversionService conversionService() {
    // DefaultFormattingConversionService 已经注册了常用格式化器
    return new DefaultFormattingConversionService();
}
```

## Spring MVC 中的格式化

### WebMvcConfigurer 配置

```java
@Configuration
public class WebConfig implements WebMvcConfigurer {

    @Override
    public void addFormatters(FormatterRegistry registry) {
        // 添加自定义 Formatter
        registry.addFormatter(new DateFormatter("yyyy-MM-dd"));

        // 添加自定义 Printer/Parser
        registry.addPrinter(new LocalDatePrinter());
        registry.addParser(new LocalDateParser());
    }
}
```

### Spring Boot 自动配置

Spring Boot 自动配置了 `FormattingConversionService`，无需手动配置。

### @InitBinder 配置

```java
@Controller
public class UserController {

    @InitBinder
    protected void initBinder(WebDataBinder binder) {
        // 注册自定义编辑器
        binder.registerCustomEditor(Date.class,
            new CustomDateEditor(new SimpleDateFormat("yyyy-MM-dd"), false));

        // 或者使用 FormattingConversionService
        binder.addCustomFormatter(new DateFormatter("yyyy-MM-dd"));
    }
}
```

## 自定义 Formatter

### 货币 Formatter

```java
public class CurrencyFormatter implements Formatter<BigDecimal> {

    private final String currencySymbol;
    private final Locale locale;

    public CurrencyFormatter(String currencySymbol, Locale locale) {
        this.currencySymbol = currencySymbol;
        this.locale = locale;
    }

    @Override
    public BigDecimal parse(String text, Locale locale) throws ParseException {
        if (text == null || text.isEmpty()) {
            return null;
        }

        // 移除货币符号和分隔符
        String cleaned = text.replace(currencySymbol, "")
                           .replace(",", "")
                           .trim();

        return new BigDecimal(cleaned);
    }

    @Override
    public String print(BigDecimal object, Locale locale) {
        if (object == null) {
            return "";
        }

        NumberFormat format = NumberFormat.getCurrencyInstance(locale);
        return format.format(object);
    }
}
```

### 枚举 Formatter

```java
public class EnumFormatter<T extends Enum<T>> implements Formatter<T> {

    private final Class<T> enumType;

    public EnumFormatter(Class<T> enumType) {
        this.enumType = enumType;
    }

    @Override
    public T parse(String text, Locale locale) throws ParseException {
        if (text == null || text.isEmpty()) {
            return null;
        }

        try {
            return Enum.valueOf(enumType, text.toUpperCase());
        } catch (IllegalArgumentException e) {
            throw new ParseException(
                "Invalid value '" + text + "' for enum " + enumType.getSimpleName(),
                0
            );
        }
    }

    @Override
    public String print(T object, Locale locale) {
        if (object == null) {
            return "";
        }
        return object.name().toLowerCase();
    }
}
```

### 使用

```java
@Configuration
public class FormattingConfig {

    @Bean
    public FormattingConversionService formattingConversionService() {
        FormattingConversionServiceFactoryBean factory =
            new FormattingConversionServiceFactoryBean();

        Set<Formatter<?>> formatters = new HashSet<>();
        formatters.add(new CurrencyFormatter("¥", Locale.CHINA));
        formatters.add(new EnumFormatter<>(OrderStatus.class));

        factory.setFormatters(formatters);

        return factory.createConversionService();
    }
}
```

## AnnotationFormatterFactory

### 接口定义

```java
public interface AnnotationFormatterFactory<A extends Annotation> {
    // 获取支持的注解类型
    Class<?>[] getFieldTypes();

    // 获取 Printer
    Printer<?> getPrinter(A annotation, Class<?> fieldType);

    // 获取 Parser
    Parser<?> getParser(A annotation, Class<?> fieldType);
}
```

### 自定义注解

```java
// 定义注解
@Target(ElementType.FIELD)
@Retention(RetentionPolicy.RUNTIME)
public @interface PhoneNumber {
    String value() default "";
}

// 创建工厂
public class PhoneNumberAnnotationFormatterFactory
        implements AnnotationFormatterFactory<PhoneNumber> {

    @Override
    public Class<?>[] getFieldTypes() {
        return new Class[]{String.class};
    }

    @Override
    public Printer<?> getPrinter(PhoneNumber annotation, Class<?> fieldType) {
        return (object, locale) -> format((String) object);
    }

    @Override
    public Parser<?> getParser(PhoneNumber annotation, Class<?> fieldType) {
        return (text, locale) -> clean(text);
    }

    private String format(String phone) {
        if (phone == null || phone.length() != 11) {
            return phone;
        }
        return phone.substring(0, 3) + "-" +
               phone.substring(3, 7) + "-" +
               phone.substring(7);
    }

    private String clean(String phone) {
        return phone == null ? null : phone.replace("-", "");
    }
}
```

### 使用

```java
public class User {
    @PhoneNumber
    private String phone;
}
```

## Thymeleaf 中的格式化

### 数字格式化

```html
<!-- 默认数字格式 -->
<span th:text="${#numbers.formatDecimal(product.price, 1, 2)}">0.00</span>

<!-- 货币格式 -->
<span th:text="${#numbers.formatCurrency(product.price)}">$0.00</span>

<!-- 百分比格式 -->
<span th:text="${#numbers.formatPercent(product.discount, 0, 0)}">0%</span>
```

### 日期格式化

```html
<!-- ISO 格式 -->
<span th:text="${#temporals.format(user.createTime)}">2024-01-15</span>

<!-- 自定义格式 -->
<span th:text="${#temporals.format(user.createTime, 'yyyy-MM-dd HH:mm')}">2024-01-15 14:30</span>
```

## 面试追问方向

**「Formatter 和 Converter 的区别？」**

- `Converter`：类型转换（S → T）
- `Formatter`：解析/打印（String ↔ T），支持 Locale

**「@NumberFormat 和 @DateTimeFormat 有什么区别？」**

- `@NumberFormat`：数字格式化（货币、百分比等）
- `@DateTimeFormat`：日期时间格式化

**「Spring Boot 如何自动配置格式化？」**

通过 `FormattingConversionServiceFactoryBean` 自动注册常用格式化器。

---

## 总结

Spring 的格式化机制：

| 组件 | 说明 |
|---|---|
| `Formatter<T>` | 解析/打印接口 |
| `Printer<T>` | 打印接口 |
| `Parser<T>` | 解析接口 |
| `@NumberFormat` | 数字格式化注解 |
| `@DateTimeFormat` | 日期时间格式化注解 |
| `FormattingConversionService` | 格式化转换服务 |

Formatter 用于 UI 层的解析和打印，Converter 用于业务层的类型转换。
