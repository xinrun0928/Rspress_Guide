## MyBatis 是什么？

MyBatis 是一个优秀的持久层框架，它封装了 JDBC 的所有繁琐操作，让开发者只需关注 SQL 本身，而不是 JDBC API 的调用。

**与 Hibernate 的区别**：

| 特性 | MyBatis | Hibernate |
|---|---|---|
| ORM 程度 | 半自动 | 全自动 |
| SQL 控制 | 开发者手写 | 框架自动生成 |
| 学习曲线 | 中等 | 陡峭 |
| 灵活性 | 高 | 中 |
| 性能 | 高 | 中 |
| 适用场景 | 复杂 SQL、精细控制 | 标准 CRUD、快速开发 |

## 核心概念

### SqlSessionFactory

SqlSessionFactory 是 MyBatis 的核心入口，负责创建 SqlSession。

```java
// 1. 创建 SqlSessionFactory
SqlSessionFactory sqlSessionFactory = new SqlSessionFactoryBuilder()
    .build(inputStream);

// 2. 创建 SqlSession
SqlSession session = sqlSessionFactory.openSession();

// 3. 获取 Mapper
UserMapper mapper = session.getMapper(UserMapper.class);

// 4. 执行 CRUD
User user = mapper.selectById(1L);

// 5. 关闭
session.close();
```

### 四大核心组件

MyBatis 的 SQL 执行涉及四大核心组件：

1. **Executor**：SQL 执行调度器
2. **StatementHandler**：SQL 语句处理器
3. **ParameterHandler**：参数处理器
4. **ResultSetHandler**：结果集处理器
