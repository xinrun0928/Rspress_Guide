# Shiro：轻量级安全框架的优雅之选

想象一下这样的场景：凌晨两点，你接到电话说系统被入侵了，用户数据疑似泄露。你火急火燎打开代码，发现安全模块一团糟——自定义的过滤器、散落各处的校验逻辑、还有不知道谁加的"临时方案"。如果你早点用上一个成熟的权限框架，这一切本可以避免。

Apache Shiro 就是来解决这个问题的。它是一个 Java 安全框架，把认证（Authentication）、授权（Authorization）、会话管理（Session Management）、密码加密（Cryptography）等安全工作封装成统一的 API，让开发者从重复造轮子中解放出来。

相比 Spring Security，Shiro 的设计理念是**简单而不简陋**——它没有 Spring 的强依赖，却能完成几乎同等的安全能力。这篇文档会带你从架构设计、认证授权流程，到 Spring Boot 集成、无状态改造，系统掌握 Shiro 的核心用法。
