# Shiro 与 Spring Boot：快速集成指南

Spring Boot 时代，一切都追求自动化配置。Spring Security 有 `spring-boot-starter-security` 开箱即用，而 Shiro 在 Spring Boot 2.x 之后需要手动配置。

好消息是：Shiro 的手动配置并不复杂。只需要几个 Bean，就能把 Shiro 集成到 Spring Boot 项目中。

本章将带你从零开始，搭建一个完整的 Shiro + Spring Boot 项目。

## 核心配置类

### ShiroConfig

```java title="src/main/java/com/example/shiro/config/ShiroConfig.java"
@Configuration
public class ShiroConfig {

    /**
     * 密码加密匹配器
     */
    @Bean
    public HashedCredentialsMatcher hashedCredentialsMatcher() {
        HashedCredentialsMatcher matcher = new HashedCredentialsMatcher();
        matcher.setHashAlgorithmName("SHA-256");
        matcher.setHashIterations(1024);
        matcher.setStoredCredentialsHexEncoded(false);
        return matcher;
    }

    /**
     * 自定义 Realm
     */
    @Bean
    public UserRealm userRealm(HashedCredentialsMatcher matcher) {
        UserRealm realm = new UserRealm();
        realm.setCredentialsMatcher(matcher);
        realm.setCachingEnabled(true);
        realm.setAuthorizationCachingEnabled(true);
        return realm;
    }

    /**
     * Session 管理器
     */
    @Bean
    public DefaultWebSessionManager sessionManager() {
        DefaultWebSessionManager sessionManager = new DefaultWebSessionManager();
        sessionManager.setGlobalSessionTimeout(1800000);  // 30 分钟
        sessionManager.setSessionIdCookieEnabled(true);
        return sessionManager;
    }

    /**
     * SecurityManager
     */
    @Bean
    public SecurityManager securityManager(UserRealm realm,
                                          DefaultWebSessionManager sessionManager,
                                          RememberMeManager rememberMeManager) {
        DefaultWebSecurityManager manager = new DefaultWebSecurityManager(realm);
        manager.setSessionManager(sessionManager);
        manager.setRememberMeManager(rememberMeManager);
        return manager;
    }

    /**
     * RememberMe 管理器
     */
    @Bean
    public RememberMeManager rememberMeManager() {
        SimpleRememberMeManager manager = new SimpleRememberMeManager();
        manager.setCipherKey(Base64.decode(
            "kPH+bIxk5D2deZiIxpm6TrYpDgF0mBJB8YdVHN2fN0Y="));
        SimpleCookie cookie = new SimpleCookie("rememberMe");
        cookie.setMaxAge(604800);  // 7 天
        cookie.setHttpOnly(true);
        manager.setCookie(cookie);
        return manager;
    }

    /**
     * Shiro 过滤器链配置
     */
    @Bean
    public ShiroFilterChainDefinition shiroFilterChainDefinition() {
        DefaultShiroFilterChainDefinition chainDefinition =
            new DefaultShiroFilterChainDefinition();

        // 公开路径
        chainDefinition.addPathDefinition("/login", "anon");
        chainDefinition.addPathDefinition("/register", "anon");
        chainDefinition.addPathDefinition("/static/**", "anon");
        chainDefinition.addPathDefinition("/api/public/**", "anon");

        // 登出
        chainDefinition.addPathDefinition("/logout", "logout");

        // 需要认证
        chainDefinition.addPathDefinition("/**", "authc");

        return chainDefinition;
    }

    /**
     * ShiroFilterFactoryBean
     */
    @Bean
    public ShiroFilterFactoryBean shiroFilterFactoryBean(SecurityManager securityManager,
                                                         ShiroFilterChainDefinition chainDefinition) {
        ShiroFilterFactoryBean factory = new ShiroFilterFactoryBean();
        factory.setSecurityManager(securityManager);
        factory.setFilterChainDefinitionMap(chainDefinition.getFilterChainMap());
        return factory;
    }

    /**
     * 启用 Shiro 注解
     */
    @Bean
    public AuthorizationAttributeSourceAdvisor authorizationAttributeSourceAdvisor(
            SecurityManager securityManager) {
        AuthorizationAttributeSourceAdvisor advisor =
            new AuthorizationAttributeSourceAdvisor();
        advisor.setSecurityManager(securityManager);
        return advisor;
    }
}
```

## UserRealm 实现

```java title="src/main/java/com/example/shiro/realm/UserRealm.java"
@Component
public class UserRealm extends AuthorizingRealm {

    @Autowired
    private UserService userService;

    @Autowired
    private RoleService roleService;

    @Autowired
    private PermissionService permissionService;

    /**
     * 认证
     */
    @Override
    protected AuthenticationInfo doGetAuthenticationInfo(
            AuthenticationToken token) throws AuthenticationException {

        UsernamePasswordToken upToken = (UsernamePasswordToken) token;
        String username = upToken.getUsername();

        User user = userService.findByUsername(username);
        if (user == null) {
            throw new UnknownAccountException("用户不存在");
        }

        if (Boolean.TRUE.equals(user.getLocked())) {
            throw new LockedAccountException("账户已被锁定");
        }

        return new SimpleAuthenticationInfo(
            user.getUsername(),
            user.getPassword(),
            ByteSource.Util.bytes(user.getSalt()),
            getName()
        );
    }

    /**
     * 授权
     */
    @Override
    protected AuthorizationInfo doGetAuthorizationInfo(
            PrincipalCollection principals) {

        String username = (String) principals.getPrimaryPrincipal();

        Set<String> roles = roleService.findRoleCodesByUsername(username);
        Set<String> permissions = permissionService.findPermissionCodesByUsername(username);

        SimpleAuthorizationInfo info = new SimpleAuthorizationInfo();
        info.setRoles(roles);
        info.setStringPermissions(permissions);
        return info;
    }
}
```

## 登录控制器

```java title="src/main/java/com/example/shiro/controller/LoginController.java"
@Controller
public class LoginController {

    @PostMapping("/login")
    public String login(@RequestParam String username,
                        @RequestParam String password,
                        @RequestParam(required = false) Boolean rememberMe,
                        RedirectAttributes attributes) {
        Subject subject = SecurityUtils.getSubject();

        UsernamePasswordToken token = new UsernamePasswordToken(username, password);
        if (Boolean.TRUE.equals(rememberMe)) {
            token.setRememberMe(true);
        }

        try {
            subject.login(token);
            return "redirect:/index";
        } catch (AuthenticationException e) {
            attributes.addFlashAttribute("error", e.getMessage());
            return "redirect:/login";
        }
    }

    @GetMapping("/logout")
    public String logout() {
        Subject subject = SecurityUtils.getSubject();
        subject.logout();
        return "redirect:/login";
    }
}
```

## 过滤器链详解

Shiro 的过滤器链是理解 Shiro Web 集成的核心。每个 URL 路径对应一个或多个过滤器：

| 过滤器 | 代码 | 说明 |
|---|---|---|
| 匿名 | `anon` | 不需要认证 |
| 认证 | `authc` | 需要登录认证 |
| 基础认证 | `authcBasic` | HTTP Basic 认证 |
| 登出 | `logout` | 登出并清除 Session |
| SSL | `ssl` | 强制 HTTPS |
| 权限 | `perms` | 需要特定权限 |
| 角色 | `roles` | 需要特定角色 |
| 用户 | `user` | 已认证或 RememberMe |

### 过滤器链顺序

过滤器按配置顺序匹配，**第一个匹配的过滤器生效**：

```java
@Bean
public ShiroFilterChainDefinition shiroFilterChainDefinition() {
    DefaultShiroFilterChainDefinition chainDefinition =
        new DefaultShiroFilterChainDefinition();

    // 必须放在前面，防止被 authc 拦截
    chainDefinition.addPathDefinition("/logout", "logout");
    chainDefinition.addPathDefinition("/login", "anon");

    // 静态资源
    chainDefinition.addPathDefinition("/static/**", "anon");

    // API 权限控制
    chainDefinition.addPathDefinition("/api/admin/**", "roles[admin]");
    chainDefinition.addPathDefinition("/api/user/**", "authc");

    // 其他请求需要认证
    chainDefinition.addPathDefinition("/**", "authc");

    return chainDefinition;
}
```

## 常见问题

### 问题一：Shiro 注解不生效

Shiro 注解需要配合 AOP 使用：

```java
@Bean
public AuthorizationAttributeSourceAdvisor authorizationAttributeSourceAdvisor(
        SecurityManager securityManager) {
    AuthorizationAttributeSourceAdvisor advisor =
        new AuthorizationAttributeSourceAdvisor();
    advisor.setSecurityManager(securityManager);
    return advisor;
}
```

如果使用 Spring Boot 3.x，还需要添加：

```xml
<dependency>
    <groupId>org.aspectj</groupId>
    <artifactId>aspectjweaver</artifactId>
    <version>1.9.20</version>
</dependency>
```

### 问题二：Session 并发控制

Spring Boot 2.x 默认不允许同一账号多处登录。如果需要允许：

```yaml
shiro:
  session:
    max-sessions: -1  # 不限制
```

### 问题三：与 Spring Security 冲突

如果同时引入了 Spring Security 和 Shiro，需要排除 Spring Security：

```java
@SpringBootApplication(exclude = {SecurityAutoConfiguration.class})
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

## 本章小结

Shiro + Spring Boot 集成的核心是三个 Bean：

```
├── SecurityManager：安全管理器
├── ShiroFilterFactoryBean：过滤器链配置
└── AuthorizationAttributeSourceAdvisor：注解支持
```

配置虽然多一些，但逻辑清晰，完全可控。
