# Shiro 与 Spring Boot：快速集成指南

Spring Boot 时代，一切都追求自动化配置。Spring Security 有 `spring-boot-starter-security` 开箱即用，而 Shiro 在 Spring Boot 2.x 之后需要手动配置。

好消息是：Shiro 的手动配置并不复杂。只需要几个 Bean，就能把 Shiro 集成到 Spring Boot 项目中。

## 为什么选择 Shiro？

| 特性 | Shiro | Spring Security |
|---|---|---|
| 学习曲线 | 平缓 | 陡峭 |
| 配置复杂度 | 简单 | 复杂 |
| 依赖 | 无 Spring 依赖 | 深度耦合 Spring |
| 功能 | 基础安全够用 | OAuth2/LDAP 等企业级功能 |
| 适用场景 | 中小型项目 | 企业级项目 |

## 本章内容

| 章节 | 核心内容 |
|---|---|
| `springboot` | Shiro + Spring Boot 基础配置 |
| `filter-chain` | ShiroFilterFactoryBean 和过滤器链 |
| `annotation` | Shiro 注解：@RequiresAuthentication 等 |
| `jsp-tag` | Shiro JSP 标签库和 Thymeleaf 整合 |
| `cache` | Shiro 缓存：CacheManager 与权限缓存 |
| `session-redis` | 分布式 Session：RedisSessionDAO |

## 快速开始

### 1. 引入依赖

```xml
<dependency>
    <groupId>org.apache.shiro</groupId>
    <artifactId>shiro-spring-boot-web-starter</artifactId>
    <version>1.12.0</version>
</dependency>
```

### 2. 配置 SecurityManager

```java
@Configuration
public class ShiroConfig {

    @Bean
    public SecurityManager securityManager(UserRealm realm) {
        DefaultWebSecurityManager manager = new DefaultWebSecurityManager(realm);
        return manager;
    }

    @Bean
    public ShiroFilterChainDefinition shiroFilterChainDefinition() {
        DefaultShiroFilterChainDefinition chain = new DefaultShiroFilterChainDefinition();
        chain.addPathDefinition("/login", "anon");
        chain.addPathDefinition("/logout", "logout");
        chain.addPathDefinition("/**", "authc");
        return chain;
    }
}
```

### 3. 实现 UserRealm

```java
@Component
public class UserRealm extends AuthorizingRealm {

    @Override
    protected AuthorizationInfo doGetAuthorizationInfo(PrincipalCollection principals) {
        // 加载权限
    }

    @Override
    protected AuthenticationInfo doGetAuthenticationInfo(AuthenticationToken token) {
        // 验证密码
    }
}
```

就这么简单，一个可用的 Shiro 安全系统就跑起来了。

## 本章小结

Shiro 与 Spring Boot 的集成关键在于三个配置：

1. **SecurityManager**：核心安全管理器
2. **ShiroFilterFactoryBean**：URL 级别的安全控制
3. **AuthorizationAttributeSourceAdvisor**：方法级别的安全注解
