# MyBatis Plus：MyBatis 的增强利器

<Badge text="MyBatis Plus" type="tip" /> <Badge text="增强框架" type="info" />

MyBatis Plus 是 MyBatis 的增强工具，在 MyBatis 的基础上只做增强不做改变，为简化开发、提高效率而生。

当你使用 MyBatis 写 CRUD 时，是否觉得大量重复的 SQL 让代码变得臃肿？MyBatis Plus 正是为了解决这个问题——它内置了大量通用 CRUD 操作，让你可以用最少的代码完成数据库操作。

## MyBatis Plus 解决了什么问题？

### MyBatis 的痛点

使用原生 MyBatis 时，单表 CRUD 仍然需要编写大量重复代码：

```java
// MyBatis：每个表都需要写这些
public interface UserMapper {
    User selectById(Long id);
    List<User> selectList();
    int insert(User user);
    int update(User user);
    int deleteById(Long id);
}
```

```xml
<!-- 还要写对应的 XML -->
<select id="selectById" resultType="User">
    SELECT * FROM user WHERE id = #{id}
</select>
<select id="selectList" resultType="User">
    SELECT * FROM user
</select>
<insert id="insert">
    INSERT INTO user (name, email) VALUES (#{name}, #{email})
</insert>
<update id="update">
    UPDATE user SET name = #{name}, email = #{email} WHERE id = #{id}
</update>
<delete id="deleteById">
    DELETE FROM user WHERE id = #{id}
</delete>
```

### MyBatis Plus 的解决方案

```java
// MyBatis Plus：继承 BaseMapper，CRUD 都有了
public interface UserMapper extends BaseMapper<User> {
    // 自定义方法可以写在这里
}

// 使用：直接调用内置方法
userMapper.selectById(1L);
userMapper.selectList(new QueryWrapper<>());
userMapper.insert(user);
userMapper.updateById(user);
userMapper.deleteById(1L);
```

**不需要写一行 SQL！**

## 核心特性一览

| 特性 | 说明 |
|------|------|
| 无侵入 | 只做增强，不改变原有 MyBatis 的设计 |
| 损耗小 | 启动即会自动注入基本 CRUD，性能无损耗 |
| 强大的 CRUD | 内置通用 Mapper、Service，无需编写任何 SQL |
| 支持 Lambda | 使用 Lambda 表达式编写查询条件，类型安全 |
| 支持主键自动生成 | 支持 4 种主键生成策略 |
| 内置分页插件 | 基于物理分页，性能优秀 |
| 内置全局拦截 | 逻辑删除、自动填充等全局生效 |
| 自带 SQL 注入 | 内置防止 SQL 注入 |

## 延伸思考

### MyBatis Plus 的设计哲学

MyBatis Plus 的核心理念是**约定优于配置**。它通过注解和全局配置，消除了大量重复的配置工作，让开发者专注于业务逻辑。

### 什么时候不用 MyBatis Plus？

1. **极简场景**：只有几张表，不需要复杂查询——用 MyBatis Plus 可能过度设计
2. **特殊 SQL**：需要使用 MyBatis 的高级特性——MyBatis Plus 可能有限制
3. **团队习惯**：团队已经习惯原生 MyBatis——迁移成本较高

### MyBatis Plus vs MyBatis-Flex vs JPA

| 特性 | MyBatis Plus | MyBatis-Flex | JPA |
|------|--------------|--------------|-----|
| 单表 CRUD | 内置 BaseMapper | 内置 BaseService | 内置 Repository |
| 动态 SQL | XML/注解 | 链式 API | Criteria API |
| 分页 | 内置 | 内置 | 依赖实现 |
| 社区活跃度 | 高 | 中 | 高 |
