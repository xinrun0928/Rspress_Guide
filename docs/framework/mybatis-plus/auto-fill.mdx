# MyBatis Plus 自动填充：createTime 不再需要手动维护

<Badge text="MyBatis Plus" type="tip" /> <Badge text="核心功能" type="info" />

每次插入数据都要手动设置 createTime？每次更新都要更新 updateTime？

MyBatis Plus 的自动填充功能，可以让你在插入或更新数据时，自动填充某些字段，无需手动处理。

## 自动填充原理

MyBatis Plus 通过 `MetaObjectHandler` 拦截 SQL 执行，在适当的时间点自动填充字段值：

```
INSERT 操作
    ↓
MyBatis Plus 拦截
    ↓
调用 MetaObjectHandler.fillInsertTime()
    ↓
自动填充字段值
    ↓
执行 SQL
```

## 配置步骤

### 第一步：实体类添加注解

```java
@TableName("user")
public class User {

    @TableId(type = IdType.AUTO)
    private Long id;

    private String name;

    private Integer age;

    // 插入时自动填充
    @TableField(fill = FieldFill.INSERT)
    private LocalDateTime createTime;

    // 插入和更新时自动填充
    @TableField(fill = FieldFill.INSERT_UPDATE)
    private LocalDateTime updateTime;

    // 更新时自动填充
    @TableField(fill = FieldFill.UPDATE)
    private LocalDateTime lastUpdateTime;
}
```

| 填充策略 | 说明 |
|---------|------|
| `DEFAULT` | 默认不处理 |
| `INSERT` | 插入时填充 |
| `UPDATE` | 更新时填充 |
| `INSERT_UPDATE` | 插入和更新时都填充 |

### 第二步：实现 MetaObjectHandler

```java
@Component
public class MyMetaObjectHandler implements MetaObjectHandler {

    @Override
    public void insertFill(MetaObject metaObject) {
        // 插入时填充
        this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());
        this.strictInsertFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }

    @Override
    public void updateFill(MetaObject metaObject) {
        // 更新时填充
        this.strictUpdateFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }
}
```

### 第三步：注册配置

```java
@Configuration
public class MyBatisPlusConfig {

    @Bean
    public MetaObjectHandler metaObjectHandler() {
        return new MyMetaObjectHandler();
    }
}
```

## 填充策略详解

### FieldFill 枚举

```java
public enum FieldFill {
    /**
     * 默认不处理
     */
    DEFAULT,
    /**
     * 插入时填充
     */
    INSERT,
    /**
     * 更新时填充
     */
    UPDATE,
    /**
     * 插入和更新时都填充
     */
    INSERT_UPDATE
}
```

### 不同场景下的填充

```java
@TableName("user")
public class User {

    // 插入时填充
    @TableField(fill = FieldFill.INSERT)
    private LocalDateTime createTime;  // 创建时间

    // 更新时填充
    @TableField(fill = FieldFill.UPDATE)
    private LocalDateTime updateTime;  // 更新时间

    // 插入和更新时都填充
    @TableField(fill = FieldFill.INSERT_UPDATE)
    private LocalDateTime modifyTime;  // 最后修改时间
}
```

## MetaObjectHandler 详解

### 基础实现

```java
@Component
public class MyMetaObjectHandler implements MetaObjectHandler {

    /**
     * 插入填充
     */
    @Override
    public void insertFill(MetaObject metaObject) {
        // 1. 获取当前时间
        LocalDateTime now = LocalDateTime.now();

        // 2. 填充创建时间
        this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, now);

        // 3. 填充更新时间
        this.strictInsertFill(metaObject, "updateTime", LocalDateTime.class, now);

        // 4. 填充创建人
        this.strictInsertFill(metaObject, "createBy", Long.class, getCurrentUserId());

        // 5. 填充更新人
        this.strictInsertFill(metaObject, "updateBy", Long.class, getCurrentUserId());
    }

    /**
     * 更新填充
     */
    @Override
    public void updateFill(MetaObject metaObject) {
        // 1. 更新时间
        this.strictUpdateFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());

        // 2. 更新人
        this.strictUpdateFill(metaObject, "updateBy", Long.class, getCurrentUserId());
    }

    /**
     * 获取当前用户 ID（需要根据实际情况实现）
     */
    private Long getCurrentUserId() {
        // 从 ThreadLocal 或 SecurityContext 获取
        return 1L;
    }
}
```

### fill 方法对比

```java
@Override
public void insertFill(MetaObject metaObject) {

    // strictInsertFill：只有字段为空时才填充
    this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());

    // fillStrategyFill：无论字段是否有值都填充
    this.fillStrategyFill(metaObject, "createTime", LocalDateTime.now());
}
```

| 方法 | 说明 |
|------|------|
| `strictInsertFill` | 只有字段为空（null）时才填充 |
| `strictUpdateFill` | 只有字段为空（null）时才填充 |
| `fillStrategyFill` | 无论字段是否有值都填充（覆盖） |

## 常见填充场景

### 创建时间、更新时间

```java
@Component
public class CommonMetaObjectHandler implements MetaObjectHandler {

    @Override
    public void insertFill(MetaObject metaObject) {
        this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());
        this.strictInsertFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }

    @Override
    public void updateFill(MetaObject metaObject) {
        this.strictUpdateFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }
}
```

### 创建人、更新人

```java
@Component
public class UserMetaObjectHandler implements MetaObjectHandler {

    @Override
    public void insertFill(MetaObject metaObject) {
        Long userId = getCurrentUserId();
        this.strictInsertFill(metaObject, "createBy", Long.class, userId);
        this.strictInsertFill(metaObject, "updateBy", Long.class, userId);
    }

    @Override
    public void updateFill(MetaObject metaObject) {
        this.strictUpdateFill(metaObject, "updateBy", Long.class, getCurrentUserId());
    }

    private Long getCurrentUserId() {
        // Spring Security 场景
        Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
        if (authentication != null && authentication.isAuthenticated()) {
            return ((LoginUser) authentication.getPrincipal()).getId();
        }
        return 1L; // 默认值
    }
}
```

### 版本号（乐观锁）

```java
@Component
public class VersionMetaObjectHandler implements MetaObjectHandler {

    @Override
    public void insertFill(MetaObject metaObject) {
        // 初始版本号为 1
        this.strictInsertFill(metaObject, "version", Integer.class, 1);
    }

    @Override
    public void updateFill(MetaObject metaObject) {
        // 更新时不处理版本号（乐观锁插件会自动处理）
    }
}
```

### 逻辑删除

```java
@Component
public class DeletedMetaObjectHandler implements MetaObjectHandler {

    @Override
    public void insertFill(MetaObject metaObject) {
        // 初始未删除
        this.strictInsertFill(metaObject, "deleted", Integer.class, 0);
    }

    @Override
    public void updateFill(MetaObject metaObject) {
        // 更新时不处理删除状态
    }
}
```

## 分布式下的时间问题

### 时钟回拨问题

雪花算法依赖时间，如果服务器时钟回拨，会产生重复 ID：

```java
// 雪花算法依赖时间
if (timestamp < lastTimestamp) {
    throw new RuntimeException("Clock moved backwards!");
}
```

**解决方案**：

1. 使用美团的 Leaf（号段 + 雪花）
2. 使用百度的 UidGenerator
3. 使用 Oracle/Redis 序列

### 时区问题

```java
// 错误：使用本地时间
this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());

// 正确：使用 UTC 时间
this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now(ZoneOffset.UTC));

// 或者使用指定时区
this.strictInsertFill(metaObject, "createTime", LocalDateTime.class,
    ZonedDateTime.now(ZoneId.of("Asia/Shanghai")).toLocalDateTime());
```

### 统一时区配置

```yaml
spring:
  jackson:
    time-zone: Asia/Shanghai
    date-format: yyyy-MM-dd HH:mm:ss

mybatis-plus:
  configuration:
    # 设置 JDBC 时区
    jdbc:
      time-zone: Asia/Shanghai
```

## 高级用法

### 条件填充

```java
@Override
public void updateFill(MetaObject metaObject) {
    // 只在特定条件下填充
    Object status = getFieldValByName("status", metaObject);
    if ("ACTIVE".equals(status)) {
        this.strictUpdateFill(metaObject, "activeTime", LocalDateTime.class, LocalDateTime.now());
    }
}
```

### 多值填充

```java
@Override
public void insertFill(MetaObject metaObject) {
    ArrayList<MetaObjectHandler> handlers = new ArrayList<>();
    handlers.add((handler, metaObj) ->
        this.strictInsertFill(metaObj, "createTime", LocalDateTime.class, LocalDateTime.now()));
    handlers.add((handler, metaObj) ->
        this.strictInsertFill(metaObj, "createBy", Long.class, getCurrentUserId()));

    for (MetaObjectHandler handler : handlers) {
        handler.insertFill(metaObject);
    }
}
```

## 注意事项

### 字段值判断

```java
@Override
public void insertFill(MetaObject metaObject) {

    // 方法一：使用 hasSetter 判断是否有 setter
    if (metaObject.hasSetter("createTime")) {
        this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());
    }

    // 方法二：使用 getFieldValByName 判断字段值
    Object createTime = getFieldValByName("createTime", metaObject);
    if (createTime == null) {
        this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());
    }
}
```

### 性能注意

```java
// 低效：每次填充都创建新的对象
this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());

// 高效：使用缓存的对象
private static final LocalDateTime NOW = LocalDateTime.now();
this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, NOW);
```

### 注解优先级

```java
@TableField(fill = FieldFill.INSERT)
private LocalDateTime createTime;
```

如果实体类有注解，会覆盖全局配置。

## 常见问题

### Q1：自动填充不生效？

1. 检查实体类是否添加了 `@TableField(fill = FieldFill.INSERT)` 注解
2. 检查 `MetaObjectHandler` 是否被 Spring 扫描到
3. 检查 `MetaObjectHandler` 是否正确注册为 Bean

### Q2：更新时不想自动填充某个字段？

```java
// 如果不想让自动填充覆盖已有值，使用 strictUpdateFill
this.strictUpdateFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
```

### Q3：多个实体类需要不同的填充策略？

可以创建多个 `MetaObjectHandler`，每个处理不同的表：

```java
@Component
public class UserMetaObjectHandler implements MetaObjectHandler {
    // 只处理 User 表的填充
}

@Component
public class OrderMetaObjectHandler implements MetaObjectHandler {
    // 只处理 Order 表的填充
}
```

## 总结

### 配置步骤

1. **实体类添加注解**：`@TableField(fill = FieldFill.INSERT)`
2. **实现 MetaObjectHandler**
3. **注册为 Bean**

### 填充策略

| 策略 | 说明 |
|------|------|
| `FieldFill.INSERT` | 插入时填充 |
| `FieldFill.UPDATE` | 更新时填充 |
| `FieldFill.INSERT_UPDATE` | 插入和更新时都填充 |

### 填充内容

| 内容 | 示例 |
|------|------|
| 时间字段 | `createTime`、`updateTime` |
| 用户字段 | `createBy`、`updateBy` |
| 版本号 | `version` |
| 逻辑删除 | `deleted` |

自动填充是 MyBatis Plus 最实用的功能之一，合理使用可以大幅减少重复代码，让代码更加整洁。
