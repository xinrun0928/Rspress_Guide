# O(n log n) 排序算法

**面试高频题：「手写一个排序算法，要求平均 O(n log n) 复杂度。」**

这道题有三种标准答案：归并排序、快速排序、堆排序。每种都有它的特点，每种都可能在面试中被要求实现。

## 归并排序

### 核心思想

**分治**：把数组分成两半，分别排序，然后合并。

```
[8, 3, 1, 7, 0, 10, 2]
    ↓ 分
[8, 3, 1, 7]    [0, 10, 2]
   ↓ 分          ↓ 分
[8, 3] [1, 7]  [0, 10] [2]
   ↓ 分          ↓ 分
[8] [3] [1] [7] [0] [10] [2]
   ↓ 合          ↓ 合
[3, 8] [1, 7]  [0, 10] [2]
    ↓ 合          ↓ 合
  [1, 3, 7, 8]   [0, 2, 10]
      ↓ 合
    [0, 1, 2, 3, 7, 8, 10]
```

### 代码实现

```java
public class MergeSort {
    public void mergeSort(int[] arr) {
        mergeSort(arr, 0, arr.length - 1);
    }

    private void mergeSort(int[] arr, int left, int right) {
        if (left >= right) return;

        int mid = left + (right - left) / 2;
        mergeSort(arr, left, mid);
        mergeSort(arr, mid + 1, right);
        merge(arr, left, mid, right);
    }

    private void merge(int[] arr, int left, int mid, int right) {
        int[] temp = new int[right - left + 1];
        int i = left, j = mid + 1, k = 0;

        while (i <= mid && j <= right) {
            if (arr[i] <= arr[j]) {
                temp[k++] = arr[i++];
            } else {
                temp[k++] = arr[j++];
            }
        }

        while (i <= mid) temp[k++] = arr[i++];
        while (j <= right) temp[k++] = arr[j++];

        // 复制回原数组
        for (int p = 0; p < k; p++) {
            arr[left + p] = temp[p];
        }
    }
}
```

### 复杂度分析

| 情况 | 时间复杂度 | 空间复杂度 |
|------|------------|------------|
| 最好 | O(n log n) | O(n) |
| 平均 | O(n log n) | O(n) |
| 最坏 | O(n log n) | O(n) |

**稳定排序**，但需要 O(n) 额外空间。

## 快速排序

### 核心思想

**分治 + 原地分区**：选择一个基准（pivot），把数组分成两部分，小于 pivot 的在左边，大于 pivot 的在右边。

```
初始：[3, 7, 8, 1, 5, 9, 2]
选择 pivot = 5
分区后：[3, 1, 2, 5, 8, 7, 9]
         ↓        ↓
      继续排序    继续排序
```

### 代码实现

```java
public class QuickSort {
    public void quickSort(int[] arr) {
        quickSort(arr, 0, arr.length - 1);
    }

    private void quickSort(int[] arr, int left, int right) {
        if (left >= right) return;

        int pivotIndex = partition(arr, left, right);
        quickSort(arr, left, pivotIndex - 1);
        quickSort(arr, pivotIndex + 1, right);
    }

    private int partition(int[] arr, int left, int right) {
        int pivot = arr[right];
        int i = left - 1;

        for (int j = left; j < right; j++) {
            if (arr[j] <= pivot) {
                i++;
                swap(arr, i, j);
            }
        }
        swap(arr, i + 1, right);
        return i + 1;
    }
}
```

### 优化：双路快排

```java
private int partition2Ways(int[] arr, int left, int right) {
    int pivot = arr[left];
    int i = left + 1, j = right;

    while (true) {
        while (i <= right && arr[i] <= pivot) i++;
        while (j >= left + 1 && arr[j] > pivot) j--;
        if (i > j) break;
        swap(arr, i, j);
        i++;
        j--;
    }
    swap(arr, left, j);
    return j;
}
```

### 优化：三数取中

```java
private int medianOfThree(int[] arr, int left, int right) {
    int mid = left + (right - left) / 2;

    if (arr[left] > arr[mid]) swap(arr, left, mid);
    if (arr[left] > arr[right]) swap(arr, left, right);
    if (arr[mid] > arr[right]) swap(arr, mid, right);

    return mid;  // 把 pivot 放到 right-1 的位置
}
```

### 复杂度分析

| 情况 | 时间复杂度 | 说明 |
|------|------------|------|
| 最好 | O(n log n) | 每次 pivot 都是中位数 |
| 平均 | O(n log n) | 随机情况下表现良好 |
| 最坏 | O(n²) | 每次 pivot 都是最值 |

## 堆排序

### 核心思想

利用堆这种数据结构进行排序。先建最大堆，然后反复把堆顶最大元素移到数组末尾。

```
堆数组：[9, 7, 8, 5, 1, 3, 2, 4, 6]
        0  1  2  3  4  5  6  7  8

完全二叉树：
        9
      /   \
     7     8
    / \   / \
   5   1 3   2
  / \
 4   6
```

### 代码实现

```java
public class HeapSort {
    public void heapSort(int[] arr) {
        int n = arr.length;

        // 1. 构建最大堆（从最后一个非叶子节点开始）
        for (int i = n / 2 - 1; i >= 0; i--) {
            heapify(arr, n, i);
        }

        // 2. 逐个取出堆顶
        for (int i = n - 1; i > 0; i--) {
            swap(arr, 0, i);    // 堆顶移到末尾
            heapify(arr, i, 0); // 重新堆化
        }
    }

    private void heapify(int[] arr, int n, int i) {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;

        if (left < n && arr[left] > arr[largest]) {
            largest = left;
        }
        if (right < n && arr[right] > arr[largest]) {
            largest = right;
        }

        if (largest != i) {
            swap(arr, i, largest);
            heapify(arr, n, largest);
        }
    }
}
```

### 复杂度分析

| 情况 | 时间复杂度 | 空间复杂度 |
|------|------------|------------|
| 最好 | O(n log n) | O(1) |
| 平均 | O(n log n) | O(1) |
| 最坏 | O(n log n) | O(1) |

**不稳定排序**，但**原地排序**（O(1) 空间）。

## 三者对比

| 维度 | 归并排序 | 快速排序 | 堆排序 |
|------|----------|----------|--------|
| **时间复杂度** | O(n log n) | O(n log n) | O(n log n) |
| **最坏情况** | O(n log n) | O(n²) | O(n log n) |
| **空间复杂度** | O(n) | O(log n) | O(1) |
| **稳定性** | 稳定 | 不稳定 | 不稳定 |
| **适用场景** | 外部排序、需要稳定 | 通用场景 | 内存受限 |

:::tip 为什么快速排序实际最快

1. **缓存友好**：顺序访问数据，缓存命中率高
2. **常数因子小**：交换操作比归并的合并操作简单
3. **原地排序**：不需要额外的大数组

:::

## 面试模板

```java
// 快速排序（面试常考）
public void quickSort(int[] arr, int left, int right) {
    if (left >= right) return;

    int pivot = arr[left + (right - left) / 2];
    int i = left, j = right;

    while (i <= j) {
        while (arr[i] < pivot) i++;
        while (arr[j] > pivot) j--;
        if (i <= j) {
            swap(arr, i, j);
            i++;
            j--;
        }
    }

    quickSort(arr, left, j);
    quickSort(arr, i, right);
}
```

## 面试追问

1. **快速排序的最坏情况怎么优化？**
   - 三数取中（median-of-three）
   - 随机化 pivot
   - tail recursive optimization

2. **归并排序和快速排序的本质区别？**
   - 归并：先分后治，先处理子问题再合并
   - 快排：先治后分，先处理分区再递归子问题

3. **什么是外部排序？为什么用归并？**
   - 数据量太大，无法一次性加载内存
   - 归并排序的分治思想天然适合外部排序

**记住：O(n log n) 排序是主流。快速排序是最常用的通用排序，归并排序适合需要稳定性的场景，堆排序适合内存受限的环境。**
