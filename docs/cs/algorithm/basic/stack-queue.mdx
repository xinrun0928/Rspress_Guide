# 栈与队列：实现与应用场景

**括号匹配**怎么判断？**函数调用栈**是怎么回事？**求最大矩形面积**用什么数据结构？**Top K 问题**怎么高效解决？

栈和队列是最基础的数据结构之一，但它们的应用远不止「先进先出」这么简单。**单调栈**、**优先级队列**这些变体，在面试中经常出现。

## 栈：后进先出（LIFO）

### 栈的底层实现

栈的本质是一种**受限的线性表**，只允许在一端（栈顶）进行插入和删除。

```java
// 数组实现栈
class ArrayStack<E> {
    private Object[] data;
    private int top; // 栈顶指针，-1 表示空栈

    public ArrayStack(int capacity) {
        data = new Object[capacity];
        top = -1;
    }

    public void push(E e) {
        if (top == data.length - 1) throw new RuntimeException("Stack full");
        data[++top] = e;
    }

    public E pop() {
        if (top == -1) throw new RuntimeException("Stack empty");
        return (E) data[top--];
    }

    public E peek() {
        if (top == -1) throw new RuntimeException("Stack empty");
        return (E) data[top];
    }
}
```

### 经典应用：括号匹配

> 题目：判断字符串中的括号是否匹配有效。如 `"()[]{}"` 合法，`"([)]"` 不合法。

```java
public boolean isValid(String s) {
    Stack<Character> stack = new Stack<>();
    Map<Character, Character> map = Map.of(')', '(', ']', '[', '}', '{');

    for (char c : s.toCharArray()) {
        if (map.containsKey(c)) {
            // 右括号：检查栈顶是否匹配
            if (stack.isEmpty() || stack.pop() != map.get(c)) {
                return false;
            }
        } else {
            // 左括号：入栈
            stack.push(c);
        }
    }
    return stack.isEmpty(); // 栈空说明全部匹配
}
```

**思路**：遍历字符串，左括号入栈，右括号出栈匹配。最后栈为空则匹配。

### 函数调用栈

函数调用栈是栈最直观的应用：

```java
void A() {
    B();
    System.out.println("A");
}

void B() {
    C();
    System.out.println("B");
}

void C() {
    System.out.println("C");
}
```

调用顺序：`A() -> B() -> C()`
打印顺序：`C B A`（栈的逆序弹出）

**递归调用**就是利用函数调用栈实现的。递归深度过深会导致栈溢出（Stack Overflow）。

## 队列：先进先出（FIFO）

### 队列的底层实现

```java
// 数组实现循环队列
class ArrayQueue<E> {
    private Object[] data;
    private int head, tail, size;

    public ArrayQueue(int capacity) {
        data = new Object[capacity];
        head = tail = 0;
        size = 0;
    }

    public void enqueue(E e) {
        if (size == data.length) throw new RuntimeException("Queue full");
        data[tail] = e;
        tail = (tail + 1) % data.length;
        size++;
    }

    public E dequeue() {
        if (size == 0) throw new RuntimeException("Queue empty");
        E result = (E) data[head];
        head = (head + 1) % data.length;
        size--;
        return result;
    }
}
```

**循环队列**用 `%` 操作实现索引的循环，避免数据搬移。

### BFS 遍历：队列的经典应用

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int levelSize = queue.size();
        List<Integer> level = new ArrayList<>();

        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
```

**层序遍历**用队列按层处理节点，每层节点数量通过 `queue.size()` 控制。

## 单调栈：打破常规的利器

单调栈是栈的进阶应用。核心思想：**栈内元素保持单调递增或递减**。

### 经典问题：柱状图中的最大矩形

> 题目：给定柱状图的高度数组，找出其中的最大矩形面积。

```java
public int largestRectangleArea(int[] heights) {
    int n = heights.length;
    int[] newHeights = new int[n + 2]; // 哨兵，简化边界处理
    System.arraycopy(heights, 0, newHeights, 0, n);

    Stack<Integer> stack = new Stack<>();
    int maxArea = 0;

    for (int i = 0; i < newHeights.length; i++) {
        // 维护单调递增栈
        while (!stack.isEmpty() && newHeights[i] < newHeights[stack.peek()]) {
            int h = newHeights[stack.pop()]; // 弹出高度
            int w = i - stack.peek() - 1;    // 宽度
            maxArea = Math.max(maxArea, h * w);
        }
        stack.push(i);
    }
    return maxArea;
}
```

**为什么用单调栈？** 对于每个柱子，我们要找它左边和右边第一个比它矮的位置。单调递增栈从左到右遍历时，栈顶元素遇到更小的值就「知道」了自己的右边界。

### 单调栈模板

```java
public void monotonicStack(int[] nums) {
    Stack<Integer> stack = new Stack<>();

    for (int i = 0; i < nums.length; i++) {
        // 单调递增：while (栈非空 && 当前值 < 栈顶) { 处理栈顶 }
        while (!stack.isEmpty() && nums[i] < nums[stack.peek()]) {
            int top = stack.pop();
            // 此时 i 是 top 右边第一个更小的
            // 栈中下一个元素是 top 左边第一个更小的
            process(top);
        }
        stack.push(i);
    }

    // 处理剩余栈中元素（右边没有更小的）
    while (!stack.isEmpty()) {
        process(stack.pop());
    }
}
```

### 更多单调栈应用

| 问题 | 核心思路 |
|------|----------|
| 下一个更大元素 | 单调递减栈 |
| 每日温度（股票问题） | 单调递减栈，记录索引差 |
| 去除重复字母 | 单调递增栈 + 计数 |

## 优先级队列：带权重的队列

优先级队列（堆）不是严格意义的「队列」，但用途极广。

### Top K 问题

> 题目：找出数组中的第 K 大的元素。

```java
// 方法一：排序，O(n log n)
public int findKthLargest1(int[] nums, int k) {
    Arrays.sort(nums);
    return nums[nums.length - k];
}

// 方法二：堆排序，O(n log k)
public int findKthLargest2(int[] nums, int k) {
    PriorityQueue<Integer> minHeap = new PriorityQueue<>();
    for (int num : nums) {
        minHeap.offer(num);
        if (minHeap.size() > k) minHeap.poll(); // 维护大小为 k 的小顶堆
    }
    return minHeap.peek();
}
```

**堆的核心特性**：小顶堆的堆顶永远是最小值。当堆大小超过 k 时，弹出最小值，最终堆内就是最大的 k 个数，堆顶就是第 k 大的数。

### 合并 K 个有序链表

```java
public ListNode mergeKLists(ListNode[] lists) {
    PriorityQueue<ListNode> heap = new PriorityQueue<>(
        Comparator.comparingInt(node -> node.val)
    );

    // 初始化：每个链表头入堆
    for (ListNode node : lists) {
        if (node != null) heap.offer(node);
    }

    ListNode dummy = new ListNode(0);
    ListNode cur = dummy;

    while (!heap.isEmpty()) {
        ListNode node = heap.poll(); // 弹出最小值
        cur.next = node;
        cur = cur.next;
        if (node.next != null) heap.offer(node.next); // 下一个节点入堆
    }
    return dummy.next;
}
```

## 总结

| 数据结构 | 核心操作 | 经典应用 |
|----------|----------|----------|
| 栈 | push/pop | 括号匹配、函数调用、单调栈 |
| 队列 | enqueue/dequeue | BFS 遍历、任务调度 |
| 优先队列 | offer/poll | Top K、合并有序序列 |

**关键理解**：栈和队列的「限制」正是它们的价值——限制带来秩序，秩序简化问题。

## 面试追问预判

1. **单调栈的时间复杂度是 O(n) 吗？**
   - 是的。每个元素最多入栈一次、出栈一次，所以是 O(n)

2. **Java 中的 `Stack` 和 `Queue` 有什么区别？**
   - `Stack` 继承自 `Vector`，是同步的（线程安全但性能差）
   - 推荐使用 `Deque` 接口的实现，如 `ArrayDeque`

3. **如何在 O(1) 时间内获取栈中的最小元素？**
   - 使用两个栈：一个存数据，一个存当前最小值
   - 入栈时比较，最小栈入栈两者中较小的

4. **如何用两个栈实现队列？**
   - 一个栈负责入队，一个栈负责出队
   - 出队时如果出栈为空，就把入栈的所有元素弹出并压入出栈
