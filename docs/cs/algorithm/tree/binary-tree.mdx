# 二叉树：遍历方式与性质

面试官问：**「二叉树有几种遍历方式？写一下中序遍历。」**

这个问题看似简单，但真正的高手不仅能写出来，还能解释**为什么需要这些遍历方式**，以及**如何用递归和迭代两种方式实现**。

二叉树是几乎所有复杂树结构的基础。掌握它的遍历，就掌握了树结构的半壁江山。

## 二叉树的基本结构

```java
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode(int val) {
        this.val = val;
    }
}
```

**二叉树**是一种每个节点最多有两个子节点的树结构。左右子节点的顺序是固定的。

## 三种深度优先遍历

### 前序遍历：根 → 左 → 右

**根节点在最前面**，所以叫「前序」。

```java
// 递归实现
public void preorder(TreeNode root) {
    if (root == null) return;
    System.out.print(root.val + " "); // 先访问根
    preorder(root.left);               // 再访问左子树
    preorder(root.right);              // 最后访问右子树
}

// 迭代实现：需要手动维护栈
public List<Integer> preorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Stack<TreeNode> stack = new Stack<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.add(node.val);

        if (node.right != null) stack.push(node.right); // 先压右子树
        if (node.left != null) stack.push(node.left);  // 再压左子树
    }
    return result;
}
```

**迭代技巧**：先压右子树，再压左子树，这样出栈时左子树先被处理。

### 中序遍历：左 → 根 → 右

**根节点在中间**，是 BST（二叉搜索树）遍历的常用方式。

```java
// 递归实现
public void inorder(TreeNode root) {
    if (root == null) return;
    inorder(root.left);               // 先访问左子树
    System.out.print(root.val + " "); // 再访问根
    inorder(root.right);              // 最后访问右子树
}

// 迭代实现
public List<Integer> inorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Stack<TreeNode> stack = new Stack<>();
    TreeNode cur = root;

    while (cur != null || !stack.isEmpty()) {
        // 一直向左走，把所有节点入栈
        while (cur != null) {
            stack.push(cur);
            cur = cur.left;
        }
        // 处理栈顶
        cur = stack.pop();
        result.add(cur.val);
        // 转向右子树
        cur = cur.right;
    }
    return result;
}
```

**中序遍历 BST 会得到有序序列**，这是 BST 最常用的操作之一。

### 后序遍历：左 → 右 → 根

**根节点在最后面**，适合需要先处理子节点的场景（如释放内存）。

```java
// 递归实现
public void postorder(TreeNode root) {
    if (root == null) return;
    postorder(root.left);              // 先访问左子树
    postorder(root.right);             // 再访问右子树
    System.out.print(root.val + " "); // 最后访问根
}

// 迭代实现：前序遍历的变形
public List<Integer> postorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Stack<TreeNode> stack = new Stack<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.add(0, node.val); // 头插法，逆序输出

        if (node.left != null) stack.push(node.left);
        if (node.right != null) stack.push(node.right);
    }
    return result;
}
```

**迭代技巧**：前序遍历是「根左右」，后序遍历是「左右根」。把前序遍历的结果逆序，就得到了后序遍历。

## 层序遍历（BFS）

**按层从上到下遍历**，用队列实现。

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int levelSize = queue.size();
        List<Integer> level = new ArrayList<>();

        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);

            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
```

### 层序遍历的变体：锯齿形遍历

```java
public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    boolean leftToRight = true;

    while (!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> level = new ArrayList<>();

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }

        // 偶数层逆序
        if (!leftToRight) Collections.reverse(level);
        result.add(level);
        leftToRight = !leftToRight;
    }
    return result;
}
```

## 二叉树的基本性质

| 性质 | 公式 |
|------|------|
| 第 i 层最多有节点数 | 2^(i-1) |
| 深度为 k 的树最多有节点数 | 2^k - 1 |
| 终端节点数 n0，度为 2 的节点数 n2 | n0 = n2 + 1 |
| n 个节点的二叉树深度 | `⌊log₂n⌋ + 1` |
| 完全二叉树的父子节点关系 | 左子节点 = 2i，右子节点 = 2i + 1 |

## 完全二叉树与满二叉树

**满二叉树**：所有层节点数都达到最大。

```
        1
      /   \
     2     3
    / \   / \
   4   5 6   7
```

**完全二叉树**：除了最后一层，其他层节点数都达到最大，且最后一层节点集中在左边。

```
        1
      /   \
     2     3
    / \   /
   4   5 6
```

## 经典应用

| 问题 | 遍历方式 | 思路 |
|------|----------|------|
| 二叉树深度 | 前/后/层序 | 递归求 max(左深度, 右深度) + 1 |
| 判断平衡树 | 前序 | 每个节点的左右子树深度差 `<=` 1 |
| 二叉树镜像 | 前/后序 | 交换每个节点的左右子节点 |
| 对称二叉树 | 层序/前序 | 比较左子树和右子树是否对称 |
| 二叉树路径和 | 前序 | 记录从根到当前节点的路径 |

## 面试追问预判

1. **递归遍历和迭代遍历如何选择？**
   - 递归：代码简洁，但有栈溢出风险（树深度大时）
   - 迭代：需要手动维护栈，但可控

2. **如何遍历一棵树，使得每个节点访问 3 次？**
   - 中序遍历中，每个节点会被访问 3 次：第一次（左子树返回）、第二次（处理节点）、第三次（右子树返回）
   - 这个特性用于实现某些需要在「返回时」做处理的操作

3. **Morris 遍历是什么？**
   - O(1) 空间遍历二叉树
   - 利用叶子节点的空指针，临时指向中序遍历的前驱节点
   - 面试不常见，了解即可

**二叉树遍历是基础中的基础。** 递归和迭代两种方式都要掌握，面试时可以先写递归实现，再问面试官是否需要迭代版本。
