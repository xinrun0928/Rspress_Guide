# 动态规划

**面试官问：「硬币找零，1 分、3 分、4 分，最少需要几枚？」**

有人会说「选面额最大的，4 分 2 枚」，但这是错的（3 枚：4+1+1 = 3 枚）。正确的解法需要**动态规划**。

动态规划（Dynamic Programming，DP）是面试中最常考、也最难掌握的算法思想。

## 什么是动态规划

### 核心思想

**把复杂问题分解为简单子问题，先求子问题的最优，再组合得到全局最优。**

关键特征：
1. **最优子结构**：全局最优包含子问题的最优
2. **无后效性**：当前状态的选择不影响未来的选择
3. **重叠子问题**：子问题会被重复计算

### 递归 vs 记忆化 vs DP

以斐波那契数列为例：

```markdown
# 递归（指数级复杂度）
fib(n) = fib(n-1) + fib(n-2)

              fib(5)
           /          \
      fib(4)           fib(3)
     /      \         /      \
  fib(3)  fib(2)   fib(2)   fib(1)
   /   \
fib(2) fib(1)
```

大量重复计算：`fib(3)` 算了 2 次，`fib(2)` 算了 3 次。

```java
// 记忆化递归（备忘录）
Map<Integer, Integer> memo = new HashMap<>();

int fib(int n) {
    if (n <= 1) return n;
    if (memo.containsKey(n)) return memo.get(n);

    int result = fib(n - 1) + fib(n - 2);
    memo.put(n, result);
    return result;
}
```

```java
// 动态规划（自底向上）
int fib(int n) {
    if (n <= 1) return n;
    int[] dp = new int[n + 1];
    dp[0] = 0;
    dp[1] = 1;
    for (int i = 2; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    return dp[n];
}
```

## DP 解决问题的步骤

### 第一步：定义状态

`dp[i]` 是什么意思？一定要明确。

```java
// 爬楼梯问题
dp[i] = 到达第 i 阶的方法数
```

### 第二步：找状态转移方程

`dp[i]` 和 `dp[i-1]`、`dp[i-2]` 的关系是什么？

```java
dp[i] = dp[i-1] + dp[i-2]  // 可以从 i-1 或 i-2 跳上来
```

### 第三步：初始化

`dp[0]`、`dp[1]` 是什么？

```java
dp[0] = 1  // 站在第 0 阶
dp[1] = 1  // 只有一种跳法
```

### 第四步：确定遍历顺序

从前往后还是从后往前？

```java
for (int i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2];
}
```

## 经典问题：打家劫舍

### 问题描述

不能偷相邻的两家，求最大金额。

```java
public int rob(int[] nums) {
    if (nums == null || nums.length == 0) return 0;
    if (nums.length == 1) return nums[0];

    int n = nums.length;
    int[] dp = new int[n];

    dp[0] = nums[0];
    dp[1] = Math.max(nums[0], nums[1]);

    for (int i = 2; i < n; i++) {
        // 状态转移：不偷 i，或偷 i（此时 i-1 不能偷）
        dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i]);
    }

    return dp[n - 1];
}
```

### 空间优化

发现 `dp[i]` 只和 `dp[i-1]`、`dp[i-2]` 有关，可以压缩空间：

```java
public int rob(int[] nums) {
    if (nums == null || nums.length == 0) return 0;
    if (nums.length == 1) return nums[0];

    int prev2 = nums[0];                    // dp[i-2]
    int prev1 = Math.max(nums[0], nums[1]); // dp[i-1]

    for (int i = 2; i < nums.length; i++) {
        int curr = Math.max(prev1, prev2 + nums[i]);
        prev2 = prev1;
        prev1 = curr;
    }

    return prev1;
}
```

## 经典问题：不同路径

### 无障碍版本

```java
public int uniquePaths(int m, int n) {
    int[][] dp = new int[m][n];

    // 初始化第一行和第一列
    for (int i = 0; i < m; i++) dp[i][0] = 1;
    for (int j = 0; j < n; j++) dp[0][j] = 1;

    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
        }
    }
    return dp[m - 1][n - 1];
}
```

### 有障碍版本

```java
public int uniquePathsWithObstacles(int[][] grid) {
    int m = grid.length, n = grid[0].length;
    int[][] dp = new int[m][n];

    for (int i = 0; i < m && grid[i][0] == 0) dp[i][0] = 1;
    for (int j = 0; j < n && grid[0][j] == 0) dp[0][j] = 1;

    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            if (grid[i][j] == 0) {
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }
    }
    return dp[m - 1][n - 1];
}
```

## 经典问题：背包问题模板

### 0-1 背包

```java
public int knapsack01(int[] weights, int[] values, int capacity) {
    int n = weights.length;
    int[][] dp = new int[n + 1][capacity + 1];

    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= capacity; w++) {
            if (weights[i - 1] <= w) {
                dp[i][w] = Math.max(
                    dp[i - 1][w],
                    dp[i - 1][w - weights[i - 1]] + values[i - 1]
                );
            } else {
                dp[i][w] = dp[i - 1][w];
            }
        }
    }
    return dp[n][capacity];
}
```

### 空间优化

```java
public int knapsack01(int[] weights, int[] values, int capacity) {
    int n = weights.length;
    int[] dp = new int[capacity + 1];

    for (int i = 0; i < n; i++) {
        // 必须从后往前！
        for (int w = capacity; w >= weights[i]; w--) {
            dp[w] = Math.max(dp[w], dp[w - weights[i]] + values[i]);
        }
    }
    return dp[capacity];
}
```

:::warning 空间优化陷阱

0-1 背包必须**从后往前**遍历，因为每个物品只能用一次。

完全背包可以从前往后，因为每个物品可以用多次。

:::

## 面试追问

1. **什么时候用二维 DP，什么时候用一维？**
   - 两个维度有关联 → 二维
   - 只和前一个状态有关 → 一维

2. **空间还能继续优化吗？**
   - 看状态转移只依赖哪些
   - 如果只依赖 `dp[i-1]` 和 `dp[i-2]`，可以压缩到几个变量

3. **DP 和贪心/回溯的区别？**
   - 贪心：局部最优 → 全局最优，不需要子问题
   - 回溯：枚举所有解，剪枝
   - DP：记录子问题最优，避免重复计算

**记住：DP 的核心是「分解问题，记录最优」。找准状态转移方程，DP 就成功了一半。**
