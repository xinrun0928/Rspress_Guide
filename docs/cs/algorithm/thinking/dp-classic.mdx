# 动态规划经典问题

**动态规划是面试的重灾区。** 背包问题、最长公共子序列、最长递增子序列、编辑距离——这些问题变换多端，但核心思路都一样。

掌握了这些经典问题，你就掌握了 DP 面试的半壁江山。

## 背包问题

### 0-1 背包

**问题**：有 n 件物品，每件重量 w[i]，价值 v[i]，容量为 C，最多能装多少价值的物品？

```java
public int knapsack01(int[] weights, int[] values, int capacity) {
    int n = weights.length;
    int[][] dp = new int[n + 1][capacity + 1];

    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= capacity; w++) {
            // 不选第 i 件
            dp[i][w] = dp[i - 1][w];

            // 选第 i 件（如果能装下）
            if (weights[i - 1] <= w) {
                dp[i][w] = Math.max(
                    dp[i][w],
                    dp[i - 1][w - weights[i - 1]] + values[i - 1]
                );
            }
        }
    }
    return dp[n][capacity];
}
```

### 完全背包

**问题**：每件物品可以选无限次。

```java
public int knapsackComplete(int[] weights, int[] values, int capacity) {
    int n = weights.length;
    int[][] dp = new int[n + 1][capacity + 1];

    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= capacity; w++) {
            // 从 0 开始，完全背包可以选多次
            for (int k = 0; k * weights[i - 1] <= w; k++) {
                dp[i][w] = Math.max(
                    dp[i][w],
                    dp[i - 1][w - k * weights[i - 1]] + k * values[i - 1]
                );
            }
        }
    }
    return dp[n][capacity];
}
```

### 多重背包

**问题**：每件物品最多选 m[i] 次。

```java
public int knapsackMultiple(int[] weights, int[] values, int[] counts, int capacity) {
    int n = weights.length;
    int[][] dp = new int[n + 1][capacity + 1];

    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= capacity; w++) {
            for (int k = 0; k <= counts[i - 1] && k * weights[i - 1] <= w; k++) {
                dp[i][w] = Math.max(
                    dp[i][w],
                    dp[i - 1][w - k * weights[i - 1]] + k * values[i - 1]
                );
            }
        }
    }
    return dp[n][capacity];
}
```

## 最长公共子序列（LCS）

### 问题

给定两个字符串，求最长的公共子序列（不要求连续）。

```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[m][n];
}
```

### 空间优化

```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[] dp = new int[n + 1];

    for (int i = 1; i <= m; i++) {
        int prev = 0;  // dp[i-1][j-1]
        for (int j = 1; j <= n; j++) {
            int temp = dp[j];  // 保存 dp[i-1][j]
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[j] = prev + 1;
            } else {
                dp[j] = Math.max(dp[j], dp[j - 1]);
            }
            prev = temp;
        }
    }
    return dp[n];
}
```

## 最长递增子序列（LIS）

### O(n²) 解法

```java
public int lengthOfLIS(int[] nums) {
    int n = nums.length;
    int[] dp = new int[n];
    int maxLen = 0;

    for (int i = 0; i < n; i++) {
        dp[i] = 1;  // 自己就是一个子序列
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = Math.max(dp[i], dp[j] + 1);
            }
        }
        maxLen = Math.max(maxLen, dp[i]);
    }
    return maxLen;
}
```

### O(n log n) 解法（贪心 + 二分）

```java
public int lengthOfLIS(int[] nums) {
    List<Integer> tails = new ArrayList<>();

    for (int num : nums) {
        // 找第一个 >= num 的位置
        int pos = binarySearch(tails, num);
        if (pos == tails.size()) {
            tails.add(num);
        } else {
            tails.set(pos, num);
        }
    }
    return tails.size();
}

private int binarySearch(List<Integer> tails, int target) {
    int left = 0, right = tails.size();
    while (left < right) {
        int mid = (left + right) / 2;
        if (tails.get(mid) < target) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    return left;
}
```

## 编辑距离

### 问题

将 word1 变成 word2，最少需要几次操作（插入、删除、替换）？

### 代码实现

```java
public int minDistance(String word1, String word2) {
    int m = word1.length(), n = word2.length();
    int[][] dp = new int[m + 1][n + 1];

    // 初始化
    for (int i = 0; i <= m; i++) dp[i][0] = i;  // 删除 i 个字符
    for (int j = 0; j <= n; j++) dp[0][j] = j;  // 插入 j 个字符

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1];  // 不需要操作
            } else {
                dp[i][j] = Math.min(
                    dp[i - 1][j] + 1,      // 删除
                    Math.min(
                        dp[i][j - 1] + 1,  // 插入
                        dp[i - 1][j - 1] + 1  // 替换
                    )
                );
            }
        }
    }
    return dp[m][n];
}
```

## 股票买卖问题

### 最多交易一次

```java
public int maxProfitOnce(int[] prices) {
    int minPrice = Integer.MAX_VALUE;
    int maxProfit = 0;

    for (int price : prices) {
        minPrice = Math.min(minPrice, price);
        maxProfit = Math.max(maxProfit, price - minPrice);
    }
    return maxProfit;
}
```

### 最多交易 k 次

```java
public int maxProfitK(int k, int[] prices) {
    if (prices.length == 0) return 0;

    int n = prices.length;
    if (k > n / 2) {  // 等价于无限次交易
        int profit = 0;
        for (int i = 1; i < n; i++) {
            if (prices[i] > prices[i - 1]) {
                profit += prices[i] - prices[i - 1];
            }
        }
        return profit;
    }

    int[][] dp = new int[k + 1][n];
    for (int t = 1; t <= k; t++) {
        int maxDiff = -prices[0];
        for (int j = 1; j < n; j++) {
            dp[t][j] = Math.max(dp[t][j - 1], prices[j] + maxDiff);
            maxDiff = Math.max(maxDiff, dp[t - 1][j - 1] - prices[j]);
        }
    }
    return dp[k][n - 1];
}
```

## 字符串DP模板

```java
public int stringDP(String s1, String s2) {
    int m = s1.length(), n = s2.length();
    int[][] dp = new int[m + 1][n + 1];

    // 初始化
    for (int i = 0; i <= m; i++) dp[i][0] = i;
    for (int j = 0; j <= n; j++) dp[0][j] = j;

    // 填表
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (s1.charAt(i - 1) == s2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1];
            } else {
                dp[i][j] = Math.min(dp[i - 1][j - 1],
                              Math.min(dp[i - 1][j], dp[i][j - 1])) + 1;
            }
        }
    }
    return dp[m][n];
}
```

## 面试追问

1. **LIS 为什么可以用二分优化？**
   - `tails[i]` 表示长度为 i+1 的递增子序列的最小结尾
   - 贪心：用更大的值替换，可以让后续更有可能延长

2. **股票问题有多少种变体？**
   - 最多交易 1 次
   - 最多交易 k 次
   - 交易冷冻期
   - 有手续费
   - 通用模板：三维 DP

3. **DP 表格怎么打印路径？**
   - 用 `pre` 数组记录选择来源
   - 从终点回溯

**记住：经典问题的核心都是「状态 + 转移」。找准 dp 数组的含义，问题就迎刃而解。**
