# 回溯算法

**八皇后问题：如何在 8×8 的棋盘上放置 8 个皇后，使得它们互不攻击？**

这个问题有 92 个解。如果用穷举，需要尝试 16,777,216 种排列。为什么回溯能高效解决？因为它会「走不通就退」，避免无谓的尝试。

## 核心思想

回溯 = **枚举 + 剪枝**

1. 枚举所有可能的解空间
2. 在枚举过程中，如果发现当前路径不满足条件，**回退**（撤销选择）
3. 继续尝试其他路径

:::tip 类比理解

回溯就像走迷宫：
- 遇到岔路口，尝试一条路
- 如果走进死胡同，退回上一个岔路口
- 尝试另一条路
- 重复直到找到出口或所有路径都尝试完

:::

## 算法框架

```java
public void backtrack(参数) {
    if (满足终止条件) {
        收集结果;
        return;
    }

    for (选择 : 当前可以做的选择) {
        做选择;
        backtrack(下一层);
        撤销选择;  // 关键：回退
    }
}
```

## 经典问题：八皇后

### 问题分析

8×8 棋盘，每行放一个皇后。

- 第 i 个皇后放在第 i 行
- 只需要考虑列和对角线是否冲突

### 代码实现

```java
public class EightQueens {
    List<List<String>> result = new ArrayList<>();

    public List<List<String>> solveNQueens(int n) {
        char[][] board = new char[n][n];
        for (char[] row : board) Arrays.fill(row, '.');

        backtrack(board, 0);
        return result;
    }

    private void backtrack(char[][] board, int row) {
        // 终止条件：所有行都放好了
        if (row == board.length) {
            result.add(boardToList(board));
            return;
        }

        int n = board[row].length;
        for (int col = 0; col < n; col++) {
            if (!isValid(board, row, col)) continue;

            board[row][col] = 'Q';      // 做选择
            backtrack(board, row + 1);  // 递归
            board[row][col] = '.';      // 撤销选择
        }
    }

    private boolean isValid(char[][] board, int row, int col) {
        int n = board.length;

        // 检查同列是否有皇后
        for (int i = 0; i < row; i++) {
            if (board[i][col] == 'Q') return false;
        }

        // 检查左上对角线
        for (int i = row - 1, j = col - 1; i >= 0 && j >= 0; i--, j--) {
            if (board[i][j] == 'Q') return false;
        }

        // 检查右上对角线
        for (int i = row - 1, j = col + 1; i >= 0 && j < n; i--, j++) {
            if (board[i][j] == 'Q') return false;
        }

        return true;
    }

    private List<String> boardToList(char[][] board) {
        List<String> list = new ArrayList<>();
        for (char[] row : board) {
            list.add(new String(row));
        }
        return list;
    }
}
```

## 经典问题：全排列

### 无重复数字

```java
public List<List<Integer>> permute(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    List<Integer> path = new ArrayList<>();
    boolean[] used = new boolean[nums.length];

    backtrack(nums, used, path, result);
    return result;
}

private void backtrack(int[] nums, boolean[] used, List<Integer> path,
                       List<List<Integer>> result) {
    if (path.size() == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;  // 已使用的跳过

        path.add(nums[i]);
        used[i] = true;
        backtrack(nums, used, path, result);
        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

### 有重复数字

```java
public List<List<Integer>> permuteUnique(int[] nums) {
    Arrays.sort(nums);  // 排序是关键
    List<List<Integer>> result = new ArrayList<>();
    List<Integer> path = new ArrayList<>();
    boolean[] used = new boolean[nums.length];

    backtrack(nums, used, path, result);
    return result;
}

private void backtrack(int[] nums, boolean[] used, List<Integer> path,
                       List<List<Integer>> result) {
    if (path.size() == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;

        // 剪枝：跳过重复数字
        if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) continue;

        path.add(nums[i]);
        used[i] = true;
        backtrack(nums, used, path, result);
        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

## 经典问题：子集

### 无重复数字

```java
public List<List<Integer>> subsets(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    List<Integer> path = new ArrayList<>();

    backtrack(nums, 0, path, result);
    return result;
}

private void backtrack(int[] nums, int start, List<Integer> path,
                       List<List<Integer>> result) {
    // 每个节点都是一个子集（不是只在叶子节点收集）
    result.add(new ArrayList<>(path));

    for (int i = start; i < nums.length; i++) {
        path.add(nums[i]);
        backtrack(nums, i + 1, path, result);
        path.remove(path.size() - 1);
    }
}
```

### 有重复数字

```java
public List<List<Integer>> subsetsWithDup(int[] nums) {
    Arrays.sort(nums);
    List<List<Integer>> result = new ArrayList<>();
    List<Integer> path = new ArrayList<>();

    backtrack(nums, 0, path, result);
    return result;
}

private void backtrack(int[] nums, int start, List<Integer> path,
                       List<List<Integer>> result) {
    result.add(new ArrayList<>(path));

    for (int i = start; i < nums.length; i++) {
        // 跳过重复元素
        if (i > start && nums[i] == nums[i - 1]) continue;

        path.add(nums[i]);
        backtrack(nums, i + 1, path, result);
        path.remove(path.size() - 1);
    }
}
```

## 组合问题

### 两数之和（简单版）

```java
public List<List<Integer>> combinationSum(int[] candidates, int target) {
    List<List<Integer>> result = new ArrayList<>();
    List<Integer> path = new ArrayList<>();

    backtrack(candidates, target, 0, path, result);
    return result;
}

private void backtrack(int[] candidates, int target, int start,
                       List<Integer> path, List<List<Integer>> result) {
    if (target == 0) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = start; i < candidates.length; i++) {
        if (candidates[i] > target) continue;  // 剪枝

        path.add(candidates[i]);
        backtrack(candidates, target - candidates[i], i, path, result);
        path.remove(path.size() - 1);
    }
}
```

## 回溯 vs DFS vs BFS

| 维度 | 回溯 | DFS | BFS |
|------|------|-----|-----|
| **核心** | 枚举 + 剪枝 | 深度优先 | 广度优先 |
| **是否回退** | 是（撤销选择） | 可选 | 否 |
| **数据结构** | 递归栈 | 栈或递归 | 队列 |
| **应用** | 解空间搜索 | 连通分量、拓扑 | 最短路径 |

## 面试追问

1. **回溯和递归的区别？**
   - 递归是「自己调用自己」
   - 回溯是「枚举 + 撤销」的过程
   - 回溯通常用递归实现

2. **如何优化回溯？**
   - **剪枝**：提前排除不可能的分支
   - **排序**：让相等的元素相邻，便于去重
   - **备忘录**：记录已计算过的状态（变为 DP）

3. **什么时候用回溯？**
   - 求所有解
   - 求是否存在解
   - 解空间相对可控

**记住：回溯的核心是「走不通就退」。做好剪枝，回溯的效率就能大幅提升。**
