# 位运算

**如何用 O(1) 空间判断一个数是否是 2 的幂？**

`n > 0 && (n & (n - 1)) == 0`

这就是位运算的魅力——用最基本的二进制操作，解决看似复杂的问题。

## 位运算基础

### 六种基本操作

| 操作 | 符号 | 说明 |
|------|------|------|
| AND | `&` | 两位都是 1 才为 1 |
| OR | `\|` | 任一位是 1 就为 1 |
| XOR | `^` | 相同为 0，不同为 1 |
| NOT | `~` | 位取反 |
| 左移 | `<<` | 左边移出，右边补 0 |
| 带符号右移 | `>>` | 右边移出，左边补符号位 |
| 无符号右移 | `>>>` | 右边移出，左边补 0 |

### 常用技巧

```java
// 取绝对值（考虑负数补码）
int abs(int x) {
    int sign = x >> 31;        // 符号位：0 或 -1
    return (x + sign) ^ sign;  // 或 (x ^ sign) - sign
}

// 取平均值，避免溢出
int avg(int a, int b) {
    return (a & b) + ((a ^ b) >> 1);
}

// 判断奇偶
boolean isOdd(int x) {
    return (x & 1) == 1;
}
```

## 常用技巧

### 1. 消除最低位的 1

```java
x & (x - 1)  // 消除 x 二进制表示中最低位的 1
```

**应用**：统计 1 的个数

```java
public int hammingWeight(int x) {
    int count = 0;
    while (x != 0) {
        x = x & (x - 1);  // 消除最低位的 1
        count++;
    }
    return count;
}
```

### 2. 获取最低位的 1

```java
x & (-x)  // 获取 x 二进制表示中最低位的 1
```

**应用**：找出只出现一次的数（其他数都出现两次）

```java
public int singleNumber(int[] nums) {
    int result = 0;
    for (int num : nums) {
        result ^= num;
    }
    return result;
}
```

### 3. 判断是否是 2 的幂

```java
boolean isPowerOfTwo(int n) {
    return n > 0 && (n & (n - 1)) == 0;
}
```

### 4. 判断是否是 4 的幂

```java
boolean isPowerOfFour(int n) {
    return n > 0 && (n & (n - 1)) == 0 && (n & 0x55555555) != 0;
}
```

## 实战应用

### 1. 统计二进制中 1 的个数（LeetCode 191）

```java
// 方法一：移位
public int hammingWeight(int n) {
    int count = 0;
    while (n != 0) {
        count += n & 1;
        n >>>= 1;
    }
    return count;
}

// 方法二：n & (n-1)
public int hammingWeight2(int n) {
    int count = 0;
    while (n != 0) {
        n &= (n - 1);
        count++;
    }
    return count;
}

// 方法三：分治
public int hammingWeight3(int n) {
    n = (n & 0x55555555) + ((n >>> 1) & 0x55555555);
    n = (n & 0x33333333) + ((n >>> 2) & 0x33333333);
    n = (n & 0x0F0F0F0F) + ((n >>> 4) & 0x0F0F0F0F);
    n = (n & 0x00FF00FF) + ((n >>> 8) & 0x00FF00FF);
    n = (n & 0x0000FFFF) + ((n >>> 16) & 0x0000FFFF);
    return n;
}
```

// 数组包含 0 到 n 中的 n 个不同的数，找出缺失的那个
public int missingNumber(int[] nums) {
    int result = nums.length;
    for (int i = 0; i < nums.length; i++) {
        result ^= i;
        result ^= nums[i];
    }

### 3. 颠倒二进制位（LeetCode 190）

```java
public int reverseBits(int n) {
    int result = 0;
    for (int i = 0; i < 32; i++) {
        result <<= 1;
        result |= (n & 1);
        n >>>= 1;
    }
    return result;
}
```

### 4. 数字范围按位与（LeetCode 201）

```java
public int rangeBitwiseAnd(int left, int right) {
    int shift = 0;
    while (left < right) {
        left >>= 1;
        right >>= 1;
        shift++;
    }
    return left << shift;
}
```

## 位运算与集合

### 用位掩码表示集合

```java
// 假设有 32 个元素，用一个 int 表示集合
int set = 0;

// 添加元素 i（1 << i）
set |= (1 << i);

// 删除元素 i
set &= ~(1 << i);

// 检查元素 i 是否存在
boolean exists = (set & (1 << i)) != 0;

// 集合交集
set1 & set2;

// 集合并集
set1 | set2;

// 集合补集
~set & ((1 << n) - 1);
```

### LeetCode 实战：子集（位运算版本）

```java
public List<List<Integer>> subsets(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    int n = nums.length;
    int total = 1 << n;  // 2^n 个子集

    for (int mask = 0; mask < total; mask++) {
        List<Integer> subset = new ArrayList<>();
        for (int i = 0; i < n; i++) {
            if ((mask & (1 << i)) != 0) {
                subset.add(nums[i]);
            }
        }
        result.add(subset);
    }
    return result;
}
```

### LeetCode 实战：全排列（位运算版本）

```java
public List<List<Integer>> permute(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    List<Integer> path = new ArrayList<>();
    int used = 0;

    backtrack(nums, used, path, result);
    return result;
}

private void backtrack(int[] nums, int used, List<Integer> path,
                       List<List<Integer>> result) {
    if (path.size() == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if ((used & (1 << i)) != 0) continue;  // 已使用

        path.add(nums[i]);
        backtrack(nums, used | (1 << i), path, result);
        path.remove(path.size() - 1);
    }
}
```

## 布隆过滤器

### 原理

用 k 个哈希函数，将元素映射到 k 个位置。只要有一个位置是 0，元素一定不存在；全是 1，可能存在（会有误判）。

```java
public class BloomFilter {
    private int[] bits;  // 位数组
    private int size;    // 位数组大小
    private int hashCount;  // 哈希函数数量

    public BloomFilter(int size, int hashCount) {
        this.size = size;
        this.hashCount = hashCount;
        this.bits = new int[(size + 31) / 32];
    }

    private int[] getHashes(int item) {
        int[] result = new int[hashCount];
        // 简单哈希函数，实际可用更好的
        for (int i = 0; i < hashCount; i++) {
            result[i] = (item * (i + 1) * 31) % size;
        }
        return result;
    }

    public void add(int item) {
        for (int pos : getHashes(item)) {
            bits[pos / 32] |= (1 << (pos % 32));
        }
    }

    public boolean mightContain(int item) {
        for (int pos : getHashes(item)) {
            if ((bits[pos / 32] & (1 << (pos % 32))) == 0) {
                return false;
            }
        }
        return true;
    }
}
```

## 面试追问

1. **位运算和算术运算哪个更快？**
   - 现代 CPU 两者差不多快
   - 但位运算更省空间，可以同时操作多位

2. **有符号右移 `>>` 和无符号右移 `>>>` 的区别？**
   - `>>` 补符号位（负数右移可能还是负数）
   - `>>>` 补 0（用于处理无符号整数）

3. **如何快速判断两个数的符号是否相同？**
   ```java
   boolean sameSign = (x ^ y) >= 0;
   ```
