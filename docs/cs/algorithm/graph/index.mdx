# 图结构

**六度空间理论**说：世界上任意两个人之间，最多只隔了六层关系。怎么用算法验证这个理论？答案是**图**。

图（Graph）是计算机科学中最强大的数据结构之一。它能够表达任意二元关系——无论是社交网络中的人际关系，还是城市之间的交通网络，甚至是网页之间的链接关系。

## 为什么需要图

回顾我们学过的数据结构：

- **数组**：按下标随机访问，O(1) 速度
- **链表**：插入删除灵活，但查找需要遍历
- **树**：层次结构，一对多关系
- **图**：网状结构，任意复杂关系

当你需要表达「A 和 B 相连，B 和 C 相连，但 A 和 C 不直接相连」这种复杂关系时，树不够用了——你需要**图**。

:::tip 现实映射

社交网络是图，微信好友、微博关注、LinkedIn 职业关系——每一个都是图结构的应用。

地图导航也是图，城市是节点，道路是边，求最短路径就是图算法。

:::

## 图的分类

### 按方向划分

| 类型 | 说明 | 例子 |
|------|------|------|
| **无向图** | 边没有方向，A 到 B 和 B 到 A 一样 | 微信好友关系 |
| **有向图** | 边有方向，A -> B 不等于 B -> A | 微博关注、网页链接 |

### 按权重划分

| 类型 | 说明 | 例子 |
|------|------|------|
| **无权图** | 所有边权重相同 | 邻接关系 |
| **有权图** | 每条边有权重 | 路程距离、费用 |

### 其他类型

- **稠密图**：边数接近节点数的平方
- **稀疏图**：边数远小于节点数的平方
- **循环图/无环图**：是否有闭环

## 图的代码表示

在代码中，图通常用以下方式表示：

```java
// 邻接表表示
class Graph {
    List<Integer>[] adj;  // adj[i] 存储节点 i 的所有邻居

    public Graph(int n) {
        adj = new ArrayList[n];
        for (int i = 0; i < n; i++) {
            adj[i] = new ArrayList<>();
        }
    }

    // 添加边（无向图）
    public void addEdge(int u, int v) {
        adj[u].add(v);
        adj[v].add(u);
    }

    // 添加边（有向图）
    public void addDirectedEdge(int u, int v) {
        adj[u].add(v);
    }
}
```
