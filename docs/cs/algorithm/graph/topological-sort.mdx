# 拓扑排序

**早上起床，你要穿衣服。穿什么？有讲究：先穿袜子再穿鞋，先穿内衣再穿外套。这些顺序怎么确定？**

这就是**拓扑排序**要解决的问题——在有依赖关系的任务中，找出一个可行的执行顺序。

## 核心概念

### 什么是拓扑排序

拓扑排序是对**有向无环图（DAG）**中所有顶点的一种线性排序，满足：

- 对于每条有向边 (u, v)，顶点 u 都在顶点 v 之前

### 关键前提

**有向无环图（DAG）**才有拓扑排序。如果有环，就形成死锁，无法确定顺序。

### 应用场景

| 场景 | 依赖关系 |
|------|----------|
| 课程选修 | 课程 A 是课程 B 的先修课 |
| 任务调度 | 任务 A 完成后才能开始任务 B |
| 编译器 | 头文件依赖 |
| 项目构建 | Makefile 中的依赖关系 |

## Kahn 算法

### 核心思想

1. 计算所有节点的入度
2. 将所有入度为 0 的节点加入队列（这些没有前置依赖）
3. 依次取出节点，将其指向的节点入度减 1
4. 如果某个节点入度变为 0，加入队列
5. 重复直到队列为空

### 实现

```java
public class TopologicalSort {
    public List<Integer> Kahn(int n, int[][] edges) {
        List<Integer>[] graph = new ArrayList[n];
        for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();

        int[] indegree = new int[n];
        for (int[] edge : edges) {
            int u = edge[0], v = edge[1];
            graph[u].add(v);
            indegree[v]++;
        }

        Queue<Integer> queue = new LinkedList<>();
        for (int i = 0; i < n; i++) {
            if (indegree[i] == 0) {
                queue.offer(i);
            }
        }

        List<Integer> result = new ArrayList<>();
        while (!queue.isEmpty()) {
            int u = queue.poll();
            result.add(u);

            for (int v : graph[u]) {
                indegree[v]--;
                if (indegree[v] == 0) {
                    queue.offer(v);
                }
            }
        }

        // 检查是否有环（结果数量 < n 说明有环）
        if (result.size() != n) {
            System.out.println("图中存在环！");
            return Collections.emptyList();
        }
        return result;
    }
}
```

### DFS 实现

```java
public class TopologicalSortDFS {
    List<Integer>[] graph;
    boolean[] visited;
    boolean[] onStack;  // 正在栈上，用于检测环
    Stack<Integer> stack;

    public List<Integer> findOrder(int n, int[][] edges) {
        graph = new ArrayList[n];
        for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();
        for (int[] e : edges) {
            graph[e[0]].add(e[1]);
        }

        visited = new boolean[n];
        onStack = new boolean[n];
        stack = new Stack<>();

        for (int i = 0; i < n; i++) {
            if (!visited[i]) {
                if (dfs(i)) {
                    return Collections.emptyList();  // 有环
                }
            }
        }

        List<Integer> result = new ArrayList<>();
        while (!stack.isEmpty()) {
            result.add(stack.pop());
        }
        return result;
    }

    private boolean dfs(int u) {
        if (onStack[u]) return true;   // 遇到环
        if (visited[u]) return false;  // 已处理过
        visited[u] = true;
        onStack[u] = true;

        for (int v : graph[u]) {
            if (dfs(v)) return true;
        }

        onStack[u] = false;
        stack.push(u);  // 后继节点先入栈，保证拓扑顺序
        return false;
    }
}
```

## 经典应用

### 1. 课程表 II（LeetCode 210）

```java
public int[] findOrder(int numCourses, int[][] prerequisites) {
    List<Integer>[] graph = new ArrayList[numCourses];
    for (int i = 0; i < numCourses; i++) graph[i] = new ArrayList<>();

    int[] indegree = new int[numCourses];
    for (int[] p : prerequisites) {
        graph[p[1]].add(p[0]);
        indegree[p[0]]++;
    }

    Queue<Integer> queue = new LinkedList<>();
    for (int i = 0; i < numCourses; i++) {
        if (indegree[i] == 0) queue.offer(i);
    }

    int[] result = new int[numCourses];
    int index = 0;

    while (!queue.isEmpty()) {
        int course = queue.poll();
        result[index++] = course;

        for (int next : graph[course]) {
            indegree[next]--;
            if (indegree[next] == 0) {
                queue.offer(next);
            }
        }
    }

    return index == numCourses ? result : new int[0];
}
```

### 2. 检测环的存在

```java
public boolean hasCycle(int n, int[][] edges) {
    List<Integer>[] graph = new ArrayList[n];
    int[] indegree = new int[n];
    for (int[] e : edges) {
        graph[e[0]].add(e[1]);
        indegree[e[1]]++;
    }

    Queue<Integer> queue = new LinkedList<>();
    for (int i = 0; i < n; i++) {
        if (indegree[i] == 0) queue.offer(i);
    }

    int count = 0;
    while (!queue.isEmpty()) {
        count++;
        int u = queue.poll();
        for (int v : graph[u]) {
            indegree[v]--;
            if (indegree[v] == 0) queue.offer(v);
        }
    }

    return count != n;  // 处理完的节点数 < n，有环
}
```

## 关键路径

### 什么是关键路径

在带权 DAG 中，从起点到终点的最长路径，称为**关键路径**。这条路径上的活动没有任何延迟余地。

### 求法

1. 拓扑排序
2. 计算每个节点的最早开始时间（按拓扑顺序）
3. 计算每个节点的最晚开始时间（逆拓扑顺序）
4. 最早 = 最晚的节点在关键路径上

```java
public int[] criticalPath(int n, List<int[]>[] graph) {
    // 1. 拓扑排序得到处理顺序
    int[] order = topologicalOrder(n, graph);

    // 2. 计算最早开始时间
    int[] earliest = new int[n];
    for (int u : order) {
        for (int[] edge : graph[u]) {
            int v = edge[0], w = edge[1];
            earliest[v] = Math.max(earliest[v], earliest[u] + w);
        }
    }

    // 3. 计算最晚开始时间
    int maxTime = Arrays.stream(earliest).max().orElse(0);
    int[] latest = new int[n];
    Arrays.fill(latest, maxTime);

    for (int i = order.length - 1; i >= 0; i--) {
        int u = order[i];
        for (int[] edge : graph[u]) {
            int v = edge[0], w = edge[1];
            latest[u] = Math.min(latest[u], latest[v] - w);
        }
    }

    // 4. 找关键路径上的边
    List<int[]> criticalEdges = new ArrayList<>();
    for (int u : order) {
        for (int[] edge : graph[u]) {
            int v = edge[0], w = edge[1];
            if (earliest[u] + w == latest[v]) {
                criticalEdges.add(new int[]{u, v});
            }
        }
    }
    return criticalEdges.stream().mapToInt(a -> a[0]).toArray();
}
```

## 面试追问

1. **拓扑排序结果唯一吗？**
   - 不一定唯一，只要满足约束即可
   - 完全有序图（链状）的拓扑排序唯一

2. **如何检测图中是否有环？**
   - Kahn 算法：处理完的节点数 `<` n
   - DFS：遇到已访问且在栈上的节点

3. **BFS 和 DFS 做拓扑排序的区别？**
   - BFS（Kahn）：用入度计数，按层处理
   - DFS：后进先出，保证所有依赖先处理

**记住：拓扑排序的核心是「有向无环图」的线性排序。Kahn 算法是入度为 0 的节点一个个出来，DFS 是后继节点先入栈。**
