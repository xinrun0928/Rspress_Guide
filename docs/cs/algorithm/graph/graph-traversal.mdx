# 图的遍历

**当你打开微信「附近的人」，它是怎么找到所有人的？**

本质上是一个图的遍历问题——从你出发，通过「好友关系」这条边，一层一层往外扩展，直到覆盖所有人。

图的遍历是所有图算法的基石。两种核心遍历方式：**DFS（深度优先）**和**BFS（广度优先）**。

## 核心概念

### 深度优先搜索（DFS）

像走迷宫一样，一条路走到黑，走不通了再回退。**递归**或**栈**实现。

### 广度优先搜索（BFS）

像涟漪一样，一圈一圈往外扩散。**队列**实现。

## DFS 实现

### 递归版本

```java
public class DFS {
    List<Integer>[] adj;
    boolean[] visited;

    public void dfs(int start) {
        visited[start] = true;
        System.out.println("访问: " + start);

        for (int neighbor : adj[start]) {
            if (!visited[neighbor]) {
                dfs(neighbor);
            }
        }
    }

    // 求解路径问题时，记录父节点
    int[] parent;

    public void dfsWithPath(int start) {
        visited[start] = true;

        for (int neighbor : adj[start]) {
            if (!visited[neighbor]) {
                parent[neighbor] = start;
                dfsWithPath(neighbor);
            }
        }
    }
}
```

### 迭代版本（用栈）

```java
public void dfsIterative(int start) {
    Stack<Integer> stack = new Stack<>();
    stack.push(start);
    visited[start] = true;

    while (!stack.isEmpty()) {
        int node = stack.pop();
        System.out.println("访问: " + node);

        for (int neighbor : adj[node]) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                stack.push(neighbor);
            }
        }
    }
}
```

## BFS 实现

```java
public class BFS {
    List<Integer>[] adj;
    boolean[] visited;

    public void bfs(int start) {
        Queue<Integer> queue = new LinkedList<>();
        queue.offer(start);
        visited[start] = true;

        while (!queue.isEmpty()) {
            int node = queue.poll();
            System.out.println("访问: " + node);

            for (int neighbor : adj[node]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.offer(neighbor);
                }
            }
        }
    }

    // 求最短路径时，记录距离
    int[] dist;

    public void bfsShortestPath(int start) {
        Queue<Integer> queue = new LinkedList<>();
        queue.offer(start);
        visited[start] = true;
        dist[start] = 0;

        while (!queue.isEmpty()) {
            int node = queue.poll();

            for (int neighbor : adj[node]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    dist[neighbor] = dist[node] + 1;
                    queue.offer(neighbor);
                }
            }
        }
    }
}
```

## DFS vs BFS

| 维度 | DFS | BFS |
|------|-----|-----|
| **数据结构** | 栈（递归调用栈） | 队列 |
| **搜索特点** | 先深后浅，一条路走到黑 | 先广后深，一圈一圈扩散 |
| **最短路径** | 不保证 | 保证（无权图） |
| **空间复杂度** | O(h)，h 为深度 | O(w)，w 为宽度 |
| **适用场景** | 找所有解、拓扑排序、连通分量 | 最短路径、分层遍历 |

:::tip 记忆口诀

DFS 深度优先，用栈或递归，一路走到底

BFS 广度优先，用队列，一圈一圈扩散

:::

## 经典应用

### 1. 岛屿数量（LeetCode 200）

```java
public int numIslands(char[][] grid) {
    if (grid == null || grid.length == 0) return 0;

    int count = 0;
    int m = grid.length, n = grid[0].length;

    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            if (grid[i][j] == '1') {
                dfs(grid, i, j);
                count++;
            }
        }
    }
    return count;
}

private void dfs(char[][] grid, int i, int j) {
    if (i < 0 || i >= grid.length || j < 0 || j >= grid[0].length
        || grid[i][j] == '0') {
        return;
    }
    grid[i][j] = '0';  // 标记已访问
    dfs(grid, i + 1, j);
    dfs(grid, i - 1, j);
    dfs(grid, i, j + 1);
    dfs(grid, i, j - 1);
}
```

### 2. 克隆图（LeetCode 133）

```java
public Node cloneGraph(Node node) {
    if (node == null) return null;

    Map<Node, Node> map = new HashMap<>();
    Queue<Node> queue = new LinkedList<>();
    queue.offer(node);
    map.put(node, new Node(node.val));

    while (!queue.isEmpty()) {
        Node cur = queue.poll();
        for (Node neighbor : cur.neighbors) {
            if (!map.containsKey(neighbor)) {
                map.put(neighbor, new Node(neighbor.val));
                queue.offer(neighbor);
            }
            // 关键：建立边的映射
            map.get(cur).neighbors.add(map.get(neighbor));
        }
    }
    return map.get(node);
}
```

## 遍历模板总结

### DFS 模板

```java
void dfs(int node) {
    if (应该停止) return;

    visited[node] = true;
    // 处理当前节点

    for (int next : adj[node]) {
        if (!visited[next]) {
            dfs(next);
        }
    }
}
```

### BFS 模板

```java
void bfs(int start) {
    Queue<Integer> queue = new LinkedList<>();
    queue.offer(start);
    visited[start] = true;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int node = queue.poll();
            // 处理当前节点

            for (int next : adj[node]) {
                if (!visited[next]) {
                    visited[next] = true;
                    queue.offer(next);
                }
            }
        }
    }
}
```

## 面试追问

1. **如果图很大，DFS 递归会栈溢出怎么办？**
   - 改用迭代版本，用显式栈
   - 或者限制递归深度，分段处理

2. **如果有环，遍历会陷入死循环吗？**
   - 不会，因为有 `visited` 数组标记已访问节点
   - 关键点：访问节点时立即标记，而不是处理完再标记

3. **为什么 BFS 能保证最短路径？**
   - BFS 按层次遍历，距离 start 为 k 的节点，一定在距离为 k+1 的节点之前处理
   - 第一次遇到某个节点，就是最短路径

**记住：遍历是图算法的基石。DFS 和 BFS 的选择，取决于你的问题需要「深度」还是「广度」。**
