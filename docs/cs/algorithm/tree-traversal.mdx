# 二叉树遍历

> 目标级别：P5

面试官问：「二叉树有哪些遍历方式？」你回答「前序、中序、后序」——然后面试官追问：「非递归怎么实现？」「层序遍历怎么实现？」「给定前序和中序怎么构造二叉树？」

二叉树遍历是算法面试的基础，每个程序员都应该掌握。

## 一、遍历方式总览

### 1.1 四种遍历方式

| 遍历方式 | 顺序 | 应用场景 |
|----------|------|----------|
| 前序（Preorder） | 根 → 左 → 右 | 树的复制、前缀表达式 |
| 中序（Inorder） | 左 → 根 → 右 | BST 排序输出 |
| 后序（Postorder） | 左 → 右 → 根 | 后缀表达式、释放树 |
| 层序（Level-order） | 按层遍历 | 最短路径、分层打印 |

## 二、递归实现

### 2.1 代码实现

```java
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode(int val) {
        this.val = val;
    }
}

public class TreeTraversal {
    // 前序遍历：根 → 左 → 右
    public void preorder(TreeNode root) {
        if (root == null) return;
        System.out.print(root.val + " ");  // 访问根
        preorder(root.left);               // 遍历左
        preorder(root.right);              // 遍历右
    }

    // 中序遍历：左 → 根 → 右
    public void inorder(TreeNode root) {
        if (root == null) return;
        inorder(root.left);                // 遍历左
        System.out.print(root.val + " ");  // 访问根
        inorder(root.right);               // 遍历右
    }

    // 后序遍历：左 → 右 → 根
    public void postorder(TreeNode root) {
        if (root == null) return;
        postorder(root.left);              // 遍历左
        postorder(root.right);             // 遍历右
        System.out.print(root.val + " ");  // 访问根
    }
}
```

### 2.2 遍历顺序示例

```
        1
       / \
      2   3
     / \
    4   5

前序遍历：1 2 4 5 3
中序遍历：4 2 5 1 3
后序遍历：4 5 2 3 1
层序遍历：1 2 3 4 5
```

## 三、非递归实现

### 3.1 前序遍历（非递归）

```java
public List<Integer> preorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Stack<TreeNode> stack = new Stack<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.add(node.val);

        // 先压右子树，再压左子树（出栈时先左后右）
        if (node.right != null) stack.push(node.right);
        if (node.left != null) stack.push(node.left);
    }

    return result;
}
```

### 3.2 中序遍历（非递归）

```java
public List<Integer> inorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Stack<TreeNode> stack = new Stack<>();
    TreeNode current = root;

    while (current != null || !stack.isEmpty()) {
        // 一路向左压栈
        while (current != null) {
            stack.push(current);
            current = current.left;
        }

        // 弹出并访问
        current = stack.pop();
        result.add(current.val);

        // 转向右子树
        current = current.right;
    }

    return result;
}
```

### 3.3 后序遍历（非递归）

```java
public List<Integer> postorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Stack<TreeNode> stack = new Stack<>();
    TreeNode prev = null;

    stack.push(root);
    while (!stack.isEmpty()) {
        TreeNode current = stack.peek();

        // 如果子节点刚被访问过，或者没有子节点
        if (prev == current.left || prev == current.right || (current.left == null && current.right == null)) {
            result.add(current.val);
            stack.pop();
            prev = current;
        } else {
            // 先压右子树，再压左子树
            if (current.right != null) stack.push(current.right);
            if (current.left != null) stack.push(current.left);
        }
    }

    return result;
}
```

### 3.4 层序遍历

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int levelSize = queue.size();
        List<Integer> level = new ArrayList<>();

        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);

            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }

        result.add(level);
    }

    return result;
}
```

## 四、面试题精讲

### 🔴 【高频】二叉树遍历

**问题**：请手写前序、中序、后序遍历的递归和非递归实现。

**标准答案**：

```
递归实现简单，只需要按照遍历顺序递归访问：

前序：根 → 左 → 右
中序：左 → 根 → 右
后序：左 → 右 → 根

非递归实现要点：
1. 前序：利用栈，根出栈时访问，先压右再压左
2. 中序：利用栈，一路向左，入栈；弹出时访问，转向右
3. 后序：利用栈，记录上一个访问的节点，判断是否可以访问
```

### 🟡 【中频】根据遍历序列构造二叉树

**问题**：给定前序和中序遍历结果，如何构造二叉树？

**标准答案**：

```
构造方法：
1. 前序第一个元素是根
2. 在中序中找到根，左边是左子树，右边是右子树
3. 递归构造左右子树

代码：
```java
public TreeNode buildTree(int[] preorder, int preStart, int preEnd,
                          int[] inorder, int inStart, int inEnd) {
    if (preStart > preEnd) return null;

    int rootVal = preorder[preStart];
    int rootIndex = find(inorder, inStart, inEnd, rootVal);

    int leftSize = rootIndex - inStart;

    TreeNode root = new TreeNode(rootVal);
    root.left = buildTree(preorder, preStart + 1, preStart + leftSize,
                          inorder, inStart, rootIndex - 1);
    root.right = buildTree(preorder, preStart + leftSize + 1, preEnd,
                           inorder, rootIndex + 1, inEnd);
    return root;
}
```

## 五、对比总结

### 遍历方式对比

| 遍历 | 顺序 | 栈深度 | 时间 | 空间 |
|------|------|--------|------|------|
| 前序 | 根 → 左 → 右 | O(h) | O(n) | O(h) |
| 中序 | 左 → 根 → 右 | O(h) | O(n) | O(h) |
| 后序 | 左 → 右 → 根 | O(h) | O(n) | O(h) |
| 层序 | 按层 | O(w) | O(n) | O(w) |

## 六、扩展思考

### 💡 Morris 遍历

```
Morris 遍历：O(1) 空间遍历二叉树

核心思想：
- 利用叶子节点的空指针
- 临时建立从叶子到祖先的连接
- 遍历完成后还原树结构
```

### 💡 二叉树遍历的应用

```
前序遍历应用：
- 树的复制
- 前缀表达式

中序遍历应用：
- BST 的有序输出
- 生成有序序列

后序遍历应用：
- 后缀表达式
- 树的删除（先删子节点）
- 计算路径和

层序遍历应用：
- 计算二叉树深度
- 分层打印
- 最短路径
```
