# 计算机基础

计算机基础是软件工程面试的核心考察领域，也是理解上层技术的底层支撑。本章节将从网络协议、系统原理、数据结构三个维度，帮你建立完整的计算机科学知识体系。

## 章节导览

### 计算机网络

从物理层到应用层，完整解析网络协议的底层原理与工程实践。TCP/UDP 的区别、三次握手与四次挥手、HTTP 各版本的演进、HTTPS 的加密原理——这些面试高频考点，都在这里。

### 网络安全

从密码学基础到应用层防护，构建系统的安全知识图谱。对称加密与非对称加密、数字签名与 PKI 体系、XSS 与 SQL 注入的防御手段……安全不是附加项，而是每个工程师的必修课。

### 操作系统

进程与线程如何调度？内存如何管理？文件系统如何组织磁盘数据？Linux 的进程管理、网络协议栈、内存分配器又有哪些门道？这些问题的答案，构成了你理解并发、性能优化的基石。

### 数据结构与算法

从数组链表到 B+ 树，从冒泡排序到动态规划。算法的本质是解决问题的方法论——理解每种结构的适用场景，比死记硬背更重要。

## 如何学习

:::tip 建议的学习顺序

1. **先网络后系统**：网络协议是系统间通信的基础
2. **边学边画**：进程状态转换图、TCP 状态机、B+ 树结构——动手画一遍才能真正理解
3. **对比记忆**：TCP vs UDP、同步 vs 异步、B+ 树 vs 红黑树——对比是区分重点的好方法
4. **结合源码**：学有余力时，阅读 JDK 源码或 Linux 内核源码，加深理解

:::
