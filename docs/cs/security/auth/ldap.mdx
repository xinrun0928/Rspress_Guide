# LDAP 目录服务

大型企业可能有几千名员工，每个员工的账户信息（姓名、邮箱、部门、职位、电话）都需要在多个系统中保持一致。如何高效管理？

**LDAP（Lightweight Directory Access Protocol）** 就是为企业目录管理而设计的协议。

## LDAP 是什么？

LDAP 是一个用于访问**目录服务**的协议。目录服务就像一个数据库，但专门优化用于**读取多、写入少**的场景。

| 特性 | LDAP 目录 | 普通数据库 |
|---|---|---|
| 优化方向 | 读性能 | 读写平衡 |
| 数据结构 | 树形结构 | 表格/文档 |
| 事务支持 | 有限 | 完整 |
| 适用场景 | 用户目录、配置信息 | 业务数据 |

## 目录结构

LDAP 使用**树形结构**（DIT - Directory Information Tree）组织数据：

```
                    dc=example,dc=com (根域)
                    /
        +-----------+-----------+
        |           |           |
    ou=People   ou=Groups   ou=Systems
        |           |
    +---+---+   +---+---+
    |   |   |   |   |   |
  uid uid uid cn=dev cn=ops
```

### 常见对象类

| 对象类 | 说明 | 常用属性 |
|---|---|---|
| inetOrgPerson | 组织人员 | cn, sn, mail, telephoneNumber |
| posixGroup | POSIX 组 | cn, memberUid, gidNumber |
| organizationalUnit | 组织单元 | ou |
| domain | 域组件 | dc |
| organization | 组织 | o |

### DN 和 RDN

```
DN（Distinguished Name）: uid=john,ou=People,dc=example,dc=com
  └── RDN（Relative Distinguished Name）: uid=john
```

## LDAP 操作

### 添加（Add）

```python
import ldap

# 连接 LDAP 服务器
conn = ldap.initialize('ldap://ldap.example.com')
conn.simple_bind_s('admin@example.com', 'password')

# 添加用户
user_dn = 'uid=john,ou=People,dc=example,dc=com'
user_attrs = [
    ('objectClass', [b'top', b'person', b'organizationalPerson', b'inetOrgPerson']),
    ('uid', b'john'),
    ('cn', b'John Doe'),
    ('sn', b'Doe'),
    ('mail', b'john@example.com'),
    ('userPassword', b'hashed_password'),
    ('departmentNumber', b'engineering'),
    ('telephoneNumber', b'+1-555-0123')
]

conn.add_s(user_dn, user_attrs)
```

### 搜索（Search）

```python
# 搜索用户
search_filter = '(uid=john)'
attrs = ['cn', 'mail', 'departmentNumber']

results = conn.search_s(
    'dc=example,dc=com',           # 搜索基准 DN
    ldap.SCOPE_SUBTREE,            # 搜索范围：子树
    search_filter,                 # 搜索过滤器
    attrs                          # 返回的属性
)

for dn, attrs in results:
    print(f"DN: {dn}")
    print(f"姓名: {attrs.get('cn', [b''])[0].decode()}")
    print(f"邮箱: {attrs.get('mail', [b''])[0].decode()}")
    print(f"部门: {attrs.get('departmentNumber', [b''])[0].decode()}")

# 搜索过滤器语法
filters = [
    '(uid=*)',                      # 所有用户
    '(uid=john)',                   # uid=john
    '(|(uid=john)(uid=jane))',      # uid=john OR uid=jane
    '(&(department=engineering)(mail=*))',  # AND 条件
    '(mail=*@example.com)',         # 通配符
    '(manager=uid=boss,ou=People,dc=example,dc=com)',  # 引用属性
]
```

### 修改（Modify）

```python
# 修改用户属性
user_dn = 'uid=john,ou=People,dc=example,dc=com'

# 添加属性
conn.modify_s(user_dn, [
    (ldap.MOD_ADD, 'mobile', b'+1-555-9999'),
])

# 删除属性
conn.modify_s(user_dn, [
    (ldap.MOD_DELETE, 'telephoneNumber', None),
])

# 替换属性
conn.modify_s(user_dn, [
    (ldap.MOD_REPLACE, 'mail', b'newjohn@example.com'),
])
```

### 删除（Delete）

```python
# 删除用户
user_dn = 'uid=john,ou=People,dc=example,dc=com'
conn.delete_s(user_dn)
```

## Active Directory

Active Directory（AD）是 Microsoft 的目录服务实现，LDAP 是其核心协议之一。

### AD 的特殊对象

| 对象类 | 说明 |
|---|---|
| user | AD 用户账户 |
| computer | 加入域的计算机 |
| group | 安全组或分发组 |
| organizationalUnit | 组织单元（OU） |
| domainDNS | 域 |

### AD 组类型

| 类型 | 用途 | 嵌套 |
|---|---|---|
| 安全组 | 权限分配、资源访问 | 支持 |
| 分发组 | 邮件列表 | 支持 |

### AD 组作用域

| 作用域 | 说明 |
|---|---|
| 域本地组 | 只能授予本域资源的权限 |
| 全局组 | 可在任何域使用，适合跨域 |
| 通用组 | 最多功能，可跨林 |

```
域本地组 -> 本域资源
全局组 -> 跨域资源访问
通用组 -> 跨林访问
```

### Python 操作 AD

```python
import ldap

# 连接 AD（使用 LDAPS）
AD_SERVER = 'ldaps://ad.example.com:636'
AD_USER = 'admin@example.com'
AD_PASSWORD = 'password'

conn = ldap.initialize(AD_SERVER)
conn.set_option(ldap.OPT_X_TLS_REQUIRE_CERT, ldap.OPT_X_TLS_NEVER)
conn.simple_bind_s(AD_USER, AD_PASSWORD)

# 搜索 AD 用户
results = conn.search_s(
    'dc=example,dc=com',
    ldap.SCOPE_SUBTREE,
    '(&(objectClass=user)(objectCategory=person)(!(userAccountControl:1.2.840.113556.1.4.803:=2)))',
    ['cn', 'mail', 'sAMAccountName', 'memberOf']
)

# 禁用用户
USER_DN = 'CN=John Doe,OU=Users,DC=example,DC=com'
# UF_ACCOUNTDISABLE = 0x0002 = 2
conn.modify_s(USER_DN, [(ldap.MOD_REPLACE, 'userAccountControl', b'514')])

# 添加用户到组
GROUP_DN = 'CN=Engineering,OU=Groups,DC=example,DC=com'
USER_DN = 'CN=John Doe,OU=Users,DC=example,DC=com'
conn.modify_s(GROUP_DN, [(ldap.MOD_ADD, 'member', USER_DN.encode())])
```

## 与应用集成

### Flask + LDAP 认证

```python
from flask import Flask, request, session
import ldap

app = Flask(__name__)

@app.route('/login', methods=['GET', 'POST'])
def login():
    if request.method == 'POST':
        username = request.form['username']
        password = request.form['password']

        try:
            # 连接 LDAP
            conn = ldap.initialize('ldap://ldap.example.com')
            conn.set_option(ldap.OPT_REFERRALS, 0)

            # 构建用户 DN
            user_dn = f'uid={username},ou=People,dc=example,dc=com'

            # 绑定验证
            conn.simple_bind_s(user_dn, password)

            # 获取用户信息
            results = conn.search_s(
                user_dn,
                ldap.SCOPE_BASE,
                '(objectClass=*)',
                ['cn', 'mail', 'ou']
            )

            if results:
                session['user'] = {
                    'username': username,
                    'cn': results[0][1]['cn'][0].decode(),
                    'mail': results[0][1]['mail'][0].decode(),
                    'ou': results[0][1]['ou'][0].decode()
                }

            conn.unbind_s()
            return redirect('/dashboard')

        except ldap.INVALID_CREDENTIALS:
            return '用户名或密码错误'
        except ldap.SERVER_DOWN:
            return '目录服务不可用'
        except Exception as e:
            return f'登录失败: {str(e)}'

    return '''
    <form method="POST">
        <input name="username" placeholder="用户名">
        <input name="password" type="password" placeholder="密码">
        <button type="submit">登录</button>
    </form>
    '''
```

## 安全考虑

### LDAPS

```bash
# 启用 LDAPS（LDAP over SSL）
# 1. 获取 CA 证书
curl -o /tmp/ca.crt http://ca.example.com/ca.crt

# 2. 配置 LDAP 使用证书
cat >> /etc/openldap/ldap.conf << EOF
TLS_CACERT /tmp/ca.crt
TLS_REQCERT demand
EOF

# 3. 连接时强制使用 SSL
ldapsearch -H ldaps://ldap.example.com:636 -D "admin@example.com" -W
```

### 访问控制

```python
# LDAP 访问控制列表（ACL）
# 在 slapd.conf 中配置
# 或者使用 OpenLDAP 的 dynamic ACLs

ACL_TEMPLATE = '''
# 默认：用户只能修改自己的密码和部分属性
access to attrs=userPassword
    by self write
    by anonymous auth
    by * none

# 用户可以读取自己的大部分信息
access to attrs=cn,sn,mail,telephoneNumber,mobile
    by self write
    by users read
    by * none

# 管理员可以读写所有用户
access to *
    by dn="cn=admin,dc=example,dc=com" write
    by * read
'''
```
