# MySQL 面试复习导航

## 快速导航

| 分类 | 核心主题 | 面试权重 |
|------|----------|----------|
| **索引原理** | B+Tree 索引、聚簇索引、覆盖索引、最左前缀、索引下推 | 🔴 高频 |
| **事务与锁** | ACID、隔离级别、MVCC、ReadView、锁分类、间隙锁 | 🔴 高频 |
| **SQL 优化** | Explain 执行计划、慢查询优化、分页优化、count 优化 | 🔴 高频 |
| **主从复制** | binlog 格式、主从原理、读写分离、分库分表 | 🟡 中频 |

## 学习路径建议

```
第一阶段（索引）
├── B+Tree 索引原理 → 聚簇索引 vs 二级索引
├── 联合索引最左前缀 → 覆盖索引与回表
└── 索引失效场景 → 索引下推（ICP）

第二阶段（事务与锁）
├── ACID 特性 → 四种隔离级别
├── MVCC 原理 → ReadView 可见性判断
└── 锁分类 → 间隙锁 → 死锁

第三阶段（优化与架构）
├── Explain 执行计划 → 慢查询优化
├── 主从复制 → 读写分离
└── 分库分表 → 分页查询优化
```

## 面试准备优先级

### 🔴 必须掌握（P6 必问）

- B+Tree 索引结构与查询流程
- 聚簇索引 vs 二级索引的区别
- 联合索引最左前缀原则
- 事务 ACID 特性与隔离级别
- MVCC 与 ReadView 原理
- 索引失效的常见场景

### 🟡 重点理解

- 覆盖索引与回表查询
- 间隙锁与临键锁
- Explain 各字段含义
- binlog 三种格式
- 死锁原因与排查方法

### 🟢 扩展了解

- 分库分表策略
- 读写分离架构
- count 查询优化
