# Oracle 数据库基础

Oracle 基础模块为你系统性地介绍 Oracle 数据库的核心概念。从数据库的整体架构出发，深入讲解实例与数据库的关系、物理存储结构（数据文件、日志文件、控制文件）、逻辑存储结构（表空间、段、区、块），以及 Oracle 独特的 SGA 和 PGA 内存架构。后台进程部分则揭示了 DBWn、LGWR、CKPT、SMON、PMON 等核心进程如何在幕后协作，确保数据库高效、稳定地运行。

通过本模块的学习，你将建立起对 Oracle 数据库的整体认知，理解数据在 Oracle 中是如何被组织、存储和管理的。这不仅是从零到一的基础，也是后续深入学习 SQL 优化、事务管理、集群架构的根基。

## 模块内容

**核心概念**

- [Oracle 数据库概述与架构](/database/oracle/basic/overview)：Oracle 在企业级市场的地位，版本演进，核心组件
- [Oracle 实例与数据库](/database/oracle/basic/instance)：Instance 与 Database 的区别，初始化参数，连接方式
- [物理结构](/database/oracle/basic/physical-structure)：数据文件、日志文件、控制文件的作用与关系
- [逻辑结构](/database/oracle/basic/logical-structure)：表空间、段、区、块的层级关系与空间管理
- [SGA 与 PGA](/database/oracle/basic/sga-pga)：内存组件详解，Buffer Cache、Shared Pool、Redo Log Buffer 的作用
- [后台进程](/database/oracle/basic/background-process)：DBWn、LGWR、CKPT、SMON、PMON 等核心进程的协作机制

:::tip 学习建议
Oracle 的物理结构和逻辑结构是面试中的高频考点。建议先理解"实例"与"数据库"这两个核心概念，再逐步深入物理存储和逻辑存储的细节。后台进程部分建议配合架构图学习，理解各进程之间的协作关系。
:::
