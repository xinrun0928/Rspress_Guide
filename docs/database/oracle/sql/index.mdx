# SQL 与 PL/SQL

SQL 与 PL/SQL 模块带你深入 Oracle 的编程世界。从基础的 SQL 语法，到强大的 PL/SQL 过程化扩展，你将掌握 Oracle 数据库的完整数据操作能力。

PL/SQL 是 Oracle 对 SQL 的过程化扩展，它不仅能执行 SQL 语句，还支持变量、循环、条件判断、异常处理等编程特性。这使得复杂的数据处理逻辑可以直接在数据库层面完成，减少应用层与数据库之间的网络往返。

## 模块内容

**SQL 基础与进阶**

- [Oracle SQL 基础](/database/oracle/sql/sql-basics)：SELECT、INSERT、UPDATE、DELETE，Oracle 特有语法
- [常用函数](/database/oracle/sql/functions)：字符、日期、数值、转换函数，聚合函数
- [分页查询](/database/oracle/sql/pagination)：ROWNUM 与 ROW_NUMBER 的使用

**PL/SQL 过程化扩展**

- [PL/SQL 块结构](/database/oracle/sql/plsql-block)：声明、执行、异常处理三部分
- [存储过程与函数](/database/oracle/sql/procedure-function)：创建、调用、参数传递
- [触发器](/database/oracle/sql/trigger)：DML 触发器、INSTEAD OF 触发器、系统触发器
- [游标](/database/oracle/sql/cursor)：显式游标、隐式游标、REF 游标
- [集合类型](/database/oracle/sql/collection)：VARRAY、嵌套表、关联数组
- [异常处理](/database/oracle/sql/exception)：预定义异常、用户定义异常
- [序列与同义词](/database/oracle/sql/sequence-synonym)：序列生成唯一键，同义词简化访问

:::tip 学习建议

:::
