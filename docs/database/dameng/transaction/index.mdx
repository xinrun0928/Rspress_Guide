# 事务与锁

本章节介绍达梦数据库的事务管理和锁机制，包括事务管理、锁类型、隔离级别以及 MVCC 实现。通过本章节的学习，你将深入理解达梦数据库的并发控制原理。

## 内容导航

| 文档 | 内容 | 难度 |
|---|---|---|
| `transaction/transaction` | 事务管理机制 | 核心 |
| `transaction/lock` | 锁类型与锁冲突处理 | 核心 |
| `transaction/isolation` | 四种隔离级别详解 | 核心 |
| `transaction/mvcc` | MVCC 多版本并发控制原理 | 进阶 |

## 学习建议

事务与锁是数据库并发控制的核心。建议先理解事务的基本概念和 ACID 特性，再学习锁的类型和作用，最后深入理解隔离级别和 MVCC 机制。每个知识点都与实际并发场景紧密相关。

开始学习吧！
