# 存储结构

本章节介绍达梦数据库的存储结构，包括堆表与 HTSP 存储引擎、索引类型、索引设计以及表分区。通过本章节的学习，你将掌握达梦数据库的存储机制和性能优化技巧。

## 内容导航

| 文档 | 内容 | 难度 |
|---|---|---|
| `storage/table-type` | 堆表与 HTSP 表存储引擎 | 核心 |
| `storage/index-type` | B 树索引、位图索引、函数索引 | 核心 |
| `storage/index-design` | 索引设计原则与优化技巧 | 进阶 |
| `storage/partition` | 表分区：范围、列表、哈希、组合分区 | 核心 |

## 学习建议

存储结构是数据库性能优化的基础。建议先理解表和索引的存储方式，再学习如何设计高效的索引，最后掌握分区表的分区策略。每个知识点都配有实际的 SQL 示例，帮助你将理论应用于实践。

开始学习吧！
