# 基础入门

本章节介绍达梦数据库的基础知识，包括达梦概述、物理结构、逻辑结构、兼容性对比以及安装配置。通过本章节的学习，你将掌握达梦数据库的核心概念和基本操作。

## 内容导航

| 文档 | 内容 | 难度 |
|---|---|---|
| `basic/overview` | 达梦概述、发展历程、市场定位 | 入门 |
| `basic/physical-structure` | 物理存储结构：数据文件、日志文件、控制文件 | 基础 |
| `basic/logical-structure` | 逻辑存储结构：表空间、段、区、页 | 核心 |
| `basic/compatibility` | 与 Oracle/MySQL 兼容性对比 | 核心 |
| `basic/install` | 安装配置与客户端工具使用 | 入门 |

## 学习建议

建议按照上述顺序依次学习，先掌握基本概念，再理解存储结构，最后学习兼容性对比和安装配置。每个文档都配有详细的代码示例和实践指导，帮助你快速上手达梦数据库。

开始学习吧！
