# 实战应用概述

Redis 的价值不仅在于缓存，更在于它丰富的数据结构可以解决各种业务问题。这一章将带你了解 Redis 在实际生产环境中的经典应用场景。

## 为什么学习 Redis 应用场景

面试中，除了问原理，还会问实践：

- Redis 除了缓存，还能做什么？
- 如何用 Redis 实现排行榜？
- 如何用 Redis 实现分布式锁？
- 如何用 Redis 实现延迟队列？

理解这些应用场景，才能真正把 Redis 用好。

## 学习路径

建议按照以下顺序学习：

1. **先掌握基础**：Session 管理、分布式锁（已讲）
2. **再理解核心**：延迟队列、排行榜（高频面试）
3. **最后扩展**：BitMap、HyperLogLog（高级特性）

:::tip 学习目标
学完本章后，你应该能够：
- 根据业务场景选择合适的数据结构
- 实现常见业务需求（排行榜、签到、UV 统计等）
- 在面试中展示 Redis 的实战能力
:::
