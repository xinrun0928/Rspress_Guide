# 排行榜与计数器

游戏积分排行、电商销量排行、内容热度排行……这些场景都离不开排行榜。Redis 的 ZSet 是实现排行榜的利器。

## 为什么用 Redis 实现排行榜

传统方案使用数据库：

```sql
SELECT * FROM game_scores 
ORDER BY score DESC 
LIMIT 10;
```

问题：
- 数据量大时查询慢
- 高并发写入性能差
- 需要额外缓存

Redis ZSet 的优势：
- 插入/更新：O(log N)
- 获取 Top N：O(log N + M)
- 天然有序

## ZSet 实现排行榜

### 基本操作

```bash
# 添加/更新玩家分数
ZADD game:rank 1500 "player1"
ZADD game:rank 2000 "player2"
ZADD game:rank 1800 "player3"

# 获取排行榜前 10
ZREVRANGE game:rank 0 9 WITHSCORES
# 1) "player2" "2000"
# 2) "player3" "1800"
# 3) "player1" "1500"

# 获取玩家排名（0-indexed）
ZREVRANK game:rank "player2"
# (integer) 0

# 获取玩家分数
ZSCORE game:rank "player1"
# "1500"

# 增加分数
ZINCRBY game:rank 100 "player1"
# "1600"
```

### Java 实现

```java
@Service
public class RankingService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    private static final String RANK_KEY = "ranking:game:";
    
    /**
     * 更新玩家分数
     */
    public void updateScore(String gameId, String playerId, double score) {
        String key = RANK_KEY + gameId;
        redisTemplate.opsForZSet().add(key, playerId, score);
    }
    
    /**
     * 增加玩家分数
     */
    public double incrementScore(String gameId, String playerId, double delta) {
        String key = RANK_KEY + gameId;
        return redisTemplate.opsForZSet().incrementScore(key, playerId, delta);
    }
    
    /**
     * 获取排行榜前 N 名
     */
    public Set<ZSetOperations.TypedTuple<String>> getTopN(String gameId, int n) {
        String key = RANK_KEY + gameId;
        // ZREVRANGE：从大到小
        return redisTemplate.opsForZSet().reverseRangeWithScores(key, 0, n - 1);
    }
    
    /**
     * 获取玩家排名
     */
    public Long getPlayerRank(String gameId, String playerId) {
        String key = RANK_KEY + gameId;
        // ZREVRANK：排名从 0 开始
        return redisTemplate.opsForZSet().reverseRank(key, playerId);
    }
    
    /**
     * 获取玩家分数
     */
    public Double getPlayerScore(String gameId, String playerId) {
        String key = RANK_KEY + gameId;
        return redisTemplate.opsForZSet().score(key, playerId);
    }
}
```

### 分段排行

```java
@Service
public class SegmentRankingService {
    
    /**
     * 获取玩家在某个分数段的排名
     * 例如：1000-2000 分段的玩家排名
     */
    public Set<String> getRankByScoreRange(String gameId, double minScore, double maxScore) {
        String key = RANK_KEY + gameId;
        // 获取 1000-2000 分段的玩家
        return redisTemplate.opsForZSet().rangeByScore(key, minScore, maxScore);
    }
    
    /**
     * 获取玩家周围的排名
     * 例如：查看玩家前后 5 名的玩家
     */
    public Map<String, Object> getPlayerNeighbors(String gameId, String playerId) {
        String key = RANK_KEY + gameId;
        
        // 获取玩家排名
        Long rank = redisTemplate.opsForZSet().reverseRank(key, playerId);
        if (rank == null) {
            return null;
        }
        
        Map<String, Object> result = new HashMap<>();
        
        // 获取前 5 名
        Set<String> top5 = redisTemplate.opsForZSet()
            .reverseRange(key, Math.max(0, rank - 5), rank - 1);
        result.put("above", top5);
        
        // 获取后 5 名
        Set<String> bottom5 = redisTemplate.opsForZSet()
            .reverseRange(key, rank + 1, rank + 5);
        result.put("below", bottom5);
        
        return result;
    }
}
```

## 排行榜去重

如果同一玩家有多条记录，需要合并：

```java
@Service
public class DeduplicateRankingService {
    
    /**
     * 场景：玩家在不同服务器有多个分数记录
     * 需要合并为最高分
     */
    public void mergeScores(String gameId, String playerId, double... scores) {
        String key = RANK_KEY + gameId;
        
        // 获取玩家当前分数
        Double currentScore = redisTemplate.opsForZSet().score(key, playerId);
        
        // 计算最高分
        double maxScore = currentScore != null ? currentScore : 0;
        for (double score : scores) {
            if (score > maxScore) {
                maxScore = score;
            }
        }
        
        // 更新为最高分
        redisTemplate.opsForZSet().add(key, playerId, maxScore);
    }
    
    /**
     * 合并多个排行榜
     */
    public void mergeRankings(String targetKey, String... sourceKeys) {
        // 使用 ZUNIONSTORE 合并多个排行榜
        redisTemplate.opsForZSet().unionAndStore(sourceKeys[0], sourceKeys[1], targetKey);
        
        for (int i = 2; i < sourceKeys.length; i++) {
            String tempKey = targetKey + ":temp";
            redisTemplate.opsForZSet().unionAndStore(targetKey, sourceKeys[i], tempKey);
            redisTemplate.rename(tempKey, targetKey);
        }
    }
}
```

## 排行榜定时清理

```java
@Service
public class RankingCleanupService {
    
    /**
     * 定时清理过期的排行榜数据
     * 例如：只保留最近 7 天的数据
     */
    @Scheduled(cron = "0 0 3 * * ?")  // 每天凌晨 3 点
    public void cleanupExpiredRankings() {
        // 获取所有排行榜 key
        Set<String> keys = redisTemplate.keys("ranking:*");
        
        for (String key : keys) {
            // 检查是否有时间戳
            if (key.matches(".*:\\d{8}$")) {
                String dateStr = key.substring(key.length() - 8);
                LocalDate date = LocalDate.parse(dateStr, 
                    DateTimeFormatter.ofPattern("yyyyMMdd"));
                
                // 删除 7 天前的排行榜
                if (date.isBefore(LocalDate.now().minusDays(7))) {
                    redisTemplate.delete(key);
                    log.info("删除过期排行榜: {}", key);
                }
            }
        }
    }
}
```

## 完整示例：游戏排行榜

```java
@RestController
@RequestMapping("/api/ranking")
public class RankingController {
    
    @Autowired
    private RankingService rankingService;
    
    /**
     * 上报玩家分数
     */
    @PostMapping("/score")
    public Result<Void> updateScore(@RequestParam String gameId,
                                    @RequestParam String playerId,
                                    @RequestParam double score) {
        rankingService.updateScore(gameId, playerId, score);
        return Result.success();
    }
    
    /**
     * 获取排行榜
     */
    @GetMapping("/top")
    public Result<List<RankingVO>> getTopRanking(
            @RequestParam String gameId,
            @RequestParam(defaultValue = "10") int limit) {
        
        Set<ZSetOperations.TypedTuple<String>> topSet = 
            rankingService.getTopN(gameId, limit);
        
        List<RankingVO> result = new ArrayList<>();
        int rank = 1;
        for (ZSetOperations.TypedTuple<String> tuple : topSet) {
            RankingVO vo = new RankingVO();
            vo.setRank(rank++);
            vo.setPlayerId(tuple.getValue());
            vo.setScore(tuple.getScore());
            result.add(vo);
        }
        
        return Result.success(result);
    }
    
    /**
     * 获取玩家排名
     */
    @GetMapping("/player")
    public Result<RankingVO> getPlayerRanking(
            @RequestParam String gameId,
            @RequestParam String playerId) {
        
        Long rank = rankingService.getPlayerRank(gameId, playerId);
        Double score = rankingService.getPlayerScore(gameId, playerId);
        
        RankingVO vo = new RankingVO();
        vo.setRank(rank != null ? rank + 1 : null);  // 转为 1-indexed
        vo.setPlayerId(playerId);
        vo.setScore(score);
        
        return Result.success(vo);
    }
}

@Data
public class RankingVO {
    private Integer rank;
    private String playerId;
    private Double score;
}
```

## 计数器实现

Redis 的 INCR 系列命令是实现计数器的利器：

```java
@Service
public class CounterService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    /**
     * 文章阅读量
     */
    public void incrementArticleViews(Long articleId) {
        String key = "article:views:" + articleId;
        redisTemplate.opsForValue().increment(key);
    }
    
    public Long getArticleViews(Long articleId) {
        String key = "article:views:" + articleId;
        String value = redisTemplate.opsForValue().get(key);
        return value != null ? Long.parseLong(value) : 0L;
    }
    
    /**
     * 用户每日行为计数
     */
    public void incrementDailyAction(Long userId, String action) {
        String key = "user:action:" + userId + ":" + LocalDate.now();
        redisTemplate.opsForValue().increment(key + ":" + action);
    }
    
    /**
     * 实时统计：每秒请求数
     */
    public void incrementRequestCount() {
        String key = "request:count:" + System.currentTimeMillis() / 1000;
        redisTemplate.opsForValue().increment(key);
        redisTemplate.expire(key, 120);  // 2 分钟后过期
    }
}
```

## 总结

排行榜是 Redis 的经典应用：

1. **ZSet 实现**：score 为分数，member 为玩家 ID
2. **高性能**：O(log N) 插入和查询
3. **多维度**：支持多种排行榜
4. **实时更新**：INCRBY 实时增加分数
5. **分段展示**：支持分页、玩家周围排名

Redis 让排行榜变得简单高效，是面试和实战中的常考知识点。
