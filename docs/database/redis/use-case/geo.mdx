# 附近的人与地理围栏

LBS（Location Based Service）是移动互联网的重要应用。「附近的人」「附近的餐厅」「附近的单车」等功能，都可以用 Redis 的 GEO 类型轻松实现。

## GEO 类型概述

Redis 的 GEO 类型基于 ZSet 实现，存储地理位置信息：

```bash
# 添加地理位置
GEOADD cities 116.4074 39.9042 "Beijing"
GEOADD cities 121.4737 31.2304 "Shanghai"
GEOADD cities 113.2644 23.1291 "Guangzhou"

# 获取城市位置
GEOPOS cities Beijing
# 1) 1) "116.4074000129699707"
#    2) "39.90419998264390884"

# 获取两城市间距离
GEODIST cities Beijing Shanghai km
# "1088.1301"

# 获取指定半径内的城市
GEORADIUS cities 116.4074 39.9042 200 km WITHDIST ASC COUNT 10
# 1) 1) "Beijing"
#    2) "0.0"
# 2) 1) "Tianjin"
#    2) "110.2894"
```

## 基本操作

### 添加位置

```java
@Service
public class GeoService {

     * 添加单个用户位置
     */

    /**
     * 批量添加位置
     */
    public void addLocations(Map<String, Point> locations) {
        String key = "geo:users";
        redisTemplate.opsForGeo().add(key, locations);
    }
}
```

```java
    }
```

## 完整实现：附近的人

```java
@Service
public class NearbyPeopleService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    /**
     * 更新用户位置
     */
    public void updateLocation(Long userId, double longitude, double latitude) {
        String key = "geo:users";
        redisTemplate.opsForGeo().add(key, new Point(longitude, latitude), userId.toString());
        
        // 保存用户基本信息到 Hash
        String userInfoKey = "geo:user:info:" + userId;
        Map<String, String> userInfo = new HashMap<>();
        userInfo.put("longitude", String.valueOf(longitude));
        userInfo.put("latitude", String.valueOf(latitude));
        userInfo.put("updateTime", String.valueOf(System.currentTimeMillis()));
        redisTemplate.opsForHash().putAll(userInfoKey, userInfo);
    }
    
    /**
     * 查询附近的人
     */
    public List<NearbyUserVO> searchNearby(Long userId, double radiusKm, int limit) {
        String key = "geo:users";
        
        // 获取当前用户位置
        List<Point> positions = redisTemplate.opsForGeo()
            .position(key, userId.toString());
        
        if (positions == null || positions.isEmpty() || positions.get(0) == null) {
            return Collections.emptyList();
        }
        
        Point currentPosition = positions.get(0);
        
        // 查询附近的人
        Circle circle = new Circle(currentPosition, new Distance(radiusKm, Metrics.KILOMETERS));
        RedisGeoCommands.GeoRadiusCommandArgs args = 
            RedisGeoCommands.GeoRadiusCommandArgs.newGeoRadiusArgs()
                .includeDistance()
                .includeCoordinates()
                .sortAscending()
                .limit(limit);
        
        GeoResults<RedisGeoCommands.GeoLocation<String>> results = 
            redisTemplate.opsForGeo().radius(key, circle, args);
        
        // 转换为 VO
        List<NearbyUserVO> nearbyUsers = new ArrayList<>();
        if (results != null) {
            for (GeoResult<RedisGeoCommands.GeoLocation<String>> result : results) {
                String nearbyUserId = result.getContent().getName();
                
                // 排除自己
                if (nearbyUserId.equals(userId.toString())) {
                    continue;
                }
                
                NearbyUserVO vo = new NearbyUserVO();
                vo.setUserId(Long.parseLong(nearbyUserId));
                vo.setDistance(result.getDistance().getValue());
                vo.setLongitude(result.getContent().getPoint().getX());
                vo.setLatitude(result.getContent().getPoint().getY());
                
                // 获取用户基本信息
                String userInfoKey = "geo:user:info:" + nearbyUserId;
                Map<Object, Object> userInfo = redisTemplate.opsForHash().entries(userInfoKey);
                if (!userInfo.isEmpty()) {
                    vo.setNickname((String) userInfo.get("nickname"));
                    vo.setAvatar((String) userInfo.get("avatar"));
                }
                
                nearbyUsers.add(vo);
            }
        }
        
        return nearbyUsers;
    }
}
```

```java
@Service
public class NearbyMerchantService {

     * 添加商户
     */
    public void addMerchant(Long merchantId, String name, 
                          double longitude, double latitude) {
        String key = "geo:merchants";
        
        // 添加位置

        // 保存商户信息

        info.put("name", name);
        info.put("longitude", String.valueOf(longitude));
        info.put("latitude", String.valueOf(latitude));
        redisTemplate.opsForHash().putAll(merchantKey, info);
    }
    
    /**
     * 按类别查询附近商户
     */
    public List<MerchantVO> searchByCategory(double longitude, double latitude,
                                           String category, double radiusKm) {
        String key = "geo:merchants:" + category;
        
        // 先查询位置
        Circle circle = new Circle(new Point(longitude, latitude), 
            new Distance(radiusKm, Metrics.KILOMETERS));
        
        GeoResults<RedisGeoCommands.GeoLocation<String>> results = 
            redisTemplate.opsForGeo().radius(key, circle);
        
        // 转换为商户信息
        List<MerchantVO> merchants = new ArrayList<>();
        if (results != null) {
            for (GeoResult<RedisGeoCommands.GeoLocation<String>> result : results) {
                Long merchantId = Long.parseLong(result.getContent().getName());
                merchants.add(getMerchantInfo(merchantId));
            }
        }
        
        return merchants;
    }
    
    /**
     * 获取商户信息
     */
    private MerchantVO getMerchantInfo(Long merchantId) {
        String merchantKey = "merchant:" + merchantId;
        Map<Object, Object> info = redisTemplate.opsForHash().entries(merchantKey);
        
        MerchantVO vo = new MerchantVO();
        vo.setId(merchantId);
        vo.setName((String) info.get("name"));
        return vo;
    }
}
```

## 地理围栏

地理围栏用于判断用户是否进入或离开某个区域：

```java
@Service
public class GeoFenceService {
    
    /**
     * 判断是否在围栏内
     */
    public boolean isInsideFence(Long userId, String fenceId) {
        // 获取用户位置
        String userKey = "geo:users";
        List<Point> positions = redisTemplate.opsForGeo()
            .position(userKey, userId.toString());
        
        if (positions == null || positions.isEmpty() || positions.get(0) == null) {
            return false;
        }
        
        Point userPoint = positions.get(0);
        
        // 获取围栏中心点
        String fenceKey = "fence:" + fenceId;
        Point fenceCenter = new Point(
            Double.parseDouble((String) redisTemplate.opsForHash().get(fenceKey, "longitude")),
            Double.parseDouble((String) redisTemplate.opsForHash().get(fenceKey, "latitude"))
        );
        
        double radius = Double.parseDouble(
            (String) redisTemplate.opsForHash().get(fenceKey, "radius"));
        
        // 计算距离
        Distance distance = redisTemplate.opsForGeo()
            .distance(userKey, userId.toString(), "fence:" + fenceId, Metrics.METERS);
        
        return distance != null && distance.getValue() <= radius;
    }
    
    /**
     * 创建围栏
     */
    public void createFence(String fenceId, String name, 
                           double longitude, double latitude, double radius) {
        String fenceKey = "fence:" + fenceId;
        
        Map<String, String> fenceInfo = new HashMap<>();
        fenceInfo.put("name", name);
        fenceInfo.put("longitude", String.valueOf(longitude));
        fenceInfo.put("latitude", String.valueOf(latitude));
        fenceInfo.put("radius", String.valueOf(radius));
        fenceInfo.put("createTime", String.valueOf(System.currentTimeMillis()));
        
        redisTemplate.opsForHash().putAll(fenceKey, fenceInfo);
        
        // 添加围栏中心点到 GEO
        String geoKey = "fence:" + fenceId;
        redisTemplate.opsForGeo().add(geoKey, new Point(longitude, latitude), "center");
    }
    
    /**
     * 批量检测用户是否进入围栏
     */
    public List<String> checkUsersInFence(List<Long> userIds, String fenceId) {
        String userGeoKey = "geo:users";
        String fenceKey = "fence:" + fenceId;
        
        // 获取围栏信息
        Map<Object, Object> fenceInfo = redisTemplate.opsForHash().entries(fenceKey);
        double centerLon = Double.parseDouble((String) fenceInfo.get("longitude"));
        double centerLat = Double.parseDouble((String) fenceInfo.get("latitude"));
        double radius = Double.parseDouble((String) fenceInfo.get("radius"));
        
        List<String> insideUsers = new ArrayList<>();
        
        for (Long userId : userIds) {
            // 获取用户位置
            List<Point> positions = redisTemplate.opsForGeo()
                .position(userGeoKey, userId.toString());
            
            if (positions != null && !positions.isEmpty() && positions.get(0) != null) {
                Point userPoint = positions.get(0);
                
                // 计算距离
                double distance = calculateDistance(
                    centerLat, centerLon, 
                    userPoint.getY(), userPoint.getX()
                );
                
                if (distance <= radius) {
                    insideUsers.add(userId.toString());
                }
            }
        }
        
        return insideUsers;
    }
    
    /**
     * Haversine 公式计算两点间距离
     */
    private double calculateDistance(double lat1, double lon1, double lat2, double lon2) {
        final double R = 6371000; // 地球半径（米）
        
        double latDistance = Math.toRadians(lat2 - lat1);
        double lonDistance = Math.toRadians(lon2 - lon1);
        
        double a = Math.sin(latDistance / 2) * Math.sin(latDistance / 2)
                + Math.cos(Math.toRadians(lat1)) * Math.cos(Math.toRadians(lat2))
                * Math.sin(lonDistance / 2) * Math.sin(lonDistance / 2);
        
        double c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
        
        return R * c;
    }
}
```

## 附近单车的实现

```java
@Service
public class BikeService {
    
    private static final String BIKE_GEO_KEY = "geo:bikes";
    private static final String AVAILABLE_BIKE_SET = "bikes:available";
    
    /**
     * 投放单车
     */
    public void addBike(Long bikeId, double longitude, double latitude) {
        // 添加到 GEO
        redisTemplate.opsForGeo().add(BIKE_GEO_KEY, 
            new Point(longitude, latitude), bikeId.toString());
        
        // 添加到可用单车集合
        redisTemplate.opsForSet().add(AVAILABLE_BIKE_SET, bikeId.toString());
        
        // 保存单车信息
        String bikeKey = "bike:" + bikeId;
        redisTemplate.opsForHash().put(bikeKey, "status", "available");
        redisTemplate.opsForHash().put(bikeKey, "longitude", String.valueOf(longitude));
        redisTemplate.opsForHash().put(bikeKey, "latitude", String.valueOf(latitude));
    }
    
    /**
     * 查找附近可用的单车
     */
    public List<BikeVO> findNearbyAvailableBikes(double longitude, double latitude, 
                                                  double radiusKm) {
        Circle circle = new Circle(new Point(longitude, latitude), 
            new Distance(radiusKm, Metrics.KILOMETERS));
        
        GeoResults<RedisGeoCommands.GeoLocation<String>> results = 
            redisTemplate.opsForGeo().radius(BIKE_GEO_KEY, circle);
        
        List<BikeVO> bikes = new ArrayList<>();
        if (results != null) {
            for (GeoResult<RedisGeoCommands.GeoLocation<String>> result : results) {
                String bikeId = result.getContent().getName();
                
                // 检查是否可用
                Boolean isAvailable = redisTemplate.opsForSet()
                    .isMember(AVAILABLE_BIKE_SET, bikeId);
                
                if (Boolean.TRUE.equals(isAvailable)) {
                    BikeVO vo = new BikeVO();
                    vo.setBikeId(Long.parseLong(bikeId));
                    vo.setDistance(result.getDistance().getValue());
                    vo.setLongitude(result.getContent().getPoint().getX());
                    vo.setLatitude(result.getContent().getPoint().getY());
                    bikes.add(vo);
                }
            }
        }
        
        return bikes;
    }
    
    /**
     * 解锁单车
     */
    public boolean unlockBike(Long bikeId, Long userId) {
        // 从可用集合移除
        Long removed = redisTemplate.opsForSet().remove(AVAILABLE_BIKE_SET, bikeId.toString());
        
        if (removed != null && removed > 0) {
            // 更新状态
            String bikeKey = "bike:" + bikeId;
            redisTemplate.opsForHash().put(bikeKey, "status", "in_use");
            redisTemplate.opsForHash().put(bikeKey, "userId", userId.toString());
            redisTemplate.opsForHash().put(bikeKey, "unlockTime", 
                String.valueOf(System.currentTimeMillis()));
            return true;
        }
        
        return false;
    }
    
    /**
     * 还车
     */
    public void returnBike(Long bikeId, double longitude, double latitude) {
        // 更新位置
        redisTemplate.opsForGeo().add(BIKE_GEO_KEY, 
            new Point(longitude, latitude), bikeId.toString());
        
        // 添加到可用集合
        redisTemplate.opsForSet().add(AVAILABLE_BIKE_SET, bikeId.toString());
        
        // 更新状态
        String bikeKey = "bike:" + bikeId;
        redisTemplate.opsForHash().put(bikeKey, "status", "available");
        redisTemplate.opsForHash().put(bikeKey, "longitude", String.valueOf(longitude));
        redisTemplate.opsForHash().put(bikeKey, "latitude", String.valueOf(latitude));
    }
}
```

## 总结

Redis GEO 是 LBS 场景的利器：

1. **GEOADD 添加位置**：经纬度 + 成员 ID
2. **GEORADIUS 查询**：圆形范围内的成员
3. **GEODIST 计算距离**：两点间距离
4. **支持排序**：按距离升序/降序
5. **底层是 ZSet**：可以复用 ZSet 的操作

基于 GEO 类型，可以轻松实现附近的人、商户查找、单车定位、地理围栏等功能。
