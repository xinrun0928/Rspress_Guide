# 签到与连续签到统计

每日签到、连续签到奖励、签到排行榜……这些功能在积分体系、用户活跃度场景中非常常见。用 Redis 的 BitMap 可以高效实现签到系统。

## BitMap 原理

BitMap 不是独立的数据类型，而是 String 类型的扩展。它通过位操作来实现高效的存储。

```
传统方式：
SET user:sign:2024-03:user123 "2024-03-15"

BitMap 方式：
SETBIT user:sign:2024-03:user123 14 1

用户 123 在 2024 年 3 月 15 日签到（3 月的第 15 天，偏移量 14）

存储对比：
- 传统方式：约 20-30 字节
- BitMap：1 位（0.125 字节）
```

## 基本操作

```bash
# 用户 123 在 2024-03-15 签到（偏移量 14）
SETBIT user:sign:2024-03:123 14 1

# 检查是否签到
GETBIT user:sign:2024-03:123 14
# (integer) 1

# 统计签到天数
BITCOUNT user:sign:2024-03:123
```

## 签到服务实现

### 基础签到

```java
@Service
public class SignService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    /**
     * 用户签到
     */
    public boolean sign(Long userId, LocalDate date) {
        // key 格式：user:sign:YYYY-MM:userId
        String key = getSignKey(userId, date);
        
        // 计算偏移量（日期 - 月初 + 1 - 1 = 日期 - 1）
        int offset = date.getDayOfMonth() - 1;
        
        // 设置签到位
        return Boolean.TRUE.equals(
            redisTemplate.opsForValue().setBit(key, offset, true)
        );
    }
    
    /**
     * 检查是否签到
     */
    public boolean isSigned(Long userId, LocalDate date) {
        String key = getSignKey(userId, date);
        int offset = date.getDayOfMonth() - 1;
        
        return Boolean.TRUE.equals(
            redisTemplate.opsForValue().getBit(key, offset)
        );
    }
    
    /**
     * 获取当月签到天数
     */
    public long getSignCount(Long userId, LocalDate date) {
        String key = getSignKey(userId, date);
        
        // 获取签到统计
        Long count = redisTemplate.opsForBitField(key, 
            BitFieldSubCommands.create().get(BitFieldSubCommands.BitFieldType.unsigned(31))
                .valueAt(0));
        
        // 统计 1 的个数
        return count != null ? bitCount(count) : 0;
    }
    
    /**
     * 获取当月签到详情
     */
    public List<Boolean> getSignDetails(Long userId, LocalDate date) {
        String key = getSignKey(userId, date);
        
        // 获取该月的签到情况
        List<Long> bitField = redisTemplate.opsForBitField(key,
            BitFieldSubCommands.create().get(BitFieldSubCommands.BitFieldType.unsigned(31))
                .valueAt(0));
        
        List<Boolean> results = new ArrayList<>();
        if (bitField != null && !bitField.isEmpty()) {
            long bits = bitField.get(0);
            for (int i = 0; i < 31; i++) {
                results.add((bits & (1L << i)) != 0);
            }
        }
        
        return results;
    }
    
    private String getSignKey(Long userId, LocalDate date) {
        return String.format("user:sign:%s:%d", 
            date.format(DateTimeFormatter.ofPattern("yyyy-MM")), userId);
    }
    
    private long bitCount(long num) {
        long count = 0;
        while (num > 0) {
            num &= (num - 1);
            count++;
        }
        return count;
    }
}
```

### 连续签到统计

```java
@Service
public class ContinuousSignService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    /**
     * 获取连续签到天数
     */
    public int getContinuousSignDays(Long userId, LocalDate date) {
        int continuousDays = 0;
        LocalDate checkDate = date;
        
        while (true) {
            String key = getSignKey(userId, checkDate);
            int offset = checkDate.getDayOfMonth() - 1;
            
            Boolean signed = redisTemplate.opsForValue().getBit(key, offset);
            
            if (Boolean.TRUE.equals(signed)) {
                continuousDays++;
                checkDate = checkDate.minusDays(1);
                
                // 如果跨月了，继续检查上个月
                if (checkDate.getMonth() != date.getMonth()) {
                    continue;
                }
            } else {
                break;
            }
            
            // 最多检查 31 天
            if (continuousDays >= 31) {
                break;
            }
        }
        
        return continuousDays;
    }
    
    /**
     * 获取本月最长连续签到天数
     */
    public int getMaxContinuousDays(Long userId, LocalDate date) {
        List<Boolean> signDetails = getSignDetails(userId, date);
        
        int maxContinuous = 0;
        int currentContinuous = 0;
        
        for (Boolean signed : signDetails) {
            if (signed) {
                currentContinuous++;
                maxContinuous = Math.max(maxContinuous, currentContinuous);
            } else {
                currentContinuous = 0;
            }
        }
        
        return maxContinuous;
    }
    
    private String getSignKey(Long userId, LocalDate date) {
        return String.format("user:sign:%s:%d", 
            date.format(DateTimeFormatter.ofPattern("yyyy-MM")), userId);
    }
    
    private List<Boolean> getSignDetails(Long userId, LocalDate date) {
        String key = getSignKey(userId, date);
        
        List<Long> bitField = redisTemplate.opsForBitField(key,
            BitFieldSubCommands.create().get(BitFieldSubCommands.BitFieldType.unsigned(31))
                .valueAt(0));
        
        List<Boolean> results = new ArrayList<>();
        if (bitField != null && !bitField.isEmpty()) {
            long bits = bitField.get(0);
            for (int i = 0; i < 31; i++) {
                results.add((bits & (1L << i)) != 0);
            }
        }
        
        return results;
    }
}
```

### 签到奖励

```java
@Service
public class SignRewardService {
    
    @Autowired
    private SignService signService;
    
    @Autowired
    private ContinuousSignService continuousSignService;
    
    /**
     * 签到并发放奖励
     */
    public SignResult signAndReward(Long userId, LocalDate date) {
        SignResult result = new SignResult();
        
        // 检查是否已签到
        if (signService.isSigned(userId, date)) {
            result.setSuccess(false);
            result.setMessage("今日已签到");
            return result;
        }
        
        // 执行签到
        boolean signed = signService.sign(userId, date);
        if (!signed) {
            result.setSuccess(false);
            result.setMessage("签到失败");
            return result;
        }
        
        result.setSuccess(true);
        result.setMessage("签到成功");
        
        // 计算奖励
        int continuousDays = continuousSignService.getContinuousSignDays(userId, date);
        int reward = calculateReward(continuousDays);
        
        result.setContinuousDays(continuousDays);
        result.setRewardPoints(reward);
        
        // 发放奖励
        // rewardService.givePoints(userId, reward);
        
        // 更新连续签到排行榜
        String rankKey = "rank:continuous:" + date.format(DateTimeFormatter.ofPattern("yyyy-MM"));
        redisTemplate.opsForZSet().add(rankKey, userId.toString(), continuousDays);
        
        return result;
    }
    
    /**
     * 计算奖励
     */
    private int calculateReward(int continuousDays) {
        int base = 10;  // 基础奖励
        int extra = 0;
        
        // 连续签到额外奖励
        if (continuousDays >= 7) {
            extra = 7;
        } else if (continuousDays >= 30) {
            extra = 30;
        }
        
        return base + extra;
    }
}
```

## 签到排行榜

```java
@Service
public class SignRankingService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    /**
     * 获取当月连续签到排行榜
     */
    public List<SignRankingVO> getMonthlyRanking(LocalDate date, int limit) {
        String rankKey = "rank:continuous:" + date.format(DateTimeFormatter.ofPattern("yyyy-MM"));
        
        // 获取 Top N
        Set<ZSetOperations.TypedTuple<String>> topSet = 
            redisTemplate.opsForZSet().reverseRangeWithScores(rankKey, 0, limit - 1);
        
        List<SignRankingVO> ranking = new ArrayList<>();
        int rank = 1;
        for (ZSetOperations.TypedTuple<String> tuple : topSet) {
            SignRankingVO vo = new SignRankingVO();
            vo.setRank(rank++);
            vo.setUserId(Long.parseLong(tuple.getValue()));
            vo.setContinuousDays(tuple.getScore().intValue());
            ranking.add(vo);
        }
        
        return ranking;
    }
    
    /**
     * 获取用户在排行榜的排名
     */
    public Integer getUserRank(Long userId, LocalDate date) {
        String rankKey = "rank:continuous:" + date.format(DateTimeFormatter.ofPattern("yyyy-MM"));
        
        Long rank = redisTemplate.opsForZSet().reverseRank(rankKey, userId.toString());
        
        return rank != null ? rank.intValue() + 1 : null;
    }
    
    /**
     * 获取用户的连续签到天数
     */
    public int getUserContinuousDays(Long userId, LocalDate date) {
        String rankKey = "rank:continuous:" + date.format(DateTimeFormatter.ofPattern("yyyy-MM"));
        
        Double score = redisTemplate.opsForZSet().score(rankKey, userId.toString());
        
        return score != null ? score.intValue() : 0;
    }
}
```

## 签到 API

```java
@RestController
@RequestMapping("/api/sign")
public class SignController {
    
    @Autowired
    private SignService signService;
    
    @Autowired
    private SignRewardService signRewardService;
    
    @Autowired
    private SignRankingService rankingService;
    
    /**
     * 签到
     */
    @PostMapping("/do")
    public Result<SignResult> sign() {
        Long userId = getCurrentUserId();
        LocalDate today = LocalDate.now();
        
        SignResult result = signRewardService.signAndReward(userId, today);
        
        return Result.success(result);
    }
    
    /**
     * 获取今日签到状态
     */
    @GetMapping("/today")
    public Result<Boolean> getTodaySignStatus() {
        Long userId = getCurrentUserId();
        boolean signed = signService.isSigned(userId, LocalDate.now());
        return Result.success(signed);
    }
    
    /**
     * 获取当月签到详情
     */
    @GetMapping("/monthly")
    public Result<Map<String, Object>> getMonthlySignDetails(LocalDate date) {
        Long userId = getCurrentUserId();
        
        List<Boolean> signDetails = signService.getSignDetails(userId, date);
        long signCount = signService.getSignCount(userId, date);
        int continuousDays = signService.getContinuousSignDays(userId, date);
        
        Map<String, Object> result = new HashMap<>();
        result.put("signDetails", signDetails);
        result.put("signCount", signCount);
        result.put("continuousDays", continuousDays);
        
        return Result.success(result);
    }
    
    /**
     * 获取签到排行榜
     */
    @GetMapping("/ranking")
    public Result<List<SignRankingVO>> getRanking(
            @RequestParam(required = false) LocalDate date,
            @RequestParam(defaultValue = "10") int limit) {
        
        if (date == null) {
            date = LocalDate.now();
        }
        
        List<SignRankingVO> ranking = rankingService.getMonthlyRanking(date, limit);
        
        return Result.success(ranking);
    }
    
    private Long getCurrentUserId() {
        // 从 Session 或 Token 获取当前用户 ID
        return 123L;  // 示例
    }
}
```

## 用户在线统计

BitMap 还可以用于统计用户在线状态：

```java
@Service
public class OnlineStatService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    /**
     * 记录用户上线
     */
    public void recordOnline(Long userId) {
        String key = getOnlineKey(LocalDate.now());
        int offset = (int) (userId % 100000);  // 用户 ID 取模，避免偏移量过大
        
        redisTemplate.opsForValue().setBit(key, offset, true);
    }
    
    /**
     * 记录用户下线
     */
    public void recordOffline(Long userId) {
        String key = getOnlineKey(LocalDate.now());
        int offset = (int) (userId % 100000);
        
        redisTemplate.opsForValue().setBit(key, offset, false);
    }
    
    /**
     * 统计当日在线人数
     */
    public long countOnlineUsers(LocalDate date) {
        String key = getOnlineKey(date);
        
        List<Long> bitField = redisTemplate.opsForBitField(key,
            BitFieldSubCommands.create().get(BitFieldSubCommands.BitFieldType.unsigned(100000))
                .valueAt(0));
        
        if (bitField != null && !bitField.isEmpty()) {
            return bitCount(bitField.get(0));
        }
        
        return 0;
    }
    
    private String getOnlineKey(LocalDate date) {
        return "online:users:" + date.format(DateTimeFormatter.ofPattern("yyyy-MM-dd"));
    }
    
    private long bitCount(long num) {
        long count = 0;
        while (num > 0) {
            num &= (num - 1);
            count++;
        }
        return count;
    }
}
```

## 总结

BitMap 是签到系统的理想选择：

1. **极省空间**：一个月签到只需 31 位（约 4 字节）
2. **高效操作**：SETBIT/GETBIT 都是 O(1)
3. **支持统计**：BITCOUNT 高效统计
4. **适合连续性**：结合连续天数判断
5. **可扩展**：用于在线统计、用户画像等

用 BitMap 实现签到，既能节省存储，又能高效查询，是面试和实战中的经典应用。
