# PostgreSQL 基础

这一章是 PostgreSQL 的入门指南，涵盖了 PostgreSQL 的历史演进、核心特性，以及与 MySQL 的关键差异。通过这一章的学习，你将理解为什么 PostgreSQL 被誉为"地表最强开源关系型数据库"。

## 内容导航

- **[PostgreSQL 概述与核心特性](overview)**：从 Ingres 到 PostgreSQL 的演进历程，MVCC、WAL、扩展性等核心特性
- **[PostgreSQL 与 MySQL 核心差异对比](mysql-compare)**：两种数据库在 MVCC 实现、索引结构、事务模型、类型系统等方面的深度对比
- **[PostgreSQL 架构](architecture)**：PostgreSQL 的整体架构，连接管理器、查询编译器、执行器、存储管理器、WAL
- **[常用数据类型与高级类型](data-type)**：数值、字符串、日期、数组、JSONB 等类型的使用
- **[分页查询](pagination)**：OFFSET 分页与游标分页的优劣对比

## 学习路径建议

对于零基础的同学，建议按以下顺序学习：

1. 先看 [概述与核心特性](overview)，了解 PostgreSQL 是什么、解决了什么问题
2. 再看 [架构](architecture)，理解 PostgreSQL 的整体设计
3. 然后学习 [数据类型](data-type)，掌握 PostgreSQL 的类型系统
4. 最后看 [MySQL 对比](mysql-compare)，理解两者的差异和选型依据

对于有 MySQL 经验的开发者，可以直接看 [MySQL 对比](mysql-compare)，快速定位差异点。
