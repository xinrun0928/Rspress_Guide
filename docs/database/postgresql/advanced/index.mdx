# PostgreSQL 高级特性

这一章介绍 PostgreSQL 的高级特性，包括触发器、存储过程、窗口函数、CTE 等。这些特性让 PostgreSQL 不仅仅是一个存储数据的容器，更是一个强大的计算平台。

## 内容导航

- **[触发器](trigger)**：自动执行的数据库逻辑
- **[存储过程 PL/pgSQL](plpgsql)**：PostgreSQL 的编程语言
- **[函数重载](overload)**：同名函数的不同实现
- **[窗口函数](window-function)**：SQL 的高级分析功能
- **[CTE 与递归查询](cte)**：复杂查询的可读性
- **[JSON/JSONB](json)**：半结构化数据处理
- **[PostGIS](postgis)**：空间数据库

## 学习路径建议

建议按以下顺序学习：

1. 先看 [窗口函数](window-function)，这是数据分析的核心工具
2. 再看 [CTE](cte)，掌握复杂查询的组织方式
3. 学习 [触发器](trigger) 和 [存储过程](plpgsql)
4. 最后看 [JSON/JSONB](json) 和 [PostGIS](postgis)
