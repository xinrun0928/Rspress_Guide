# PostgreSQL PostGIS 空间数据库

PostGIS 是 PostgreSQL 的空间数据库扩展，提供了对地理空间数据的存储、查询和分析能力。它是开源领域最强大的地理信息系统之一。

## PostGIS 基础

### 安装

```sql
-- 安装扩展
CREATE EXTENSION IF NOT EXISTS postgis;

-- 验证安装
SELECT PostGIS_Version();
```

### 常用几何类型

| 类型 | 说明 |
|---|---|
| `POINT` | 点 |
| `LINESTRING` | 线 |
| `POLYGON` | 多边形 |
| `MULTIPOINT` | 点集合 |
| `MULTILINESTRING` | 线集合 |
| `MULTIPOLYGON` | 多边形集合 |
| `GEOMETRYCOLLECTION` | 几何集合 |

### 创建空间表

```sql
-- 创建空间表
CREATE TABLE locations (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    geom GEOMETRY(Point, 4326)
);

-- 添加空间索引
CREATE INDEX idx_locations_geom ON locations USING GIST (geom);
```

## 几何构造

```sql
-- 构造点
SELECT ST_MakePoint(116.4, 39.9);
SELECT ST_GeomFromText('POINT(116.4 39.9)', 4326);

-- 构造线
SELECT ST_GeomFromText('LINESTRING(116.3 39.9, 116.5 40.0)', 4326);

-- 构造多边形
SELECT ST_GeomFromText('POLYGON((116.3 39.9, 116.5 39.9, 116.5 40.1, 116.3 40.1, 116.3 39.9))', 4326);
```

## 常用函数

### 距离计算

```sql
-- 计算两点间距离（米）
SELECT ST_Distance(
    ST_MakePoint(116.4, 39.9)::GEOGRAPHY,
    ST_MakePoint(116.5, 40.0)::GEOGRAPHY
);
```

### 空间关系

```sql
-- 点是否在多边形内
SELECT ST_Within(
    ST_MakePoint(116.4, 39.9),
    ST_GeomFromText('POLYGON((116.3 39.8, 116.5 39.8, 116.5 40.0, 116.3 40.0, 116.3 39.8))', 4326)
);

-- 两几何是否相交
SELECT ST_Intersects(
    ST_GeomFromText('LINESTRING(116.3 39.9, 116.5 40.0)', 4326),
    ST_GeomFromText('LINESTRING(116.4 39.8, 116.4 40.1)', 4326)
);
```

### 缓冲区分析

```sql
-- 创建点周围 1 公里的缓冲区
SELECT ST_Buffer(
    ST_MakePoint(116.4, 39.9)::GEOGRAPHY,
    1000
)::GEOMETRY;
```

## 空间查询

### 最近邻查询

```sql
-- 查找最近的 10 个地点
SELECT
    id,
    name,
    ST_Distance(
        geom,
        ST_MakePoint(116.4, 39.9)::GEOGRAPHY
    ) AS distance
FROM locations
ORDER BY geom <-> ST_MakePoint(116.4, 39.9)
LIMIT 10;
```

### 范围查询

```sql
-- 查询矩形范围内的地点
SELECT *
FROM locations
WHERE geom && ST_MakeEnvelope(116.3, 39.8, 116.5, 40.0, 4326);

-- 查询圆形范围内的地点
SELECT *
FROM locations
WHERE ST_DWithin(
    geom,
    ST_MakePoint(116.4, 39.9)::GEOGRAPHY,
    5000  -- 5 公里
);
```

## 地理计算

### 面积计算

```sql
-- 计算多边形面积（平方米）
SELECT ST_Area(
    ST_GeomFromText('POLYGON((116.3 39.8, 116.5 39.8, 116.5 40.0, 116.3 40.0, 116.3 39.8))', 4326)::GEOGRAPHY
);
```

### 长度计算

```sql
-- 计算线段长度（米）
SELECT ST_Length(
    ST_GeomFromText('LINESTRING(116.3 39.9, 116.5 40.0)', 4326)::GEOGRAPHY
);
```

---

## 总结

PostGIS 是 PostgreSQL 的空间数据库扩展：

1. **几何类型**：POINT、LINESTRING、POLYGON 等
2. **空间索引**：使用 GIST 索引加速空间查询
3. **空间函数**：距离、交集、包含等
4. **地理计算**：面积、长度、最近邻

PostGIS 让 PostgreSQL 成为地表最强的开源空间数据库，广泛应用于 GIS、物流、位置服务等领域。
