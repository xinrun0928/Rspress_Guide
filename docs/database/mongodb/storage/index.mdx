## 核心概念

MongoDB 的存储引擎负责数据的持久化存储，WiredTiger 是默认的存储引擎。

| 组件 | 说明 |
|---|---|
| WiredTiger | 默认存储引擎，支持文档级锁 |
| B+ 树索引 | 默认索引结构 |
| WT Cache | WiredTiger 缓存，默认占 RAM 的 50% |
| Journal | WAL 日志，保证持久性 |
