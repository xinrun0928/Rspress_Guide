# TiKV 存储引擎总览

TiKV 是 TiDB 的分布式事务型 KV 存储引擎，承载着 TiDB 的数据存储重任。

理解 TiKV 的原理，是理解 TiDB 高性能、高可用、分布式特性的关键。

---

## TiKV 概述

### TiKV 的定位

TiKV 是 TiDB 的存储层，负责：

- 数据的持久化存储
- 分布式事务支持
- 数据复制与一致性保证
- 水平扩展能力

### 与 TiDB 的关系

```
TiDB 架构：

┌─────────────────────────────────────────────────────────────┐
│                      TiDB Server                           │
│            (SQL 层：解析、优化、执行)                        │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       PD Cluster                            │
│              (调度器：元数据管理、调度)                      │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       TiKV Cluster                          │
│         (存储层：分布式 KV 存储、Raft 复制)                  │
└─────────────────────────────────────────────────────────────┘
```

---

## 核心特性

| 特性 | 说明 |
|---|---|
| 分布式架构 | 数据自动分片和调度 |
| 强一致性 | Raft 共识协议保证 |
| 高可用 | 多副本自动故障转移 |
| 水平扩展 | 在线扩容，不影响服务 |
| 事务支持 | 分布式 ACID 事务 |
| MVCC | 多版本并发控制 |
