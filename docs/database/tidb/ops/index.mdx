# TiDB 运维概述

TiDB 的运维涉及集群管理、性能监控、故障排查等多个方面。本章节涵盖：

- PD 调度组件
- 扩容与缩容
- Binlog 同步与 DM
- 备份恢复
- 性能监控
- 故障排查

---

## 运维架构

```
TiDB 运维架构：

┌─────────────────────────────────────────────────────────────┐
│                      监控平台                               │
│              (Prometheus + Grafana)                        │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      TiDB Server                           │
│              (SQL 层，状态上报)                             │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       PD Cluster                            │
│              (调度器，集群管理)                             │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       TiKV Cluster                          │
│              (存储层，数据存储)                              │
└─────────────────────────────────────────────────────────────┘
```
