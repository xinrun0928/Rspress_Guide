# TiSpark

TiSpark 是 TiDB 的 Spark 连接器，允许使用 Spark 执行复杂的数据分析和机器学习任务。

## TiSpark 概述

### 什么是 TiSpark

TiSpark 将 Spark 与 TiDB/TiKV 集成，提供了 Spark SQL 和 MLlib 的能力。

```

┌─────────────────────────────────────────────────────┐

│   └────┬─────┘ └────┬─────┘ └────┬─────┘          │
│        └─────────────┴─────────────┘                │
│                      │                               │
│        ┌─────────────┴─────────────┐                │
│        ▼                           ▼                │

│   │            TiSpark                              │  │
│   │  - Spark SQL 与 TiDB 集成                        │  │
```

## 基本使用

### Spark SQL

```scala
import org.apache.spark.sql.SparkSession

val spark = SparkSession.builder()
  .appName("TiSpark Demo")
  .config("spark.tispark.pd.addresses", "192.168.1.1:2379")
  .config("spark.sql.extensions", "org.apache.spark.sql.extensions.TiExtensions")
  .getOrCreate()

// 读取 TiDB 表
val orders = spark.sql("SELECT * FROM orders WHERE created_at >= '2024-01-01'")

// 查看 Schema
orders.printSchema()

// 执行聚合
val result = orders
  .groupBy("customer_id")
  .agg(
    count("*").as("order_count"),
    sum("amount").as("total_amount")
  )

// 显示结果
result.show()

// 保存结果
result.write
  .format("tidb")
  .option("tidb.addr", "192.168.1.1")
  .option("tidb.port", "4000")
  .option("tidb.user", "root")
  .option("tidb.password", "password")
  .option("database", "analytics")
  .option("table", "customer_summary")
  .mode("overwrite")
  .save()
```

### DataFrame API

```scala
import org.apache.spark.sql.functions._

// 读取表
val orders = spark.read
  .format("tidb")
  .option("tidb.addr", "192.168.1.1")
  .option("tidb.port", "4000")
  .option("database", "shop")
  .option("table", "orders")
  .load()

// DataFrame 转换
val customerStats = orders
  .filter(col("created_at") >= lit("2024-01-01"))
  .groupBy("customer_id")
  .agg(
    count("*").as("order_count"),
    sum("amount").as("total_amount"),
    avg("amount").as("avg_amount"),
    min("created_at").as("first_order"),
    max("created_at").as("last_order")
  )

// 添加派生字段
val withRetention = customerStats
  .withColumn("days_active", datediff(lit("2024-12-31"), col("first_order")))
  .withColumn("order_frequency", col("order_count") / col("days_active") * 30)

// 保存到 TiDB
withRetention.write
  .mode("overwrite")
  .saveAsTable("customer_analysis")
```

## 复杂分析

### 多表关联

```scala
// 关联分析
val analysis = spark.sql("""
  SELECT
    o.customer_id,
    c.customer_name,
    COUNT(DISTINCT o.id) as order_count,
    SUM(o.amount) as total_amount,
    SUM(oi.quantity) as total_items,
    COUNT(DISTINCT p.category_id) as category_count
  FROM orders o
  JOIN customers c ON o.customer_id = c.id
  JOIN order_items oi ON o.id = oi.order_id
  JOIN products p ON oi.product_id = p.id
  WHERE o.created_at >= '2024-01-01'
  GROUP BY o.customer_id, c.customer_name
""")

// 进一步分析
val topCustomers = analysis
  .filter(col("order_count") > 10)
  .orderBy(col("total_amount").desc)

topCustomers.show()
```

### 窗口函数

```scala
// 使用窗口函数
val withRank = spark.sql("""
  SELECT
    customer_id,
    order_month,
    order_count,
    total_amount,
    RANK() OVER (PARTITION BY order_month ORDER BY total_amount DESC) as monthly_rank,
    SUM(total_amount) OVER (
      PARTITION BY customer_id
      ORDER BY order_month
      ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW
    ) as cumulative_amount
  FROM monthly_stats
""")

withRank.show()
```

### 用户行为分析

```scala
import spark.implicits._

// 用户行为序列分析
val userEvents = spark.read
  .format("tidb")
  .option("table", "user_events")
  .load()

// 计算用户路径
val userPaths = userEvents
  .groupBy("user_id")
  .agg(collect_list(col("event")).as("event_sequence"))
  .withColumn("path", concat_ws(" -> ", col("event_sequence")))

// 统计路径分布
val pathStats = userPaths
  .groupBy("path")
  .agg(count("*").as("user_count"))
  .orderBy(col("user_count").desc)

pathStats.show(20)
```

## 机器学习

### MLlib 集成

```scala
import org.apache.spark.ml.clustering.KMeans
import org.apache.spark.ml.feature.VectorAssembler

// 用户聚类分析
val customerFeatures = spark.sql("""
  SELECT
    customer_id,
    order_count,
    total_amount,
    avg_amount,
    category_diversity,
    days_since_first_order,
    days_since_last_order
  FROM customer_features
""")

// 特征向量化
val assembler = new VectorAssembler()
  .setInputCols(Array("order_count", "total_amount", "avg_amount",
                      "category_diversity", "days_since_first_order", "days_since_last_order"))
  .setOutputCol("features")

val featureData = assembler.transform(customerFeatures)

// K-Means 聚类
val kmeans = new KMeans()
  .setK(5)
  .setSeed(42)
  .setFeaturesCol("features")
  .setPredictionCol("cluster")

val model = kmeans.fit(featureData)

// 保存模型结果
val predictions = model.transform(featureData)
predictions.select("customer_id", "cluster").write
  .format("tidb")
  .option("table", "customer_clusters")
  .save()
```

### 协同过滤

```scala
import org.apache.spark.ml.recommendation.ALS

// 商品推荐
val ratings = spark.sql("""
  SELECT
    customer_id,
    product_id,
    rating
  FROM user_product_ratings
""")

// 训练推荐模型
val als = new ALS()
  .setMaxIter(10)
  .setRegParam(0.01)
  .setUserCol("customer_id")
  .setItemCol("product_id")
  .setRatingCol("rating")

val model = als.fit(ratings)

// 生成推荐
val userRecs = model.recommendForAllUsers(10)
userRecs.write
  .format("tidb")
  .option("table", "product_recommendations")
  .save()
```

## 性能优化

## 总结

TiSpark 要点：

- **Spark 集成**：使用 Spark SQL 和 DataFrame API
- **复杂分析**：多表关联、窗口函数、用户行为分析
- **机器学习**：MLlib 集成，聚类、推荐等
- **性能优化**：Spark 配置和 TiSpark 特定优化
- **使用场景**：大数据分析、机器学习、数据湖分析
