# TiDB HTAP 概述

TiDB 的 HTAP（Hybrid Transactional/Analytical Processing）能力由 TiDB 和 TiFlash 共同提供。

---

## HTAP 概述

### 什么是 HTAP

HTAP 是同时支持事务处理（OLTP）和分析处理（OLAP）的数据库架构：

| 场景 | 说明 | 组件 |
|---|---|---|
| OLTP | 事务处理，高并发写入 | TiDB |
| OLAP | 分析处理，复杂查询 | TiFlash |

### TiDB HTAP 架构

```
TiDB HTAP 架构：

┌─────────────────────────────────────────────────────────────┐
│                      TiDB Server                           │
│            (统一入口，自动路由)                                │
└─────────────────────────────────────────────────────────────┘
                              │
              ┌───────────────┼───────────────┐
              ▼               ▼               ▼
      ┌───────────┐   ┌───────────┐   ┌───────────┐
      │  TiKV     │   │  TiFlash  │   │  TiFlash  │
      │ (行存)    │   │ (列存)    │   │ (列存)    │
      └───────────┘   └───────────┘   └───────────┘
```
