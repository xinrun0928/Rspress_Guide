# Docker 进阶

本模块面向有一定 Docker 基础的开发者，深入探讨容器运行机制、网络通信原理、存储驱动对比与选型、资源限制机制、安全加固、日志与健康检查、性能调优，以及构建缓存优化。

## 模块内容

**深入原理** — 理解容器运行机制

| 章节 | 内容 |
|---|---|
| 容器间通信原理 | veth pair、docker0 网桥、iptables 规则、NAT 与端口转发 |
| 存储驱动原理 | overlay2、devicemapper、btrfs、zfs、vfs 的对比与选型 |
| 资源限制机制 | cgroups 原理、CPU/内存/IO 限制、PID 限制 |
| 容器安全 | 用户命名空间映射、seccomp、Linux capabilities、AppArmor |

**运维实践** — 生产环境必备

| 章节 | 内容 |
|---|---|
| 日志处理 | 日志驱动配置、ELK 集成、日志轮转 |
| 健康检查 | HEALTHCHECK 指令、健康状态监控、编排集成 |
| 性能优化 | 镜像优化、层缓存、并行容器、资源调优 |
| 构建缓存 | 缓存失效机制、BuildKit 缓存挂载、Registry 缓存 |

## 学习建议

进阶内容偏原理性和生产实践，建议：

1. 学习每个章节前，先动手操作对应的命令验证效果
2. 理解「为什么这样做」比「怎么做」更重要
3. 遇到问题使用 `docker inspect` 和 `docker stats` 获取详细信息
4. 参考官方文档确认具体参数和行为

学完这一模块，你将能够深入理解 Docker 的运行机制，排查生产环境问题，并进行针对性优化。
