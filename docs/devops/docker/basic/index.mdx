# Docker 基础

本模块面向 Docker 初学者，从容器化的基本概念讲起，逐步深入到镜像构建、数据管理、网络配置和私有仓库搭建。通过这一模块的学习，你将掌握日常开发中必备的容器化技能，能够独立完成从编写 Dockerfile 到部署容器化应用的完整流程。

## 模块内容

这一模块包含以下核心章节：

**概念入门** — 理解容器化的本质

| 章节 | 内容 |
|---|---|
| 虚拟机与容器的区别 | 从架构层面理解两种虚拟化方案的差异，明确各自适用场景 |
| Docker 核心概念 | 镜像、容器、仓库的的定义与相互关系，UnionFS 文件系统简介 |

**镜像构建** — 掌握 Dockerfile 语法

| 章节 | 内容 |
|---|---|
| Dockerfile 入门 | 常用指令详解：FROM、RUN、COPY、CMD、ENTRYPOINT 等 |
| 镜像分层原理 | UnionFS 与 OverlayFS 的工作方式，Copy-on-Write 机制 |
| Dockerfile 最佳实践 | 多阶段构建、镜像优化、安全加固、BuildKit 新特性 |
| .dockerignore 文件 | 排除构建上下文中不需要的文件，减小镜像体积 |

**数据与网络** — 容器持久化与通信

| 章节 | 内容 |
|---|---|
| 数据卷 | Volume 的三种类型及其适用场景，tmpfs 内存文件系统 |
| Docker 网络 | 五种网络驱动详解，容器间通信与端口映射原理 |

**编排与仓库** — 多人协作必备

| 章节 | 内容 |
|---|---|
| Docker Compose | 多容器应用的 YAML 定义，up/down/logs 命令速查 |
| Docker Registry | 私有镜像仓库部署，镜像推送、拉取与维护 |
| Harbor 私有仓库 | 企业级镜像仓库，镜像扫描、签名与高可用配置 |

## 学习建议

在学习过程中，建议你边学边练：

1. 动手写 Dockerfile，尝试将一个简单的 Web 应用容器化
2. 使用 Docker Compose 定义一个包含数据库和后端服务的完整应用栈
3. 搭建一个私有 Registry，体验镜像推送和拉取的完整流程
4. 在学习网络章节时，使用 `docker network inspect` 命令查看容器网络详情

当你能够独立完成这些实践操作，就说明 Docker 基础已经扎实掌握了。
