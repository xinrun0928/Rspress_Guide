# 容器编排对比

这一章节探讨容器编排的不同方案和工具。

## 学习路径

```
容器编排对比
│
├── 主流方案对比        ← Docker Swarm、Kubernetes、Mesos
├── K8s vs Compose    ← 何时选择哪种方案
├── K3s 轻量 K8s      ← 资源受限环境的选择
└── k0s / MicroK8s   ← 其他轻量级 Kubernetes
```

## 你将学到什么

1. **理解主流编排方案**：Kubernetes、Docker Swarm、Mesos 的差异
2. **做出合适选择**：根据场景选择合适的编排工具
3. **掌握轻量方案**：K3s、k0s、MicroK8s 的使用
4. **了解迁移路径**：Compose 到 K8s 的过渡策略

:::tip 学习建议
如果你已经有 Kubernetes 基础，可以直接跳到轻量级 Kubernetes 部分。如果你是容器新手，建议从 Docker Swarm 开始理解。
:::
