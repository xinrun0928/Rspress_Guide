# KubeSphere 核心功能

这一章节深入讲解 KubeSphere 的核心功能模块。

## 学习路径

```
KubeSphere 核心功能
│
├── 应用商店        ← 基于 OpenPitrix 的应用生命周期管理
├── CI/CD 流水线    ← Jenkins 集成的可视化流水线
├── 告警管理        ← 多租户告警和通知
├── 日志系统        ← 统一的日志收集和查询
├── 审计日志        ← 操作审计和合规
├── 服务网格        ← Istio 集成的微服务治理
└── 可观测性        ← 监控指标和仪表盘
```

## 你将学到什么

通过这一章节的学习，你将能够：

1. **掌握应用商店**：发布和管理 Helm 应用，实现一键部署
2. **熟练使用 CI/CD**：创建和管理 Jenkins 流水线，自动化构建部署
3. **配置告警规则**：设置多维度告警，及时发现系统问题
4. **使用日志系统**：跨租户日志查询，快速定位问题
5. **理解服务网格**：配置流量管理，实现灰度发布
6. **使用监控能力**：自定义监控仪表盘，掌握集群状态

:::tip 学习建议
核心功能涉及较多实战操作，建议在已有 KubeSphere 环境中边学边练。如果还没有安装环境，可以先通读理解，在脑海中构建知识框架。
:::
