# OpenFeign 集成 Spring Boot

Feign 最初是 Netflix 开发的独立库，后来被整合到 Spring Cloud 中，成为 Spring Cloud OpenFeign。通过与 Spring Boot 的深度集成，OpenFeign 变得更加易用。

## Spring Cloud OpenFeign 概述

Spring Cloud OpenFeign 提供了：

```yaml
# OpenFeign 核心特性
特性                  说明
─────────────────────────────────────────────────────────────────
自动配置              基于 Spring Boot 自动配置，开箱即用
Spring MVC 注解       支持 @GetMapping、@PostMapping 等
负载均衡集成          自动集成 Ribbon/Spring Cloud LoadBalancer
熔断降级集成          自动集成 Hystrix/Sentinel
日志增强              更详细的日志配置
请求压缩              支持 GZIP 压缩
重试机制              灵活的重试配置
```

## 快速开始

### 引入依赖

```xml
<!-- Spring Cloud OpenFeign -->
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>
```

### 启用 OpenFeign

```java
@SpringBootApplication
@EnableFeignClients
public class OrderApplication {
    public static void main(String[] args) {
        SpringApplication.run(OrderApplication.class, args);
    }
}
```

### 定义 Feign Client

```java
// 使用 Spring MVC 注解
@FeignClient(name = "user-service", path = "/api/users")
public interface UserClient {

    @GetMapping("/{id}")
    User getUser(@PathVariable Long id);

    @GetMapping
    Page<User> listUsers(
        @RequestParam int page,
        @RequestParam int size
    );

    @PostMapping
    User createUser(@RequestBody @Valid CreateUserRequest request);
}
```

### 使用 Client

```java
// 直接注入使用
@Service
public class OrderService {

    @Autowired
    private UserClient userClient;

    public Order createOrder(Long userId, List<Long> productIds) {
        // 调用远程服务
        User user = userClient.getUser(userId);

        // 创建订单
        return Order.builder()
                .userId(userId)
                .userName(user.getName())
                .build();
    }
}
```

## 配置详解

### application.yml 配置

```yaml
# OpenFeign 完整配置
spring:
  application:
    name: order-service

  cloud:
    # OpenFeign 配置
    openfeign:
      # 启用 HTTP 客户端
      client:
        config:
          # 全局默认配置
          default:
            connectTimeout: 3000
            readTimeout: 5000
            loggerLevel: basic
          # 指定服务配置
          user-service:
            connectTimeout: 5000
            readTimeout: 10000
      # 启用 Hystrix
      hystrix:
        enabled: true
      # 请求压缩
      compression:
        request:
          enabled: true
          mime-types: text/xml,application/xml,application/json
          min-request-size: 2048
        response:
          enabled: true
      # 日志
      logging:
        level:
          com.example.feign: DEBUG
```

### Java 配置类

```java
// 自定义配置类
@Configuration
public class FeignConfiguration {

    @Bean
    public Logger.Level feignLogger() {
        return Logger.Level.FULL;
    }

    @Bean
    public Retryer feignRetryer() {
        // 重试配置：间隔 100ms，间隔最大 1000ms，最多重试 3 次
        return new Retryer.Default(100, 1000, 3);
    }

    @Bean
    public Contract feignContract() {
        // 使用 Spring MVC 注解（默认已启用）
        return new SpringMvcContract();
    }

    @Bean
    public RequestInterceptor requestInterceptor() {
        return new RequestInterceptor() {
            @Override
            public void apply(RequestTemplate template) {
                // 添加公共请求头
                template.header("X-Request-Id", UUID.randomUUID().toString());
                template.header("X-Source", "order-service");
            }
        };
    }
}

// 在 Client 上应用配置
@FeignClient(
    name = "user-service",
    path = "/api/users",
    configuration = FeignConfiguration.class
)
public interface UserClient {
    // ...
}
```

## 复杂请求处理

### 文件上传

```java
// Feign Client 定义文件上传
@FeignClient(name = "storage-service")
public interface StorageClient {

    @PostMapping(value = "/files/upload", consumes = MediaType.MULTIPART_FORM_DATA_VALUE)
    FileUploadResponse uploadFile(@RequestPart("file") MultipartFile file);

    @PostMapping(value = "/files/batch-upload", consumes = MediaType.MULTIPART_FORM_DATA_VALUE)
    BatchFileUploadResponse batchUpload(@RequestPart("files") MultipartFile[] files);
}

// 服务实现
@Service
public class FileService {

    @Autowired
    private StorageClient storageClient;

    public String upload(MultipartFile file) {
        FileUploadResponse response = storageClient.uploadFile(file);
        return response.getFileId();
    }
}
```

## 拦截器详解

### 认证拦截器

```java
// 认证拦截器
@Component
public class AuthInterceptor implements RequestInterceptor {

    @Autowired
    private TokenService tokenService;

    @Override
    public void apply(RequestTemplate template) {
        // 获取当前请求的上下文
        SecurityContext context = SecurityContextHolder.getContext();
        Authentication auth = context.getAuthentication();

        if (auth != null && auth.getCredentials() != null) {
            String token = (String) auth.getCredentials();
            template.header("Authorization", "Bearer " + token);
        }
    }
}
```

### 日志拦截器

```java
// 请求日志拦截器
@Component
public class LoggingInterceptor implements RequestInterceptor {

    private final Logger log = LoggerFactory.getLogger(getClass());

    @Override
    public void apply(RequestTemplate template) {
        log.info("Feign Request: {} {}", template.method(), template.url());
        log.debug("Feign Headers: {}", template.headers());
        log.debug("Feign Body: {}", template.body());
    }
}
```

## 错误处理

### 自定义 ErrorDecoder

```java
// 自定义错误解码器
@Configuration
public class FeignErrorDecoder implements ErrorDecoder {

    private final Logger log = LoggerFactory.getLogger(getClass());

    @Override
    public Exception decode(String methodKey, Response response) {
        try {
            String body = Util.toString(response.body().asReader(StandardCharsets.UTF_8));

            switch (response.status()) {
                case 400:
                    return new BadRequestException(body);
                case 401:
                    return new UnauthorizedException(body);
                case 403:
                    return new ForbiddenException(body);
                case 404:
                    return new NotFoundException(body);
                case 500:
                    return new InternalServerException(body);
                case 503:
                    return new ServiceUnavailableException(body);
                default:
                    return new FeignException(body);
            }
        } catch (IOException e) {
            log.error("解析 Feign 响应失败", e);
            return new FeignException("解析响应失败");
        }
    }
}

// 自定义异常
public class BadRequestException extends RuntimeException {
    public BadRequestException(String message) {
        super(message);
    }
}
```

### 全局异常处理

```java
// 全局异常处理
@RestControllerAdvice
public class FeignExceptionHandler {

    @ExceptionHandler(UnauthorizedException.class)
    public ResponseEntity<ErrorResponse> handleUnauthorized(UnauthorizedException ex) {
        return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
                .body(new ErrorResponse("UNAUTHORIZED", ex.getMessage()));
    }

    @ExceptionHandler(NotFoundException.class)
    public ResponseEntity<ErrorResponse> handleNotFound(NotFoundException ex) {
        return ResponseEntity.status(HttpStatus.NOT_FOUND)
                .body(new ErrorResponse("NOT_FOUND", ex.getMessage()));
    }

    @ExceptionHandler(ServiceUnavailableException.class)
    public ResponseEntity<ErrorResponse> handleUnavailable(ServiceUnavailableException ex) {
        return ResponseEntity.status(HttpStatus.SERVICE_UNAVAILABLE)
                .body(new ErrorResponse("SERVICE_UNAVAILABLE", ex.getMessage()));
    }
}
```

## 性能优化

### 连接池配置

```yaml
# 连接池配置
spring:
  cloud:
    openfeign:
      httpclient:
        enabled: true              # 启用 Apache HttpClient
        pool:
          max-connections: 200     # 最大连接数
          max-connections-per-route: 50  # 每路由最大连接数
          time-to-live: 900       # 连接空闲时间（秒）
          reuse-connections: true # 复用连接
```

```xml
<!-- 引入 HttpClient 依赖 -->
<dependency>
    <groupId>io.github.openfeign</groupId>
    <artifactId>feign-httpclient</artifactId>
</dependency>
```

### 实践二：使用 FallbackFactory 获取异常信息

```java
// FallbackFactory 可以获取失败原因
@Component
public class UserClientFallbackFactory implements FallbackFactory<UserClient> {

    @Override
    public UserClient create(Throwable cause) {
        return new UserClient() {
            @Override
            public User getUser(Long id) {
                log.error("调用 user-service.getUser 失败", cause);
                throw new ServiceException("用户服务暂时不可用");
            }
        };
    }
}

@FeignClient(
    name = "user-service",
    fallbackFactory = UserClientFallbackFactory.class
)
public interface UserClient {
    @GetMapping("/{id}")
    User getUser(@PathVariable Long id);
}
```

## 总结

Spring Cloud OpenFeign 提供了声明式 REST 客户端的完整解决方案，与 Spring Boot 的深度集成让开发变得更加简单。

**核心要点回顾：**

- `@EnableFeignClients` 启用 OpenFeign
- 使用 Spring MVC 注解定义接口
- 支持配置：超时、日志、压缩、重试
- 可自定义拦截器处理认证、日志等
- 支持 Hystrix/Sentinel 降级
- 支持连接池优化性能
