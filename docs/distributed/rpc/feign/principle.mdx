# Feign 声明式 HTTP 客户端原理

你想调用一个 HTTP 接口，传统的做法是：用 RestTemplate 或者 HttpClient，写一堆 URL、参数、Header……代码又长又难维护。

有没有更优雅的方式？像调用本地方法一样调用远程接口？

这就是 Feign 的设计初衷——**声明式 HTTP 客户端**。

## 什么是 Feign

Feign 是 Netflix 开源的声明式 HTTP 客户端。它的核心思想是：**通过注解定义 HTTP 接口，框架自动生成实现类，调用远程接口就像调用本地方法一样。**

```yaml
# Feign vs 传统 HTTP 调用对比
# 传统方式
@RestTemplate
public String call() {
    String url = "http://user-service/users/1";
    HttpHeaders headers = new HttpHeaders();
    headers.set("Authorization", "Bearer token");
    HttpEntity<String> entity = new HttpEntity<>(headers);
    ResponseEntity<User> response = restTemplate.exchange(
        url,
        HttpMethod.GET,
        entity,
        User.class
    );
    return response.getBody();
}

// Feign 方式
@FeignClient("user-service")
public interface UserFeignClient {
    @GetMapping("/users/{id}")
    User getUser(@PathVariable Long id);
}
```

## Feign 工作原理

Feign 通过动态代理实现声明式调用：

```yaml

├─────────────────────────────────────────────────────────────┤
│                                                             │

│     │  @FeignClient("user-service")                  │   │
│     │  public interface UserClient {                  │   │
│     │      @GetMapping("/users/{id}")                │   │
│     │      User getUser(@PathVariable Long id);       │   │

│                                                             │
│  2. Feign 创建代理                                          │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Feign 注解详解

### 服务定义注解

```java
// @FeignClient 注解
@FeignClient(
    name = "user-service",              // 服务名（必须）
    url = "http://user-service:8080",  // 直接指定 URL
    fallback = UserClientFallback.class, // 降级类
    configuration = FeignConfig.class,   // 配置类
    path = "/users",                    // 路径前缀
    primary = true                     // 将代理设为 Primary Bean
)
public interface UserClient {
    // ...
}
```

### HTTP 方法注解

```java
// HTTP 方法注解
public interface UserClient {

    @RequestLine("GET /users/{id}")
    User getUser(@PathVariable Long id);

    @RequestLine("POST /users")
    User createUser(@RequestBody User user);

    @RequestLine("PUT /users/{id}")
    User updateUser(@PathVariable Long id, @RequestBody User user);

    @RequestLine("DELETE /users/{id}")
    void deleteUser(@PathVariable Long id);

    // 带查询参数的 GET
    @RequestLine("GET /users?page={page}&size={size}")
    Page<User> listUsers(
        @Param("page") int page,
        @Param("size") int size
    );

    // 带 Header
    @RequestLine("GET /users/{id}")
    @Headers("Authorization: {token}")
    User getUserWithToken(
        @PathVariable Long id,
        @Param("token") String token
    );
}
```

### 参数注解

```java
// 参数注解
public interface UserClient {

    // @Param: URL 参数
    @RequestLine("GET /users?name={name}")
    List<User> searchUsers(@Param("name") String name);

    // @QueryMap: 动态查询参数
    @RequestLine("GET /users")
    List<User> searchUsers(@QueryMap Map<String, Object> params);

    // @RequestBody: 请求体
    @RequestLine("POST /users")
    User createUser(@RequestBody User user);

    // @Header: 请求头
    @RequestLine("GET /users")
    @Headers("Authorization: {token}")
    List<User> getUsers(@Header("Authorization") String token);

    // @PathVariable: 路径参数
    @RequestLine("GET /users/{id}")
    User getUser(@PathVariable Long id);
}
```

## Feign 请求执行流程

```java
// Feign 请求执行流程源码解析

// 1. 创建代理
public class ReflectiveFeign extends Feign {

    @Override
    public <T> T newInstance(Target<T> target) {
        // 解析接口上的方法
        Map<String, MethodHandler> nameToHandler = new HashMap<>();

        for (Method method : type.getInterfaces()) {
            // 解析 @RequestLine 等注解
            Contract contract = new Contract.Default();
            List<Metadata> metadata = contract.parseAndValidate(target, method);

            // 创建 MethodHandler
            MethodHandler handler = new SynchronousMethodHandler(
                client,                         // HTTP Client
                target,                         // 目标 URL
                metadata,                        // 解析后的元数据
                options,                         // 配置
                retryer                         // 重试策略
            );
            nameToHandler.put(method.getName(), handler);
        }

        // 创建 JDK 动态代理
        InvocationHandler handler = new InvocationHandler() {
            @Override
            public Object invoke(Object proxy, Method method, Object[] args) {
                // 调用对应的 MethodHandler
                MethodHandler handler = nameToHandler.get(method.getName());
                return handler.invoke(args);
            }
        };

        return (T) Proxy.newProxyInstance(type.getClassLoader(), new Class[]{type}, handler);
    }
}

// 2. MethodHandler 执行调用
class SynchronousMethodHandler implements MethodHandler {

    @Override
    public Object invoke(Object[] args) throws Throwable {
        // 构建请求
        Request request = Request.Builder()
            .method(metadata.method)
            .uri(metadata.uri)
            .headers(metadata.headers)
            .body(requestBody)
            .build();

        // 执行请求
        Response response = client.execute(request, options);

        // 解码响应
        return decode(response);
    }
}
```

## Feign 配置

### 全局配置

```yaml
# application.yml
feign:
  client:
    config:
      default:
        connectTimeout: 5000          # 连接超时
        readTimeout: 10000           # 读取超时
        loggerLevel: basic          # 日志级别
        retryer: Retryer.NEVER_RETRY # 重试策略
        encoder: GsonEncoder         # 编码器
        decoder: GsonDecoder         # 解码器
```

### 指定客户端配置

```yaml
feign:
  client:
    config:
      user-service:                  # 指定服务名
        connectTimeout: 3000
        readTimeout: 5000
        loggerLevel: full
      order-service:
        connectTimeout: 10000
        readTimeout: 30000
```

### Java 配置

```java
// Java 配置
@Configuration
public class FeignConfig {

    @Bean
    public Feign.Builder feignBuilder() {
        return Feign.builder()
                .logger(new Slf4jLogger())
                .encoder(new GsonEncoder())
                .decoder(new GsonDecoder())
                .retryer(new Retryer.Default(100, 1000, 3))
                .requestInterceptor(new MyRequestInterceptor())
                .errorDecoder(new MyErrorDecoder())
                .decode404()
                .options(new Request.Options(3000, 5000));
    }
}

// 自定义请求拦截器
@Component
public class MyRequestInterceptor implements RequestInterceptor {

    @Override
    public void apply(RequestTemplate template) {
        // 添加 Header
        template.header("X-Request-Id", UUID.randomUUID().toString());
        template.header("Authorization", getToken());

        // 添加公共参数
        template.query("appId", "my-app");
    }
}

// 自定义错误处理
public class MyErrorDecoder implements ErrorDecoder {

    @Override
    public Exception decode(String methodKey, Response response) {
        if (response.status() == 404) {
            return new NotFoundException("资源不存在");
        }

        if (response.status() == 500) {
            return new InternalServerException("服务器内部错误");
        }

        return new FeignException();
    }
}
```

### 启用 Feign

```java
// 启用 Feign Client
@SpringCloudApplication

public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

### 定义 Feign Client

```java
// 定义 Feign Client
@FeignClient(
    name = "user-service",
    path = "/api/users",
    fallback = UserClientFallback.class
)
public interface UserClient {

    @GetMapping("/{id}")
    User getUser(@PathVariable Long id);

    @GetMapping
    Page<User> listUsers(
        @RequestParam int page,
        @RequestParam int size
    );

    @PostMapping
    User createUser(@RequestBody CreateUserRequest request);

    @PutMapping("/{id}")
    User updateUser(@PathVariable Long id, @RequestBody UpdateUserRequest request);

    @DeleteMapping("/{id}")
    void deleteUser(@PathVariable Long id);
}

// 降级实现
@Component
public class UserClientFallback implements UserClient {

    private static final Logger log = LoggerFactory.getLogger(UserClientFallback.class);

    @Override
    public User getUser(Long id) {
        log.warn("调用 user-service.getUser 降级，id={}", id);
        return User.builder()
                .id(id)
                .name("降级用户")
                .build();
    }

    @Override
    public Page<User> listUsers(int page, int size) {
        log.warn("调用 user-service.listUsers 降级");
        return Page.empty();
    }

    // 其他方法...
}
```

```java
// Hystrix 降级
@FeignClient(
    name = "user-service",
    fallbackFactory = UserClientFallbackFactory.class
)
public interface UserClient {
    @GetMapping("/{id}")
    User getUser(@PathVariable Long id);
}

// 使用 FallbackFactory 获取异常信息
@Component
public class UserClientFallbackFactory implements FallbackFactory<UserClient> {

    @Override
    public UserClient create(Throwable cause) {
        return new UserClient() {
            @Override
            public User getUser(Long id) {
                log.error("调用 user-service 失败", cause);
                return User.builder()
                        .id(id)
                        .name("降级用户")
                        .build();
            }
        };
    }
}
```

## 总结

Feign 通过声明式的方式简化了 HTTP 客户端的编写，让远程调用像本地方法调用一样简洁。

**核心要点回顾：**

- Feign 是声明式 HTTP 客户端，通过注解定义接口
- 核心原理是 JDK 动态代理
- 支持多种注解：`@FeignClient`、`@RequestLine`、`@Headers` 等
- 与 Spring Cloud 深度集成，自动支持负载均衡和熔断降级
- 支持自定义配置：编码器、解码器、拦截器、错误处理器等
