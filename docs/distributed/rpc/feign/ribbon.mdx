# Feign 负载均衡：Ribbon 集成

你用 Feign 调用服务，但服务有多个实例。Feign 怎么知道该调用哪一个？

这就要靠负载均衡组件了。

## Feign 与负载均衡的关系

## Spring Cloud Ribbon（已停止维护）

Spring Cloud Ribbon 是 Netflix Ribbon 的 Spring Cloud 封装，目前已停止维护，推荐使用 Spring Cloud LoadBalancer。

### Ribbon 核心组件

```yaml
# Ribbon 核心组件
组件                  类名                           说明
─────────────────────────────────────────────────────────────────
ServerList           ServerList                    服务列表提供者
ServerListFilter     ServerListFilter              服务列表过滤器
IRule                IRule                         负载均衡策略
IPing                IPing                         健康检查策略
LoadBalancer         ILoadBalancer                 负载均衡器
```

### Ribbon 配置

```yaml
# application.yml 中配置 Ribbon
# 服务名需要大写
user-service:
  ribbon:
    # 服务列表
    NIWSServerListClassName: com.netflix.loadbalancer.ConfigurationBasedServerList
    # 负载均衡策略
    NFLoadBalancerRuleClassName: com.netflix.loadbalancer.RoundRobinRule
    # 健康检查
    NFLoadBalancerPingClassName: com.netflix.loadbalancer.PingUrl
    # 关联的 ServerListFilter
    NIWSServerListFilterClassName: com.netflix.loadbalancer.ZoneAffinityServerListFilter
```

### Ribbon 负载均衡策略

```yaml
# Ribbon 内置负载均衡策略
策略                  类名                           特点
─────────────────────────────────────────────────────────────────
RoundRobinRule       轮询                          默认
RandomRule           随机                          随机选择
WeightedResponseTimeRule  加权响应时间              响应时间越短，权重越高
BestAvailableRule    最低并发                      选择并发请求最少的
AvailabilityFilteringRule 可用性过滤                过滤掉连接失败和熔断的服务
ZoneAvoidanceRule    区域感知                      根据区域选择
RetryRule            重试                          在其他策略基础上重试
```

### 配置 Ribbon 策略

```java
// 方式一：配置文件
user-service:
  ribbon:
    NFLoadBalancerRuleClassName: com.netflix.loadbalancer.WeightedResponseTimeRule

// 方式二：Java 配置
@Configuration
public class RibbonConfig {

    @Bean
    public IRule ribbonRule() {
        // 加权响应时间策略
        return new WeightedResponseTimeRule();
    }

    @Bean
    public IPing ribbonPing() {
        // URL Ping 健康检查
        return new PingUrl(false, "/health");
    }
}

// 方式三：注解配置
@RibbonClient(name = "user-service", configuration = RibbonConfig.class)
public class RibbonClientConfig {
}
```

## Spring Cloud LoadBalancer（推荐）

Spring Cloud LoadBalancer 是 Spring Cloud 官方推荐的负载均衡组件，替代了 Ribbon。

### 启用 LoadBalancer

```xml
<!-- 引入 LoadBalancer 依赖 -->
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-loadbalancer</artifactId>
</dependency>
```

### LoadBalancer 配置

```yaml
# application.yml
spring:
  cloud:
    loadbalancer:
      # 启用重试
      retry:
        enabled: true
      # 配置 zones
      zones: us-east-1,us-west-1
```

### 自定义负载均衡策略

```java
// 方式一：替换默认策略
@Configuration
public class CustomLoadBalancerConfig {

    @Bean
    public ReactorLoadBalancer<ServiceInstance> randomLoadBalancer(
            ServiceInstanceListSupplier supplier) {
        return new RandomLoadBalancer(supplier);
    }
}

// 启用自定义配置
@LoadBalancerClient(
    name = "user-service",
    configuration = CustomLoadBalancerConfig.class
)
public interface UserClient {
    @GetMapping("/{id}")
    User getUser(@PathVariable Long id);
}

// 方式二：使用 @LoadBalancerExchange
@Configuration
public class LoadBalancerExchangeConfig {

    @Bean
    public LoadBalancerExchangeFunctionBuilder loadBalancerExchangeFunctionBuilder() {
        return new LoadBalancerExchangeFunctionBuilder();
    }
}

// 使用
@RestController
public class ProductController {

    @Autowired
    private LoadBalancerExchangeFunctionBuilder builder;

    public Product getProduct(Long id) {
        return builder.build()
            .exchange("http://product-service/products/" + id)
            .bodyTo(Product.class);
    }
}
```

### 自定义区域感知策略

```java
// 区域感知负载均衡
@Configuration
public class ZoneAwareLoadBalancerConfig {

    @Bean
    public ReactorLoadBalancer<ServiceInstance> zoneAwareLoadBalancer(
            ServiceInstanceListSupplier supplier) {
        return new ZoneAwareLoadBalancer<>(supplier);
    }
}
```

## Feign 与 LoadBalancer 配合

### 默认配置

```yaml
# Feign 默认使用 LoadBalancer
spring:
  cloud:
    openfeign:
      client:
        config:
          default:
            # Feign 使用 LoadBalancer 进行服务发现
            loadBalancer: true
```

### 禁用 LoadBalancer

```yaml
# 禁用 LoadBalancer，使用直连
spring:
  cloud:
    openfeign:
      client:
        config:
          user-service:
            url: http://192.168.1.100:8080
```

### 配置负载均衡超时

```yaml
# 完整配置
spring:
  cloud:
    openfeign:
      client:
        config:
          default:
            connectTimeout: 3000
            readTimeout: 5000

    loadbalancer:
      ribbon:
        enabled: false    # 禁用 Ribbon，启用 LoadBalancer

# LoadBalancer 超时配置
user-service:
  spring:
    cloud:
      loadbalancer:
        ribbon:
          connectionTimeout: 3000
          readTimeout: 5000
```

## 实际使用示例

### 定义带负载均衡的 Feign Client

```java
// 使用服务名，LoadBalancer 自动选择实例
@FeignClient(
    name = "user-service",
    path = "/api/users"
)
public interface UserClient {

    @GetMapping("/{id}")
    User getUser(@PathVariable Long id);

    @GetMapping
    List<User> listUsers(@RequestParam int page, @RequestParam int size);
}

// 使用
@Service
public class OrderService {

    @Autowired
    private UserClient userClient;

    public Order createOrder(Long userId) {
        // LoadBalancer 会自动选择 user-service 的一个实例
        User user = userClient.getUser(userId);
        return Order.builder()
                .userId(userId)
                .userName(user.getName())
                .build();
    }
}
```

### 自定义 Feign + LoadBalancer 配置

```java
// 完整配置类
@Configuration
public class FeignLoadBalancerConfig {

    @Bean
    public Logger.Level feignLogger() {
        return Logger.Level.FULL;
    }

    @Bean
    public ReactorLoadBalancer<ServiceInstance> feignLoadBalancer(
            ServiceInstanceListSupplier supplier) {
        // 自定义负载均衡策略
        return new WeightedResponseTimeLoadBalancer(supplier);
    }
}

// 应用到 Client
@FeignClient(
    name = "user-service",
    path = "/api/users",
    configuration = FeignLoadBalancerConfig.class
)
public interface UserClient {
    @GetMapping("/{id}")
    User getUser(@PathVariable Long id);
}
```

## 总结

Feign 本身不提供负载均衡能力，需要配合 LoadBalancer 使用。

**核心要点回顾：**

- Spring Cloud Ribbon 已停止维护，推荐使用 Spring Cloud LoadBalancer
- LoadBalancer 自动从注册中心获取服务列表
- 支持多种负载均衡策略：轮询、随机、加权响应时间等
- Feign Client 使用服务名，LoadBalancer 自动选择实例
- 可通过配置或 Java 代码自定义负载均衡策略
