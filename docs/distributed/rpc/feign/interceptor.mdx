# Feign 拦截器

你需要在每次 Feign 调用时自动带上认证 Token，或者记录调用日志，或者修改请求参数……

这些需求，用拦截器就能轻松实现。

## 拦截器概述

Feign 拦截器（RequestInterceptor）是 Feign 提供的一种扩展机制，可以在请求发送前和响应返回后进行拦截处理。

```yaml

│     ▼                                                      │

│  │ 1. Interceptor.apply(RequestTemplate)              │  │

```

## RequestInterceptor

`RequestInterceptor` 是最常用的拦截器接口，在请求发送前执行。

### 基础用法

```java
// 定义拦截器
@Component
public class MyRequestInterceptor implements RequestInterceptor {

    private final Logger log = LoggerFactory.getLogger(getClass());

    @Override
    public void apply(RequestTemplate template) {
        // 添加 Header
        template.header("X-Request-Id", UUID.randomUUID().toString());
        template.header("X-Source", "order-service");

        // 添加查询参数
        template.query("appId", "my-app");
        template.query("timestamp", String.valueOf(System.currentTimeMillis()));

        // 记录日志
        log.info("Feign Request: {} {}", template.method(), template.url());
    }
}

// 全局配置，所有 Client 都会应用
@Configuration
public class FeignInterceptorConfig {

    @Bean
    public RequestInterceptor requestInterceptor() {
        return new MyRequestInterceptor();
    }
}
```

### 认证拦截器

```java
// 认证拦截器
@Component
public class AuthInterceptor implements RequestInterceptor {

    private final ThreadLocal<String> tokenHolder = new ThreadLocal<>();

    // 设置 Token
    public void setToken(String token) {
        tokenHolder.set(token);
    }

    // 清除 Token
    public void clearToken() {
        tokenHolder.remove();
    }

    @Override
    public void apply(RequestTemplate template) {
        String token = tokenHolder.get();
        if (token != null) {
            template.header("Authorization", "Bearer " + token);
        }
    }
}

// 使用
@Service
public class OrderService {

    @Autowired
    private UserClient userClient;

    @Autowired
    private AuthInterceptor authInterceptor;

    public void createOrder() {
        // 设置 Token
        authInterceptor.setToken(getCurrentToken());

        try {
            userClient.getUser(1L);
        } finally {
            // 清除 Token
            authInterceptor.clearToken();
        }
    }
}
```

### Spring Security 集成

```java
// Spring Security 认证拦截器
@Component
public class SecurityAuthInterceptor implements RequestInterceptor {

    private final TokenService tokenService;

    public SecurityAuthInterceptor(TokenService tokenService) {
        this.tokenService = tokenService;
    }

    @Override
    public void apply(RequestTemplate template) {
        // 从 Security Context 获取认证信息
        Authentication auth = SecurityContextHolder.getContext().getAuthentication();

        if (auth != null && auth.isAuthenticated()) {
            // 获取当前用户的 Token
            String token = tokenService.getToken(auth.getName());
            if (token != null) {
                template.header("Authorization", "Bearer " + token);
            }
        }

        // 添加租户信息
        String tenantId = TenantContext.getCurrentTenant();
        if (tenantId != null) {
            template.header("X-Tenant-Id", tenantId);
        }
    }
}
```

### 传递上下文信息

```java
// 传递 TraceId
@Component
public class TraceInterceptor implements RequestInterceptor {

    private static final String TRACE_ID_HEADER = "X-Trace-Id";

    @Override
    public void apply(RequestTemplate template) {
        // 获取或生成 TraceId
        String traceId = template.headers().getOrDefault(TRACE_ID_HEADER, List.of())
                .stream().findFirst()
                .orElseGet(() -> {
                    String newTraceId = generateTraceId();
                    template.header(TRACE_ID_HEADER, newTraceId);
                    return newTraceId;
                });

        // 设置到 MDC，方便日志追踪
        MDC.put("traceId", traceId);
    }

    private String generateTraceId() {
        return UUID.randomUUID().toString().replace("-", "");
    }
}
```

### 日志拦截器

```java
// 日志拦截器
@Component
public class FeignLoggingInterceptor implements RequestInterceptor {

    private final Logger log = LoggerFactory.getLogger(getClass());

    @Override
    public void apply(RequestTemplate template) {
        long startTime = System.currentTimeMillis();

        // 在请求上存储开始时间
        template.request().requestTemplate().variable("feignStartTime", startTime);

        log.info("==> Feign Request: {} {}", template.method(), template.url());
        log.debug("Headers: {}", template.headers());
        log.debug("Body: {}", template.body() != null ? new String(template.body()) : "N/A");
    }
}

// 响应日志需要使用 Logger 后处理器
@Component
public class FeignLogger implements Logger {

    private final Logger log = LoggerFactory.getLogger("FeignClient");

    @Override
    protected void logRequest(String configKey, Level level, Request request, Response response) {
        log.info("<== {} {} {}", request.httpMethod(), request.url(), response.status());
    }

    @Override
    protected void logRetry(String configKey, Level level, Request request, Response response, RetryableException retryableException) {
        log.warn("Retry {} {} attempt {}", request.httpMethod(), request.url(), retryableException.retryAfter());
    }

    @Override
    protected void logError(String configKey, Logger.Errorf field, String format, Object... args) {
        log.error("Error " + format, args);
    }
}
```

## 响应拦截器

Feign 没有直接的响应拦截器接口，但可以通过 `Retryer` 或自定义 `Client` 实现。

### 通过 Retryer 实现响应拦截

```java
// 响应拦截器（通过 Retryer）
@Component
public class ResponseInterceptor implements Retryer {

    private final Logger log = LoggerFactory.getLogger(getClass());

    @Override
    public void continueOrPropagate(RetryableException e) {
        // 响应处理逻辑
        throw e;
    }

    @Override
    public Retryer clone() {
        return this;
    }
}
```

### 通过自定义 Client 实现响应拦截

```java
// 自定义 Client 实现响应拦截
@Configuration
public class CustomFeignClient {

    @Bean
    @Primary
    public Client feignClient(Client client, ResponseInterceptor interceptor) {
        return new Client() {
            @Override
            public Response execute(Request request, Options options) throws IOException {
                long startTime = System.currentTimeMillis();

                // 执行请求
                Response response = client.execute(request, options);

                long duration = System.currentTimeMillis() - startTime;

                // 响应拦截
                interceptor.intercept(request, response, duration);

                return response;
            }
        };
    }
}

// 响应拦截器
@Component
public class ResponseInterceptor {

    private final Logger log = LoggerFactory.getLogger(getClass());

    public void intercept(Request request, Response response, long duration) {
        log.info("<== {} {} {} ({}ms)",
                request.httpMethod(),
                request.url(),
                response.status(),
                duration);

        if (response.status() >= 400) {
            log.warn("Response error: {}", response.body());
        }
    }
}
```

## 条件拦截器

```java
// 条件拦截器
@Component
public class ConditionalInterceptor implements RequestInterceptor {

    private final Environment env;

    public ConditionalInterceptor(Environment env) {
        this.env = env;
    }

    @Override
    public void apply(RequestTemplate template) {
        // 只对特定服务添加 Header
        if (template.url().contains("admin-service")) {
            // 检查权限
            if (!hasAdminRole()) {
                throw new ForbiddenException("需要管理员权限");
            }
            template.header("X-Admin", "true");
        }

        // 只在生产环境添加特定 Header
        if ("prod".equals(env.getActiveProfile())) {
            template.header("X-Environment", "production");
        }
    }
}
```

## 最佳实践

### 实践一：拦截器要轻量

```java
// 拦截器要轻量，不要在其中执行耗时操作
@Component
public class LightInterceptor implements RequestInterceptor {

    @Override
    public void apply(RequestTemplate template) {
        // ✅ 轻量操作：添加 Header
        template.header("X-Request-Id", UUID.randomUUID().toString());

        // ❌ 耗时操作不要在拦截器中执行
        // - 不要查询数据库
        // - 不要调用其他服务
        // - 不要执行复杂计算
    }
}
```

### 实践二：使用 ThreadLocal 管理上下文

```java
// 使用 ThreadLocal 管理上下文
@Component
public class ContextInterceptor implements RequestInterceptor {

    private static final ThreadLocal<RequestContext> contextHolder = new ThreadLocal<>();

    public static void setContext(RequestContext context) {
        contextHolder.set(context);
    }

    public static RequestContext getContext() {
        return contextHolder.get();
    }

    public static void clear() {
        contextHolder.remove();
    }

    @Override
    public void apply(RequestTemplate template) {
        RequestContext context = getContext();
        if (context != null) {
            template.header("X-Trace-Id", context.getTraceId());
            template.header("X-User-Id", context.getUserId());
            template.header("X-Tenant-Id", context.getTenantId());
        }
    }
}
```

### 实践三：拦截器要有日志

```java
@Component
public class DebugInterceptor implements RequestInterceptor {

    private static final Logger log = LoggerFactory.getLogger(DebugInterceptor.class);

    @Override
    public void apply(RequestTemplate template) {
        log.debug("Feign Request - Method: {}, URL: {}",
                template.method(),
                template.url());
    }
}
```

## 总结

Feign 拦截器是实现请求增强的强大工具。

**核心要点回顾：**

- `RequestInterceptor` 在请求发送前执行，可修改请求
- 常见用途：认证、追踪、日志、监控
- 可通过实现多个拦截器完成不同功能
- 拦截器要轻量，避免耗时操作
- 使用 `ThreadLocal` 管理请求上下文
