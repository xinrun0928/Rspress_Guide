# 分布式批量任务框架设计：分片任务与结果聚合

凌晨 2 点，你的数据同步任务要处理 1000 万条数据。如果用单机串行处理，需要 10 个小时。但老板要求 6 点前完成。

分布式批量任务，就是来解决这个问题的——把大任务拆分成小任务，分给多个机器并行处理。

## 为什么需要分布式批任务？

## 分片任务设计

### 分片策略

```java
/**
 * 分片任务抽象
 */
public interface ShardingTask {

    /**
     * 获取分片总数
     */
    int getTotalShards();

    /**
     * 获取当前分片编号
     */
    int getShardIndex();

    /**
     * 执行分片任务
     */
    void execute() throws Exception;

    /**
     * 任务唯一标识
     */
    String getTaskId();
}

/**
 * 数据分片任务
 */
public abstract class DataShardingTask<T> implements ShardingTask {

    private final String taskId;
    private final int totalShards;
    private final int shardIndex;
    private final List<T> data;

    public DataShardingTask(String taskId, int totalShards, int shardIndex, List<T> data) {
        this.taskId = taskId;
        this.totalShards = totalShards;
        this.shardIndex = shardIndex;
        this.data = data;
    }

    @Override
    public int getTotalShards() {
        return totalShards;
    }

    @Override
    public int getShardIndex() {
        return shardIndex;
    }

    @Override
    public void execute() throws Exception {
        // 计算当前分片的数据范围
        int start = calculateStart();
        int end = calculateEnd();

        List<T> shardData = data.subList(start, end);

        for (T item : shardData) {
            processItem(item);
        }
    }

    private int calculateStart() {
        int pageSize = data.size() / totalShards;
        return shardIndex * pageSize;
    }

    private int calculateEnd() {
        if (shardIndex == totalShards - 1) {
            return data.size();
        }
        return (shardIndex + 1) * (data.size() / totalShards);
    }

    /**
     * 处理单条数据，由子类实现
     */
    protected abstract void processItem(T item) throws Exception;
}
```

### 分片计算策略

```java
/**
 * 分片计算器
 */
@Service
public class ShardingCalculator {

    /**
     * 按 ID 范围分片
     */
    public List<ShardRange> calculateByIdRange(long minId, long maxId, int shardCount) {
        long range = maxId - minId + 1;
        long perShard = range / shardCount;

        List<ShardRange> shards = new ArrayList<>();
        for (int i = 0; i < shardCount; i++) {
            long start = minId + i * perShard;
            long end = (i == shardCount - 1) ? maxId : start + perShard - 1;
            shards.add(new ShardRange(i, start, end));
        }

        return shards;
    }

    /**
     * 按时间范围分片
     */
    public List<ShardRange> calculateByTimeRange(
            LocalDateTime startTime,
            LocalDateTime endTime,
            int shardCount) {

        long totalMinutes = java.time.Duration.between(startTime, endTime).toMinutes();
        long perShard = totalMinutes / shardCount;

        List<ShardRange> shards = new ArrayList<>();
        for (int i = 0; i < shardCount; i++) {
            LocalDateTime shardStart = startTime.plusMinutes(i * perShard);
            LocalDateTime shardEnd = (i == shardCount - 1)
                ? endTime
                : shardStart.plusMinutes(perShard);

            shards.add(new ShardRange(i, shardStart, shardEnd));
        }

        return shards;
    }

    /**
     * 按哈希分片
     */
    public int calculateByHash(String key, int shardCount) {
        return Math.abs(key.hashCode()) % shardCount;
    }
}

@Data
@AllArgsConstructor
public class ShardRange {
    private int shardIndex;
    private Object start;
    private Object end;
}
```

```java
/**
 * 分布式任务调度器
 */
@Service
public class DistributedTaskScheduler {

    private final TaskExecutor taskExecutor;
    private final TaskRepository taskRepository;
    private final ShardingCalculator shardingCalculator;
    private final RedisTemplate<String, Object> redisTemplate;

    private static final String TASK_LOCK_PREFIX = "task:lock:";
    private static final String TASK_PROGRESS_PREFIX = "task:progress:";

    /**
     * 提交分片任务
     */
    public String submitTask(TaskRequest request) {
        // 1. 生成任务 ID
        String taskId = IdGenerator.nextTaskId();

        // 2. 创建任务记录
        Task task = new Task()
            .setId(taskId)
            .setType(request.getType())
            .setStatus(TaskStatus.PENDING)
            .setTotalShards(request.getShardCount())
            .setCreatedAt(System.currentTimeMillis())
            .setCreatedBy(request.getUserId());

        taskRepository.save(task);

        // 3. 发布分片任务
        for (int i = 0; i < request.getShardCount(); i++) {
            publishShardTask(taskId, i, request.getParams());
        }

        return taskId;
    }

    /**
     * 发布分片任务到消息队列
     */
    private void publishShardTask(String taskId, int shardIndex, Map<String, Object> params) {
        ShardTaskMessage message = new ShardTaskMessage()
            .setTaskId(taskId)
            .setShardIndex(shardIndex)
            .setParams(params);

        kafkaTemplate.send("task-topic", taskId + "-" + shardIndex, message);
    }

    /**
     * 取消任务
     */
    public void cancelTask(String taskId) {
        // 1. 检查任务状态
        Task task = taskRepository.findById(taskId);
        if (task.getStatus() == TaskStatus.COMPLETED) {
            throw new IllegalStateException("已完成的任务不能取消");
        }

        // 2. 更新状态
        task.setStatus(TaskStatus.CANCELLED);
        taskRepository.save(task);

        // 3. 发送取消消息
        for (int i = 0; i < task.getTotalShards(); i++) {
            kafkaTemplate.send("task-cancel-topic", taskId + "-" + i,
                new CancelTaskMessage(taskId, i));
        }
    }
}
```

### 任务执行器

```java
/**
 * 分片任务执行器
 */
@Component
public class ShardTaskExecutor {

    private final Map<String, TaskProcessor> processors = new ConcurrentHashMap<>();

    @KafkaListener(topics = "task-topic", groupId = "task-executor")
    public void executeShardTask(ShardTaskMessage message) {
        String taskId = message.getTaskId();
        int shardIndex = message.getShardIndex();

        // 1. 检查是否已取消
        if (isTaskCancelled(taskId)) {
            log.info("任务 {} 分片 {} 已被取消", taskId, shardIndex);
            return;
        }

        // 2. 获取任务处理器
        Task task = taskRepository.findById(taskId);
        TaskProcessor processor = processors.get(task.getType());

        try {
            // 3. 初始化分片进度
            initShardProgress(taskId, shardIndex);

            // 4. 执行分片任务
            ShardingTask shardTask = createShardingTask(task, shardIndex, message.getParams());
            shardTask.execute();

            // 5. 更新分片状态
            updateShardProgress(taskId, shardIndex, ShardStatus.COMPLETED);

            // 6. 检查是否所有分片完成
            checkTaskCompletion(taskId);

        } catch (Exception e) {
            log.error("执行分片任务失败: taskId={}, shard={}", taskId, shardIndex, e);
            updateShardProgress(taskId, shardIndex, ShardStatus.FAILED);

            // 标记任务失败
            markTaskFailed(taskId);
        }
    }

    /**
     * 创建分片任务
     */
    private ShardingTask createShardingTask(Task task, int shardIndex, Map<String, Object> params) {
        switch (task.getType()) {
            case "DATA_SYNC":
                return new DataSyncShardTask(task.getId(), shardIndex, params);
            case "DATA_EXPORT":
                return new DataExportShardTask(task.getId(), shardIndex, params);
            case "DATA_IMPORT":
                return new DataImportShardTask(task.getId(), shardIndex, params);
            default:
                throw new IllegalArgumentException("未知任务类型: " + task.getType());
        }
    }
}
```

```java
/**
 * 数据同步分片任务
 */
public class DataSyncShardTask extends DataShardingTask<DataSyncItem> {

    private final DataSource sourceDataSource;
    private final DataSource targetDataSource;

    public DataSyncShardTask(String taskId, int shardIndex, Map<String, Object> params) {
        super(taskId, getShardCount(), shardIndex, fetchData(params));
        this.sourceDataSource = getSourceDataSource(params);
        this.targetDataSource = getTargetDataSource(params);
    }

    @Override
    protected void processItem(DataSyncItem item) throws Exception {
        // 1. 从源数据库读取
        DataSyncItem source = sourceDataSource.get(item.getId());

        // 2. 数据转换
        DataSyncItem target = transform(source);

        // 3. 写入目标数据库
        targetDataSource.save(target);

        // 4. 更新进度
        updateItemProgress(item.getId(), "SYNCED");
    }

    private DataSyncItem transform(DataSyncItem source) {
        // 数据转换逻辑
        DataSyncItem target = new DataSyncItem();
        target.setId(source.getId());
        target.setName(source.getName());
        target.setValue(source.getValue());
        target.setSyncTime(System.currentTimeMillis());
        return target;
    }
}
```

## 结果聚合

### 聚合策略

```java
/**
 * 任务结果聚合器
 */
@Service
public class ResultAggregator {

    private final RedisTemplate<String, Object> redisTemplate;
    private static final String SHARD_RESULT_PREFIX = "task:result:";

    /**
     * 收集分片结果
     */
    public void collectShardResult(String taskId, int shardIndex, ShardResult result) {
        String key = SHARD_RESULT_PREFIX + taskId + ":" + shardIndex;

        // 存储分片结果
        redisTemplate.opsForValue().set(key, result, Duration.ofHours(24));
    }

    /**
     * 聚合所有分片结果
     */
    public TaskResult aggregateResults(String taskId, int totalShards) {
        List<ShardResult> shardResults = new ArrayList<>();

        for (int i = 0; i < totalShards; i++) {
            String key = SHARD_RESULT_PREFIX + taskId + ":" + i;
            ShardResult result = (ShardResult) redisTemplate.opsForValue().get(key);

            if (result == null) {
                throw new RuntimeException("分片结果缺失: " + i);
            }

            shardResults.add(result);
        }

        return aggregate(shardResults);
    }

    /**
     * 聚合逻辑
     */
    private TaskResult aggregate(List<ShardResult> shardResults) {
        TaskResult result = new TaskResult();

        // 1. 合并统计数据
        int totalSuccess = 0;
        int totalFailed = 0;
        int totalSkipped = 0;
        long totalCost = 0;

        for (ShardResult shard : shardResults) {
            totalSuccess += shard.getSuccessCount();
            totalFailed += shard.getFailedCount();
            totalSkipped += shard.getSkippedCount();
            totalCost += shard.getCostTime();
        }

        result.setSuccessCount(totalSuccess);
        result.setFailedCount(totalFailed);
        result.setSkippedCount(totalSkipped);
        result.setTotalCost(totalCost);

        // 2. 合并失败记录
        List<FailedItem> allFailed = shardResults.stream()
            .flatMap(r -> r.getFailedItems().stream())
            .collect(Collectors.toList());
        result.setFailedItems(allFailed);

        // 3. 生成报告
        result.setReport(generateReport(result));

        return result;
    }
}
```

### 聚合监听器

```java
/**
 * 任务完成监听器
 */
@Component
public class TaskCompletionListener {

    private final TaskRepository taskRepository;
    private final ResultAggregator resultAggregator;
    private final NotificationService notificationService;

    /**
     * 检查任务是否完成
     */
    @TransactionalEventListener(phase = TransactionPhase.AFTER_COMMIT)
    public void onShardCompleted(ShardCompletedEvent event) {
        String taskId = event.getTaskId();
        Task task = taskRepository.findById(taskId);

        // 1. 检查是否所有分片都完成
        if (areAllShardsCompleted(taskId, task.getTotalShards())) {
            // 2. 聚合结果
            TaskResult result = resultAggregator.aggregateResults(
                taskId, task.getTotalShards());

            // 3. 更新任务状态
            task.setStatus(TaskStatus.COMPLETED);
            task.setResult(result);
            task.setCompletedAt(System.currentTimeMillis());
            taskRepository.save(task);

            // 4. 发送通知
            notificationService.sendTaskCompletedNotification(task, result);
        }
    }

    private boolean areAllShardsCompleted(String taskId, int totalShards) {
        for (int i = 0; i < totalShards; i++) {
            ShardStatus status = getShardStatus(taskId, i);
            if (status != ShardStatus.COMPLETED && status != ShardStatus.FAILED) {
                return false;
            }
        }
        return true;
    }
}
```

```java
/**
 * 任务仓储

@Repository
public class TaskRepository {

    @Autowired
    private JdbcTemplate jdbcTemplate;

    public void save(Task task) {
        if (task.getId() == null) {
            insert(task);
        } else {
            update(task);
        }
    }

    private void insert(Task task) {
        jdbcTemplate.update(
            "INSERT INTO task (id, type, status, total_shards, " +
            "completed_shards, failed_shards, created_at, created_by) " +
            "VALUES (?, ?, ?, ?, ?, ?, ?, ?)",
            task.getId(),
            task.getType(),
            task.getStatus().name(),
            task.getTotalShards(),
            task.getCompletedShards(),
            task.getFailedShards(),
            task.getCreatedAt(),
            task.getCreatedBy()
        );
    }

    private void update(Task task) {
        jdbcTemplate.update(
            "UPDATE task SET status = ?, completed_shards = ?, " +
            "failed_shards = ?, completed_at = ?, result = ? WHERE id = ?",
            task.getStatus().name(),
            task.getCompletedShards(),
            task.getFailedShards(),
            task.getCompletedAt(),
            toJson(task.getResult()),
            task.getId()
        );
    }
}
```

```java
/**
 * 任务监控服务
 */
@Service
public class TaskMonitor {

    @Autowired
    private TaskRepository taskRepository;

    @Scheduled(fixedRate = 60000)  // 每分钟检查
    public void checkRunningTasks() {
        List<Task> runningTasks = taskRepository.findByStatus(TaskStatus.RUNNING);

        for (Task task : runningTasks) {
            // 1. 检查任务是否超时
            if (isTaskTimeout(task)) {
                handleTaskTimeout(task);
            }

            // 2. 检查任务进度
            checkTaskProgress(task);
        }
    }

    private boolean isTaskTimeout(Task task) {
        long runningTime = System.currentTimeMillis() - task.getCreatedAt();
        long timeout = getTaskTimeout(task.getType());
        return runningTime > timeout;
    }

    private void handleTaskTimeout(Task task) {
        log.warn("任务超时: taskId={}, type={}, runningTime={}",
            task.getId(), task.getType(), runningTime);

        // 发送告警
        alertService.sendAlert("任务超时", task);

        // 自动处理：失败或继续
        if (shouldAutoFail(task)) {
            task.setStatus(TaskStatus.FAILED);
            task.setRemark("自动超时失败");
            taskRepository.save(task);
        }
    }
}
```

## 常见问题与解决

### 问题一：分片不均匀

```java
// 使用动态分片策略
public List<ShardRange> calculateDynamicSharding(
        String taskId,
        int targetShardCount,
        long totalCount) {

    // 如果数据量小，减少分片数
    if (totalCount < targetShardCount * 1000) {
        targetShardCount = (int) Math.max(1, totalCount / 1000);
    }

    // 使用 ID 取模分片，更均匀
    return calculateByModSharding(taskId, targetShardCount, totalCount);
}

private List<ShardRange> calculateByModSharding(
        String taskId,
        int shardCount,
        long totalCount) {

    List<ShardRange> shards = new ArrayList<>();
    long perShard = totalCount / shardCount;

    for (int i = 0; i < shardCount; i++) {
        long start = i * perShard;
        long end = (i == shardCount - 1) ? totalCount - 1 : (i + 1) * perShard - 1;
        shards.add(new ShardRange(i, start, end));
    }

    return shards;
}
```

## 总结

分布式批量任务的核心是**分而治之 + 结果聚合**：

| 组件 | 作用 |
|---|---|
| 分片器 | 将大任务拆分成小任务 |
| 调度器 | 分配任务到各执行节点 |
| 执行器 | 并行执行分片任务 |
| 聚合器 | 收集并聚合分片结果 |
| 监控器 | 监控任务状态，处理异常 |

分片策略选择：

| 策略 | 适用场景 | 优点 | 缺点 |
|---|---|---|---|
| ID 范围 | 连续 ID | 均匀 | 可能不连续 |
| 时间范围 | 时序数据 | 均匀 | 时区问题 |
| 哈希 | 随机数据 | 均匀 | 不支持范围查询 |
| 动态 | 通用场景 | 自适应 | 实现复杂 |

记住：**分片数不是越多越好**。分片太多会增加调度开销，分片太少又无法充分利用资源。一般每个分片耗时 1-5 分钟为宜。
