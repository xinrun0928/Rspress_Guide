# 无状态 Token 方案：JWT 生成、刷新、注销与黑名单

用户登录后，服务器如何知道请求来自哪个用户？在 Session 方案中，服务器通过 Session ID 查询内存/Redis 获取用户信息。但 Redis 会成为瓶颈，Session 数据会占用内存。

JWT（JSON Web Token）来了。它把用户信息直接编码到 Token 里，服务器不需要存储任何状态。

## JWT 是什么？

### JWT 的结构

JWT 由三部分组成，用 `.` 连接：

```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
```

| 部分 | 说明 | 示例 |
|---|---|---|
| Header | Token 类型和签名算法 | `{"alg":"HS256","typ":"JWT"}` |
| Payload | Claims（声明），存放用户信息 | `{"sub":"123","name":"John"}` |
| Signature | 签名，确保 Token 未被篡改 | `HMACSHA256(base64(header) + "." + base64(payload), secret)` |

### Base64 URL 编码

JWT 使用 URL 安全的 Base64 编码（`+` 变 `-`，`/` 变 `_`）：

```java
// 标准 Base64 vs URL-safe Base64
Standard: eyJhbGciOiJIUzI1NiJ9+/wE
URL-Safe: eyJhbGciOiJIUzI1NiJ9-_wE
```

### JWT 工具类

```java
public class JwtUtil {

    // 签名密钥（生产环境从配置中心获取）
    private static final String SECRET_KEY = "your-256-bit-secret-key-here-must-be-long-enough";

    // Token 有效期（毫秒）
    private static final long ACCESS_TOKEN_EXPIRE = 1000 * 60 * 30;       // 30 分钟
    private static final long REFRESH_TOKEN_EXPIRE = 1000 * 60 * 60 * 24 * 7;  // 7 天

    // 签名算法
    private static final SignatureAlgorithm ALGORITHM = SignatureAlgorithm.HS256;

    /**
     * 生成 Access Token
     */
    public static String generateAccessToken(Long userId, String username, List<String> roles) {
        return Jwts.builder()
            // 标准声明
            .subject(userId.toString())
            .issuedAt(new Date())
            .expiration(new Date(System.currentTimeMillis() + ACCESS_TOKEN_EXPIRE))
            // 自定义声明
            .claim("username", username)
            .claim("roles", roles)
            .claim("type", "access")
            // 签名
            .signWith(KEY, ALGORITHM)
            .compact();
    }

    /**
     * 生成 Refresh Token
     */
    public static String generateRefreshToken(Long userId) {
        return Jwts.builder()
            .subject(userId.toString())
            .issuedAt(new Date())
            .expiration(new Date(System.currentTimeMillis() + REFRESH_TOKEN_EXPIRE))
            .claim("type", "refresh")
            // 使用不同的密钥或算法
            .signWith(KEY, ALGORITHM)
            .compact();
    }

    /**
     * 解析 Token，验证签名
     */
    public static Claims parseToken(String token) {
        return Jwts.parser()
            .verifyWith(KEY)
            .build()
            .parseSignedClaims(token)
            .getPayload();
    }

    /**
     * 获取 Token 中的用户 ID
     */
    public static Long getUserId(String token) {
        Claims claims = parseToken(token);
        return Long.parseLong(claims.getSubject());
    }

    /**
     * 获取用户名
     */
    public static String getUsername(String token) {
        Claims claims = parseToken(token);
        return claims.get("username", String.class);
    }

    /**
     * 获取角色列表
     */
    @SuppressWarnings("unchecked")
    public static List<String> getRoles(String token) {
        Claims claims = parseToken(token);
        return claims.get("roles", List.class);
    }

    /**
     * 判断 Token 类型
     */
    public static String getTokenType(String token) {
        Claims claims = parseToken(token);
        return claims.get("type", String.class);
    }

    /**
     * 判断 Token 是否过期
     */
    public static boolean isTokenExpired(String token) {
        try {
            Claims claims = parseToken(token);
            return claims.getExpiration().before(new Date());
        } catch (ExpiredJwtException e) {
            return true;
        }
    }

    /**
     * 验证 Token
     */
    public static boolean validateToken(String token) {
        try {
            parseToken(token);
            return true;
        } catch (JwtException e) {
            return false;
        }
    }
}
```

### Token 服务接口

```java
public interface TokenService {

    /**
     * 生成 Token 对
     */
    TokenPair generateTokens(Long userId, String username, List<String> roles);

    /**
     * 刷新 Access Token
     */
    String refreshAccessToken(String refreshToken);

    /**
     * 注销 Token（加入黑名单）
     */
    void revokeToken(String token);

    /**
     * 验证 Token
     */
    boolean validateToken(String token);

    /**
     * 获取当前用户信息
     */
    UserInfo getCurrentUser(String token);
}

@Data
public class TokenPair {
    private String accessToken;
    private String refreshToken;
    private long expiresIn;
}

@Data
public class UserInfo {
    private Long userId;
    private String username;
    private List<String> roles;
}
```

### Token 服务实现

```java
@Service
public class JwtTokenServiceImpl implements TokenService {

    private final RedisTemplate<String, Object> redisTemplate;
    private static final String BLACKLIST_PREFIX = "jwt:blacklist:";
    private static final String USER_TOKEN_PREFIX = "user:token:";

    public JwtTokenServiceImpl(RedisTemplate<String, Object> redisTemplate) {
        this.redisTemplate = redisTemplate;
    }

    @Override
    public TokenPair generateTokens(Long userId, String username, List<String> roles) {
        // 1. 生成 Access Token 和 Refresh Token
        String accessToken = JwtUtil.generateAccessToken(userId, username, roles);
        String refreshToken = JwtUtil.generateRefreshToken(userId);

        // 2. 存储 Refresh Token（用于撤销）
        String refreshKey = USER_TOKEN_PREFIX + userId + ":refresh";
        redisTemplate.opsForValue().set(refreshKey, refreshToken, Duration.ofDays(7));

        // 3. 记录用户当前有效的 Token（用于单点登录或强制下线）
        String activeKey = USER_TOKEN_PREFIX + userId + ":active";
        redisTemplate.opsForValue().set(activeKey, accessToken, Duration.ofMinutes(30));

        return new TokenPair()
            .setAccessToken(accessToken)
            .setRefreshToken(refreshToken)
            .setExpiresIn(30 * 60);  // 30 分钟
    }

    @Override
    public String refreshAccessToken(String refreshToken) {
        // 1. 验证 Refresh Token
        if (!JwtUtil.validateToken(refreshToken)) {
            throw new TokenException("Refresh Token 无效");
        }

        // 2. 检查是否在黑名单
        if (isTokenBlacklisted(refreshToken)) {
            throw new TokenException("Token 已注销");
        }

        // 3. 获取用户信息
        Long userId = JwtUtil.getUserId(refreshToken);

        // 4. 检查 Refresh Token 是否是当前有效的
        String storedRefreshToken = (String) redisTemplate.opsForValue()
            .get(USER_TOKEN_PREFIX + userId + ":refresh");

        if (!refreshToken.equals(storedRefreshToken)) {
            throw new TokenException("Refresh Token 已过期，请重新登录");
        }

        // 5. 获取用户最新信息（可能有变更）
        User user = getUserById(userId);

        // 6. 生成新的 Access Token
        return JwtUtil.generateAccessToken(user.getId(), user.getUsername(), user.getRoles());
    }

    @Override
    public void revokeToken(String token) {
        // 1. 解析 Token 获取过期时间
        Claims claims = JwtUtil.parseToken(token);
        Date expiration = claims.getExpiration();

        // 2. 计算剩余有效期
        long ttl = expiration.getTime() - System.currentTimeMillis();

        if (ttl > 0) {
            // 3. 加入黑名单
            String blacklistKey = BLACKLIST_PREFIX + token;
            redisTemplate.opsForValue().set(blacklistKey, "revoked", Duration.ofMillis(ttl));

            // 4. 如果是 Refresh Token，清除用户关联的 Token
            String type = claims.get("type", String.class);
            if ("refresh".equals(type)) {
                Long userId = Long.parseLong(claims.getSubject());
                redisTemplate.delete(USER_TOKEN_PREFIX + userId + ":refresh");
                redisTemplate.delete(USER_TOKEN_PREFIX + userId + ":active");
            }
        }
    }

    @Override
    public boolean validateToken(String token) {
        // 1. 检查格式和签名
        if (!JwtUtil.validateToken(token)) {
            return false;
        }

        // 2. 检查黑名单
        if (isTokenBlacklisted(token)) {
            return false;
        }

        return true;
    }

    @Override
    public UserInfo getCurrentUser(String token) {
        if (!validateToken(token)) {
            throw new TokenException("Token 无效");
        }

        return new UserInfo()
            .setUserId(JwtUtil.getUserId(token))
            .setUsername(JwtUtil.getUsername(token))
            .setRoles(JwtUtil.getRoles(token));
    }

    private boolean isTokenBlacklisted(String token) {
        String blacklistKey = BLACKLIST_PREFIX + token;
        return Boolean.TRUE.equals(redisTemplate.hasKey(blacklistKey));
    }
}
```

## 过滤器集成

### JWT 认证过滤器

```java
@Component
public class JwtAuthenticationFilter extends OncePerRequestFilter {

    private final TokenService tokenService;
    private final UserDetailsService userDetailsService;

    public JwtAuthenticationFilter(TokenService tokenService,
                                  UserDetailsService userDetailsService) {
        this.tokenService = tokenService;
        this.userDetailsService = userDetailsService;
    }

    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain)
            throws ServletException, IOException {

        // 1. 从请求头获取 Token
        String token = extractToken(request);

        if (token != null) {
            try {
                // 2. 验证 Token
                if (tokenService.validateToken(token)) {
                    // 3. 获取用户信息
                    UserInfo userInfo = tokenService.getCurrentUser(token);

                    // 4. 加载用户详情并创建认证信息
                    UserDetails userDetails = userDetailsService.loadUserByUsername(userInfo.getUsername());

                    UsernamePasswordAuthenticationToken authentication =
                        new UsernamePasswordAuthenticationToken(
                            userDetails,
                            null,
                            userDetails.getAuthorities()
                        );

                    authentication.setDetails(
                        new WebAuthenticationDetailsSource().buildDetails(request)
                    );

                    // 5. 设置到 Security Context
                    SecurityContextHolder.getContext().setAuthentication(authentication);

                } else {
                    response.sendError(HttpServletResponse.SC_UNAUTHORIZED, "Token 无效");
                    return;
                }

            } catch (TokenException e) {
                response.sendError(HttpServletResponse.SC_UNAUTHORIZED, e.getMessage());
                return;
            }
        }

        filterChain.doFilter(request, response);
    }

    private String extractToken(HttpServletRequest request) {
        // 优先从 Authorization 头获取
        String bearerToken = request.getHeader("Authorization");
        if (bearerToken != null && bearerToken.startsWith("Bearer ")) {
            return bearerToken.substring(7);
        }

        // 也可以从 Cookie 获取
        Cookie[] cookies = request.getCookies();
        if (cookies != null) {
            for (Cookie cookie : cookies) {
                if ("access_token".equals(cookie.getName())) {
                    return cookie.getValue();
                }
            }
        }

        return null;
    }
}
```

### Spring Security 配置

```java
@Configuration
@EnableWebSecurity
public class SecurityConfig {

    @Autowired
    private JwtAuthenticationFilter jwtAuthenticationFilter;

    @Autowired
    private JwtAuthenticationEntryPoint authenticationEntryPoint;

    @Override
    protected void configure(HttpSecurity http) throws Exception {
        http
            // 禁用 CSRF（API 场景下通常不需要）
            .csrf(csrf -> csrf.disable())

            // 禁用 Session
            .sessionManagement(session ->
                session.sessionCreationPolicy(SessionCreationPolicy.STATELESS)
            )

            // 配置权限
            .authorizeHttpRequests(auth -> auth
                .antMatchers("/api/auth/**").permitAll()
                .antMatchers("/public/**").permitAll()
                .antMatchers("/actuator/health").permitAll()
                .anyRequest().authenticated()
            )

            // 异常处理
            .exceptionHandling(ex -> ex
                .authenticationEntryPoint(authenticationEntryPoint)
            )

            // 添加 JWT 过滤器
            .addFilterBefore(jwtAuthenticationFilter, UsernamePasswordAuthenticationFilter.class);
    }
}
```

### 登录与注销接口

```java
@RestController
@RequestMapping("/api/auth")
public class AuthController {

    private final AuthService authService;
    private final TokenService tokenService;

    public AuthController(AuthService authService, TokenService tokenService) {
        this.authService = authService;
        this.tokenService = tokenService;
    }

    /**
     * 登录
     */
    @PostMapping("/login")
    public ResponseEntity<?> login(@RequestBody LoginRequest request) {
        // 1. 验证用户名密码
        User user = authService.authenticate(request.getUsername(), request.getPassword());
        if (user == null) {
            return ResponseEntity.status(401).body("用户名或密码错误");
        }

        // 2. 生成 Token
        TokenPair tokenPair = tokenService.generateTokens(
            user.getId(),
            user.getUsername(),
            user.getRoles()
        );

        return ResponseEntity.ok(tokenPair);
    }

    /**
     * 刷新 Token
     */
    @PostMapping("/refresh")
    public ResponseEntity<?> refresh(@RequestParam String refreshToken) {
        try {
            String accessToken = tokenService.refreshAccessToken(refreshToken);
            return ResponseEntity.ok(Map.of("accessToken", accessToken));
        } catch (TokenException e) {
            return ResponseEntity.status(401).body(e.getMessage());
        }
    }

    /**
     * 注销
     */
    @PostMapping("/logout")
    public ResponseEntity<?> logout(@RequestHeader("Authorization") String authHeader) {
        if (authHeader != null && authHeader.startsWith("Bearer ")) {
            String token = authHeader.substring(7);
            tokenService.revokeToken(token);
        }
        return ResponseEntity.ok("已注销");
    }

    /**
     * 强制下线（管理员操作）
     */
    @PostMapping("/force-logout/{userId}")
    public ResponseEntity<?> forceLogout(@PathVariable Long userId) {
        // 清除用户的所有有效 Token
        String refreshKey = "user:token:" + userId + ":refresh";
        String activeKey = "user:token:" + userId + ":active";
        redisTemplate.delete(refreshKey);
        redisTemplate.delete(activeKey);

        return ResponseEntity.ok("用户已被强制下线");
    }
}
```

## Token 黑名单策略

### 黑名单存储优化

```java
// 使用 Bloom Filter 优化黑名单查询
public class TokenBlacklistService {

    private final RedisTemplate<String, Object> redisTemplate;
    private final BloomFilter<String> bloomFilter;

    public TokenBlacklistService(RedisTemplate<String, Object> redisTemplate) {
        this.redisTemplate = redisTemplate;
        // 预估 1000 万 Token，误判率 1%
        this.bloomFilter = BloomFilter.create(
            Funnels.stringFunnel(StandardCharsets.UTF_8),
            10_000_000,
            0.01
        );
    }

    /**
     * 添加到黑名单
     */
    public void addToBlacklist(String token, long expirationTime) {
        // 1. 添加到 Bloom Filter（用于快速判断）
        bloomFilter.put(token);

        // 2. 存储到 Redis（用于精确验证）
        String key = "jwt:blacklist:" + token;
        long ttl = expirationTime - System.currentTimeMillis();
        if (ttl > 0) {
            redisTemplate.opsForValue().set(key, "1", Duration.ofMillis(ttl));
        }
    }

    /**
     * 检查是否在黑名单
     */
    public boolean isBlacklisted(String token) {
        // 1. 先用 Bloom Filter 快速判断
        if (!bloomFilter.mightContain(token)) {
            return false;  // 一定不在黑名单
        }

        // 2. Bloom Filter 误判，精确查询 Redis
        String key = "jwt:blacklist:" + token;
        return Boolean.TRUE.equals(redisTemplate.hasKey(key));
    }
}
```

## 常见问题与解决

### 问题一：Token 泄露怎么办？

```java
// 解决方案：Token 追踪 + 立即撤销
public class TokenTracker {

    private final RedisTemplate<String, Object> redisTemplate;

    // 每个 Token 绑定唯一 ID
    public String generateTokenWithId(Long userId, String tokenId) {
        // tokenId 用于追踪和撤销
        return JwtUtil.generateAccessTokenWithId(userId, tokenId);
    }

    // 撤销特定 Token
    public void revokeByTokenId(String tokenId) {
        String key = "token:id:" + tokenId;
        String token = (String) redisTemplate.opsForValue().get(key);
        if (token != null) {
            tokenBlacklistService.addToBlacklist(token, JwtUtil.getExpiration(token));
        }
    }

    // 撤销用户所有 Token
    public void revokeAllUserTokens(Long userId) {
        String pattern = "user:tokens:" + userId + ":*";
        Set<String> keys = redisTemplate.keys(pattern);
        if (keys != null && !keys.isEmpty()) {
            redisTemplate.delete(keys);
        }
    }
}
```

### 问题二：如何实现单点登录？

```java
// 单点登录：同一用户只能有一个有效 Token
public class SingleSignOnService {

    private final RedisTemplate<String, Object> redisTemplate;
    private static final String SSO_KEY_PREFIX = "sso:user:";

    public String generateTokenWithSSO(Long userId) {
        String ssoKey = SSO_KEY_PREFIX + userId;

        // 获取当前有效的 Token
        String currentToken = (String) redisTemplate.opsForValue().get(ssoKey);

        if (currentToken != null) {
            // 撤销旧 Token
            Claims claims = JwtUtil.parseToken(currentToken);
            Date expiration = claims.getExpiration();
            tokenBlacklistService.addToBlacklist(currentToken, expiration.getTime());
        }

        // 生成新 Token
        String newToken = JwtUtil.generateAccessToken(userId);

        // 存储新 Token
        redisTemplate.opsForValue().set(
            ssoKey,
            newToken,
            Duration.ofMinutes(30)
        );

        return newToken;
    }
}
```

## 总结

无状态 JWT 的核心优势：

| 特性 | Session | JWT |
|---|---|---|
| 存储方式 | 服务端存储 | 客户端存储 |
| 扩展性 | 需要 Redis 共享 | 天然支持集群 |
| 性能 | 每次请求查 Redis | 无需查询 |
| 撤销 | 立即生效 | 需要黑名单 |

JWT 的安全最佳实践：

1. **Token 不要存储敏感信息**（密码等）
2. **使用 HTTPS 传输**
3. **设置合理的过期时间**（Access Token 短，Refresh Token 长）
4. **实现 Token 撤销机制**（黑名单或短 Token 策略）
5. **使用强签名密钥**（HS256 至少 256 位，RSA 至少 2048 位）

记住：**JWT 是签名的，不是加密的**。如果需要存储敏感信息，应该使用 JWE（JSON Web Encryption）而不是 JWT。
