# CAS 单点登录原理与流程

你在公司内部系统登录了 OA，发了一封邮件，顺手点开了财务系统——咦，为什么不需要再登录了？

这就是单点登录（Single Sign-On，SSO）的魅力。用户一次登录，处处通行。CAS（Central Authentication Service）就是实现 SSO 的经典方案之一。

## SSO 的核心问题

在没有 SSO 的世界里，每个系统都有自己的登录模块。用户每访问一个新系统，都要输入用户名密码。这带来三个问题：

1. **用户体验差**：员工可能需要记忆十几个系统的账号密码
2. **安全风险高**：用户在多个系统使用相同密码，一个泄露全部遭殃
3. **运维成本高**：密码重置、权限变更要在每个系统单独处理

SSO 要解决的核心问题是：**用户登录一次，如何让其他系统知道？**

## CAS 协议原理

CAS 是 Yale 大学开源的单点登录方案，其核心思想是：**所有系统的认证都集中到一个认证中心**。

### 三个核心票据

| 票据 | 全称 | 作用 | 存储位置 |
|---|---|---|---|
| TGT | Ticket Granting Ticket | 长期凭证，证明已登录 | 认证中心（Session） |
| ST | Service Ticket | 短期凭证，一次性使用 | 仅在 URL 中传递 |
| TGC | Ticket Granting Cookie | TGT 的 Cookie | 浏览器 Cookie |

## CAS 服务端实现

### 核心控制器

```java
// CAS 认证中心核心服务
@Service
public class CasAuthenticationService {

    @Autowired
    private TicketRegistry ticketRegistry;

    @Autowired
    private UserService userService;

    /**
     * 登录接口
     * 1. 验证用户名密码
     * 2. 生成 TGT
     * 3. 设置 Cookie
     */
    public LoginResult login(HttpServletRequest request, HttpServletResponse response,
                            String username, String password) {
        // 1. 验证用户
        User user = userService.authenticate(username, password);
        if (user == null) {
            return LoginResult.failure("用户名或密码错误");
        }

        // 2. 创建 TGT（Ticket Granting Ticket）
        String tgtId = generateTGTId();
        TicketGrantingTicket tgt = new TicketGrantingTicket(tgtId, user, Duration.ofHours(24));

        // 3. 存储到 Session Registry
        ticketRegistry.addTGT(tgt);

        // 4. 设置 Cookie
        Cookie ticketGrantingCookie = new Cookie("CASTGC", tgtId);
        ticketGrantingCookie.setPath("/cas");
        ticketGrantingCookie.setHttpOnly(true);
        ticketGrantingCookie.setSecure(true);
        ticketGrantingCookie.setMaxAge(24 * 60 * 60);  // 24 小时
        response.addCookie(ticketGrantingCookie);

        return LoginResult.success(user);
    }

    /**
     * 验证 ST 并返回用户信息
     * 这个接口由应用系统调用，用于验证用户身份
     */
    public AuthenticationResult verifyServiceTicket(String service, String serviceTicket) {
        // 1. 从 ST 中解析出 TGT ID
        String tgtId = extractTGTIdFromST(serviceTicket);

        // 2. 验证 TGT 是否存在且有效
        TicketGrantingTicket tgt = ticketRegistry.getTGT(tgtId);
        if (tgt == null || tgt.isExpired()) {
            return AuthenticationResult.failure("登录已过期");
        }

        // 3. 生成 PGT（Proxy Granting Ticket）用于代理
        String pgtId = generatePGTId();
        String pgt = tgt.grantProxyGrantingTicket(pgtId, service);
        ticketRegistry.addPGT(pgt);

        // 4. 验证通过，返回用户信息
        return AuthenticationResult.success(tgt.getUser(), pgtId);
    }

    /**
     * 退出登录
     * 清除 TGT 和相关票据
     */
    public void logout(HttpServletRequest request, HttpServletResponse response) {
        Cookie cookie = findCookie(request, "CASTGC");
        if (cookie != null) {
            String tgtId = cookie.getValue();
            ticketRegistry.removeTGT(tgtId);

            // 清除 Cookie
            cookie.setMaxAge(0);
            response.addCookie(cookie);
        }
    }
}
```

### 票据注册表

```java
// 票据存储接口
public interface TicketRegistry {
    void addTGT(TicketGrantingTicket tgt);
    TicketGrantingTicket getTGT(String tgtId);
    void removeTGT(String tgtId);

    void addST(ServiceTicket st);
    ServiceTicket getST(String stId);
    void removeST(String stId);

    void addPGT(ProxyGrantingTicket pgt);
    String getPGT(String pgtId);
}

// 基于 Redis 的实现（生产环境推荐）
@Service
public class RedisTicketRegistry implements TicketRegistry {

    private final RedisTemplate<String, Object> redisTemplate;

    public RedisTicketRegistry(RedisTemplate<String, Object> redisTemplate) {
        this.redisTemplate = redisTemplate;
    }

    @Override
    public void addTGT(TicketGrantingTicket tgt) {
        String key = "tgt:" + tgt.getId();
        redisTemplate.opsForValue().set(key, tgt, Duration.ofHours(24));
    }

    @Override
    public TicketGrantingTicket getTGT(String tgtId) {
        String key = "tgt:" + tgtId;
        return (TicketGrantingTicket) redisTemplate.opsForValue().get(key);
    }

    @Override
    public void removeTGT(String tgtId) {
        String key = "tgt:" + tgtId;
        redisTemplate.delete(key);
    }

    @Override
    public void addST(ServiceTicket st) {
        String key = "st:" + st.getId();
        // ST 有效期较短，通常 10 秒
        redisTemplate.opsForValue().set(key, st, Duration.ofSeconds(10));
    }

    @Override
    public ServiceTicket getST(String stId) {
        String key = "st:" + stId;
        ServiceTicket st = (ServiceTicket) redisTemplate.opsForValue().get(key);
        // 读取后立即删除，实现一次性使用
        if (st != null) {
            redisTemplate.delete(key);
        }
        return st;
    }
}
```

## CAS 客户端实现

### Spring Security 集成

```java
// CAS 客户端配置
@Configuration
@EnableWebSecurity
public class SecurityConfig extends WebSecurityConfigurerAdapter {

    @Value("${cas.server.url}")
    private String casServerUrl;

    @Value("${cas.server.callback-url}")
    private String serverCallbackUrl;

    @Value("${app.server.url}")
    private String appServerUrl;

    @Override
    protected void configure(HttpSecurity http) throws Exception {
        http
            .authorizeRequests()
                .antMatchers("/public/**", "/login/**").permitAll()
                .anyRequest().authenticated()
                .and()
            .csrf()
                .disable()
            .addFilterAt(
                casAuthenticationFilter(),
                UsernamePasswordAuthenticationFilter.class
            );
    }

    @Bean
    public CasAuthenticationFilter casAuthenticationFilter() {
        CasAuthenticationFilter filter = new CasAuthenticationFilter();
        filter.setAuthenticationManager(authenticationManager());
        filter.setServiceProperties(serviceProperties());
        return filter;
    }

    @Bean
    public ServiceProperties serviceProperties() {
        ServiceProperties sp = new ServiceProperties();
        sp.setService(appServerUrl + "/login/cas");
        sp.setSendRenew(false);  // 是否强制使用新票据
        sp.setArtifactParameterName("ticket");
        return sp;
    }

    @Bean
    public CasAuthenticationProvider casAuthenticationProvider() {
        CasAuthenticationProvider provider = new CasAuthenticationProvider();
        provider.setTicketValidator(cas20ServiceTicketValidator());
        provider.setUserDetailsService(userDetailsService());
        provider.setKey("CasAuthenticationProvider");
        return provider;
    }

    @Bean
    public Cas20ServiceTicketValidator cas20ServiceTicketValidator() {
        return new Cas20ServiceTicketValidator(casServerUrl);
    }
}
```

### 认证过滤器

```java
// 自定义 CAS 认证过滤器
public class CustomCasAuthenticationFilter extends GenericFilterBean {

    private AuthenticationManager authenticationManager;
    private ServiceProperties serviceProperties;

    @Override
    public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain)
            throws IOException, ServletException {

        HttpServletRequest httpRequest = (HttpServletRequest) request;
        HttpServletResponse httpResponse = (HttpServletResponse) response;

        // 从请求中获取 ST（ticket 参数）
        String ticket = httpRequest.getParameter("ticket");

        if (ticket != null) {
            // 有票据，尝试认证
            try {
                ServiceRequestDTO serviceRequest = new ServiceRequestDTO();
                serviceRequest.setTicket(ticket);
                serviceRequest.setService(getServiceUrl(httpRequest));

                Authentication auth = authenticationManager.authenticate(
                    new TicketAuthenticationToken(serviceRequest)
                );

                // 认证成功，存入 SecurityContext
                SecurityContextHolder.getContext().setAuthentication(auth);
                httpRequest.getSession().setAttribute("SPRING_SECURITY_CONTEXT", auth);

            } catch (AuthenticationException e) {
                // 认证失败
                SecurityContextHolder.clearContext();
            }
        }

        // 检查 Session 中是否有认证信息
        if (SecurityContextHolder.getContext().getAuthentication() == null) {
            // 未认证，重定向到 CAS 登录页
            String casLoginUrl = casServerUrl + "/login"
                + "?service=" + URLEncoder.encode(getServiceUrl(httpRequest), "UTF-8");
            httpResponse.sendRedirect(casLoginUrl);
            return;
        }

        chain.doFilter(request, response);
    }
}
```

## CAS 注销与单点登出

### 单点登出协议（SLO）

当用户在任何一个系统注销时，需要通知其他系统也注销：

```java
// CAS 注销监听器
@Bean
public ListenerServlet listenerServlet() {
    return new ListenerServlet()
        .addListener(new LogoutListener() {
            @Override
            public void onLogout(LogoutEvent event) {
                // 清除本地 Session
                HttpSession session = event.getSession();
                if (session != null) {
                    session.invalidate();
                }
                SecurityContextHolder.clearContext();
            }
        });
}
```

### 注销接口

```java
// 统一注销接口
@RestController
@RequestMapping("/logout")
public class LogoutController {

    @Autowired
    private CasLogoutService casLogoutService;

    @GetMapping
    public ResponseEntity<?> logout(HttpServletRequest request, HttpServletResponse response) {
        // 1. 清除本地认证信息
        SecurityContextHolder.clearContext();
        request.getSession().invalidate();

        // 2. 通知 CAS 注销
        String redirectUrl = casLogoutService.getRedirectUrl(request);

        return ResponseEntity.ok()
            .header("Location", redirectUrl)
            .build();
    }
}
```

## 常见问题与解决

### 问题一：跨域 Cookie 无法写入

CAS 的 TGC Cookie 需要设置在 CAS 域下，而应用系统在子域下：

```java
// 方案：设置父域 Cookie
Cookie cookie = new Cookie("CASTGC", tgtId);
cookie.setDomain(".company.com");  // 设置为父域
cookie.setPath("/cas");
response.addCookie(cookie);
```

## 总结

CAS 单点登录的核心是**票据机制**：

| 组件 | 作用 |
|---|---|
| TGT | 长期票据，存储在认证中心 |
| ST | 短期票据，一次性使用 |
| TGC | Cookie，用于传递 TGT |
| Service | 应用系统，需要在认证中心注册 |

CAS 适合的场景：

- 企业内部多系统统一认证
- 需要严格的统一登出
- 对安全性要求高的场景

不适用的场景：

- 面向外部用户的认证（用 OAuth2 更合适）
- 需要第三方登录的场景
