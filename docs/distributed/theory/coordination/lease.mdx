# Lease 机制

<Badge text="分布式系统" type="tip" /> <Badge text="租约" type="warning" /> <Badge text="缓存管理" type="info" />

在分布式系统中，如何优雅地管理缓存授权？你可能会想到设置一个过期时间，但如果在过期前缓存服务器崩溃了，其他服务器怎么知道这个缓存已经失效了？

**Lease 机制**就是来解决这个问题的——它提供了一种有租约的授权方式，授权者在租约有效期内保证授权有效，租约过期后自动失效。

## 为什么需要 Lease？

```java
/**
 * 传统缓存失效的问题
 */

// 场景：缓存服务器 S1 缓存了数据，D1、D2 是数据消费者
// 如果数据被更新，传统 TTL 的问题：

// 1. 数据在 T1 时刻被更新
// 2. 缓存服务器 S1 还没来得及失效
// 3. D1 在 T2 时刻读取，可能读到旧数据

// 解决方案：设置缓存失效时间
// 但这带来了新问题：

// 1. 失效时间太短：缓存频繁失效，热点数据被反复加载
// 2. 失效时间太长：数据更新后，用户可能长时间看到旧数据

// Lease 的解决方案：
// - 授权者（数据源）给缓存服务器发放一个「租约」
// - 租约有明确的有效期
// - 在租约有效期内，授权者承诺数据不会被更新
// - 租约过期后，授权者可以自由更新数据
```

## Lease 的基本原理

```java
/**
 * Lease 机制的核心
 */
public class LeaseMechanism {
    
    /**
     * Lease 结构
     */
    static class Lease {
        long startTime;    // 租约开始时间
        long duration;     // 租约时长
        String holder;     // 持有者 ID
        
        public boolean isValid() {
            return System.currentTimeMillis() - startTime < duration;
        }
        
        public long remainingTime() {
            long elapsed = System.currentTimeMillis() - startTime;
            return Math.max(0, duration - elapsed);
        }
    }
    
    /**
     * Lease 的语义
     * 
     * 1. 授权者承诺：
     *    在 Lease 有效期内，不会单方面撤销授权
     * 
     * 2. 持有者承诺：
     *    在 Lease 过期后，不再使用授权
     * 
     * 3. 自动过期：
     *    Lease 到期后自动失效，无需额外通知
     */
}

/**
 * Lease vs 锁
 */

// 锁：
// - 需要主动释放
// - 持有者崩溃会导致死锁
// - 需要复杂的死锁检测

// Lease：
// - 自动过期
// - 持有者崩溃，Lease 自动失效
// - 无死锁风险
```

## Lease 的应用场景

### 1. 分布式缓存一致性

```java
/**
 * Lease 在缓存中的应用
 */
public class LeaseCacheClient {
    
    private LeaseServer leaseServer;
    private Map<String, Lease> localLeases = new ConcurrentHashMap<>();
    private static final long LEASE_DURATION = 10 * 1000;  // 10 秒
    
    /**
     * 读取数据
     */
    public Data read(String key) {
        // 1. 检查本地 Lease
        Lease localLease = localLeases.get(key);
        if (localLease != null && localLease.isValid()) {
            // Lease 有效，直接返回本地数据
            return localCache.get(key);
        }
        
        // 2. Lease 无效或不存在，从数据源获取
        Data data = dataSource.read(key);
        
        // 3. 申请 Lease
        Lease lease = leaseServer.grantLease(key, LEASE_DURATION);
        localLeases.put(key, lease);
        localCache.put(key, data);
        
        return data;
    }
    
    /**
     * 数据源更新数据
     */
    public void update(String key, Data newData) {
        // 1. 等待所有 Lease 过期
        waitForLeasesToExpire(key);
        
        // 2. 更新数据
        dataSource.write(key, newData);
        
        // 3. 通知客户端（可选）
        notifyClients(key);
    }
    
    private void waitForLeasesToExpire(String key) {
        // 策略：等待 Lease 最长有效期
        // 或者使用其他机制（如版本号）检测 Lease 失效
        try {
            Thread.sleep(LEASE_DURATION);
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
    }
}

/**
 * Lease 服务器
 */
public class LeaseServer {
    
    private Map<String, Lease> activeLeases = new ConcurrentHashMap<>();
    
    /**
     * 发放 Lease
     */
    public Lease grantLease(String resource, long duration) {
        Lease lease = new Lease(
            System.currentTimeMillis(),
            duration,
            getCurrentHolderId()
        );
        
        activeLeases.put(resource, lease);
        
        return lease;
    }
    
    /**
     * 检查 Lease 是否有效
     */
    public boolean isLeaseValid(String resource) {
        Lease lease = activeLeases.get(resource);
        return lease != null && lease.isValid();
    }
    
    /**
     * 获取资源的 Lease 信息
     */
    public LeaseInfo getLeaseInfo(String resource) {
        Lease lease = activeLeases.get(resource);
        if (lease == null) {
            return new LeaseInfo(LeaseStatus.NO_LEASE, 0);
        }
        if (lease.isValid()) {
            return new LeaseInfo(LeaseStatus.VALID, lease.remainingTime());
        }
        return new LeaseInfo(LeaseStatus.EXPIRED, 0);
    }
}
```

### 2. 主节点选举

```java
/**
 * Lease 在主节点选举中的应用
 */
public class LeaseBasedLeaderElection {
    
    private LeaseRegistry leaseRegistry;
    private static final String LEADER_RESOURCE = "cluster:leader";
    private static final long LEASE_DURATION = 5 * 1000;  // 5 秒
    
    private volatile boolean isLeader = false;
    
    /**
     * 尝试成为领导者
     */
    public boolean tryBecomeLeader() {
        // 1. 尝试获取 Lease
        Lease lease = leaseRegistry.tryAcquire(LEADER_RESOURCE, LEASE_DURATION);
        
        if (lease != null) {
            isLeader = true;
            startLeaderHeartbeat(lease);
            return true;
        }
        
        return false;
    }
    
    /**
     * 领导者心跳：续租
     */
    private void startLeaderHeartbeat(Lease lease) {
        ScheduledExecutorService executor = 
            Executors.newSingleThreadScheduledExecutor();
        
        // 心跳间隔为 Lease 时长的 1/3
        long heartbeatInterval = LEASE_DURATION / 3;
        
        executor.scheduleAtFixedRate(() -> {
            if (!isLeader) {
                executor.shutdown();
                return;
            }
            
            // 续租
            boolean renewed = leaseRegistry.renew(LEADER_RESOURCE, LEASE_DURATION);
            if (!renewed) {
                // 续租失败，失去领导权
                isLeader = false;
                onLoseLeadership();
                executor.shutdown();
            }
        }, heartbeatInterval, heartbeatInterval, TimeUnit.MILLISECONDS);
    }
    
    /**
     * 失去领导权后的处理
     */
    private void onLoseLeadership() {
        isLeader = false;
        
        // 等待一段时间后重新尝试
        try {
            Thread.sleep(1000 + new Random().nextInt(2000));
            tryBecomeLeader();
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
    }
}
```

### 3. 分布式文件系统元数据管理

```java
/**
 * Lease 在分布式文件系统中的应用
 * 
 * 典型场景：GFS/HDFS 的 Chunk Lease
 */
public class ChunkLeaseManager {
    
    /**
     * Chunk Lease 结构
     */
    static class ChunkLease {
        String chunkHandle;      // Chunk 句柄
        String primaryServer;    // 主副本服务器
        long expirationTime;    // 过期时间
        int version;            // Chunk 版本号
    }
    
    /**
     * 主副本选举
     */
    public void electPrimary(ChunkHandle chunk) {
        // 1. 获取所有副本服务器
        List<String> replicas = chunk.getReplicas();
        
        // 2. 计算过期时间
        long now = System.currentTimeMillis();
        long leaseDuration = 60 * 1000;  // 60 秒
        long expirationTime = now + leaseDuration;
        
        // 3. 选择第一个可用的副本作为主副本
        for (String replica : replicas) {
            if (isServerAlive(replica)) {
                ChunkLease lease = new ChunkLease(
                    chunk.getHandle(),
                    replica,
                    expirationTime,
                    chunk.getVersion()
                );
                
                // 4. 发放 Lease
                grantLease(lease);
                
                // 5. 通知所有副本谁是主副本
                notifyPrimaryToAll(chunk, replica);
                
                return;
            }
        }
        
        throw new NoAvailableReplicaException();
    }
    
    /**
     * 主副本的数据写入流程
     */
    public void write(ChunkHandle chunk, ByteBuffer data) {
        ChunkLease lease = getLease(chunk);
        
        // 只有主副本可以处理写请求
        if (!lease.getPrimaryServer().equals(getCurrentServer())) {
            throw new NotPrimaryException();
        }
        
        // 1. 主副本接收数据
        // 2. 主副本将数据推送给所有从副本
        for (String replica : lease.getReplicaServers()) {
            if (!replica.equals(lease.getPrimaryServer())) {
                pushDataToReplica(replica, data);
            }
        }
        
        // 3. 等待所有副本确认
        waitForAcks(chunk);
        
        // 4. 更新版本号，续租
        chunk.incrementVersion();
        renewLease(lease);
    }
}
```

## Lease 的续租机制

```java
/**
 * Lease 续租
 */
public class LeaseRenewal {
    
    /**
     * 续租策略
     */
    
    // 1. 固定续租
    // - 以固定间隔续租
    // - 简单但可能浪费资源
    public void fixedRenewal(Lease lease, long interval) {
        while (lease.isValid()) {
            sleep(interval);
            renew(lease);
        }
    }
    
    // 2. 智能续租
    // - 根据成功率动态调整续租间隔
    // - 成功率高时延长间隔，失败时缩短
    public void adaptiveRenewal(Lease lease) {
        long interval = INITIAL_INTERVAL;
        double successRate = 1.0;
        
        while (lease.isValid()) {
            sleep(interval);
            
            boolean success = tryRenew(lease);
            if (success) {
                successRate = 0.9 * successRate + 0.1 * 1.0;
            } else {
                successRate = 0.9 * successRate;
            }
            
            // 根据成功率调整间隔
            interval = Math.max(MIN_INTERVAL, 
                Math.min(MAX_INTERVAL, 
                    interval * successRate * 1.1));
        }
    }
    
    // 3. 异步续租
    // - 在后台线程续租，不阻塞主线程
    public void asyncRenewal(Lease lease) {
        CompletableFuture.runAsync(() -> {
            while (!Thread.currentThread().isInterrupted() && lease.isValid()) {
                try {
                    renew(lease);
                    Thread.sleep(RENEWAL_INTERVAL);
                } catch (InterruptedException e) {
                    break;
                }
            }
        });
    }
}
```

## Lease 与时钟偏移

```java
/**
 * Lease 与时钟偏移
 * 
 * 问题：如果持有者的时钟快于授权者
 * Lease 可能看起来还没过期，但授权者已经过期了
 */
public class ClockSkewHandling {
    
    private static final long CLOCK_SKEW_MARGIN = 1 * 1000;  // 1 秒
    
    /**
     * 安全地检查 Lease 有效性
     * 
     * 考虑时钟偏移
     */
    public boolean isLeaseValidWithSkew(Lease lease) {
        // 如果 Lease 马上就要过期，保守地认为已过期
        long remaining = lease.remainingTime();
        return remaining > CLOCK_SKEW_MARGIN;
    }
    
    /**
     * 计算安全的 Lease 有效时间
     */
    public long getSafeRemainingTime(Lease lease) {
        long remaining = lease.remainingTime();
        return Math.max(0, remaining - CLOCK_SKEW_MARGIN);
    }
    
    /**
     * 提前刷新策略
     * 
     * 在 Lease 还剩 1/3 有效期时就开始续租
     */
    public void proactiveRenewal(Lease lease) {
        long remaining = lease.remainingTime();
        long duration = lease.getDuration();
        
        // 提前 1/3 时长续租
        if (remaining < duration * 2 / 3) {
            renew(lease);
        }
    }
}
```

## Lease 的优缺点

```java
/**
 * Lease 的优点
 */

// 1. 自动过期，无需显式释放
// - 持有者崩溃后，Lease 自动失效
// - 不需要死锁检测机制

// 2. 授权者有明确的承诺期
// - 在 Lease 有效期内，授权者不会单方面撤销
// - 持有者可以放心使用授权

// 3. 灵活性高
// - 可以设置不同的 Lease 时长
// - 可以续租、转让、提前撤销

// 4. 性能好
// - Lease 检查是 O(1) 操作
// - 不需要复杂的协调机制

/**
 * Lease 的缺点
 */

// 1. 时间精度依赖时钟
// - 时钟偏移可能导致 Lease 提前或延后失效
// - 需要额外的时钟偏移保护机制

// 2. Lease 有效期难以确定
// - 太长：故障恢复慢
// - 太短：续租开销大

// 3. 不适合强一致性场景
// - Lease 有效期内数据可能已过时
// - 需要额外的版本检查机制
```

## 总结

Lease 机制是一种优雅的分布式授权协议：

1. **有租约的授权**：授权者承诺在租约有效期内不单方面撤销
2. **自动过期**：持有者崩溃无需担心锁释放问题
3. **灵活续租**：可以根据需要续租或提前释放

从分布式缓存到主节点选举，从文件系统到分布式锁，Lease 机制在众多分布式系统中发挥着重要作用。
