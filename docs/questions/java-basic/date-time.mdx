## 一、旧 API 的问题

### 1.1 Date 类的问题

```java
// [!code warning] Java 1.0 的 Date 类
Date date = new Date(2024, 1, 1);  // [!code warning] 月份从 1 开始？错！从 0 开始！

// [!code warning] Date 可变，线程不安全
Date date = new Date();
date.setTime(date.getTime() + 1000);  // 修改了原对象
```

### 1.2 SimpleDateFormat 的问题

```java
// [!code warning] SimpleDateFormat 线程不安全
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");

// 多线程环境下共享会导致异常
ExecutorService pool = Executors.newFixedThreadPool(10);
pool.submit(() -> {
    Date date = sdf.parse("2024-01-01");  // [!code warning] 可能抛出异常
});
```

:::warning 旧 API 的三大问题
1. **可变**：Date 可被修改
2. **线程不安全**：SimpleDateFormat 不是线程安全的
3. **设计混乱**：月份从 0 开始，难以理解
:::

## 二、新 API 的核心类

### 2.1 三个核心类

| 类 | 说明 | 示例 |
|------|------|------|
| LocalDate | 日期（年-月-日） | 2024-01-01 |
| LocalTime | 时间（时:分:秒） | 14:30:00 |
| LocalDateTime | 日期时间 | 2024-01-01 14:30:00 |
| Instant | 时间戳 | 1704112200000 |
| Duration | 时间间隔 | 2小时30分 |
| Period | 日期间隔 | 2天 |

### 2.2 创建对象

```java
// 获取当前日期时间
LocalDate date = LocalDate.now();
LocalTime time = LocalTime.now();
LocalDateTime dateTime = LocalDateTime.now();

// 指定日期时间
LocalDate date = LocalDate.of(2024, 1, 15);
LocalTime time = LocalTime.of(14, 30, 0);
LocalDateTime dateTime = LocalDateTime.of(2024, 1, 15, 14, 30);
```

## 三、常用操作

### 3.1 日期计算

```java
LocalDate today = LocalDate.now();
LocalDate tomorrow = today.plusDays(1);
LocalDate nextWeek = today.plusWeeks(1);
LocalDate nextMonth = today.plusMonths(1);
LocalDate nextYear = today.plusYears(1);

LocalDate yesterday = today.minusDays(1);
```

### 3.2 日期比较

```java
LocalDate date1 = LocalDate.of(2024, 1, 1);
LocalDate date2 = LocalDate.of(2024, 1, 2);

boolean isAfter = date1.isAfter(date2);   // false
boolean isBefore = date1.isBefore(date2); // true
boolean isEqual = date1.isEqual(date2);   // false
```

### 3.3 日期组件提取

```java
LocalDate date = LocalDate.of(2024, 1, 15);

int year = date.getYear();       // 2024
int month = date.getMonthValue(); // 1
DayOfWeek dayOfWeek = date.getDayOfWeek(); // MONDAY
int day = date.getDayOfMonth(); // 15
```

## 四、格式化与解析

### 4.1 DateTimeFormatter

```java
import java.time.format.DateTimeFormatter;

// 格式化
LocalDate date = LocalDate.of(2024, 1, 15);
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
String formatted = date.format(formatter);  // "2024-01-15"

// 解析
LocalDate parsed = LocalDate.parse("2024-01-15", formatter);
```

### 4.2 常用格式

```java
DateTimeFormatter f1 = DateTimeFormatter.ofPattern("yyyy-MM-dd");           // 2024-01-15
DateTimeFormatter f2 = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");  // 2024-01-15 14:30:00
DateTimeFormatter f3 = DateTimeFormatter.ofPattern("yyyy年MM月dd日");        // 2024年01月15日
DateTimeFormatter f4 = DateTimeFormatter.ISO_LOCAL_DATE;                    // 2024-01-15
```

## 五、时区处理

### 5.1 ZonedDateTime

```java
// 当前时区
ZonedDateTime now = ZonedDateTime.now();

// 指定时区
ZonedDateTime tokyoTime = ZonedDateTime.now(ZoneId.of("Asia/Tokyo"));
ZonedDateTime utcTime = ZonedDateTime.now(ZoneOffset.UTC);

// 时区转换
ZonedDateTime tokyo = ZonedDateTime.of(2024, 1, 15, 14, 30, 0, 0, ZoneId.of("Asia/Tokyo"));
ZonedDateTime utc = tokyo.withZoneSameInstant(ZoneOffset.UTC);
```

### 5.2 Instant

```java
// Instant 是时间戳
Instant now = Instant.now();
long epochSecond = now.getEpochSecond();  // 秒
long epochMilli = now.toEpochMilli();    // 毫秒

// Instant 与 Date 互转
Date date = Date.from(instant);
Instant instant = date.toInstant();
```

## 七、常见错误与陷阱

### ⚠️ 陷阱 1：误用旧 API

```java
// [!code warning] 错误：使用旧的 Date
Date date = new Date();
date.setMonth(0);  // 1 月
date.setYear(2024 - 1900);  // [!code warning] 需要减去 1900

// [!code highlight] 正确：使用新 API
LocalDate date = LocalDate.of(2024, 1, 1);
```

### ⚠️ 陷阱 2：日期格式化

```java
// [!code error] 错误：月份用大写 MM
DateTimeFormatter.ofPattern("yyyy-mm-dd");  // [!code error] mm 是分钟！

// [!code highlight] 正确：月份用大写 MM
DateTimeFormatter.ofPattern("yyyy-MM-dd");
```

### ⚠️ 陷阱 3：线程安全问题

```java
// [!code warning] DateTimeFormatter 线程安全，但仍不建议共享可变状态
// [!code highlight] 如果需要共享，使用 ThreadLocal
ThreadLocal<DateTimeFormatter> formatter = ThreadLocal.withInitial(
    () -> DateTimeFormatter.ofPattern("yyyy-MM-dd")
);
```

## 八、加分回答

💡 **超出预期的深度**：

### 1. Duration vs Period

```java
// Duration：时间间隔（用于机器时间）
Duration duration = Duration.between(
    Instant.now(),
    Instant.now().plusSeconds(3600)
);

// Period：日期间隔（用于人类时间）
Period period = Period.between(
    LocalDate.of(2024, 1, 1),
    LocalDate.of(2024, 12, 31)
);
```

### 2. 时间窗口计算

```java
// 计算一天的开始和结束
LocalDate date = LocalDate.of(2024, 1, 15);
LocalDateTime start = date.atStartOfDay();
LocalDateTime end = date.plusDays(1).atStartOfDay().minusNanos(1);
```
