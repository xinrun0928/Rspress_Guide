# Java 基础 40 题

> **目标级别**：P5/P6
> **面试频率**：Java 基础题是所有技术面试的第一道门槛，无论你是校招还是社招，这 40 道题都是绕不过去的基础关卡。

面试官问：「面向对象的三大特性是什么？」你脱口而出「封装、继承、多态」——然后面试官紧接着追问「多态是怎么实现的？向上转型有什么代价？」你沉默了。

这就是 Java 基础题的真实面貌：表面上问的是概念，实际上考的是原理深度。90% 的候选人在第一层回答得头头是道，但被追问到第二层、第三层时就露馅了。

本专题精选 Java 基础中最具面试价值的 40 道高频题，从「是什么」到「为什么」再到「怎么答」，帮助你建立完整的 Java 基础知识体系。

## 学习路径建议

### P5 级别（1-3 年经验）

重点掌握前 25 题，确保每道题能回答到第二层追问。重点关注：
- 面向对象三大特性
- `equals` 与 `hashCode` 关系
- 异常处理机制
- 反射与泛型基础

### P6 级别（3-5 年经验）

全部 40 题都需要掌握，确保能回答到第三层追问。重点关注：
- 泛型类型擦除的底层实现
- 反射的性能优化
- Stream 与 Lambda 的原理
- 设计模式中的多态应用

## 常见面试场景

面试官通常不会直接问「Java 基础怎么样」，而是把基础题融入到场景题或源码分析中：

- 「看你用过 HashMap，那 `equals` 和 `hashCode` 有什么关系？」
- 「你在项目中用过反射吗？为什么用它？」
- 「String 为什么设计成不可变的？」
- 「try-with-resources 的原理是什么？」

这些追问才是真正考验 Java 基础的时候。所以不要只背答案，要理解背后的原理。
