## 场景切入

面试官问：「方法引用是什么？」你说「像 `String::length` 这种」——然后面试官追问「那 `System.out::println` 是什么类型的方法引用？」你愣住了。

方法引用是 Lambda 表达式的简化写法，理解它的四种类型是基础要求。

## 一、方法引用的四种类型

### 1.1 类型对照表

| 类型 | 语法 | 示例 |
|------|------|------|
| 静态方法引用 | `ClassName::staticMethod` | `String::valueOf` |
| 实例方法引用（特定对象） | `object::instanceMethod` | `System.out::println` |
| 实例方法引用（任意对象） | `ClassName::instanceMethod` | `String::length` |
| 构造器引用 | `ClassName::new` | `ArrayList::new` |

---

## 二、静态方法引用

### 2.1 基本语法

```java
// Lambda 形式
Function<Integer, String> lambda = x -> String.valueOf(x);

// [!code highlight] 方法引用形式
Function<Integer, String> ref = String::valueOf;  // [!code highlight]

// 使用
System.out.println(ref.apply(100));  // "100"
```

### 2.2 示例

```java
List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// Lambda
numbers.stream()
    .map(x -> String.valueOf(x))
    .collect(Collectors.toList());

// [!code highlight] 方法引用
numbers.stream()
    .map(String::valueOf)  // [!code highlight]
    .collect(Collectors.toList());
```

## 三、实例方法引用（特定对象）

### 3.1 基本语法

```java
String prefix = "Hello, ";

// Lambda
Function<String, String> lambda = x -> prefix.concat(x);

// [!code highlight] 方法引用
Function<String, String> ref = prefix::concat;  // [!code highlight]

// 使用
System.out.println(ref.apply("World"));  // "Hello, World"
```

### 3.2 示例

```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

// Lambda
names.forEach(name -> System.out.println(name));

// [!code highlight] 方法引用
names.forEach(System.out::println);  // [!code highlight]
```

:::tip println 的方法签名
`System.out.println(String s)` 的签名是 `(String) -> void`，与 `Consumer<String>` 匹配。
:::

## 四、实例方法引用（任意对象）

### 4.1 基本语法

```java
// Lambda
Function<String, Integer> lambda = s -> s.length();

// [!code highlight] 方法引用
Function<String, Integer> ref = String::length;  // [!code highlight]

// 使用
System.out.println(ref.apply("Hello"));  // 5
```

### 4.2 原理

```java
// String::length 的调用方式
// 相当于把第一个参数作为调用对象
BiFunction<String, ?, Integer> f = String::length;
// f.apply("Hello", null) 等价于 "Hello".length()
```

### 4.3 与 compareTo 结合

```java
List<String> names = Arrays.asList("Bob", "Alice", "Charlie");

// [!code highlight] 按长度排序
names.sort(String::compareTo);  // 自然顺序

// [!code highlight] 按长度比较
names.sort(Comparator.comparingInt(String::length));
```

## 五、构造器引用

### 5.1 基本语法

```java
// Lambda
Supplier<ArrayList<String>> lambda = () -> new ArrayList<>();

// [!code highlight] 构造器引用
Supplier<ArrayList<String>> ref = ArrayList::new;  // [!code highlight]

// 使用
ArrayList<String> list = ref.get();  // []
```

### 5.2 带参数的构造器

```java
// ArrayList(int initialCapacity)
Function<Integer, ArrayList<String>> ref = ArrayList::new;  // [!code highlight]

// 使用
ArrayList<String> list = ref.apply(10);  // 空列表，预留容量 10
```

### 5.3 示例：创建对象列表

```java
// 创建对象
List<String> names = Arrays.asList("Alice", "Bob");

// [!code highlight] 方法引用创建对象
List<User> users = names.stream()
    .map(User::new)  // [!code highlight] User::new 调用 User(String) 构造器
    .collect(Collectors.toList());

// 自定义构造器
class User {
    private String name;

    public User(String name) {
        this.name = name;
    }
}
```

## 七、常见错误与陷阱

### ⚠️ 陷阱 1：方法签名不匹配

```java
// [!code error] 编译错误：方法签名不匹配
Consumer<String> consumer = String::length;  // [!code error] length 返回 int，不是 void

// [!code highlight] 正确
ToIntFunction<String> function = String::length;
```

### ⚠️ 陷阱 2：混淆引用类型

```java
// [!code warning] 错误：应该是 String::length
Function<String, Integer> func = str -> str.length();

// [!code highlight] 正确
Function<String, Integer> func = String::length;
```

### ⚠️ 陷阱 3：构造器引用参数

```java
class User {
    private String name;
    private int age;

    User(String name) { this.name = name; }
    User(String name, int age) { this.name = name; this.age = age; }
}

// [!code highlight] 无参构造器
Supplier<User> s1 = User::new;

// [!code highlight] 单参构造器
Function<String, User> s2 = User::new;

// [!code highlight] 双参构造器
BiFunction<String, Integer, User> s3 = User::new;
```

## 八、加分回答

💡 **超出预期的深度**：

### 1. 数组构造器引用

```java
// [!code highlight] 创建 String 数组
Function<Integer, String[]> arrayFactory = String[]::new;

// 使用
String[] arr = arrayFactory.apply(10);  // 创建长度为 10 的数组
```

### 2. 方法引用的推导

```java
// String::compareTo
// 等价于 (String s1, String s2) -> s1.compareTo(s2)
// 类型：Comparator<String>

// String::compareToIgnoreCase
// 等价于 (String s1, String s2) -> s1.compareToIgnoreCase(s2)
// 类型：Comparator<String>

// String::toUpperCase
// 等价于 (String s) -> s.toUpperCase()
// 类型：Function<String, String>
```

### 3. 方法引用的优化

```java
// 方法引用比 Lambda 更简洁
// JVM 优化相同，性能几乎一致

// 推荐：代码意图明确时使用方法引用
list.forEach(System.out::println);  // 意图明确
list.forEach(x -> x * 2);  // 简单计算，Lambda 更清晰
```
