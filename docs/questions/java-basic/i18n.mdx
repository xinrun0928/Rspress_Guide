## 一、基本概念

### 1.1 术语

| 术语 | 含义 |
|------|------|
| i18n | Internationalization（国际化） |
| l10n | Localization（本地化） |

### 1.2 核心类

| 类 | 用途 |
|------|------|
| Locale | 表示地区和语言 |
| ResourceBundle | 加载资源文件 |
| MessageFormat | 格式化消息 |

---

## 二、ResourceBundle

### 2.1 资源文件结构

```properties title="messages.properties (默认)"
greeting=Hello
farewell=Goodbye
```

```properties title="messages_zh_CN.properties (中文)"
greeting=你好
farewell=再见
```

### 2.2 使用方式

```java
// 加载资源包
ResourceBundle bundle = ResourceBundle.getBundle("messages", Locale.CHINA);

String greeting = bundle.getString("greeting");  // 你好
```

---

## 三、MessageFormat

### 3.1 格式化消息

```java
String pattern = "Hello, {0}! Today is {1,date,long}.";
MessageFormat.format(pattern, "Alice", new Date());

// 输出：Hello, Alice! Today is 2024年1月15日.
```
