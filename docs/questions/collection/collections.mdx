## 一、排序方法

### 🔴 sort 系列

```java
// 1. 自然排序（元素需实现 Comparable）
List<Integer> list = new ArrayList<>(Arrays.asList(3, 1, 2));
Collections.sort(list);  // [1, 2, 3]

// 2. 自定义排序
List<String> names = new ArrayList<>(Arrays.asList("Tom", "Alice", "Bob"));
Collections.sort(names, (a, b) -> b.compareTo(a));  // 倒序

// 3. 逆序排序
Collections.sort(list, Collections.reverseOrder());  // 倒序

// 4. 部分排序
List<Integer> list2 = new ArrayList<>(Arrays.asList(5, 2, 8, 1, 9));
Collections.sort(list2.subList(1, 4));  // 只排序索引 1-3
```

---

## 二、查找方法

### 🔴 binarySearch 系列

```java
List<Integer> sorted = new ArrayList<>(Arrays.asList(1, 3, 5, 7, 9));

// 二分查找（列表必须有序）
int index = Collections.binarySearch(sorted, 5);  // 返回 2
int notFound = Collections.binarySearch(sorted, 6);  // 返回 -3（插入点-1）

// 自定义比较器查找
int index2 = Collections.binarySearch(sorted, 5, Collections.reverseOrder());
```

## 三、同步包装

### 🔴 synchronizedXXX 系列

```java
// 将普通集合包装成线程安全的
List<String> syncList = Collections.synchronizedList(new ArrayList<>());
Map<String, Integer> syncMap = Collections.synchronizedMap(new HashMap<>());
Set<String> syncSet = Collections.synchronizedSet(new HashSet<>());

// 使用时需要同步
synchronized (syncList) {
    Iterator<String> it = syncList.iterator();
    while (it.hasNext()) {
        // ...
    }
}
```

### ⚠️ 注意事项

```java
// Collections.synchronizedList 的迭代器需要手动同步
synchronized (list) {
    Iterator<String> it = list.iterator();  // 必须在同步块内
    while (it.hasNext()) {
        // ...
    }
}
```

## 四、不可变集合

### 🔴 unmodifiableXXX 系列

```java
// 创建只读集合
List<String> readonly = Collections.unmodifiableList(Arrays.asList("A", "B"));
readonly.add("C");  // UnsupportedOperationException

Map<String, Integer> readonlyMap = Collections.unmodifiableMap(map);
Set<String> readonlySet = Collections.unmodifiableSet(set);
```

### 💡 JDK 9+ 的工厂方法

```java
// JDK 9+ 更简洁的写法
List<String> list = List.of("A", "B", "C");
Map<String, Integer> map = Map.of("A", 1, "B", 2);
Set<String> set = Set.of("A", "B", "C");
```

## 五、其他常用方法

### 🔴 fill、copy、reverse

```java
List<String> list = new ArrayList<>(Arrays.asList("A", "B", "C"));

// fill：填充所有元素
Collections.fill(list, "X");  // [X, X, X]

// copy：复制集合
List<String> copy = new ArrayList<>(Arrays.asList("1", "2", "3"));
Collections.copy(copy, list);  // copy 变为 [X, X, X]

// reverse：反转列表
Collections.reverse(list);  // [X, X, X]（无变化因为都相同）
List<Integer> nums = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));
Collections.reverse(nums);  // [5, 4, 3, 2, 1]
```

### 🔴 shuffle、rotate、swap

```java
List<Integer> nums = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));

// shuffle：随机打乱
Collections.shuffle(nums);

// rotate：旋转列表
Collections.rotate(nums, 2);  // [4, 5, 1, 2, 3]

// swap：交换元素
Collections.swap(nums, 0, 4);  // [3, 5, 1, 2, 4]
```

## 六、max、min、frequency、disjoint

### 🔴 聚合方法

```java
List<Integer> nums = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));

// max/min：最值
int max = Collections.max(nums);  // 5
int min = Collections.min(nums);  // 1

// 自定义比较器
String longest = Collections.max(Arrays.asList("A", "AB", "ABC"), 
                              (a, b) -> a.length() - b.length());  // "ABC"

// frequency：元素出现次数
int count = Collections.frequency(nums, 3);  // 1

// disjoint：是否有相同元素
boolean hasCommon = Collections.disjoint(list1, list2);  // false 表示有交集
```

## 七、面试题精讲

### 🔴 第一层：Collections 工具类有哪些常用方法？

> **参考答案**：
>
> 主要有几类：
> 1. **排序**：sort、reverseOrder
> 2. **查找**：binarySearch
> 3. **同步包装**：synchronizedList/Map/Set
> 4. **不可变集合**：unmodifiableList/Map/Set
> 5. **操作**：fill、copy、reverse、shuffle、swap
> 6. **聚合**：max、min、frequency

### ⚠️ 面试官挖坑点

| 陷阱 | 错误回答 | 正确回答 |
|------|---------|----------|
| 「Collections 可以排序数组」 | 搞混 Arrays | Collections 排序 List，Arrays.sort 排序数组 |
| 「synchronizedList 不需要同步迭代」 | 忽略文档 | 迭代时仍需手动同步 |
| 「unmodifiableList 可以修改」 | 不了解不可变 | 修改会抛出 UnsupportedOperationException |
