## 一、排序方法

### 🔴 sort 系列

```java
int[] arr = {3, 1, 2, 5, 4};

// 1. 基本类型排序（快速排序 Dual-Pivot Quicksort）
Arrays.sort(arr);  // [1, 2, 3, 4, 5]

// 2. 对象数组排序（归并排序，稳定）
Integer[] nums = {3, 1, 2};
Arrays.sort(nums, Collections.reverseOrder());  // 倒序

// 3. 部分排序
int[] partial = {5, 4, 3, 2, 1};
Arrays.sort(partial, 1, 4);  // 只排序索引 1-3：[5, 2, 3, 4, 1]

// 4. 并行排序（JDK 8+）
Arrays.parallelSort(arr);  // 多线程排序
```

### 💡 基本类型 vs 对象数组

| 类型 | 排序算法 | 稳定性 |
|------|---------|--------|
| int、long、double | Dual-Pivot Quicksort | 不稳定 |
| Object | TimSort（归并排序） | 稳定 |

## 二、查找方法

### 🔴 binarySearch

```java
int[] arr = {1, 2, 3, 4, 5};

// 二分查找（数组必须有序）
int index = Arrays.binarySearch(arr, 3);  // 2
int notFound = Arrays.binarySearch(arr, 6);  // -6（插入点-1）

// 自定义比较器
Integer[] nums = {5, 3, 1, 4, 2};
Arrays.sort(nums);  // 先排序
int idx = Arrays.binarySearch(nums, 3, Collections.reverseOrder());
```

## 三、填充和复制

### 🔴 fill 和 copyOf

```java
int[] arr = new int[5];

// fill：填充所有元素
Arrays.fill(arr, 10);  // [10, 10, 10, 10, 10]

// fill 部分
int[] arr2 = new int[5];
Arrays.fill(arr2, 1, 3, 20);  // [0, 20, 20, 0, 0]

// copyOf：复制数组（自动处理边界）
int[] original = {1, 2, 3};
int[] copied = Arrays.copyOf(original, 5);  // [1, 2, 3, 0, 0]
int[] extended = Arrays.copyOf(original, 10);  // [1, 2, 3, 0, 0, 0, 0, 0, 0, 0]

// copyOfRange：复制指定范围
int[] range = Arrays.copyOfRange(original, 1, 3);  // [2, 3]
```

## 四、asList

### 🔴 Arrays.asList 的注意点

```java
// asList 返回固定大小的列表
List<String> list = Arrays.asList("A", "B", "C");
list.add("D");  // UnsupportedOperationException
list.remove(0);  // UnsupportedOperationException
list.set(0, "X");  // 允许修改
```

### ⚠️ 陷阱一：基本类型数组

```java
// 基本类型数组会被当作单一元素
int[] primitive = {1, 2, 3};
List<int[]> list1 = Arrays.asList(primitive);  // List<int[]>，不是 List<Integer>

// 正确做法：包装成 Integer 数组
Integer[] wrapper = {1, 2, 3};
List<Integer> list2 = Arrays.asList(wrapper);  // List<Integer>
```

### ⚠️ 陷阱二：可变长参数

```java
// asList 使用可变长参数
String[] arr = {"A", "B", "C"};
List<String> list = Arrays.asList(arr);  // 正确

// 直接传元素
List<String> list2 = Arrays.asList("A", "B", "C");  // 正确

// 传可变长参数
String[] arr2 = new String[]{"X", "Y"};
// Arrays.asList("A", arr2) 会被解析为 (String[], String)
// 而 Arrays.asList(arr2) 是 (String...)
```

## 五、equals 和 deepEquals

### 🔴 数组比较

```java
int[] a = {1, 2, 3};
int[] b = {1, 2, 3};
int[] c = {1, 2, 4};

Arrays.equals(a, b);  // true
Arrays.equals(a, c);  // false

// 多维数组要用 deepEquals
int[][] d = {{1, 2}, {3, 4}};
int[][] e = {{1, 2}, {3, 4}};
Arrays.equals(d, e);  // false（比较引用）
Arrays.deepEquals(d, e);  // true（递归比较）
```

## 六、toString 和 deepToString

### 🔴 数组��字符串

```java
int[] arr = {1, 2, 3};
Arrays.toString(arr);  // "[1, 2, 3]"

int[][] matrix = {{1, 2}, {3, 4}};
Arrays.toString(matrix);  // "[[I@..., [I@...]"（数组引用）
Arrays.deepToString(matrix);  // "[[1, 2], [3, 4]]"
```

## 七、stream 和并行计算

### 💡 JDK 8+ 流式操作

```java
int[] nums = {1, 2, 3, 4, 5};

// 转 Stream
int sum = Arrays.stream(nums).sum();  // 15

// 过滤
int evenSum = Arrays.stream(nums)
                    .filter(n -> n % 2 == 0)
                    .sum();  // 6

// 映射
int[] doubled = Arrays.stream(nums)
                      .map(n -> n * 2)
                      .toArray();  // [2, 4, 6, 8, 10]
```

## 八、面试题精讲

### 🔴 第一层：Arrays.asList 有什么注意点？

> **参考答案**：
>
> 主要注意点：
> 1. **返回的是固定大小列表**：不能 add/remove，只能 set
> 2. **基本类型数组问题**：int[] 会被当作单一元素，结果是 `List<int[]>`
> 3. **内部数组引用**：修改数组元素会影响列表

### ⚠️ 面试官挖坑点

| 陷阱 | 错误回答 | 正确回答 |
|------|---------|----------|
| 「asList 可以 add」 | 不了解固定大小 | 抛 UnsupportedOperationException |
| 「int[] asList 后是 List」 | 不了解基本类型 | 是 `List<int[]>` |
| 「Arrays 和 Collections 一样」 | 搞混了 | Arrays 操作数组，Collections 操作集合 |
