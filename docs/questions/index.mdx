# 高频面试题 300 题

## 每篇文档结构

每道面试题的文档都包含以下内容：

1. **快速自测**：面试官最关心的 3 个问题
2. **核心原理**：知识点讲解 + Mermaid 图解
3. **源码分析**：关键源码片段逐行解读
4. **高频面试题**：题目 + 参考答案 + 追问链
5. **对比表格**：与同类技术的对比总结
6. **陷阱警示**：⚠️ 标记常见错误
7. **加分回答**：💡 标记超预期答案
